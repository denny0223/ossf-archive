傳 IBM 有意購併 Sun

Created at Thursday, 19 March 2009 08:00      Last Updated on Friday, 03 April 2009 02:37 Written by 謝良奇/編譯
華爾街日報 (Wall Street Journal) 的消息宣稱，Sun 即將出售給 IBM。報導中指出，假如兩家公司交涉順利，這筆交易很快就會簽訂。據稱 IBM 向 Sun 出價每股 10 到 11 美元，因此總交易金額將達 80 億美元。儘管 Sun 易手給 IBM 將為深藍巨人在資料中心市場面對 HP 等對手時，帶來產品線上的優勢，然而 IBM 為何看上在平價伺服器硬體風潮興起後漸走下坡的 Sun，市場上仍充斥各種揣測。

假設 IBM 真有意要買 Sun，外界的第一個問題是，為什麼要買？顯然兩家公司的產品線有許多重疊之處，看不出互補效果。IBM 另一個要購併 Sun 的重要理由，似乎是削減競爭。Sun 部份業務雖然已經無足輕重，但部份軟體資產與智財權仍能為 IBM 所用。

特別是 Sun 在開放源碼軟體上的成果能嘉惠 IBM 成為該領域的領導者，而 Sun 近期宣佈將進軍雲端運算 (cloud computing) 的 Sun Cloud 產品，也能擴大 IBM 在此一成長領域的機會。

儘管有來源指出，IBM 想要的是 Sun 的軟體業務，而非其 x86 與 SPARC 伺服器產品線。但也有不少分析指出，雖然 IBM 已由硬體銷售轉型為軟體與服務為重，Sun 高階伺服器仍然提供 IBM 快速擴展此一市場佔有率的機會。在用戶轉往 Dell 與 HP 等公司供應的伺服器後，Sun 的高階伺服器因而流失市佔率，但仍握有約 10% 的伺服器市場。

對此有部份看法認為，IBM 早已轉型服務模式多時，由於其服務部門具獲利能力，且硬體採購較服務更易受到經濟蕭條影響，除了減少競爭外，IBM 並無理由取得 Sun 的硬體業務。一旦如此，購併後 Sun 的硬體業務很可能繼續以硬體伺服器公司經營，另一個可能是，IBM 會將 Sun 的 x86 伺服器與其 System x 產品線結合，然後將 SPARC 放在 System p 部門底下。

如果 IBM 有辦法將 Sun 的企業用戶轉往 IBM 的陣營，那麼在面對 HP 或 Cisco 等對手時，IBM 會更有勝算。Barclays 分析師 Ben Reitzes 表示，Sun 客戶轉往 IBM Unix 機器上，不僅能有助 IBM 的銷售數字，還可增加 IBM 自有晶片製造作業部門的利用率。

Reitzes 指出，Sun 在開放源碼軟體上的獲利能力並不理想，然而 IBM 一向很擅長將自家的智財權轉為獲利來源，Java 和 Solaris 這兩個 Sun 手上的重要軟體，會為 IBM 的軟體和服務業務帶來額外的獲利挹注。

IBM 取得 Sun 的開放源碼產品後，部份的軟體授權可能進行更動。例如，IBM 並不樂見 Sun 以 GPL 方式釋出 Java，一旦 IBM 購併 Sun，該公司有可能對 Java 採取多重授權的方式，在 GPL 外提供如 Apache Software License 的授權方案，使 Java 技術更容易被整合到如 Apache Harmony 專案等現有的開放源碼 Java 實作，第三方也能更輕易將 Java 納入私有應用。

Linux 和 OpenSolaris 同是自由的 Unix-like 作業系統，購併 Sun 之後 IBM 是否會嘗試建立 OpenSolaris 社群，或是會全力壓寶 Linux？目前的看法是，IBM 對於 Linux 的支持仍舊不會動搖，OpenSolaris 需要 IBM 投入許多人力，從組織的觀點來看，同時維持 OpenSolaris、IBM 自有的 Unix 品牌 AIX，並且同時支援 Linux 似乎不太可行。

對於深藍巨人來說，提供 OpenSolaris/Solaris 的用戶一個轉往 Linux 的途徑，會是較為可行的策略。無獨有偶的，最近 IBM 剛剛購併了一家專於將 Solaris 軟體轉換至 Linux 上的公司，Transitive。IBM 似乎已經準備就緒。

IBM 一直以來透過 Eclipse 支持 Java。購併 Sun 可以為 IBM 取得更多 Java 的控制權。鼓勵開發者在 Linux 與 AIX/Solaris 上使用 Java，有助 IBM 將微軟排除於伺服器開發圈之外。

Sun 另一個不可忽視的重要軟體資產是知名的資料庫 MySQL，Sun 購併 MySQL 雖然看似一步好棋，但是後續處理不當，例如 MySQL 首席開發者離開，MySQL 創辦人 Michael 'Monty' Widenius 公開表示 MySQL 5.1 GA 版臭蟲多多。在 IBM 的正確管理下，例如 DB2 部門，MySQL 夾帶在 Web 2.0 世代的高普及率，仍然具有潛力。

並非每個人都樂見 IBM 購併競爭對手的影響。電腦與通訊產業協會 (Computer & Communications Industry Association，CCIA) 就 IBM 此舉動機發表聲明，懷疑 IBM 有獨佔傾向。CCIA 執行長 Ed Black 表示，有關此一可能交易的分析中，IBM 對高階電腦市場持續的獨霸地位都扮演了關鍵地位。一旦合併案宣佈，由於兩家公司跨足了廣泛的科技產品，反托拉斯政府當局必須進行仔細詳盡的檢視。IBM 與 Sun 是長久以來的競爭對手，合併案將削減關鍵競爭者，從而影響多項 IT 產品的選擇與價格。
