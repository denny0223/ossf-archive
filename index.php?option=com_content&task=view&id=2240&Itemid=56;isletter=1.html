Google Chrome 瀏覽器超越 Safari 成為第三大瀏覽器

建立日期 2010-01-05 08:00      最近更新在 2010-01-26 19:37 作者是 謝良奇/編譯
根據最新瀏覽器市佔率數字，Google 的 Chrome 瀏覽器日前首次超越 Apple 的 Safari，成為全球第三大的瀏覽器軟體，僅落後於 Internet Explorer 與 Firefox。根據網路調查公司 Net Applications 的數據，Google Chrome 在 2009 年 12 月市佔率上升 0.7 個百分點，來到 4.63%，超越 Safari 的 4.46%。此一數字為 Chrome 釋出以來的最大成長，根據 PC World 推測，要歸功於 12 月 Mac 與 Linux 平台的 Chrome Beta 版本釋出。

不過對於 Apple 和 Safari 來說，去年 12 月也並不全是壞消息。桌面 Safari 市佔率上升了 0.1 的百分點，而 iPhone 版的 Safari 相較於上個月則成長了 20%。更何況 Chrome 與 Safari 並非純然的競爭者關係。事實上，這兩套瀏覽器全都以 Apple 的開放源碼 Webkit 引擎為基礎。某一方成功帶來更多 Web 開發者在網頁中為 Webkit 加入最佳化程式碼，使整體市佔率增加，對另一方也會有好處。

Mozilla 的開放源碼瀏覽器 Firefox 微幅衰退 0.1 個百分點，市佔率為 24.6%，再度拖延該全球第二大瀏覽器跨越 25% 大關的時間。Opera Software 的私有免費軟體 Opera 則在 12 月份有所斬獲，以 2.4% 市佔率創下該瀏覽器新高。

12 月的瀏覽器戰役中，遭到慘敗的是微軟的 Internet Explorer (IE)。儘管 IE 仍以 62.69% 市佔率排名第一，但卻有 0.92 個百分點的跌幅，創下該瀏覽器市佔率新低。而且過去 6 個月來，IE 每個月的市佔率都掉了近乎 1 個百分點。假如此一趨勢持續下去，Net Applications 預測，IE 將在 2010 年中跌破 50%。對於那些只能用 IE 瀏覽的網站，這將是一大警訊。

值得注意的是，IE 去年下半年每個月平均 0.94 個百分點的跌幅，幾乎是上半年 0.36 個百分點的 3 倍之多，微軟去年 3 月正式推出 IE8 以來，至今並未挽回這股跌勢。2009 年12 月份的數據顯示，IE8 首度成為最多人使用的微軟瀏覽器。將 IE8 的相容模式一併計算在內的話，IE8 佔微軟所有瀏覽器使用率的 37.8%。過去第一名的 IE6 衰退到 33.5% 佔據第二。

和微軟相同，Mozilla 在說服用戶轉換到最新瀏覽器版本 Firefox 3.5 上，也有不錯的成果。12 月只有不到 7% 的用戶還在使用 Firefox 3.0。由於 Mozilla 將停止該版本的安全更新，這一股跳船至 Firefox 3.5 的趨勢，或許也因 Firefox 3.0 即將從市場上退役所造成。

Net Applications 的統計資料來自該公司監控的 4 萬個網站，並追蹤拜訪這些網站的系統。每個月約可追蹤到 1 億 6 千萬的個別使用者。

在作業系統方面，Windows 7 已經突破 5% 關卡，其市佔率從 11 月的 4% 上升到 5.17%。不過 Windows 7 所取得的，似乎多來自 XP 與 Vista 的用戶，因為 Mac 與 Linux 用戶的數字仍然各自維持在大約 5.11% 與 1.02%。

儘管有所斬獲，Google 並未就此鬆懈。最近該公司為 Chrome 瀏覽器釋出了一個只提供給 Windows 用戶使用的延伸按鈕，可以讓用戶在瀏覽器中獲得連接類似網頁的功能。名為 Google Similar Pages 的附加元件目前還是 beta 版本，提供和 Google 搜尋結果中類似網頁 (Similar Pages) 一樣的功能。

此舉不禁令人想起，當 Google 在瀏覽器市場影響力日增後，該公司釋出的眾多產品，有多少還會像過去一樣為所有瀏覽器提供支援。
