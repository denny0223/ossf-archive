Topic Information 議程簡介

厲害的 Perl 程式設計師，常常被人稱為魔術師。因為他們寫著沒有人看得懂的符咒文字，用潦草幾行，輕輕鬆鬆，變出別人辛苦寫幾十行、幾百行還變不出的神奇魔法。魔法的秘訣之一，就是「常規表示法」（Regular Expression）。Regular Expression 是一種簡潔有彈性的字串表示法，用簡單的記號，就可以表示出複雜的字串比對規則，功能強大。本課程帶著您進入 Regular Expression 的魔幻世界，讓初學 Perl，或被主管強迫學 Perl 多年卻一直不知道 Perl 是在幹嘛的你，一窺由程式設計師晉昇為魔法師的奧秘。

詳細的內容時間描述與報名資訊如下：

  • 對象：大專院校資訊相關系所的學生，或是對自由軟體有興趣者。
    （本課程要求具備基本 Perl 程式撰寫能力，Linux/Unix 終端機操作經驗、文字編輯程式 Text Editor 使用經驗。）
  • 人數：約 28 人。
  • 費用：全程免費。
  • 平台：提供 28 台電腦實機操作，使用 Windows 模擬 Ubuntu 10.04 系統。
