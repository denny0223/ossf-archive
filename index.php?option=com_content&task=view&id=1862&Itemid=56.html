Aspire One 邀您一同加入開放共享行列

建立日期 2008-12-10 08:00      最近更新在 2011-01-11 18:40 作者是 Arthur
2008 年是小筆電 (Netbook) 激戰元年，也是台灣 IT 廠商最值得驕傲的一年，因為從小筆電目前的全球銷售量來看，台灣兩大本土品牌 ACER 與 ASUS 的小筆電，已穩站全球 70% 以上市佔率。這項光榮戰績打破了過往 Desktop 及 Notebook 時代 IT 業界規格制定掌握於日本、美國…等國際大廠中的迷思，首度由台灣廠商取得完整的軟、硬體規格主導權。此也顯示台灣廠商已具備創新市場品牌與高度附加價值之商品，進而擁有快速行銷全球的能力。而在這波全球不景氣的大環境影響下，小筆電能因應市場需求而起，自由軟體的應用與對成本的影響，無庸置疑地佔了關鍵的地位。

成功運用自由軟體之優點，創下銷售亮眼佳績的 ACER Aspire One 於今年 7 月開始銷售，光 9 月單月就創下全球 120 萬台的銷售成績。值得一提的是，這台 Aspire One 軟、硬體，都是道地 Made in Taiwan！隨機出貨預設 Linux 版本採用百資科技的 Linpus Lite，此點筆者於日前有親自與百資科技靜副總培文以原型機確認，於 OS 系統螢幕最後 Page 有清楚標示台灣百資科技的公司資訊。有台灣 Acer 與百資科技掛 2 個 Made in Taiwan 保證，無怪乎甫上市不到 3 個月，已締造全球多項銷售紀錄並獲得廣大使用者熱烈好評。

中央研究院自由軟體鑄造場 (OSSF) 本著促進國內軟體發展，活絡自由軟體社群、校園及企業交流互動為己任，長期持續舉辦各類型工作坊。日前舉辦『台灣社群企業 BoF-Netbook』活動，邀請 ACER Aspire One SW 與 RD 團隊，面對面與社群朋友自由的交流與分享。與會之社群夥伴涵蓋了 Browser、Linux OS、Linux 套件、UI、OOo.org、USB 應用及資安…等。活動以愉快之步調，順利達成社群與企業之交流，除了討論原定命題外，也彼此交到了共同興趣的良師益友。隨著簡單輕鬆的流程安排與人人平等的前提下，與會者都確確實實的『人材兩得』：結交到跨領域志同道合之友，也交流共同興趣的話題。

本次代表 Aspire One 團隊與社群聯繫的窗口尤焙麟 (Bey Yu)，其職務為 Software PM，主要負責程式取得與管理 Netbook 的應用軟體。透過現場 Bey 的口述，表示 Acer 透過與自由軟體鑄造場 (OSSF) 正式合作，加入台灣自由軟體發展行列，未來也將積極與社群朋友交流互動。為讓更多的社群朋友有機會參與 Aspire One 開放共享的行列，Bey 特別於百忙之中，接受筆者以電話及 Mail 方式進行訪談，以下為訪談內容：

問：Acer 對於 Open Source Software 看法與政策為何？未來如何與 Community 合作？

答：自由軟體成功應用於 Aspire One 的經驗讓 Acer 清楚體悟作業系統不單單只有一種選擇，而應該被鼓勵與支持朝多元化發展，這才是使用者的福音，也發覺了一條未來新的軟體應用路線。很感謝全球開放源碼社群同好無私的貢獻，讓 Linux 與自由軟體得以蓬勃發展，讓大家都能有更好用更成熟的軟體自由選用，也降低了商業軟體的成本，打破一家壟斷的現象。Acer 將秉持取之於 OSS 用之於 OSS 之信念，希望能與社群朋友廣結善緣，未來相關 Linux 平台之開發與程式之應用也會盡量朝向與社群合作，並多支持社群聚會，一同為振興台灣軟體業與正體中文化而努力。

問：小筆電崛起速度極快，Acer 短期內最需要的 Open Source 人才種類為何？是否有長期培養 Developer 人才之規劃？

答：Acer 成功打響了 Netbook 第一砲之後，後續尚有其他形態的 PC 與 embedded devices 產品等著推出。因此對於 Linux 軟體開發之人才可說是求才若渴。從上層的人機介面到底層的 kernel 與 driver 開發修改都是我們想要網羅的人才。熟悉 Kernel 與任何程式語言或技術 (Java、Perl、Python、GTK+ 或 QT...等) 的有志之士我們都非常歡迎。目前架構在 Linpus Lite 下的應用程式尚不多，若社群夥伴能夠提供好的程式或提供蒐集可用程式的方法，也歡迎交流。至於人才的培育，Acer 的 RD Team 已經在 Linux 默默耕耘 5 年以上，在嵌入式系統、智慧型手機、PC 軟體與韌體都有過實戰經驗。未來，Acer 希望能網羅更多業界有經驗的開發人員、專案管理人員、測試工程師等等。

問：現階段最迫切需要社群或夥伴們共同努力之項目為何？

答：Aspire one 大部分的使用者都是屬於一般使用者，他們的使用習慣與思維大都來自於 Windows 的使用經驗。為了讓更多人習慣 Linux、常用 Linux、進而喜歡 Linux，我們需要社群的夥伴幫忙打破 Windows 與 Linux 的轉換障礙。舉凡瀏覽器的相容性、Office 文件的相容性、字型支援、周邊硬體的驅動程式支援。再者，如果社群夥伴能把自己的使用經驗轉換成淺顯易懂的教學文件或懶人包，以幫助更多入門級的使用者踏入 Linux 的世界，也能對於 Linux 的推廣有很大的幫助。

問：Bey 參加『台灣社群企業 BoF-Netbook』的心得感想。

答：感謝自由軟體鑄造場舉辦的各種工作坊，特別是這種類型的 BoF 聚會，讓企業能與社群朋友直接談話與認識，性質上與舉辦中大型的研討會不太一樣。透過這種社群分享式的小型 BoF 聚會，讓我們思考到目前 Aspire One 軟體服務的策略優先順序，讓原屬於硬體廠商的 Acer 能快速接觸到不同領域的軟體開發者，共同激起創意與靈感的火花。更重要的是，我們樂於遵循自由軟體開放分享的精神，讓對於 Netbook 感興趣之夥伴，能加入開發或推廣行列。再次謝謝自由軟體鑄造場與來參加本次 BoF 聚會的社群夥伴們，Bey 代表 Aspire One 團隊竭誠邀請廣大的社群朋友一同加入 Netbook 開放分享的行列，歡迎有興趣的朋友立即與我(Bey)聯繫，或委由自由軟體鑄造場代為轉達這份合作分享的熱誠！
