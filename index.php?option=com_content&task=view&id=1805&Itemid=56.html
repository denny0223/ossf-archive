何謂「智慧型注音輸入法」？

建立日期 2008-11-06 08:00      最近更新在 2008-11-13 18:04 作者是 thkuo
所謂智慧型注音輸入法正是為了避免使用者反覆選字，利用中文斷詞技術，取代繁複的中文輸入選字動作而來的。目前筆者周遭的朋友多數都選用即微軟新注音輸入法以及自然輸入法。而兩者均只能運行於 Windows 平台，在這個開放的年代，難免是一種遺憾。

◎ 跨平台輸入需求

隨著自由軟體蓬勃發展，無論開發者或是一般使用者對於自由的作業系統接受度越來越高，我們開始想像我們的日常工作，非在特定平台不可嗎？輸入法會是最大考量的因素，有多少新注音的習慣，我們如何能夠轉換至 Linux 上使用？有多少自然輸入法的好，我們因為轉換系統必須犧牲？難道轉換系統，就得不斷改變長久以來的輸入習慣嗎？

因此本期專欄將介紹一套選字智慧介於自然與新注音輸入法之間的輸入軟體—新酷音輸入法，簡述新酷音的發展歷史與跨平台共通的操作特色，期待使用者能更深刻體會自由軟體所帶來的魔力。

◎ 新酷音輸入法發展簡史

新酷音輸入法係由酷音輸入法為基礎發展而來，酷音最早是教授指導下的專題題目，僅能在 Linux 下運作，經由一群熱心的社群朋友通力合作下，以此為基礎發展為一個在 Linux、Windows 甚至 Mac OS X 都能運作順暢的輸入法。為了加強使用者參與亦分支出詞庫的維護團隊，智慧型輸入法仰賴詞庫運作是非常緊密的，有了這樣發展模式，如今新酷音輸入法已快速發展為眾多智慧型輸入法之中的優先選擇。

◎ 新酷音輸入法特色

1. 選字技術解析

相對於微軟新注音來說，新酷音在選字智慧上有明顯的加強，這樣說來相當抽象。改以設計架構來解說新酷音的優點，新酷音並沒有新注音沈重的『輸入學習』包袱，新酷音主要是以字詞頻率統計模型來作選字判斷，判斷的依據，以詞庫為基準，加以頻率分析來顯現使用者最可能想輸入的文字。

為何新注音的輸入學習是沈重包袱呢？對於使用者而言在二次輸入可能會有很大幫助，可是三次、四次後便發覺，電腦學習的其實只是短暫的輸入習慣，並非一體通用，因此打完第一篇文章，換下篇文章電腦學習下來的習慣，學習下來的詞彙，反而造成選字的困擾，我們得花更多時間修正這些錯誤，懶惰選字的人也因此鬧出許多新注音的笑話。

2. 新酷音詞庫

如上段所述，新酷音係由字詞頻率作為選字基準，因此詞庫的維護是選字正確率提昇的不二法門，新酷音提供了非常便利的快速加詞功能，亦提供簡便的詞庫維護及詞庫匯入與匯出介面，當使用一段期間後，這陣子常打的親朋好友姓名，就可透過這樣的方式分享給其他朋友使用，相信打對姓名對方也會感受到尊重，更是一種基本的禮貌。

3. 人工斷詞

我們打字的時候，總會有一口氣打下一長串句子的情形，電腦交叉判讀經常沒有顯示正確的字詞，新酷音提供了 Tab 斷詞功能，將選字指標移至要斷詞的字前，按下Tab 即可將詞切開，讓後面的輸入字串重新判讀，如此一來又可大幅減少選字操作的頻率。

4. 全形符號輸入

一篇正式文章中，全形符號使用是必然的，對於文字工作者而言，快速輸入全形標點符號是很必要的，而新酷音在中文輸入的狀態下，即可輕易打出『，。、：；！』等常見符號，亦可編輯常見的符號表，以供挑選，讓工作者可以將精神花在文字輸入上，而非大量的符號調整與替換。

5. 選字操作

相信新注音的使用者一定不陌生使用上下左右鍵來選字的過程，但對於快速輸入而言，過多的手部移動不但造成效率降低，亦可能造成心情上的煩躁。因此輸入的時候讓手指多數停留於鍵盤符號熱區是首要的目標，而新酷音提供了 Space 鍵觸發選字列表的功能，亦提供 jk 鍵來作選字指標左右移動，這對於輸入速度提昇都有明顯的效果。

◎ 結語

看完以上說明，不知是否勾起讀者的興趣了呢？相信在這個資訊年代，打得一手好字，充分表達您想用文字傳達的感情，是一件非常有成就感的事。我們不必再讓眼睛遭受錯字的襲擊，也不必再為了選字錯誤所鬧的笑話悔恨，相信新酷音輸入法可以帶來不同的輸入體驗。
