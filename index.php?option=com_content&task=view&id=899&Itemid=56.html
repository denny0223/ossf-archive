IBM、紅帽進行 Linux 創新研發

Created at Sunday, 18 March 2007 08:00      Last Updated on Thursday, 05 July 2007 18:36 Written by 謝良奇
IBM 與紅帽 (Red Hat) 兩家全球最大的商業 Linux 開發廠商，宣佈雙方的 Linux 實驗室將進行合作，整合最新技術到 Linux 核心中，將協助用戶提升效率以取得競爭優勢。
IBM Linux 技術中心位於全球 40 個不同地點，共 600 餘位開發者，將聯同紅帽的工程團隊在 Linux 核心中，整合先進的虛擬技術、安全改善及即時 Linux 功能。數百位 IBM 與紅帽的開發者會全時工作投入 Linux 的改進上。

雙方在聲明中表示，Linux 核心中新的虛擬與安全功能也會出現在紅帽企業版 Linux 5 (Red Hat Enterprise Linux 5，RHEL 5) 中。至於即時 Linux 的功能，則正在整合進 Linux 核心的階段，未來可望提供給 IBM 與紅帽用戶。

此外雙方也表示 IBM 正在其標準中介軟體與伺服器上，測試與驗證 RHEL 5，IBM 也計劃提供 RHEL 5 的服務與支援。

IBM 與紅帽指出的三大 Linux 創新開發，包括 Xen 虛擬技術、Linux 安全強化與即時 Linux。

IBM、紅帽與開放源碼社群目前已經把 Xen 虛擬技術整合到 Linux 核心中，Xen 是一套開放源碼的虛擬軟體，有助用戶提高伺服器使用率，更能有效地管理以 Linux 為基礎的資訊系統。該項技術已整合到 RHEL 5 中，IBM 在 Intel 與 Opteron 處理器為主的伺服器及 IBM BladeCenter 系統上，提供了 IBM Virtualization 一系列產品的支援。

因應需要更高資料保全等級的用戶，IBM 與紅帽開發者連同開放源碼社群已為紅帽的核心作業系統建構了 Security-Enhanced Linux，可提供多層次的安全功能。這些功能讓企業用戶，特別是政府單位的用戶，能夠建立強制的存取控管策略，讓資訊分級為極機密、機密、保密與不需保密等層級。這套 Linux 核心的安全系統將應用軟體與服務劃分成不同的安全領域，為各類型的用戶提供高安全性的應用軟體與資料平台。

最後，IBM 與紅帽正在打造新的即時 Linux 應用軟體開發及部署平台，用戶將可藉此在執行系統時，取得更高的處理效率及可靠性。此一新系統包括了 IBM 的即時 J2SE Java 虛擬機器 WebSphere Real Time，以及在 IBM System x 與 BladeCenter 的 AMD、Intel 伺服器上執行的 RHEL 5 即時版本。藉著這套新的運算架構，Java 開發社群將能發展出執行時間精確到毫秒級的應用軟體。

美國海軍是 IBM 即時 Java 技術的早期採用者之一，並以 IBM 部署的客製化即時 Linux 為執行平台，提供提供命令控管、探索、目標鎖定、武器控制與雷達系統等應用軟體，一套單一且基於開放標準的運算環境。

除了上述技術外，IBM 也將持續與紅帽及開放源碼社群合作，藉由 SystemTap 工具套件，提供效能調教與系統除錯功能。雙方將投入的其它技術還包括 Samba、核心延展性、用戶測試等。

紅帽工程執行副總裁 Paul Cormier 表示，其企業平台接下來的釋出不只是單純的 Linux 升級而已，更重要的，這將是一套完整企業架構的誕生，其中包括了來自 IBM 等合作夥伴經過驗證的高可靠性解決方案。
