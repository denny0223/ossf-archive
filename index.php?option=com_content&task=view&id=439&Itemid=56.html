WordPress 2.0 釋出 Yahoo! 提供 WordPress 服務

Created at Tuesday, 27 December 2005 08:00      Last Updated on Thursday, 08 March 2007 08:30 Written by 謝良奇
日前 WordPress 社群發表了兩大部落格（blogging）解決方案之一的 WordPRess 2.0 版，代號 “Duke”。此一新一代的 WordPress 特別在撰寫介面上做出改善。Yahoo! 也在與 Six Apart 合作推出 Movable Type 主機代管服務後，提供 WordPress 解決方案給專業部落客選擇。 WordPress 社群的釋出聲明表示，在這版為了向爵士鋼琴家艾靈頓公爵（Duke Ellington）致意而名為 "Duke" 的 WordPress 2.0 中，他們對於部落格核心部份，也就是撰寫介面，花了許多心血。

在使用上，WordPress 的後端經過重新設計，登入後可以發現在視覺效果與可用性上都已完全翻新。這也是 Shuttle 設計師團隊的努力成果。往後可以期待 WordPress 在視覺上會有更好的表現。

WordPress 採用了 AJAX、DHTML 技術，用戶可以更順暢地管理自家的 blog。當用戶撰寫部落格時，可以立即加入類別，有如 Flickr 上的 tagging。此外，簡單（simple）與進階（advanced）各自分開的發表介面，在這一個版本中合而為一，且用戶可藉由拖拉對話框的方式，自 訂網頁版面（layout）。

Andy Skelton 與 TinyMCE 團隊合作為 WordPress 加入了所見即所得（WYSIWYG）的編輯介面。有效地節省用戶的編寫時間。用戶也可以選擇關閉此功能。此外，2.0 版的WordPress 還包含了相當普遍的 plugins：可以用來備份資料庫的 Skippy DB backup，以及分散式的垃圾郵件過濾器 Akismet。

Yahoo! 主機代管服務在推出 Movable Type 服務，也宣佈與 WordPress 合作，提供專業部落客 WordPress 解決方案選項。不但包括 WordPress 2.0 的安裝，往後也會負責為用戶升級新的 WordPress。
