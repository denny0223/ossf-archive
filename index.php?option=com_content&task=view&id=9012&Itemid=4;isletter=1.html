Ubuntu 13.10 可能改用 Chromium 作為預設瀏覽器

建立日期 2013-06-25 11:09      最近更新在 2013-06-25 11:19 作者是 謝良奇／編譯

如果 Canonical 部份開發者的計劃得以順利進行，從 13.10 版本開始，Ubuntu 中預設且預先安裝的網路瀏覽器將不再是 Firefox。Canonical 內部以 Chromium 取代 Firefox 的支持聲浪正不斷高漲，許多開發者相信，透過 Chromium 可以提供更好的終端用戶體驗。

由於 Firefox 與 Chromium 都屬於開放源碼，納入 Chromium 並不違反 Ubuntu 作為百分百開放源碼作業系統的理念。但是別把 Chromium 與 Google Chrome 給搞混了，後者是以 Chromium 程式碼為基礎的免費瀏覽器，其中部分屬於私有軟體。

更換瀏覽器的想法，首次在 5 月舉行的第二次線上 Ubuntu 開發者大會中，由某些 Canonical 開發者提出。在某 Hangouts 議程上，Chromium 瀏覽器維護者 Jason Warner、Chad Miller 與其他人，和與會者討論到這個想法並徵求使用者意見。根據統計數字顯示 Google Chrome 已成為當前最普遍網路瀏覽器，有鑑於此，不難想見多數用戶也對以 Chromium 更換 Firefox 表示贊同。

對於此一發展，Jason Warner 最近發出了一封後續的電子郵件。節錄如下：

過去數周內，對此可能的轉換有許多文章與評論，針對於此，我想要讓此一討論著重在某些層面。

  1. 這跟哪個瀏覽器比較好無關。
  2. 這無關乎哪個瀏覽器功能比較多，或者有某些功能。
  3. 開放性與自由仍是我們核心價值之一。不過我不想讓此一討論串，變成誰較開放與自由的爭論。

重要且最終將為決定因素的是，普遍的終端使用者體驗。哪個瀏覽器，在普遍情況下，會是一般終端用戶較佳的選擇？和此一討論有關的，我想應該是品質、穩定性、穩固性、熟悉度、易用性與整體使用者體驗。

Jason 在電郵最終再次保證，不論最後決定為何，即使其中之一不會出現在下一版 Ubuntu 釋出中，Ubuntu App Store 還是會持續提供 Firefox 與 Chromium。
