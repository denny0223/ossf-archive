請教相關免費軟體使用於「商業用途」上的問題 2010/04/05 10:27  (9 Years, 7 Months ago) Karma: 0  
Dear Sir

您好！我從自由軟體鑄造場的網站上得知一些關於免費軟體、自由軟體的相關資訊，感覺非常的專業及可以信任

又看到貴站有提供免費的法律諮詢服務，需要請您協助解答相關免費軟體在公司內使用與法律上可能產生的問題

最近在公司內推行軟體合法化的相關政策，除了目前有合法購買版權的軟體外，仍有一些是使用所謂的Freeware、Shareware

但從軟體裡面的說明、或官方網站上的說明，都無法確定該軟體在一般商業的公司內使用，是否就是定義為「商業用途」(商業上使用)

End User License Agreement, EULA 這個規範應該算是只要不修改重製軟體，關於「商業用途」(商業上使用)是指公司內使用就算嗎？

還是說像我們使用它的軟體，設計或是產出其他成品有販賣交易行為才算是「商業用途」(商業上使用)呢？

在一般企業裡面，他們應該也會用到相關免費的軟體，不可能都是買要花錢的軟體的，有聽說相關免費軟體使用的配套方法嗎？

目前可能在公司內想審議下列8套網路上廣稱免費軟體(Freeware or Shareware)

想瞭解這些軟體有哪些是可以不需另外花費、註冊即可合法在一般商業的公司內正常使用，當然不會拿它販售或是反編譯(重製)軟體

1.Irfanview
2.FileZilla
3.PDFCreator
4.PicPick
5.WinRAR
6.Adobe Reader
7.GOM Player

下面含有自行先查軟體版本、授權方式，有的軟體說明及官方網站都查不到，希望透由貴站協助審議相關軟體的性質

請協助我們一般公司於內部可以合法化的使用免費軟體！謝謝您！！

Hi elvis,

要回答以下的問題，首先我將「Freeware、Shareware，以及Free Software」做一個簡單的定義：

1、Freeware，所使用的Free指的是「免費」的意思，中文常翻作「免費軟體」，這類的軟體簡單來說就是「Free-downloaded Software)」，使用者可以免費的下載到這樣的軟體，然而、免費軟體的著作權利人還是保有其著作權利，所以雖然可以免費下載，但要看下載者的立場來判定是否合法使用，部份的免費軟體是允許使用者(即使這個使用者是營利法人-公司)做任何方式的利用，但也有很大部份的免費軟體，是限定只讓終端使用者「個人免費使用的(End-user only)。

2、Shareware，中文常翻作「共享軟體」，這類的軟體可以說就是「進階版的Freeware」，使用者可以免費的下載到共享軟體來使用，但軟體使用上可能有二個附帶限制，第1個附帶限制就是使用時間可能受限，例如只能讓使用者使用30天，30天後若不付費的話就只能停止使用；第2個附帶限制可能是功能上的限制，例如免費的版本只有限定功能，付費後使用者才會得到程式完整功能的使用權。

3、Free Software & Open Source Software，所使用的Free指的是「自由」的意思。近似的中譯字會有「自由軟體」以及「開放源碼軟體」、「開放原始碼軟體」這樣的名詞，以下我將這樣的名詞統稱為「自由軟體」。所有自由軟體的授權方式都是容許他人商業利用的，但不同類型的自由軟體授權條款，同時會要求使用者善盡一些預先設定好的義務性規定，所以精確一點來說，自由軟體是可以拿來做商業利用的，但它同時要求使用者盡相當的義務，以最著名的GPL授權條款為例，您可以將GPL授權的程式拿來做商業利用，但如果散布該程式的程式執行檔(Binary Code)的話，同時也被賦予要散布程式源碼(Source Code)的義務。

好、那第二個我們要釐清的是，所謂的「商業利用」「為商業目的之利用」的定義為何：

事實上，何謂「商業使用」的具體界定範圍，這是見仁見智的，也就是說、不同的解讀者，有不同的解釋角度，各國的法律上，並沒有直接的成文法是去介定何謂商業使用，何不謂商業使用，所以、若是從尊重軟體著作權利人的觀點來說，個別軟體的商業使用範圍，要去從著作權利人的觀點來解釋，也就是說、要看那個軟體專案細部的授權說明；所以關於在「公司內部使用」算不算是一種商業利用軟體的作法，也是要看使用了哪個軟體專案而有不同的差異。

以下就依您所列舉的軟體專案，擇要說明：

1、Irfanview：https://www.irfanview.com/

Irfanview的授權說明網頁如右：https://www.irfanview.com/，它是一個Freeware沒錯，從它的授權聲明看來，僅供個人居家式免費使用，所以公司內部使用是會有一些爭議的。

USAGE AGREEMENT

IrfanView is provided as freeware, but only for private, non-commercial use (that means at home).→此段說明IrfanView僅供「個人」「非商業性」「居家式」使用。

IrfanView is free for educational use (schools, universities and libraries) and for use in charity or humanitarian organisations.→此段說明IrfanView亦可供「中小學、大專院校、圖書館、公益團體或是一般人道推廣組織免費使用」，由此可知，明示公益團體，排除營利公司，所以公司內部使用就這個專案的著作權利人來說，是認定這算是商業利用，而不能免費使用的。

2、FileZilla

SourceForge上的專案授權網頁：https://sourceforge.net/projects/filezilla/ (點擊「Show project details」即可看到完整資訊)

Filezilla是以GPL2(GNU General Public License Version 2)授權釋出的自由軟體，它的授權聲明隨著檔案一併散布，壓縮檔裡的GPL.html即為授權書面文字。所以、這個軟體可以在商業公司內部使用，甚至商業性散布至公司外亦可，但請記得、散布GPL授權軟體時，需一併提供程式原始碼給收受程式的後手，這個責任是散布GPL授權程式所無法規避的重要義務。

3、PDFCreator

SourceForge上的專案授權網頁：https://sourceforge.net/projects/pdfcreator/ (點擊「Show project details」即可看到完整資訊)

PDFCreator同樣是以GPL2(GNU General Public License Version 2)授權釋出的自由軟體，它的授權聲明隨著檔案一併散布，詳細的授權資訊可參照安裝後子目錄裡的「GNU License.txt、AFPL License.txt，以及FairPlay License.txt」這三個檔案。這個軟體一樣可以在商業公司內部使用。

4、PicPick

PicPick首頁裡有關授權部份的說明：https://www.picpick.org/home
Furthermore, this software is distributed as freeware. So you may use this software on any number of computers for as long as you like.

再加上donation頁面的說明：https://www.picpick.org/donate
As you probably already know, PicPick is completely free software that anyone can use anywhere.

結論：PicPick是一個典型的Freeware，並且可以免費使用的對象，原則上並沒有將商業公司排除在外，
因為其強調「anyone can use anywhere」，然而、這樣的授權態度嗣言並不是沒有變更的可能性，所以建議商業公司如要內部使用，需將此頁面printscreen留存以資證明，如果有其他的自由軟體可以代換掉PicPick這個專案，也建議以自由軟體的方案替換掉，因為Freeware的授權方式嗣後是可能會變更的，而自由軟體(Free/Open Source Software)的授權方式並不會嗣後撤銷或是向後失效。

5、WinRAR

WinRAR就是典型的shareware(共享軟體)，其在試用期間可以讓使用者免費試用，逾期則授權終止，軟體程式是受到著作權利保護，一般預設為「權利人保有全部權利(All rights reserved)，所以WinRAR40天的試用期屆至，使用人其實並無任何授權基礎可以繼續使用。建議可以7-Zip這套GPL授權的自由軟體代換掉。

7-Zip專案首頁：https://www.7-zip.org/

OpenFoundry電子報的介紹專文：https://www.openfoundry.org/component/option,com_content/Itemid,362/id,1927/task,view/

6、Adobe Reader9

其授權聲明全文如文末所附，其進階說明網頁如下：https://www.adobe.com/tw/products/reader/rdr_distribution1.html，簡單分析這些授權文件得到的結論如下。

(1) Adobe Reader9性質偏向讓使用者免費下載安裝的Freeware(免費軟體)。
(2)個人至Adobe網站下載Adobe Reader9並安裝使用沒有任何問題，但不得再行散布，此使用的環境不限居家個人使用，亦可於公司內部使用。
(3)營利公司若要自行於公司內部散布Adobe Reader9給員工使用，需線上簽署AdobeReader散發合約，並完成此份文件的傳送，讓Adobe公司作為記錄：https://www.adobe.com/cfusion/mmform/index.cfm?name=rdr_distribution2&loc=tw

7、GOM Player

GOM Player專案繁體首頁：https://www.gomlab.com/tw/

GOM Player性質偏向Freeware(免費軟體)，它的GOM Player 最終用戶使用授權協議如文末所附，重點句在以下二項：
a) GOM PLAYER是一款非商業用途的免費軟體,任何個人,企業,學校等單位可以在任意地點自由安裝和使用本軟體播放自己的媒體檔。
b) 本軟體只能用於私人的非盈利性用途. 在沒有得到授權的情況下不得將本軟體用於任何商業用途.不得擅自將本軟體的拷貝分發給第三者。

上面二項告訴我們：
(1)企業可以安裝在企業內部進行使用。
(2)但是企業不能再把這個軟體包裝至產品進行商業牟利。

所以結論是，照這樣的授權聲明，營業公司將GOM安裝於其內部供員工偏向個人性質的使用是沒有太大爭議，只要安裝的目的不是為了讓公司生產影音播放方面的產品。

或者您也可以改用GPL授權的VLC影音播放軟體：https://www.videolan.org/vlc/

VLC在OpenFoundry網頁上的介紹專文：https://www.openfoundry.org/component/option,com_content/Itemid,402/id,1910/task,view/

其他更多以自由軟體授權方式釋出的應用軟體可參閱：https://www.openfoundry.org/component/option,com_content/Itemid,402/id,268/task,blogcategory/
