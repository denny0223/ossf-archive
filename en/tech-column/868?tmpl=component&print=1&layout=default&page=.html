Enfold Desktop 測試報告

Created at Monday, 12 March 2007 23:22      Last Updated on Thursday, 15 March 2007 18:13 Written by 紀博文（數聯資安工程師）

{mosimage} Enfold Desktop（以下簡稱 ED）是由 Enfold System 公司所開發。Enfold System 是一間提供 Zope/Plone 解決方案的公司，本身也包括 Zope/Plone 的教育訓練。


ED 特別之處，就在於他能夠將原本只能在 Web 介面操作的內容，以 類似網路資料夾的形式嵌入到檔案總管之中。

安裝 ED 需包含兩個部分，分別是 ED Client 以及 ED Server 。ED Server 被包裝成一個 Zope Product 的形式，必須安裝到 Zope Server 中。而 ED Client 則必須安裝到各台 Client 端電腦。

此外，由於 ED 透過 WebDAV 協定與 Zope/Plone 連結，但 Zope 預 設的 WebDAV 功能是關閉的，因此必須把 zope.conf 中的 WebDAV 設定開啟。 

  <webdav-source-server>

#   # valid keys are "address" and "force-connection-close"

    address $PLONE_DAVSERVER_PORT

#    force-connection-close off

  </webdav-source-server>

當設定好 ED 之後，Zope/Plone 的內容會在檔案總管中變成一個檔 案夾。接下來就跟一般的檔案操作方式一樣，可以直接複製/貼上檔 案，也可以用 Dreamweaver 編輯網頁檔案，或者直接用文字編輯器 修改檔案內容。

{mosimage}

修改後的檔案，並不會立即回存到 Zope/Plone 內，必需要經過一個 Check-in 的動作。檔案才會正式存入。

另外一方面，ED 對於中文支援性也相當不錯，筆者試著在 Title 與 Description 欄位輸入中文字，都能夠正常的運作。

{mosimage}

ED 對於文件管理相當有幫助，例如對行政人員而言，他並不需要實 際弄懂如何檔案放上 Plone 或者要怎麼編輯原有的檔案，整個過程 就像在使用一般的檔案夾一樣。

為了凸顯 ED 的功能性，在 Enfold System 的網站上，特別製作了 一段 ED 的展示影片。有興趣的朋友可去看看他的神奇之處。

然而 ED 本身是商業軟體，好消息是非商業行為則可免費使用。
