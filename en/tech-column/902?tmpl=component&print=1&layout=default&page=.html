入門 Plone 自訂樣式

Created at Saturday, 24 March 2007 08:00      Written by 紀博文

* 授權條款：Creative Commons 姓名標示－相同方式分享 2.0 台灣
* 教學目標：由 Plone 定義好的樣式表格介面，自訂 Plone 樣式。以背景與文字顏色為例。

圖 1: 自訂顏色樣式前 

 {mosimage}

 

圖 2: 自訂顏色樣式後

* 操作步驟：

1.登入 ZMI 介面後，點選 Plone 網站的位置。

 {mosimage}

 

圖 3: 點選 Plone 網站

2.點選 portal_skins (Controls skin behaviour (search order etc))。

 {mosimage}

 圖 4: 點選 portal_skins (Controls skin behaviour (search order etc))

3.點選 plone_styles

 {mosimage}

  圖 5: 點選 plone_styles

4.點選 base_properties (Plone's color, font, logo and border defaults)

 {mosimage}

  圖 6: 點選base_properties

5.選取 custom 之後，點選 Customize 按鈕。

 {mosimage}

  圖 7: 選取 custom 之後，點選 Customize 按鈕

6.在 globalBackgroundColor 填入 yellow(黃色)，GlobalFontColor 填入 black(黑色)

 {mosimage}

  圖 8: 在適當欄位填入適當顏色

7.點選 Save Changes

 {mosimage}

  圖 9: 點選 Save Changes

8.重新瀏覽 Plone 網站之後，即可看到樣式已經更新。

 {mosimage}

 圖 10: 自訂完成之後
