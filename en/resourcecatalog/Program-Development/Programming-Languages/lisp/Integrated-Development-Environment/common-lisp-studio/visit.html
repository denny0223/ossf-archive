Ufasoft Common Lisp  4.37

Common Lisp development system. Includes command-line and IDE Lisp interpreters. Possibility of creating EXE-files

Licensed under GPL

Ufasoft Common Lisp is fork of GNU CLISP implementation.

Principal difference is that CLISP core implemented on C language. Ufasoft Common Lisp keeps most of CLISP's .lisp files unmodified, but reimplements the core on C++.
