Komodo Edit vs Komodo IDE

Komodo Edit is a great editor if you’re looking for something powerful, yet simple.

If you need more professional functionality such as debugging, unit testing, code refactoring, code profiling, plus integrations with other technologies (version control systems, Gulp, Grunt, PhoneGap, Docker, Vagrant, etc.), Komodo IDE is the solution for you.
