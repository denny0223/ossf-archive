What is Resource Catalog ?

The OpenFoundry Resource Catalog lists professional resources and applications related to the development of open source software. If you have any recommendation listing / category  or bug for this resource catalog, please do not hesitate to contact us.

Embedded Star

Embedded Star covers the embedded system industry. It includes articles, papers, vendors, blogs, jobs, journals and so on. The contents cover real-time operating systems (rtos), single board computers (sbc), unified modeling language (uml), wireless tools, microcontrollers, dsp, fpga, ip cores, test solutions, networking, and more.
Platform: unknown; License: unknown
References: https://www.embeddedstar.com/

Learn-c.com

This site uses hands-on experiments to show the basics of how real things in the real world are controlled with computers. That means it's about most of the computers in the world. Most computers don't sit on desks, but are used to control things that don't look anything like a computer, such as factories, spacecraft, toys and appliances. This site provides the opportunity to learn basic control and embedded system concepts while taking advantage of the low cost and conveneince of using a PC as a platform.

Platform: unknown; License: unknown

References: https://www.learn-c.com/

Linux for PowerPC Embedded Systems

 

這份文件試圖告訴你，為了在以PowerPC為基礎的嵌入式系統上使用Linux，你必須要了解的東西。這些內容是從linuxppc-embedded郵件列表的集體智慧中萃取而來的。

適用平台: Linux ;原始碼授權類型: GNU Free Documentation License

資料來源: https://penguinppc.org/embedded/
