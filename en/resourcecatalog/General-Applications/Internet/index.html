What is Resource Catalog ?

Evolution

Evolution is a free software licensed under the terms of the GNU General Public License. It is the official personal information manager and workgroup information management tool for GNOME since version 2.8 in Septemeber 2004. It combines many functions: email, calendar, address book, and task list management. Users can use it easily because its user interface is similiar to Microsoft Outlook.

smb4k

Smb4k is a networking software tool based on the KDE desktop environment. Smb4k helps transfer files between computers, regardless of the operating systems involved SMB or CIFS, It is also compatible with "Network Neighborhood" in Windows.

PCManX

PCManX is the Linux release of PCMan, a famous BBS software used in Windows. Because PCManX is largely a ported version of PCMan, it may have fewer functions than the original Windows version. It should still however maintain all basic functions. It is also worth mentioning that PCManX makes use of pop-up message windows, just like MSN. This helps users retain some sense of familiarity.

PuTTY

PuTTY is a free , open source and light-weight terminal emulator application which can act as a client with Telnet, SSH and SSH2 protocols. This program provides an encrypted environment for users to conduct remote access or exchange data behind the firewall while connected to a Telnet or SSH supported computer. Moreover, it also provides private keys for users to use in order to prevent data leakage.

PieTTY

PieTTY is a free and open source terminal emulator application which can act as a client for the SSH and Telnet computing protocols. It is developed originally from PuTTY. This light-weight program provides better support for Chinese language text, as well as simplifies the original use interface. It also provides an encrypted environment for users to conduct remote access, exchange data behind the firewall while connected to a Telnet or SSH supported computer. Moreover, it also provides private keys for users to use to prevent data leakage.

Free Download Manager

Free Download Manager is an open source download management tool under GNU GPL.  It supports HTTP, FTP, and BitTorrent protocol. It accelerates downloads by splitting files into sections and then downloading them simultaneously. According to their website, the download speed increases up to 600%, or even more! Free Download Manager can also resume broken downloads so users needn`t start downloading from the beginning after casual interruption. It also provides a built-in upload manager for users to share files easily.

gFTP

gFTP is distributed under the terms of the GNU Public License Agreement. It is a free multi-threaded file transfer client for UNIX-like based machines. It is written in C and has a text interface and a GTK+ 1.2/2.x interface. It supports the FTP, FTPS, HTTP, HTTPS, SSH and FSP protocols, and it supports UNIX, EPLF, Novell, MacOS, VMS, MVS and NT (DOS) style. It supports bookmarks menu to allow users to quickly connect to remote sites just like CuteFTP.

KompoZer

KompoZer is a WYSIWYG (what you see is what you get) web builder. Formerly called NVU, KompoZer was officially discontinued in 2005, however some developers have continued updating the unofficial version. KompoZer is an easy-operating web editor. It can be used to create clear and good-looking webpages without the user needing to understand any HTML-based computer languages. KompoZer also includes a CSS editor to help users edit the CSS in webpage more easily.

Firefox

Mozilla Firefox is a free, cross-platform, graphical web browser developed by the Mozilla Foundation and hundreds of volunteers. It is a lightweight, fast, and highly extensible standalone browser. It also now becomes the Mozilla Foundation's main development focus. This program includes an integrated pop-up blocker, tabbed browsing, live bookmarks, built-in Phishing protection, support for open standards, an extension mechanism for adding functionality and localization for Firefox in different languages.

FileZilla

FileZilla  is a free FTP solution. It is a fast and reliable cross-platform (Runs on Windows, Linux, *BSD, Mac OS X and more) FTP, FTPS and SFTP client with lots of useful features and an intuitive graphical user interface. Users can use it easily by drag & drop. It is also an open source software distributed free of charge under the terms of the GNU General Public License. It has a client and a server release.

PCMan

PCMan is an open source software developed by a Chinese software developer in 2001. A famous free BBS software for Windows! PCMan comes in two release - PCMan Combo (both Web and BBS) and PCMan Lite (only BBS). Both versions have standard built-in unicode complement fonts, Traditional Chinese and Simplified Chinese conversion capabilities, and a worldwide Chinese BBS list. Furthermore, PCMan supports tabbed browsing and customizable hotkeys, buttons, and icons.
