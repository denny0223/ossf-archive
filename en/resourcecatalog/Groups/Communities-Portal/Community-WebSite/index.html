What is Resource Catalog ?

The OpenFoundry Resource Catalog lists professional resources and applications related to the development of open source software. If you have any recommendation listing / category  or bug for this resource catalog, please do not hesitate to contact us.

TOSSUG 台北開放原始碼使用者社群

TOSSUG 是一群熱愛開放原始碼軟體的使用者，每週二在台北有免費無線網路的店家聚會，平時則是交流電腦、軟體、程式、通訊、網頁與技術的相關資訊，每月的第一個星期二會舉辦心得分享，邀請各領域高手分享使用經驗與作品發表，TOSSUG參與者中軟體開發者的密度相當高，亦不時會有外國友人來訪。TOSSUG亦不時舉辦一般性聚會如節日禮物交換、遠足踏青等等。

Ubuntu Taiwan

於2005年6月10日，由一群愛好Ubuntu的人所成立的台灣Ubuntu Linux推廣組織—致力於改善Ubuntu的中文使用環境、提供更多中文軟體及文件資源、幫助台灣地區的使用者更方便地使用Ubuntu Linux，同時提供愛好者一個交流討論的園地。目前社群主要分為數個工作小組，有網頁維護、文件、套件包裝、中文環境等。另外也有活動推廣組，負責規劃台灣地區的活動規劃與文件製作。

Study Area 酷！學園

Study-Area 酷學園教學團隊是一個致力推廣教育訓練的社群組織。每個月在台北、台中、台南皆會請到專業人士為大家免費講解較為進階的電腦技術。從網路佈線、Linux或Windows系統管理、資訊安全、到各式程式語言等各形各色的資訊技術議題。

moztw.org Mozilla 台灣辦事處

Mozilla Taiwan - moztw.org 是台灣專屬於 Mozilla 相關資訊的網站， 於 2004 年九月正式開張。目前本站管理、維護 Mozilla 的正體中文語言套件（Language Pack）跟已在地化（Localized）的安裝檔，提供 WWW (下載、文件)與 討論區 等服務。Mozilla Taiwan 本著推廣主力 — Firefox 易學常用的特性，自許為使用者接觸自由文化的起點。目前持續辦理推廣自由文化的課程與活動，若有相關需求或合作提案，請連絡社群連繫長。
