Open Source Software Application Consulting Centre

教育部致力於校園資訊應用多元發展的推動，以培養學生正確的資訊應用觀念以及善用資訊科技解決問題的能力， 並希望透過自由軟體的推動，縮短數位落差，實現教育機會均等的理想，除支援各縣市自由軟體的推動， 並成立「校園自由軟體應用諮詢中心」，執行校園自由軟體推動計畫、建置並運作推廣諮詢中心，協助解決中小學校園使用自由軟體時所遇之使用、推廣及教育訓練等相關問題。

Chinese Open Systems Association

一群有心的學者專家與社會人士，有感於「以顧客需求為導向的開放式系統」資訊時代即將來臨，乃於八十年三月二十七日結合了政府單位、學術機構、電腦公會，以及國內外各大資訊服務業者，共同發起成立了中華民國開放系統協會(COSA)。致力推廣「開放系統」的運用，同時結合主管機關的力量，為國內中文開放作業系統相關標準規格催生。經過了六年辛勤地耕耘，在推廣國內「開放系統」的環境，蒐集並整理國際間開放系統環境之各類標準，以及影響中文標準的制定。

Software Liberty Association of Taiwan

軟體自由協會是由一群熱心的開發者發起，在 Linux 和 FreeBSD 等自由軟體起飛的時代，以宣揚軟體自由的理念為職志、以推廣自由軟體與開放源碼的使用和開發為己任；希望藉著一個非營利組織的努力，使軟體自由的理念深入人心，更使從事創造的工程師、科學家、教育家等在完成一件作品時，會以自由軟體的形式散播，使該作品成為可以繼續成長的有機體，促進知識交流，除去知識障壁。

Open Source Software Foundry

即本單位，成立目的在促進與提昇台灣產、學、研、開發社群乃至於個人使用者，對於自由軟體研究、開發與使用的程度。計畫成果包含專案開發平台、自由軟體人才資料庫、自由軟體資源連結列表、自由軟體鑄造場電子報，自由軟體相關教材與知識庫系統等。本計畫並提供如軟體專案開發管理、網站託管、自由軟體法律授權問題解答、版本控制教學，自由軟體商業運用模式諮詢等服務，成功協助Mozilla、Perl、Python、Wordpress、LifeType等軟體專案建立穩定成長的技術社群，並應答多家竹科廠商的法務諮詢。本計畫定期舉辦有關企業運算中介軟體與嵌入式軟體等重要領域的國內及國際性研討會與工作坊。其他成果包括主辦國際性的自由軟體競賽，與國內嵌入式應用廠商合辦自由軟體社群交流活動，並至業界、社群以及大專院校舉辦教學性質的推廣演講。

Open Source Software Portal

願景旨在發展台灣成為全球高質化自由軟體應用典範，並朝發展環境優質化、產業體系規模化、開發社群國際化、自主產品高值化的方向推動，期能帶動週邊衍生性資訊服務之產業發展。台灣自由軟體產業在產官學研共同努力下，結合產官學研各界資源與力量，以健全產業價值鏈體系、建立產品自主的高附加價值地位以及建構社群國際知名度。

National Center for High-Performance Computing

國家實驗研究院高速網路與計算中心—又簡稱「國研院國網中心」為台灣一具備高速計算能量和學術網路資源。—『我們不斷嘗試以全觀的角度活用能量。以對人、對生活、對世界的關懷，希望整合運用國研院國網中心研發科技為各領域的生活應用加值，讓科技與生活有更多對話，使科學有更多新發現、科技不斷再創新。』

國研院國網中心於1991年成立，2003年轉制為財團法人，隸屬於國家實驗研究院，簡稱為「國研院國網中心」，為高效能計算、儲存、網路、平台整合之技術先導本位。在新竹、台中、台南設置有三大事業群，以服務、研發、育才的理念在台灣紮根，並於 2005年起，陸續取得ISO 9001:2000品質管理、ISO 27001:2005資訊安全管理國際認證及國家級「科技組織績效評鑑」的認證。
