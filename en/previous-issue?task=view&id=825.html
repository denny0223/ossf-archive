JStock－開源的股票看盤軟體
作者  林雪凡
軟體名稱：JStock
介紹版本：1.0.7
官網：https://jstock.sourceforge.net/
授權：GPL-2.0

 

簡介

JStock 是一款開源的股票看盤與管理軟體。

以一般市面上的股票軟體為標準，JStock 的功能介面可說是相當地簡單。不過正因為簡單，這也反而容易上手。

您可以用 JStock 管理股票的買入賣出狀況，可以用它監視一系列您有興趣的股票，可以在股價低於或高於某數值時送出警告。此外您還可以設計一些「指標」（如 CCI 買進訊號），用這些指標來一口氣過濾分析大量的股票。當然，基本的技術分析線圖也是有的。

關於 JStock，就聽我介紹一下心得吧。

Python Software Foundation 透過協商守住歐洲的商標
作者  黃郁文／編譯

The Python Software Foundation (PSF) 日前宣布，與英國公司 PO Box Hosting（其股票上市名稱為 Veber）就 Python 商標爭議一事已達成和解協商！在此之前 Veber 於 1997 年租用了 python.co.uk 網域名稱，並直接以 Python 字樣來稱呼其提供的雲端服務，同時於近期申請 Python 商標在歐洲的註冊，此一舉措讓已使用 Python 字樣為其程式語言名稱的 PSF 大為震驚，從而自今年 2 月份開始，向四面八方尋求協助，以對抗 Veber 此一商標註冊行動。而令人欣慰的是，經過眾多自由開源軟體開發社群與商用公司的表態支援後，該爭議已於日前圓滿解決。

4 月 20 日是 2013 年的硬體自由日
作者  謝良奇／編譯

硬體自由日 (Hardware Freedom Day) 的主辦單位已敲定 4 月 20 日為該活動第二屆的日期。由非營利組織數位自由基金會 (Digital Freedom Foundation) 於 2012 年所發起的硬體自由日，是為了建立軟體自由日 (Software Freedom Day) 的硬體版本。軟體自由日這個自由軟體的慶祝活動，同樣是由數位自由基金會主辦。該組織已經在世界各地舉辦過許多活動，讓人們了解並參與那些在他們周圍發展的自由軟體、開放源碼與社群。硬體自由日是為了促進開放硬體 (Open Hardware) 的理念，鼓勵人們不加限制地分享設計，並基於這些設計創造實際硬體。

LinuxCon Japan 2013 一般票價報名期間至 4/30 日止！
作者  OSSF／編述

LinuxCon Japan 為 Linux Foundation Japan 每年固定於日本橫濱市或東京市舉辦的大會，其亦為 Linux Foundation 最早於亞洲地區常態性舉辦的綜合年會，議程規劃採多軌方式進行，以容納 Linux 領域裡不同性質的「軟體開發者、資訊專家、企業用戶、資深經理人、公司營運者、系統建置師，以及對 Linux 學習與應用感興趣的學生或媒體」，都可以在這個活動裡發掘到符合自己向性的議程，並進而能夠更深度的應用與推廣 Linux 作業系統。而經過了多年的細心經營，該活動已成為亞洲地區 Linux 相關技術、法律，以及營運議題最佳表現與表演的舞台之一！

自由軟體達人募集活動開跑！
作者  自由軟體鑄造場電子報

行政院研究發展考核委員會與自由軟體鑄造場為了讓全台民眾對於自由軟體的便利與自由性有更深的認識，舉辦了「體驗開源－自由軟體達人甄選活動」。期盼透過：自由軟體的概念與推廣介紹的短片創作、自由或開放原始碼軟體的實際開發與創作、以及自由軟體相關技術與應用的教學文章及部落格的推廣達人等三種方式，以輕鬆活潑的創作內容，讓民眾更能夠接受及學習平常比較少接觸的自由或開源軟體，進而更加地認識、使用它們，進而參與各種自由軟體社群的討論與交流活動。

歡迎各界對於自由／開源軟體有興趣以及熱忱的朋友們報名參加！
