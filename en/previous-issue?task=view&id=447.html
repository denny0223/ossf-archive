【牛】轉乾坤 時來運轉！
李貞臻    2009/01/15

再過幾天就是除夕夜啦！電子報編輯小組先來給大家拜年，希望大家在新的一年裡－身體健康，萬事如意，也謝謝大家一直以來對於電子報的支持與鼓勵。

面對嶄新的 2009，電子報編輯小組會持續秉持專業，將最新國內外自由軟體相關新聞、自由軟體法律授權訊息與案例討論、實用的操作技巧等呈現給大家，另外亦會邀請產官學界的專家，將自我使用自由軟體的心得、技術成果、好站推薦等訊息，透過電子報與各位讀者分享。

對於本報的任何想法，歡迎您隨時來信指教，您寶貴的建議，都是讓電子報更臻完美的動力。最後祝福大家－【牛】轉乾坤 時來運轉！

電子報編輯小組 敬賀
michelle@iis.sinica.edu.tw

EMC 購併 SourceLabs 增添開放源碼動力
謝良奇/編譯    2009/01/06

企業資訊儲存公司 EMC 正試圖打入開放源碼界，日前購併 SourceLabs 的部份資產並獲得該公司的主要人才。據外界猜想，SourceLabs 優秀的開發者團隊可能是購併主要理由之一。

位於西雅圖的 SourceLabs 是家開放源碼新創公司，所開發的軟體技術為系統管理者、開發者與品管經理，簡化 Linux 與開放源碼 Java 相關問題的解決過程。該公司的 SourceLabs Support Suite 運用開放源碼協助用戶發現、定義並解決和 Linux 與 Java 有關的系統錯誤、系統當機、缺陷等等，其資料庫包含 1600 萬已知的 Linux 與 Java 問題。

越南政府計畫全面改用開放源碼
謝良奇/編譯    2009/01/13
越南資訊與通訊部 (Ministry of Information and Communications) 正計畫在 2010 年之前，讓該國政府全面改用開放源碼軟體。根據報導，越南資訊與通訊部長 Le Doan Hop，根據報導已經要求中央與地方政府 IT 人員，在 6 月 30 日前更換所有伺服器為開放源碼軟體。Hop 更要求在 2010 年之前，所有政府工作人員都要使用開放源碼軟體。
