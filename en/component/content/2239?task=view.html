Mac clone 製造商 Psystar 轉而銷售 Linux 電腦

Created at Thursday, 31 December 2009 08:00      Written by 謝良奇/編譯

製造 Mac 兼容機器 (Mac clone) 的 Psystar 在 Apple 提出的著作權訴訟中敗退後，為履行法院判決，稍早已經停止了該公司所有硬體產品的銷售。如今該公司打算放棄其 Mac 產品而轉進銷售 Linux 系統。同時 Psystar 也暫時下架了該公司另一款協助用戶在 PC 上安裝 Mac OS X 的軟體產品。

除了銷售安裝 Mac OS X 的一般個人電腦，Psystar 也販售名為 Rebel EFI 的軟體產品。該軟體用來檢驗裸機系統 (bare-bones system) 是否能夠安裝 Mac OS X，並推薦適合的第三方驅動程式。

由於 Rebel EFI 是在 Psystar 開始販售其 "Hackintosh" 個人電腦之後才推出， 因此並未納入 Apple 的著作權侵害訴訟之中。所以 Apple 儘管在該訴訟上擊敗 Psystar，Rebel EFI 因為訴訟時間點之故，得以排除在外。

Psystar 在公司網站上做出宣佈。該公司表示法院命令不影響其產品保固，現有用戶不須擔心。只有用戶擁有有效保固，該公司會持續履行硬體保固。

之前 Psystar 宣佈將遵守不得販售 Hackintosh 系統的永久禁制令時，曾表示該公司將等待法院裁定是否將 Rebel EFI 納入其中。如今 Psystar 宣佈將暫停 Rebel EFI 銷售。該公司表示，Rebel EFI 現有用戶仍可透過電子郵件或軟體內建的 ticket 介面，獲得支援。

除此之外，Psystar 也表示不同意法院的判決，而他們的行為並非真的違反著作權。該公司宣稱，用戶擁有軟體，應該有權力決定其使用場合。假如用戶購買了架上的 OS X Snow Leopard，就有權使用該軟體。這就像出版商不能禁止你在浴室閱讀書籍，或一邊騎腳踏車一邊聽 CD 一樣。不論 Apple 或任何人砸了多少錢，軟體領域不該有所差別。

儘管如此，很明顯地在爭取消費者權利上，Psystar 做得並不好。另外，Psystar 從何處取得 Rebel EFI 的程式碼，也是一個引人關注的問題。盡管該公司宣稱軟體是自行開發，Netkas 與 OSx86 社群中的其他人卻認為那是他們的軟體程式碼。

但是這一切不代表 Psystar 就要放棄 Rebel EFI。一旦法院同意 Rebel EFI 並不包含在禁制令中，Psystar 將持續銷售該軟體。該公司表示 Rebel EFI 會加入新的功能，包括針對 Dell 9 Mini 與 HP Mini 小筆電的硬體支援。

此外，在硬體方面，Psystar 計畫重新開始銷售電腦產品，但是以 Linux 取代 Mac OS X。該公司表示，未來該公司將再次銷售完整系統，價格會有所折扣，而且將搭載用戶選擇的 Linux 作業系統。除了採用頂級元件，該系統更將特殊挑選相容於 OS X 的硬體。
