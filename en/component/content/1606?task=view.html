Acer 看好 Linux 超可攜筆記型電腦市場

Created at Friday, 13 June 2008 08:00      Written by 謝良奇/編譯
世界第三大個人電腦製造商 Acer 正瞄準逐漸增溫的小型筆記型電腦市場。根據該公司高層說法，Acer 認為 Linux 將成為推動此一低價筆記型電腦市場的關鍵，該公司並希望為這類型設備發展出周邊社群。 Acer 行銷副總裁 Gianpiero Morbello 則更進一步表示，Acer 在 Linux 市場有相當大的計畫，不僅包含低價的超可攜 (ultraportable) 裝置，也包括該公司的筆記型電腦產品。

不少近來推出的新低價筆記型電腦紛紛採用微軟 Windows XP 或 Linux 作業系統。以 Acer 為例，該公司就開發了 One Page，做為筆記型電腦載入作業系統前的使用環境，減少用戶開始使用電腦前的等待時間。One Page 提供像是音樂播放、即時通訊、遊戲、網路通訊等功能。

另一家積極採用 Linux 的個人電腦製造商 Asus，也在其部份筆記型電腦上，搭載 Express Gate 這套 Linux 軟體，Express Gate 與 One Page 相似，可以在開機時直接載入起始環境，快速提供網路存取、遊戲、音樂等功能。

Acer 想進一步在這樣的軟硬體搭配上，發展周邊社群。Acer 資深組織副總裁 Jim Wong 表示，Acer 將會與更多夥伴建立合作關係，Linux 一直是孤軍奮鬥的代名詞，他們需要發展出產業社群。

在低價筆電上的 Linux 社群發展，另一方面凸顯出各大個人電腦製造商要求微軟持續提供 Windows XP 支援的動作，儘管微軟希望由 Windows Vista 取代 XP，由於 Vista 比 XP 需要更多昂貴元件，將增加低價筆電的成本。

從微軟的角度，當然不願失去這塊版圖，並已積極提供 XP 支援，甚至修改該作業系統，以便在一學童一筆電 (One Laptop Per Child) 基金會的 XO 筆電上執行。儘管如此，XP 授權費用仍然使得使用 XP 的裝置，在成本上高於採用 Linux 的產品。

Acer 日前在台灣舉辦的 Computex 展覽會上，公開了新的低價小尺寸筆記型電腦 Aspire one。要價美金 399 至 499 的這款小型筆電主攻需要便宜易用行動裝置的用戶。 Aspire one 採用了 Intel 的新 Atom N270 晶片。

Aspire one 配備 512MB 或 1GB 記憶體，搭載 Linpus Linux Lite 或微軟 Windows XP，內建了 802.11b/g Wi-Fi。

Acer 發言人 Jean Zhu 表示，HP、Dell 等大廠的關注，顯示此一市場所具有的潛力。Acer 預期明年此一市場會是目前的雙倍。

除了 Acer，Intel 也看好被稱為 netbook 的新超可攜電腦市場潛力。Intel 發言人 Chris Tulley 指出，新興市場家庭多半沒有個人電腦，而成熟市場家庭多半擁有一部個人電腦，Intel 認為他們有機會讓擴大家庭市場，甚至讓每個人都擁有一部裝置。Tulley 表示該公司預期 netbook 與 nettop 的銷售，會超越傳統的筆記型電腦與桌面電腦。

netbook 或稱為超可攜筆記型電腦，是價格相對低廉且小尺寸的筆記型電腦，可提供基本應用軟體如網路瀏覽、電子郵件與文書功能。nettop 則是功能類似的桌面設備。

Acer 之外，Linux 也受到其他台灣廠商的青睞。Asus 掀起低價筆電風潮的 Eee PC，同時有 Linux 與 Windows XP 的版本。MSI 的低價筆電，也有 Windows XP 與 Linux 版本。
