兩大行動 Linux 組織 LiPS 與 LiMo 宣佈整合 共推行動 Linux 系統

Created at Tuesday, 01 July 2008 08:00      Written by 謝良奇/編譯
當行動 Linux 市場競爭逐漸昇溫至高點，日前兩大行動 Linux 陣營為整合雙方資源進而宣佈整合。關注行動 Linux 不同但可相互補足層面的 Linux 手機標準論壇 (Linux Phone Standards Forum，LiPS) 與 Linux 行動基金會 ( Linux Mobile Foundation，LiMo) 兩大組織，同意整合各自資源，為 Linux 行動發展發展更為強大的產業群。

LiPS 於 2005 年底成立，由 Palmsource、France Telecom 與 Orange 等公司創立，一直以來致力於行動 Linux 正式標準的建立。LiMo 則是於 2007 年 1 月成立，14 個月以來成員已經達 40 個行動市場的重要廠商。相對於 LiPS，LiMo 的目標是為開放源碼行動平台開發共享實作。

LiMo 至今已獲得數家重要電信業者與手機製造商的支持，如 Motorola、NEC、Panasonic、Samsung、LG、Vodafone、NTT DoCoMo、Red Bend Software、Sagem Mobiles、SK Telecom、Kvaleberg 與 Infineon Technologies。美國第二大行動服務商 Verizon Wireless，日前也宣佈支持 Linux 行動手機作業系統，加入 LiMo 基金會。

行動 Linux 大會 (Mobile Linux Conference) 主席也是 LiPS 總經理的 Bill Weinberg 表示， 儘管所採取的方式不同，LiMo 與 LiPS 確有相同的目標，那就是發展強大的行動 Linux 平台與社群。LiPS 強調標準，LiMo 則注重實作。

LiPS 去年釋出了部份標準規格。然而正當該市場面臨競爭壓力日盛，許多公司更加關心實際的產品而非正式標準。如 Nokia 不久前宣佈將買下 Symbian 行動作業系統的其他部份，並將以 Eclipse Public License 開放源碼釋出。另一個外界關注已久的，Google 的 Android 平台，根據官方說法，將於今年底前首次釋出。外界預料實際釋出時間可能將延至明年。

Weinberg 指出，在此快速變遷的市場之下，今日的挑戰來自於快速反應的機動性。他表示，更有效率運用資源以支持實際目標，是 LiMo 與 LiPS 合併主要目的。

Linux 行動平台已可見於美國市場的許多智慧型手機中，例如 Motorola 的 Razrs 產品。中國市場推出 Linux 手機已有 4 年之久。Weinberg 表示，Linux 已經搭載於 5 千萬支智慧型手機上。根據某些分析師指出，到了 2010 年，Linux 行動平台將與 Windows Mobile 及 Symbian 勢均力敵。藉由 LiPS 和 LiMO 的結合，他們能為此一產業群創造更大動力。

此外，另一個促成雙方整合的原因，來自於對 Linux 行動平台分歧化的擔憂。Weinberg 表示，缺乏單一來源的完整行動 Linux 標準，對於許多人來說是一大問題。

在 LiPS 與 LiMO 雙方的目標下，兩大團體希望為已經開發的應用軟體，建構更完善的中介軟體。Weinberg 指出，有許多功能有待定義並相互整合。

兩大陣營整合的消息，已獲得來自社群的肯定。Linux 社群貢獻者同時也是 LiMO 會員之一的 MontaVista Software，該公司創辦人兼任技術長 Jim Ready 表示，在他們與重要客戶共同合作 Linux 手機過程中的一大問題，正是相關組織數量過多。他表示，例如 Motorola，就加入了每一個組織。珍貴的產業資源也因此分散開來。

VirtualLogix 共同創辦人兼任組織策略執行副總裁 Michel Gien 同樣針對這次雙方合作給予掌聲。Gien 認為，我們不想要有兩個基本上擁有共同目標，僅在方法上有所不同的組織同時存在。藉由資源協同合作，Gien 認為兩大團體的會員可以加強 Linux 行動手機產業圈，並發展更大的成長動能。

在此同時，Nokia 日前也宣佈有意與 Symbian 基金會 (Symbian Foundation) 的其他電子產品製造商進行合作。Nokia 執行長 Nigel Clifford 表示，其目標是讓 Symbian 成為最廣為採用的軟體平台。該基金會有來自各大電子產品公司的支持，例如 AT&T、NTT Docomo、Samsung Electronics、LG Electronics、Motorola、Sony Ericsson、STMicroelectronics、Vodafone 與 Texas Instruments。
