Access 公開 Access Linux Platform 3.0 介面功能皆加強並相容 LiMo 標準

Created at Thursday, 30 October 2008 08:00      Written by 謝良奇/編譯

眾所期待的 Palm OS 新一代繼任者遲遲不出現之際，下一代 ALP (Access Linux Platform) 加上針對低價手機的輕量級作業系統版本，日前已經對外公開。

繼任 Palm OS 的新一代作業系統平台已經讓市場等待許久，簡單且具用戶親和性的 Palm OS (Garnet)，已逐漸不敵近來層出不窮各家行動平台吸引目光的能力。Palm 曾經宣佈該公司代號 Nova 的下一代 Palm OS，將於 2009 年上市。以 Linux 為基礎的 Nova 行動作業系統承諾要讓 Palm 品牌回復過去產業領先者的光輝。

然而在 2005 年購併 PalmSource，並買下後來改名為 Garnet 的 Palm OS 的日本公司 Access，日前對外公開了他們對於 Palm OS 平台繼任者的最新解答，也就是 ALP 3.0 版本。Access 技術長 Tomihisa Kamada 在該公司年度展示會上，公開展示了 ALP 3.0。

Access 宣稱，ALP 3.0 完全符合 LiMo 基金會 (LiMo Foundation) 的標準。LiMo 基金會是致力於建立標準化行動 Linux 中介軟體層的主要產業組織。相容 LiMo 標準將有助開發者撰寫的應用軟體，能跨越各類型行動 Linux 裝置。

此外 ALP 3.0 將相容於 Palm Garnet OS 應用軟體與 Java 應用軟體，降低改換 ALP 3.0 平台的轉換成本。

從 ALP 3.0 公開的螢幕快照畫面看出，ALP 3.0 採用的使用者介面相當直覺，設計美感上可以看到 Palm OS 的影子。根據 Access 的聲明，ALP 3.0 的使用者介面包含動畫效果與視覺加強。伴隨著新增的功能，以及加入更多介面美觀元素，ALP 3.0 可說是一大改進。

當天一同公開的還包括了 ALP mini，根據該公司表示，有意鎖定價格敏感與新興市場的製造商，已可立即取得 ALP mini。Access 指出此一輕量版的 ALP，並非僅僅針對低價智慧型手機與多功能手機 (feature phones)，更包括 PC 之外的各式網際網路裝置，如可持式導航裝置、可持式媒體播放器與機上盒。

Access 執行長兼總裁 Toru Arakawa 表示，Linux 的開放性與彈性正主導手機與網際網路裝置的創新，藉由 ALP mini 與相容於 LiMo 的 ALP 3.0，該公司提供其用戶完整的解決方案，包括功能手機、低價智慧型手機，一路擴展到高階智慧型手機與網際網路裝置，並將進一步開啟新一代的行動網際網路應用。

ALP 採用之 LiMo 標準的競爭對手，包括了 Google 的 Android 平台與即將開放源碼的 Symbian 作業系統。
