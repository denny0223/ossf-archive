使用者心得專欄新登場 歡迎各界踴躍投稿
ossf 2007/01/29

從本期開始，OpenFoundry 電子報新專欄 [源碼心得] 登場！OSSF 自由軟體鑄造場 專案平台可說是國內主要自由軟體專案的寶庫，為了讓讀者們了解 OSSF 進駐專案的軟體，特邀請 OSSF 進駐專案的下載使用者，談一 談他們的使用經驗，近期先從 OSSF 專案下載排行榜的前十名開始介紹，本期刊載的是 OSSF 專案平台上人氣 No.1 的中文 WordPress 專案，請目前實際用 WordPress 架站的使用者談一談他們的使用心 得。如果您對中文 WordPress 有興趣，請別遲疑，現在就到 專案平台 下載！ 在此也歡迎各界支持 OSSF 專案平台的使用者徵求使用心得，只要您 正在使用 OSSF 專案平台上的進駐專案開發出的軟體，如 pLog、 OpenVanilla、SpaceChewingOSX、Mozilla 與 Firefox 中文化、svk 等，歡迎您投稿使用心得給 editor@foundry.iis.sinica.edu.tw， 一經採用者，我們會致贈小禮物給您，期待各位的來信 :) 電子報小編 敬上 2005/08/05

藉由自由軟體推動科學研究－英特爾國際科展冠軍團隊專訪
翁千婷 2005/07/25

由台灣大學應用力學研究所李世光教授指導，於今年五月在英特爾國 際科學展大放異彩的麗山高中邢本元與黃維綱二人組，以「以熱聲效 應改善微電子裝置散熱的研究」奪下五項大獎。雖然不顯眼，但是在 精密複雜的研究背後，開放源碼軟體也扮演著推手的角色。為了了解 開放源碼軟體在其中的助力，我們特地訪問了李教授、邢本元與黃維 綱兩位同學。

問：能否先介紹一下你們的研究？ 答：我們的研究起始於學校必修的「專題研究課」，學生必須自己找 老師指導。之前我們也做過其他以散熱為主的主題，後來找到李教授， 才做這個題目。

授權條款中免責聲明的法律意義
林誠夏 2005/08/04
稍稍瀏覽各式自由軟體的授權條款，您可以發現條文的最後必然有一 段標題是 Limitations of Liability，就是俗稱的「免責聲明」、 「免責條款」，是程式的設計者、著作權人對於損害責任的預免聲明， 強調在任何狀況下，對於使用程式所帶來的各種損害均不負賠償責任。
