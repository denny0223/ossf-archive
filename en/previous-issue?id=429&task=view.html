Wine 歷時 15 年完成第一個正式版本
謝良奇/編譯    2008/06/19
歷時 15 年，Wine 的第一個穩定版本終於現身，許多開放源碼開發者日前齊聚歡慶 Wine 專案釋出 1.0 正式版本。

Wine 可以讓 Windows 應用軟體在其他作業系統上以原生方式執行，支援作業系統包括 Linux、Unix、Mac OS。Wine 在這些作業系統上加入 Windows 相容層，以便讓 Windows 應用軟體在其他作業系統上執行。與當前熱門的虛擬化技術 (Virtualization) 不同的是，Wine 並不需要在本機上，再安裝一套 Windows。

紅帽推出新 hypervisor 計畫於 Linux 內建虛擬機器管理功能
謝良奇/編譯    2008/06/26
紅帽 (Red Hat) 日前在年度大會上宣佈為其 Linux 平台改善彈性、開放性與管理功能，將推出新的 hypervisor，並為 Red Hat Network (RHN) Linux 管理平台開放源碼。紅帽正計畫將虛擬機器管理功能建置於 Linux 作業系統中。

紅帽在 oVirt 專案上發表了以 Linux 為基礎的 hypervisor，嵌入式 Linux Hypervisor (Embedded Linux Hypervisor)，可置於 64MB 的 flash 磁碟中，於 x86 硬體上執行虛擬啟動。紅帽同時也釋出了 RHN Satellite 的原始碼。Satellite 是 Linux 的自動化管理工具，用來管理包括實體與虛擬伺服器的內容更新、系統更新與監測、權限與排程。

保護密度的高與低：侵權與違約差異之我見
自由軟體鑄造場 林誠夏 專案經理/文    2008/06/27
◎ 「侵權」與「違約」伴隨出現不表示完全同質

一般用語上「侵權」與「違約」兩字常常伴隨出現，尤其在著作權法的體制上，因為「權利人保留所有著作權利 (All rights reserved)」的預設，導致著作權授權契約的違反，往往無可避免的造成「既侵權又違約」的結果，就如同打雷與劈閃電一樣，出現 A 結果伴隨出現 B 結果，此種伴隨現象造成一般人對於「侵權」與「違約」在概念上的差異無所知覺，更常常因為慣用語法之故將二者混為一談。其實許多爭訟案件並不必然牽涉到法定權利的侵犯，有些案件雖然有擬似權利侵害的外觀，但實際分析後卻與法定權利侵害無關，而只能論以契約義務的違反。

def NAME( LIST OF PARAMETERS ):

第十一屆中央松濤資工營，即將在七月六日展開！
李貞臻    2008/06/27
● 營期時間：7/6(日)-7/10(四)
● 活動地點：國立中央大學校區(桃園縣中壢市中大路 300 號)
● 招生對象：全國各高中職生及國中應屆畢業生，共 96 位學員
● 報名網址：https://camp.csie.ncu.edu.tw/

每年固定在暑假舉辦的中央松濤資工營，主要提供給高中生一個初步體驗大學生活的環境。今年將邁入第十一個年頭，本屆營隊主題在『中古世紀』的氛圍下，規劃一系列的「多媒體」課程，包括短片剪輯、Flash、影像處理等，搭配晚會表演、大地遊戲、深夜探險、舞會活動，讓學員在活潑有趣的情境下，提前感受資工系課程，並建立對電腦的正確認知。

推廣開放原始碼不遺餘力的中研院資訊所自由軟體鑄造場（以下簡稱 OSSF），今年應國立中央大學資訊工程學系的邀請，在五天四夜的資訊營隊中，規劃半天的自由軟體教學，課程內容將介紹自由軟體的歷史演進、基本概念、還有最火紅的開放源碼瀏覽器『Firefox』等，這是 OSSF 第三年參與營隊活動，希望讓學員在潛移默化中加深對自由軟體的印象，運用自由軟體解決問題，學習開放原始碼協同開發的優勢，為台灣教育播下更多自由軟體的種子。
