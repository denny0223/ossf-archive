RealNetworks 釋出媒體播放軟體源碼

Created at Wednesday, 13 August 2003 08:00      Last Updated on Thursday, 08 March 2007 06:39 Written by 謝良奇
RealNetworks 日前公開宣佈釋出旗下影音播放程式原始程式碼，該計畫將在 Linux、Unix、Solaris 平台上，建構支援能力廣泛的開放源碼媒體播放軟體。RealNetworks 表明此舉乃是為了證實對 Linux 社群和開放軟體的承諾，然而，藉由預期帶來的版圖擴張效應，該公司將可望在無線、行動裝置的影音解決方案上取得收益。

在著名的 LinuxWorld 大會上，RealNetworks 公開了 Helix Player 專案，該專案將釋出可在 Linux 上執行的影音播放軟體的原始碼。該公司表示，此專案的目標，是在 Linux、Unix、Solaris 上發展一套包容性廣泛的開放源碼媒體播放軟體，並在流通、發展 Linux 軟體的企業與組織間，取得廣泛的產業支援。根據 RealNetworks 表示，這套開放源碼的 Helix Player，加上額外的 RealAudio 和 RealVideo 二進位檔，將可讓 Linux、Unix、Solaris 用戶，首次享受到與 Windows、Mac 數百萬 RealOne Player 消費者相同等級的媒體錄音和 SMIL 2.0 支援。這項程式碼釋出計畫，將可作為 RealNetworks 在私有系統間，推廣其多格式系統的策略。過去數年內，微軟在媒體軟體上挖走不少市佔率，RealNetworks 看來有意以此挽回損失的市場。RealNetworks 這些年來已陸續釋出旗下的 Helix 影音壓縮技術與 Helix DNA 媒體伺服器，這些技術可支援眾多檔案格式，如 MPEG-4、Windows Media。開發者可自由地將這些技術使用於研究用途上，或者取得授權以發展商業產品。Sony 日前便在 Altair 家庭娛樂設備中，使用了授權的 Helix 伺服器與 audio-video codecs。

 

如今最新的 Helix Player 計畫，將釋出一套建構在跨平台開放源碼媒體引擎 Helix DNA Client 上的媒體播放軟體。該播放軟體的最初版本將於 Linux、Solaris 上發展，遵循 freedesktop.org 標準，並採用 GTK+ 工具套件。對於 RealNetworks 而言，這是一個相對較新的市場。Jupiter Research 研究經理 Michael Gartenberg 表示，其網路效應將擴展 RealNetworks 的版圖。RealNetworks 的 Helix technology 總經理 Kevin Foreman 認為，此舉對他們並非一項防衛性策略，而是攻擊性策略。這也恰好成為他們對於 Linux 社群與開放源碼軟體承諾的另一項證明。研究公司 Directions on Microsoft 的分析師 Matt Rosoff 則對於 Linux 媒體播放軟體的市場規格，持懷疑的看法。他說，對於客戶而言，仍需要高度的技術素養，才有能力在桌面上執行 Linux。他不確定此舉對於 RealNetworks 的財務是否會產生可觀的效果，

 

不過這的確和他們的多平台策略相當一致。日前 RealNetworks 公佈了第二季的損失，幾乎為去年同時期的兩倍之多。該公司公佈的淨損失為 960 萬美元，而去年則為 490 萬美元。時值第二季虧損，該公司執行長 Rob Glaser 將行動影音媒體軟體視為該公司在獲利上的轉機。RealNetworks 相當重視無線市場的爆發力與創造力，在無線市場上，開發者正在尋求能支援各類型檔案格式的單一媒體引擎。藉著釋出原始碼，RealNetworks 可從授權金方面獲得收益。Glaser 提到該公司日前與 Nokia、Ericsson 和 Vodafone 的合作協議，這些公司已獲得在其 3 億 5 千萬支行動裝置上，使用 Helix Universal Mobile Server 以及 Gateway 的授權。Foreman 也很重視這次機會。他說，行動與家居設備，都是影音方面的熱門發展領域，這個地方還沒有贏家出現，他們是這個世界的競爭者之一。RealNetworks 同時也宣佈，已登錄的 Helix 開發者已達 2 萬人。該公司原先預估到年底時才能達到該數字。該公司表示，他們將鼓勵並歡迎對於開放源碼懷有熱情的開發者加入 Helix 社群。為了鼓勵這群開發者進行創新，RealNetworks 成立了一套 Helix 社群資助計畫（Helix Community Grant Program），提供了 7 萬 5 千美元的獎助金，給予具有發展潛力的創新提案。
