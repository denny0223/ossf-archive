中國 Linux 發展如火如荼 紅旗執中國 Linux 市場牛耳

Created at Monday, 04 August 2003 08:00      Last Updated on Wednesday, 14 April 2010 23:03 Written by 謝良奇

Linux 在全球發展迅速，而中國可能就是成長最為快速的 Linux 市場之一。紅旗是中國最早的 Linux 廠商，該公司的紅旗 Linux 是中國主要的 Linux 散佈套件之一。紅旗已和 HP、IBM 等世界主要廠商合作，一方面在中國推動 Linux 相關培訓，另一方面則試圖在各家產品中提供穩定的 Linux 解決方案。

紅旗是中國 Linux 市場上主要的 Linux 廠商之一，專責於開發與推展 Linux 作業系統與應用軟體。對於想與 Wintel、AMD 競爭的中國資訊產業而言，擺脫獨占性的作業系統，擁有自己的作業系統有其必要。由紅旗推出的紅旗 Linux，便是中國的 Linux 散佈套件版本。

散佈套件僅是 Linux 產業的一環，除了 Linux 散佈套件之外，專門人才的培訓，對於在中國推動該作業系統，亦是不可或缺。為此，紅旗設置了培訓部門與各地的培訓中心，過去也曾和 IBM 合作，在中國發展紅旗 Linux 的認證培訓，並於日前在浙江大學軟體學院寧波分院，建立了紅旗 Linux 開放系統技術學院，以培養專門的 Linux 人才。

日前 Culturecom、Transmeta 和 eForce 則在香港，正式同意共同開發與管理 Midori Linux 系統平台。此外，中國更以 Linux 和 AMD Opteron 處理器為基礎，嘗試建構超級電腦。

紅旗日前與 Oracle China 建立策略合作關係，將共同在中國市場，提供企業級的 Linux 解決方案。Oracle China 是世界最大企業級軟體供應商 Oracle 的子公司，紅旗則是中國重要的 Linux 廠商，兩者都是中國 Linux 市場的主要企業。這次策略性合作將可讓中國廣大的用戶群，得以迅速地在紅旗的 Linux 平台上，部展穩固的 Linux 基本架構，並取得完整的技術支援。

紅旗執行長劉博表示，由於 Linux 在中國的應用步調極為快速，企業級產品越加成熟，紅旗與 Oracle 的合作時機相當正確。這項策略合作不僅僅對紅旗與 Oracle 意義深遠，更加是 Linux 在中國發展的重要里程碑。

Oracle 與紅旗對於 Linux 在中國的應用有著相同的共識，在合作初期，雙方將選定政府單位、通訊運輸、公共與能源事業，作為目標客戶，目前雙方已開始招募 Linux 軟體供應商。

而繼 IBM、Intel 等硬體廠商對 Linux 提供支援，紅旗日前亦同 Acer 達成合作，Acer 將於在南亞銷售的個人電腦中，預先安裝紅旗 Linux 作業系統。在此合作案中，紅旗將提供紅旗 Linux 桌面版 3.2 版，該系統採用 KDE 桌面環境，並擁有相當優異的穩定與可靠性，對於網際網路與辦公室應用提供了可靠的保證。此外，其使用者界面十分個性化，並且易於使用。
