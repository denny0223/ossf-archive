「2003 夏季源碼營」之一

Created at Friday, 31 January 2003 08:00      Last Updated on Wednesday, 12 January 2011 18:58 Written by 林鈺維
台灣政府在日內瓦 WSIS (資訊社會世界高峰會) 之後，對於資訊科技、自由軟體和 NGOs 組織之間的合作特別感興趣，這個可行的合作除了提供一個另類管道讓台灣在中國的打壓下仍能透過提供資訊科技技術的方式和國際上的 NGOs 組織合作外，也提供了台灣自由軟體社群壯大的一個方式--和台灣本土的社運團體合作。2003 年八月底我參加了荷蘭 Technical Tech 在克羅埃西亞威斯島舉行的夏季源碼營，這個夏令營的用意最主要在促進自由軟體和 NGOs 社群交集的合作互惠。自己親身參與的經驗或許可以提供出來成為台灣未來進行類似活動的參考。 (編按：作者林鈺維將參與「2003 夏季源碼營」所見所聞寫成長篇雜記，與大家分享參與國際性社群聚會的經驗。全文將分三期刊登。)

活動主旨

為什麼自由軟體和 NGOs 社群的合作是互惠的呢？簡單地說，自由軟體社群需要開拓新使用者增加市場佔有率 (總不能一直讓微軟視窗一枝獨秀吧)，而 NGOs 組織經費有限、但卻極需要先進的資訊科技來運作(好比說 NGOs 組織很多是全球化的運動，各地區組織間的聯絡就需要網路科技，而且有時 NGOs 組織的運作 (好比說一些人權組織) 涉及對抗獨裁政府或是全球化的大企業，他們所用的資訊科技和傳遞的訊息必須更著重在加密保護，才能保護這些 NGOs 工作者的人身安全，而如果有頂尖的駭客幫助無異是為他們加了一雙翅膀)，這樣看，NGOs 和自由軟體社群的合作真的是有互補的效果，各取所需。對很多自由軟體的工作者來說，能和 NGOs 合作也是福氣啦，他們有機會了解不同的使用者，也有可能再開發更好的產品，對程式設計者來說，和使用者直接接觸能更激發出創作的能量。而且，事實上，自由 軟體社群所崇尚的理念某種程度上也和 NGOs 組織不謀而合，自由軟體 Copyleft 的理念其實充滿社會主義理想，和 NGOs 組織一樣，自由軟體社群也在為創造一個知識共享的人類社會努力，兩者的結合還真是天生一對，一拍即合。

2003 年八月底至九月初，Tactical Tech 與 OSI (不是 Eric Raymond 和 Bruce Perens 發起的 Open Source Initiative；是那位因操作期貨引發 97 年東南亞金融風暴的富豪 George Soros 的 Open Society Institute) 合辦了自由軟體與 NGOs 組織交流的夏令營 Summer Source Camp，結合自由軟體與非營利/非政府組織兩陣營，將雙方人馬聚集在克羅埃西亞的威斯島 (Island of Vis, Croatia) 上，了解彼此的需求與長處並互相學習合作。其實在這個營隊之前，國際上雙方陣營的合作早就如火如荼的展開了，會議上有許多簡報來自前輩們在非洲中亞洲各地 的開發中國家進行專案計畫的心得，只是從來沒有直接的管道建立社會網絡，這個營隊因應目前雙方陣營最迫切的需求—人脈資料庫—而生，和許多社群活動組織一 樣，自由軟體社群目前所面臨的當務之急之一就是建立一個合作交流的社群網絡，成員想知道遇到問題的時候找誰解決，滿懷抱負充滿理想但是苦無識貨的伯樂賞識 自己，因此這個營隊除了讓NGOs和自由軟體社群的成員 都更了解彼此之外互相學習彼此的知識，最重要的目的是在一個融洽和睦的環境下讓建立起一個永續合作的社群環境，因為，和你一起在孤島上生活一個星期的人， 就算不成為朋友也很難忘記他/她吧。

生活方式

說威斯島是個孤島是有點誇張，這個島以前是軍事基地，不對外開放觀光，一直到克羅埃西亞 1998 年脫離南斯拉夫獨立之後，才開放給遊客觀光，也因此島上明媚的風光仍維持著非常原始的風貌，並不非常的觀光商業化，不管來自何方，與會者都必須搭船才能抵 達，或從克羅埃西亞出發，或從義大利出發。在克羅埃西亞的首都 Zagreb 搭夜車到港口城市 Split 的旅途中，就遇到十幾個同路人，從 Split 搭船到 Vis 的途中，儘管前一夜在夜車上沒睡好，但是一見到許多來自不同國家的與會人 (奇怪的是我們似乎都辨認的出來是不是要去開會的還是去觀光的)，仍睜著一雙惺忪的睡眼和大夥兒聊天，到了威斯島，主辦單位的地陪就已經在港口等我們，我 搭的那艘船同行者近三十個人，光是行李就塞滿一卡車，一行人就浩浩蕩蕩地步行往夏令營所在的前軍事基地出發，沒錯，這真的是像救國團的夏令營一樣，住在前 軍事基地裡，男女分房 (不過女生只有十五個人所以才一個房間，最後經過票選是所有房間中最芬芳的:-))，睡行軍床，廁所也不足，女生還好，十五個人共用一個淋浴和一個廁所， 男生人數眾多，一百個人才共用兩個淋浴兩個廁所，真慘，而且熱水有限(其實整個島都缺水，更慘的是克羅埃西亞今年夏天因為天氣太熱頻生森林大火，水源更是 不足)，很多人就乾脆不洗澡了，不然就直接到地中海去游泳去，早餐過後有早點名(其實不是點名啦，就是早會，公佈事項並且交換心得，但是以一種非常有趣的 方式進行)，我們的營長 Gunner 帶隊帶的很有一套，每天早上都會問問大家的活力度 (Energy Level)，然後想一個中心題目，讓與會者用一個字來描述他/她的意見，好比說，用一個字描述昨天所學到的東西，這樣讓大家都有說話表達的機會，所以每 天早上的早會挺令人期待的，而且前一個晚上酒喝太多的話，這個早會也可以振奮一下精神)，每個房間都依照一種程式語言命名 (女生的房間命名為 Ruby)，每個與會者，不管是與會者或是「輔導員」(facilitators -- 這裡的用語非常救國團，但是其實 facilitators 都是自由軟體社群的高手，將帶領與會者討論自由軟體的科技社會法律課題和實際操作軟體)，都會被分配到打掃和廚房的工作，幫忙分配飯菜，就像學生時代吃營 養午餐時分工合作的情況一樣，不讓你感覺到有階級 (不管是從種族性別或是從 digital divide 的角度去看)的差異。

第一天下午就有人忍不注到海邊去曬太陽游泳去了，地中海的誘惑實在是擋不住，第一天晚上是歡迎晚會，所有來參加會議的人之前就被通知要攜帶居住地最有名的 飲料 (可含也可不含酒精) 與大家分享，想像一下總共有一百二十個人來自三十五個國家，你就知道第一天晚上我們有多醉了 (給不喝酒的人了解一下，把多種酒混在一起喝是最容易醉的情況，這樣不但很容易醉，隔天的宿醉還會很糟糕) (唯一美中不足的是音樂，因為現場播放的是克羅埃西亞駭客自己 mix 的電子音樂，可惜不怎麼 high 得起來)，而且還得再這種情況下學十幾種乾杯的語言，這種娛樂社交真是促進國際交流最方便的方式了…

雖然主辦單位請了一位波蘭廚師準備三餐，但是大部分的與會者在吃了夏令營的伙食一個星期之後都瘦了一圈，這位波蘭廚師不知道為什麼獨厚豆乾 (Soya)，但是他煮豆乾的方式非常奇特，煮成義大利麵醬也就算了，但是每天吃一樣的東西也會膩吧，加上天氣熱，真的是讓人食慾不佳，每天想進食的大概 只有水和咖啡而已，記得有兩三天沒有咖啡了，很多人都快活不下去了，但是最後一天的晚餐吃的不錯。為了落實環保，所有與會者都被要求攜帶自己的餐盤餐具和 杯子，可惜還是有懶惰或健忘的與會者沒有帶，現場雖然還是提供了免洗杯，但是有鑒於不停地使用免洗杯會製造很多垃圾而且又很浪費，我自願幫大家洗免洗杯再 利用，當我提出這項建議時，獲得許多人讚賞的眼光，這就是 Summer Source Camp 的精神，有聲就發，同意的人就會配合，每個人都有講話的機會，彼此尊重不同的意見，沒有性別種族和階級的差別。

與會者介紹

學不學的到東西是一回事，但是夏令營好不好玩卻關係著與會者有沒有意願參加下次活動，而這個好不好玩的關鍵在於一起生活一星期的人是誰。先來做個人口普查 吧，如同先前提到的，與會者有 120 人，分別來自 35 個不同的國家，歐洲和北美的與會人員約 1/3，其他的大部分來自中亞 (大部分是脫離前蘇聯獨立的新國家，很多中亞國家的名字我還沒聽說過呢，像 Azerbaijan，Tajikistan，Belarus，還有 Georgia -- 不是美國的喬治亞州喔，是脫離前蘇聯獨立的喬治亞共合國)和一些非洲國家，其中輔導員和主辦單位工作人員佔了 1/4 (想當然爾幾乎是和歐洲和北美地區的人，尤其因為出錢的是索羅斯的基金會，因此工作人員大半是美加來的人)，輔導員主要都是自由軟體工作者。整體而言，與 會人員分為兩部分：第一部分是從事 NGOs 組織的工作者，稱為 Implementors，主要在會後將所學的自由軟體知識和技術應用到NGOs的日常工作中，像非政府非營利組織的執行人員，e-Riders，文教基 金會的顧問 (例如，福特基金會的顧問、Summit Collaborative 的 Dot Org Media Project)，第二部分是軟體開發者，稱為 Developers，泛指所有軟體工程師，這其中很多是非政府組織的網路管理員，例如，綠色和平 (GreenPeace) 組織的網管，英國 Appropriate Software Foundation 的網管，自由軟體的設計者研發者)，正因為大家的文化背景和專業不同，這種異質性也讓與會者之間的互動更有趣。

設備

營地有兩間電腦教室五十台桌上型電腦 (規格約是 1 Ghz，128MB RAM)，其中一半裝有 Windows 系統，雖然活動是有關自由軟體，但是仍不能強迫所有人都使用 Linux 或其他自由軟體作業系統，畢竟有些人還是習慣用 Windows，除了 LAN 以外，整個營區有 ISDN 撥接網路，還有 802.11g/b 的 WiFi network，但是因為整個營區全部只靠一條寬頻網路上線，第一天啟用的時候非常擁擠，營區的網路大塞車，後來靠在場的駭客想辦法改了 gateway 另外設了 proxy 的方式解決，輔導員另外有工作室，派有專人駐防守衛喔，裡面還真是藏了高科技的機密呢！所謂的會議室由幾張摺疊式的鐵椅為成圈，地上擺了幾床床墊，不想坐 鐵椅或是佔不到鐵椅位的人就可側臥床墊上，這麼輕鬆的環境卻進行著非常嚴肅的腦力激盪，因為室內討論空間不足，很多分組討論活動其實必須移到戶外進行， 喔，不僅僅場地原始，與會人穿的保證你看不出來大家是來開會的，很多男生禁不住熱打了赤膊，大部分的人是 t-shirt 短褲涼鞋，但是很多女生是穿著 bikini 加熱褲，想像一下東歐妹妹各個身材窈窕花容月貌，金髮碧眼，又是在男特多女特少的環境下，看官您大概可以想像得到在威斯島這個地中海小島上那些男生所受的 煎熬了。
