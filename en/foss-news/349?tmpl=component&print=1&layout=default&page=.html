Linux 延伸影音娛樂設備領域

Created at Sunday, 27 July 2003 08:00      Last Updated on Thursday, 08 March 2007 06:38 Written by 謝良奇
除了伺服器、桌面系統、可攜式設備外，Linux 的觸角仍不斷向外擴展，最近延伸到了影音娛樂設備的應用上。包含 Philips、NEC、Sony、Motorola 等電子設備大廠，皆推出了以 Linux 作業系統為核心的影音娛樂設備。

由於 Linux 擁有的高度客製化能力，以及開放源碼本身的可用性，許多電子大廠開始尋求 Linux 在娛樂設備上的應用可能。PRISMIQ 的 MediaPlayer 是一款機上盒（set-top box）產品，該公司稱為網路化的娛樂通道，使用 Linux 作為核心的該款設備，可以播放音樂與影像媒體檔案，從家用電腦與網路上，取得資訊播放至電視或立體音響。使用者可藉由該設備在電視上瀏覽網路、進行即時通訊以及存取多種網路服務。PRISMIQ 工程副總裁 Leon Hounshell 表示，他們在 MediaPlayer 上選用 Linux，是因為 Linux 能適切地支援多種其客戶需要的應用軟體，如串流媒體、網路瀏覽和即時傳訊，再者，成本也相當漂亮。他補充說，還有其他因素像是開放源碼的可用性，以及活躍的開發者社群，不過 Linux 的品質與特點，致使它成為該專案的最佳選擇。Hounshell 對嵌入式 Linux 的未來發展有著樂觀的看法，

 

他認為，Linux 很顯然會持續佔領嵌入式市場的版圖，特別是當設備更為連線化，客戶需要越來越多的功能，如串流媒體，情況更是如此。這是 Linux 能夠一展所長的領域。日本電子設備大廠 Sony 亦推出過一款採用 MontaVista Linux 專業版為作業系統的個人錄影設備（PVR），稱為 CoCoon Channel Server。Sony 並非將 Linux 應用於 PVR 市場的第一家廠商，PVR 市場領導廠 TiVo 早在其產品中採用了由該公司工程師自行客製開發的嵌入式 Linux 作業系統。在機上盒領域亦有產品推出的 Motorola，也在旗下的 DCT5000 中，採用了 Lineo 公司的 Embedix Digital Media 這套 Linux 系統。

 

該公司選擇 Linux 軟體，是因為這類軟體提供了強大的系列功能，並且可以透過允許進階數位程式化與服務的驅動程式和延伸元件，加強系統本身。Linux 環境能夠提供寬頻網路業者許多可觀的利多，像是較低的作業系統與開發成本、對於新硬體功能更為迅速的設備驅動程式支援、較少的中介解決方案移植時程，並且能更容易地移植額外的新互動式應用軟體。Motorola Broadband 的 DigiCable 業務副總裁兼總經理 Carl McGrath 表示，Embedix Digital Media 核心提供了開放且穩定的作業系統軟體解決方案，適合其先進機上盒平台，同時有助壓低開發成本。日前由 MIT 博士班學生 James Patten 和 Ben Recht 開發的 Audiopad，則是一款採用了 Debian Linux 的音樂作曲、演奏樂器。該設備能追蹤其表面上的物件位置，將其動作轉為音樂。該設備軟體以著名的開放源碼語言 Python 語言開發，在低階的追蹤程式碼上使用了部份的 C 與 C++。
