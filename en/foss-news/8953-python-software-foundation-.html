Python Software Foundation 透過協商守住歐洲的商標

Created at Friday, 12 April 2013 09:53      Written by 黃郁文／編譯

The Python Software Foundation (PSF) 日前宣布，與英國公司 PO Box Hosting（其股票上市名稱為 Veber）就 Python 商標爭議一事已達成和解協商！在此之前 Veber 於 1997 年租用了 python.co.uk 網域名稱，並直接以 Python 字樣來稱呼其提供的雲端服務，同時於近期申請 Python 商標在歐洲的註冊，此一舉措讓已使用 Python 字樣為其程式語言名稱的 PSF 大為震驚，從而自今年 2 月份開始，向四面八方尋求協助，以對抗 Veber 此一商標註冊行動。而令人欣慰的是，經過眾多自由開源軟體開發社群與商用公司的表態支援後，該爭議已於日前圓滿解決。

PSF 就 Python 這個字樣在資訊與軟體開發領域下的使用，已經在美國取得商標權 (trademark) 的註冊，不過過往因為經費有限，而未一併申請在歐洲的商標註冊，從而 Veber 才仍能在歐洲提出同一字樣的商標申請，依商標基於國際協議與各國法制的互惠原則來看，PSF 仍有機會主張已使用 Python 字樣多年，且於資訊領域已獲得廣大使用群眾的認同，而可以援引其具有著名商標、標章 (well-known trademark or mark) 的地位，以事後撤銷 Veber 在歐洲申請的 Python 商標，不過透過正式司法訴訟或是異議程序總是曠日費時，故在 PSF 公開徵詢各界的支援與協助後，Veber 也重新檢討了此一商標申請舉措的合宜性，雖然部份網路支持者失序發表了對 Veber 公司成員的人身威脅或是無理謾罵，但可幸最後雙方可以透過協商的方式來理性對談！經協議之後，Veber 同意改以新的品牌名稱行銷其雲端伺服器與備份服務，並同意撤回商標申請案，中止其原已運作的「https://python.co.uk」網址，並支持 PSF 在歐洲繼續使用 Python 一詞來推廣 Python 程式語言的相關專案。PSF 的主席 Van Lindberg 表示：「PSF 建置的最主要的目的，就是要對全球的 Python 開發社群提供支援，而他很欣慰 Veber 願意承認 PSF 使用 Python 字樣的合法與合宜地位，及同意迅速完成和解，並期待 Veber 產品在市場上也能有亮眼表現！」Van Lindberg 同時對熱情參與此一活動以促成和解協商的廣大 Python 社群表示感激，感謝其等提供金錢與行動上的支援，以協助爭議能儘速落幕。

而雖然 PSF 使用 Python 字樣為其標章，在資訊軟體領域已甚為著名，但為免往後再發生類似的爭端，Python Software Foundation 已透過正式流程進行歐洲商標的註冊。
