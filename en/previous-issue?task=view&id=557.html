Vim - 自動補齊
Cornelius    2009/12/13

自動補齊 (Auto-Completion) 一直是每個整合環境編輯器 (IDE) 都必備的一項功能，因此 Vim 也不例外的支援了自動補齊 (Auto-Completion) 的功能。

其中最大的不同就是使用者可以撰寫或設定自己的補齊函式 (Completion Function)，其中還包含了使用者定義補齊 (User-defined completion)、全補齊 (Omni-completion)、關鍵字補齊 (Keyword completion)、檔案名稱補齊 (Filename completion)、字典補齊 (Dictionary completion)、詞庫補齊 (Thesaurus completion) 以及 Vim 指令補齊 (Vim command completion ) 等等各種的自動補完方式。

OSSF新聞
OSDC.TW 2010 開放徵稿！
簡信昌    2009/12/12

OSDC.TW 2010 開放徵稿囉～

OSDC.TW 2010 預計在 2010/4/24-4/25 舉行，地點假中央研究院人文館會議廳。這次會議的籌辦除了由台北 Perl 推廣組持續擔任主辦的角色之外，中央研究院自由軟體鑄造場也投入大量的協助，促成這次活動場地的訂定。

OSDC.TW (Open Source Developers' Conference)是台灣幾個主要推動開放源碼的大型研討會之一，和其他活動不同的是，OSDC.TW 主要參與的對象是程式開發者，每年都有不少台灣及國外的開發者與會進行交流與分享。讓人興奮的是，OSDC.TW 2010 已經獲得 Jacascript 教父級人物 Douglas Crockford 的首肯，將親身抵台進行公開演講。

OSDC.TW 2010 於十二月初已經正式開始對外徵稿，只要您過去一年在開放源碼領域有什麼有趣的開發成果、經驗，或是您目前心目中有什麼有趣的點子想找人一起合作，都歡迎您來 OSDC.TW 跟大家分享，徵稿的截止日為 2010/1/31，會議的時程表也將在徵稿截止後儘快公布。

● OSDC.TW 2010 徵稿
● 聯絡人：簡信昌 hcchien@osdc.tw

Nokia 釋出 Qt 4.6 版 支援 Symbian、 Maemo 平台
謝良奇/編譯    2009/12/08

Nokia 最近更新了 Qt 開發套件。Qt 是基於 C++ 的開放源碼使用者介面與應用軟體框架，被用來為許多裝置如智慧型手機、機上盒與電腦，撰寫應用軟體和使用者介面。Qt 4.6 釋出版本新增加許多功能，讓開發者擴大其應用軟體的使用者族群。

Qt 最引人注意的特色，就是其跨平台能力，這一次釋出的 4.6 版再一次擴大其平台範圍，引入對 Symbian S60 和以 Linux 為基礎的 Maemo 行動作業系統的支援。如此一來內容創造者就能夠在無須修改程式的情況下，讓同一個軟體在多種設備上執行。

Intel 釋出 Atom 軟體開發套件測試版本
謝良奇/編譯    2009/12/10

針對旗下的 Atom 開發者計畫 (Atom Developer Program)，Intel 日前釋出了其軟體開發套件 (Software Development Kit，SDK) 的測試版本。此舉是該公司透過應用軟體商店網路，協助開發者銷售 Atom 軟體的計畫之一。

於 9 月時發佈的該項開發者計畫，目的是推廣針對 Intel Atom 晶片做過最佳化的軟體。該晶片主要應用於小筆電 (netbook) 上。這一類的小型筆電和一般筆電一樣，其市場主要由 WinTel 稱霸的局面，也就是在 Intel 晶片上執行 Windows。如今 Intel 似乎決定加深在軟體層面的參與程度，特別是 Moblin 專案。
