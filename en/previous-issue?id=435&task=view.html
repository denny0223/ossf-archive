OSSF新聞
OpenFoundry 脫胎換骨正式上線囉！
李貞臻 2008/08/08

OpenFoundry 是一個供自由軟體愛好者討論、編寫、管理專案內容所設置的共同開發平台，使用者可以透過此平台下載所需的自由軟體，並回報軟體的改良意見。OSSF 技術團隊在本周二(2008/8/5)完成系統升級及介面更新，煥然一新的平台將帶來更穩定的服務功能及更便利的操作方式。

本次升級主要的變更有：

1. OpenFoundry (主系統): 使用 Ruby on Rails 為系統主體，並與各模組整合；
2. RT (待辦事項) : 變成獨立的模組，並升級至 3.6.6 版；
3. Wiki (共同筆記) : 此版未做升級；
4. VCS (版本控制) : Subversion 升級至 1.4.6 版；
5. Sympa (通信論壇) : 升級至 5.4.3 版；
6. Web Hosting (專案網頁空間) : 此版未做升級；
7. Release (規劃釋出) : 升級後支援 FTP 方式上傳檔案。

全新 OpenFoundry 採用 Ruby on Rails 開發，在效能及穩定性上優於舊版本；操作介面的規劃，新增活潑的圖示作為專案與專案管理員的識別方式，使用者也可以自訂個性化的圖示，外觀設計則強調與 OSSF 主網站的一致性；釋出的軟體在新版網站上，可以選用 FTP 上傳，改善以往網頁無法續傳大型檔案與上傳效率不穩定的問題。

OpenFoundry 程式碼採用 MIT License 對公眾授權釋出，詳細程式原始碼可前往 https://of.openfoundry.org/projects/openfoundry進行下載，任何對 OpenFoundry 程式碼研究改寫有興趣的朋友，皆歡迎一同加入開發除錯的行列。升級後如果您原本營運的專案遇到問題，或對新的版本有任何建議，請不吝透過 contact@openfoundry.org 與我們聯繫！

如何像電腦科學家一樣思考 | 4. 條件式
自由軟體鑄造場 張凱慶/翻譯 徐孟逵/校訂 2008/08/10

4.1 餘數運算子

餘數運算子運作在整數（及整數運算式）中，並在第一個運算元除以第二個運算元時，得出餘數。在 Python 裡，餘數運算子用百分比符號（%）表示。語法則與其他運算子相同：

>>> quotient = 7 / 3
>>> print quotient
2
>>> remainder = 7 % 3
>>> print remainder
1
所以 7 除以 3 得到的商是 2，餘數為 1。

餘數運算子的運算結果出人意料地有用。例如，你可以用來檢查一個數是否能被另一個數整除---如果 x % y 的結果是 0，x 就能被 y 整除。

你也可以用它抽取出一個數值中的最右側的數字。例如，x % 10 可得出 x 最右側的數字（在十進位時）。同樣地，而 x % 100 則會產生最右側的兩位數字。

謝良奇/編譯 2008/07/29
今年在奧勒岡州 (Oregon) 波特蘭 (Portland) 舉辦的 OSCON 2008，聚集超過 2500 位開放源碼開發者、黑客、專家、IT 管理者與使用者。OSCON 幾乎囊闊了與開放源碼相關的所有議題，每年都會吸引各領域最重要最有趣的人們共襄盛舉，除了到場發掘最新趨勢，也要為開放原則與計算產業的開放源碼運用大力發聲。
