香港生產力促進局支援軟件業 開拓內地「開放源碼軟體」市場

Created at Monday, 30 July 2007 07:57      Last Updated on Wednesday, 12 January 2011 19:40 Written by Linuxpilot

為協助香港軟體公司掌握內地發展開放源碼軟體的新機遇，香港生產力促進局在今年推出一系列開放源碼軟體支援項目。香港生產力促進局總經理（資訊科技業發展）容啟泰介紹開放源碼軟體市場的商機時表示：「開放源碼技術可減輕企業在資訊科技的投資成本，因此近年在全球各地的應用率持續上升，而有關應用領域亦橫跨公營及工商機構。據研究公司 IDC 的調查顯示，開放源碼的應用率正以每年平均 47% 的速度增長。而據業界估計，2005 年亞太區的開放源碼軟件市場有 78% 的增長。在內地方面，由於政府大力支持採用及開發開放源碼軟件，在 2005 年的市場增長亦超過 30%。」

◎『十一、五』規劃的發展重點

他續說：「在中國的『十一、五』規劃下，廣東省政府也將開放源碼軟體定為其中一項重點推動的信息化技術，而粵港信息合作專責小組也已經把開放源碼技術定為合作範圍之一。香港軟件業者應配合 CEPA 的優惠政策，積極開拓內地開放源碼軟體市場。」容啟泰表示，香港生產力促進局將加強有關開放源碼軟體的支援工作，協助香港軟體業爭取中國內地市場的新機遇。他說：「成立『香港開放源碼軟件中心』的目的，是為軟體業界與企業用戶提供技術支援、培訓、技術轉移、商業配對、技術推廣等全面的支援服務。而推動香港與泛珠三角軟件業的合作，是其中一個重點工作。」

◎共同開發華南的軟件市場

談及香港生產力促進局與內地及香港業界的合作計畫時，他表示：「香港生產力促進局將與廣東省 Linux 公共服務技術支持中心，就 Linux 開源技術的測評認證、移植開發、技術支援和培訓等方面合作，協助業界開發華南的龐大市場。此外，生產力局也會聯同 IBM，合力提升本地開放源碼軟體技術人員的實力，並致力推廣開放源碼軟體應用方案，當中包括 Linux 作業系統。」

◎加速移植應用方案至 Linux 平台

國際商業機器中國香港有限公司 Linux 業務總監徐成業表示將支援「香港開放源碼軟件中心」的工作。他說：「IBM 除了提供硬體的支援外，還會提供香港及北京 Linux 中心的技術專家支援，並協助香港軟體商把現有的應用方案移植到 Linux 平台上。」

◎粵港軟件業合作的開始

廣東省 Linux 公共服務技術支持中心是廣東省政府重點推動 Linux 產業化，企業及行業信息化的服務平台。中心主任袁泉博士說：「是次與香港生產力促進局合作，可為粵港軟體業提供一個理想的發展平台。在廣東省政府的大力支持下，Linux 及其他開放源碼軟體技術將更廣泛地應用於電子政府及其他行業，為粵港企業帶來大量合作良機。」目前廣東省政府已強制規定伺服器作業系統需使用 Linux，比例達 9 成以上，包括使用於辦公、公安及醫療系統等，桌面端方面的 Linux 應用也陸續開始。香港生產力促進局亦分別與廣東省 Linux 公共服務技術支持中心及 IBM 簽署合作備忘錄，前者由容啟泰及袁泉博士主持，而香港生產力促進局與 IBM 的合作備忘錄則由容啟泰及徐成業簽署。

◎香港中華煤氣分享成功經驗

香港中華煤氣有限公司工程系統部系統發展經理曾帆亦出席了簽署儀式，並向與會者分享了該公司在內地推介其開放源碼應用方案 SCADA 系統的成功經驗。有關的系統監控和資料擷取軟體主要用於監控戶外煤氣設施、管道、調壓站等，定時將煤氣流量、氣壓及溫度等數據經過網路傳回監控中心分析，早期系統是在 UNIX 上運行，但考慮到專屬軟硬體會限制應用系統開發的自由度，所以由第三代系統開始便轉用Linux，公司可以在軟硬體供應商方面有更多選擇。現時香港中華煤氣有限公司在國內有很多合資企業，他們將透過附屬公司提供有關解決方案，大大增強 Linux 在中國公用事業的應用。

◎香港生產力促進局推廣開放源碼軟體不遺餘力

自 2003 年起，香港生產力促進局積極推廣開放源碼軟件，當中包括 Linux 的商業應用。為提高企業對 Linux 的認識及鼓勵更廣泛應用 Linux 平台，生產力局在 2003 年及 2005 年舉辦了 Linux 商業應用大獎，而在 2005 年推行了「中小企 Linux 起動計畫」，向中小企推廣 Linux 的應用。計劃包括香港企業 Linux 應用調查，以及測試不同的 Linux 工具及產品。
