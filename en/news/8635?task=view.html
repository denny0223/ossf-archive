Android 版 Ubuntu 消息發布！

Created at Monday, 05 March 2012 14:04      Last Updated on Monday, 23 April 2012 11:26 Written by 黃湘婷／編譯

Mark Shuttleworth 先前發布了令人興奮的消息︰一個能夠在多核心 CPU 的 Android 手機上使用的 Ubuntu 作業系統。

這套作業系統讓手機不只可以撥打電話，也變成一台攜帶型電腦。當手機與外接螢幕連接，即可在螢幕上操作手機的應用程式，兩者為同步進行且同時共享。

這表示我們可以在 Ubuntu 開啟手機的聯絡人及應用程式，搜尋通訊錄，發送電子郵件，甚至撥打電話。

除了可開啟手機程式外，Ubuntu for Android 還可連接並啟動桌面版的 Ubuntu 作業系統，並提供完整的桌面環境與熱門的應用程式，例如 Chromium 網頁瀏覽器、VLC 影音播放器、Thunderbird 郵件收發軟體、Google 日曆及文件、Ubuntu 的音樂播放器和圖片瀏覽器等，讓使用者可利用 Android 手機加上外接螢幕操作 Ubuntu 系統。

▲ 圖片來源：OMG! Ubuntu!

相對來說，因為手機就是提供作業系統運作的來源，要能用 Android 手機操作 Ubuntu 系統，至少需要 Android 2.3 以上的作業系統、雙核心 1GHz 的 CPU、2GB 的儲存空間、512 MB 的 RAM，以及可以接 HDMI 及 USB 的接口，只要擁有以上的配備，智慧型手機便可執行PC的工作，對於經常出差旅遊的人就方便多了。
