Novell 由 Attachmate 購併 出售部份智財權

Created at Tuesday, 30 November 2010 08:00      Last Updated on Monday, 13 December 2010 19:36 Written by 謝良奇／編譯
Attachmate 宣佈以 22 億美元價格購併 Linux 廠商 Novell 的消息，在自由軟體界引發了一陣評論與分析。此外，或許更重要的是，Novell 同時要將數量可觀的智財權，以 4.5 億美元出售給由微軟建立的科技公司財團 CPTN Holdings。

針對許多懸而未答的問題，Attachmate 發出公開聲明藉以澄清外界若干疑慮。Attachmate 計畫持續 SUSE 平台的開發，而且將支持社群主導的 openSUSE 專案。該公司也證實將保有位居 SCO 爭議核心的智財權，也就是 UNIX 的著作權。

事實上，Novell 被購併並非出人意料之外。關於該公司尋找買主的說法已流傳一時。令人訝異的是，買主並不是許多人預期的 VMware。至於以 22 億美元買下 Novell 的 Attachmate，究竟是何方神聖？

根據對於 Attachmate 有著長期了解人士的說法，Attachmate 多年來的主要業務是軟體終端機模擬器。自從 Web 於 90 年代初期蓬勃發展後，這個產品線可說等於宣告死亡。僅有少部份人仍然需要這類產品。即使加上其 NetIQ 系統和安全管理子公司，Attachmate 依然是一個衰退產業中的一家利基公司。讓人不禁懷疑該公司哪來一大筆錢用來購併 Novell。

再者，Attachmate 與 Novell 的產品之間毫無共同利害關係可言，從商業觀點看來，這次購併放到 80 年代後期或 90 年代初可能還說得通，至於在目前的背景下，可說是一點道理也沒有。

於是外界有了微軟才是這場購併案背後真正金主的猜測說法。首先，對微軟而言這次購併免於讓 Novell SUSE Linux 落入 VMware 之手，在戰術上極有價值。因為微軟最不想看到的，就是讓 VMware 這個主要的雲端與虛擬化競爭者，取得主要的作業系統。該公司於是有了充分時間完善其 Hyper-V 虛擬化和 Azure 雲端的行銷策略，而無須面對 VMware 和 SUSE Linux 的聯手夾擊。

此外，從戰術上而言，一旦 VMware 或其他公司取得 SUSE Linux，這家公司可能不再延續 Novell 和微軟之間的合作關係。但是微軟需要這道跨入 Linux 的橋樑。鑑於由微軟出面購併 Novell 可能引發的反拖拉斯訴訟，以及 Novell 握有的 Unix 智財權，微軟透過 Attachmate 作為代理者取得 Novell，在必要時用來攻擊紅帽 (Red Hat)、Android、Ubuntu。

Novell 出售其智財權給微軟成立的 CPTN Holdings，由於部份原始 UNIX 著作權正位於 Novell 手上的智財權之列，引發了微軟是否運用這些智財權來對付 Linux 平台的疑慮。SCO 針對 Linux 侵害UNIX 智財權的指控之所以不曾被法庭加以評估，是因為 Novell 證明該公司才是相關著作權的實際合法擁有者，這使得 SCO 沒有提起著作權侵害訴訟的立場。

一旦這些著作權被對 Linux 懷有惡意的第三方取得，他們很可能接手 SCO 所作的一切。雖然至今沒有侵害發生的實際證據，但訴訟仍有可能被用來在市場上產生對 Linux 的疑惑和不確定感，這個戰術很符合微軟的反 Linux 手法。

儘管 Novell 證實該公司將保有 UNIX 著作權，意味著隨著購併案結束，該著作權將落入 Attachmate 之手。外界沒有任何辦法阻止 Attachmate 或其他 Novell 的後續擁有者，以訴訟方式從中賺取獲利。畢竟 Attachmate 是由投資集團所主導，當他們認為其中有利可圖時，沒有什麼能阻止他們採取這種做法。

此外，雖然 Novell 部份的商業決定不為 Linux 支持者所樂見，該公司過去的確為其平台周圍的獨立社群發展做出努力。該公司為了擴大 SUSE 的發展，於 2005 年成立 openSUSE 專案，在過去幾年內吸引了若干貢獻者。鑑於和獨立社群維持良好關係所需的可觀投資，不禁使人懷疑 Attachmate 是否會支持 openSUSE 並持續與該專案的合作。

Attachmate 執行長 Jeff Hawn 對此表示，openSUSE 專案是SUSE 業務的重要部份之一，Attachmate 在購併案完成後，將不會在 SUSE 業務與 openSUSE 專案間的關係上有所變動。

儘管這對於 openSUSE 用戶與自願貢獻者來說是個好消息，但並沒有回答 Attachmate 是否將和 Novell 過去一樣對開放源碼軟體主動貢獻。除了 Linux 桌面外，Novell 也是 LibreOffice、MeeGo 等專案的關鍵貢獻者。Attachmate 計畫讓 SUSE 以獨立事業單位營運，雖然表示該 Linux 散佈套件可免於結束的風險，但是外界並不清楚 Attachmate 是否對開放源碼辦公室套件和行動平台有著相同的關注。
