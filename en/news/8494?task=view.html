新聞傳播科系該教，也該學的內容管理系統 (CMS)

Created at Friday, 28 October 2011 16:50      Last Updated on Friday, 18 November 2011 09:44 Written by 鄭國威

◎本文轉載自數位文化學院，網址為 https://d-academy.tw/archives/1644/

過去有的新聞傳播系所會教學生架設網站，例如使用 Dreamweaver 從 HTML 開始寫起，然後一頁一頁上傳到 FTP，不過現在有那麼多正點、架構完整的自由開放源碼軟體，如果不善用豈不是浪費也落伍？問題不在於該不該教，而是要教什麼，或學什麼？

我知道愈來愈多傳播系所的學生透過主動學習，已經擅長多個內容管理系統 (Content Management System)，因為他們知道這些入門技巧就跟拿攝影機或剪接一樣，已經成為是傳播人必備基本知識，但就我個人所知，還沒有傳播系所專門開設課程教導學生如何架設跟使用 CMS。在傳播實踐上，最多就是將架站列為研討會、重要活動或是畢業展覽的要求之一，但這種一次性的架站跟經營工作通常都落在幾個本來就會的同學身上，知識很難外溢。正好，我看見 Andrew Lih（USC Annenberg 新聞傳播學院教授，線上新聞與維基專家）在部落格發表了這篇：Choosing a Content Management System in 2011，我覺得與我自己的使用經驗頗相符。

WordPress、Drupal、跟 Joomla 三大 CMS 我都用過，其中以 WordPress 用得最熟，Drupal 次之，Joomla 最近才開始碰；至於曾經最流行的 Moveble Type，雖然目前還是我個人部落格的後台系統，但是已經久未更新，現在的版本即使不是最新版，我也摸得很痛苦而不再熟悉。對我來說，WordPress 最簡單，我相信對大多數人來說也是，而當我覺得 WordPress 不夠用，得學學其他的 CMS 時，WordPress 總是能再次證明它夠用。Drupal 讓我跌跌撞撞好一段時間，現在沒有繼續跌下去，其實是因為很少使用了，但之前鑽研過一陣子，我認為它的社群管理跟自訂彈性是 WordPress 所不及的。至於 Joomla…我前陣子還在推特上發訊說「如果 Joomla 是一個人，我一定馬上把他打得半死」，此言可佐證我在初接觸這段時間的困惑，老實說，現在還在想要不要放棄算了，儘管 Joomla 的首頁及呈現都會讓人有「這才是一個網站，其他的只是部落格而已」的突出感。

如果傳播科系學生想要 3 個都學也不賴，但如果目前都還不熟稔，或許先挑一個適合的來上手比較好，接下來讓我們看看 Andrew Lih 教授提出的幾個選擇標準：

使用性：就最高管理者的角度來說，WordPress「幾乎」省去了使用 FTP 跟 shell/command line access 來進行維護跟自訂的需求，這就差很大了，不管是要教還是要學，能不去碰 php、css 這些東西而是填內容勾選項的話當然是比較輕鬆。

容量：Joomla 跟 Drupal 內建的功能能夠優雅地降低網站表現，也就是說，如果主機負荷不了的時候， Drupal 跟 Joomla 的網站能夠自動關閉使用量太高的功能，讓訪客還是能夠快速正確地載入網頁。如果你認為你的網站會需要這樣的能耐，就該把 Drupal 跟 Joomla 考慮進去。

社群串聯：Drupal 在這方面表現特別好，具備彈性的系統非常適合用來架設社群網站或是替網站增加社群元素，包括協同篩選跟電子商務。只要簡單的設定，就能創建一系列規則，讓網站的訪客都能擁有自己的部落格、推文跟噓文。就連美國白宮跟 2008 馬英九選總統的競選網站也是用 Drupal 網站架的喔。

外觀跟自訂：Joomla 的首頁最正點，因為可以用在首頁上的功能特多，欄位的變化也非常多樣。Usaviah 網路防災研究室在調整的時候，讓我最有收穫而不那麼苦惱的也是在調首頁的時候，看首頁要三欄五欄欄中欄欄外欄都輕鬆自在。

使用者角色跟工作流程：一般來說，Joomla 跟 Drupal 對於「工作流程」的支援較 WordPress 佳，不過近來只要用 WordPress MU 搭配一些外掛，要建立一個多人共筆，分角色權限編輯的發稿系統也很容易，不輸給 Joomla 或 Drupal，而且通常也夠用了。

主題跟外掛：WordPress 上能選的主題跟外掛實在太多了，遠遠多過其他的系統，所以在這個項目得到完全勝利。Andrew 教授表示，Joomla 比 Drupal 被更多網站採用，主題跟外掛應該更多，不過因為 Drupal 受到許多重要大網站的採用，因此額外加分，與 Joomla 不相上下。

我個人的建議是：對學生來說，能全學就全學，起碼這 3 個系統要很熟悉其中一個，然後另外兩個要了解基本架構跟操作。等到像我這樣開始需要在工作上頻繁用上時才學習，真的太浪費時間了，而且又沒基礎，只能不斷從錯中學。而傳播系所應該開設專門課程教導學生如何使用，並介紹案例，然後讓學生把這些知識應用在傳播議題上、打造媒體性質的網站。
