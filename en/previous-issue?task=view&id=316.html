2007 國際企業運算研討會 (ICESA 2007) 報名 享有優惠價格喔！
OSSF 編輯小組    2007/10/26

◎ 活動時間：2007/12/12-13
◎ 活動地點：中央研究院 學術活動中心 二樓
◎ 報名網址：https://icesa.openfoundry.org/

中研院資訊所自由軟體鑄造場擬於 12/12、12/13 在院內學術活動中心舉辦 2007 國際企業運算研討會 (2007 International Conference on Enterprise Systems and Applications；ICESA 2007)。兩天的研討會中，包含企業運算原理與實務的論文發表及海報展示，此外，主辦單位特別邀請了國立清華大學榮譽講座教授劉炯朗院士及資策會黃彥男副執行長擔綱主題演講者，分別從學術及產業的角度，親身闡述企業應用軟體的未來趨勢。另外自由軟體鑄造場亦安排國際大廠 IBM 及國內 JAVA 工具軟體領導廠商普奇科技分別進行兩場寶貴的技術教學交流。在面對面的座談會上，延攬了來自韓國、香港、新加坡等地的專家，跟與會來賓針對在地的經驗與亞洲區域軟體發展的機會與挑戰進行討論。

歡迎各界先進一同來共襄盛舉，機會難得，席次有限，11/30 前完成報名繳費者，還享有優惠價格喔！

2007 亞洲區 DSP 暨嵌入式系統應用競賽台灣區分區競賽
OSSF 工作小組    2007/11/23

◎ 報名時間：即日起～97年1月31日止
◎ 活動網址：https://www.cn.nctu.edu.tw/event/TI/index.htm

凡在學之大學生、研究生均可以組隊參加，每個隊由1-5名隊員，1名指導教授組成。競賽分成系統應用與軟體演算兩個大組。系統應用組：要求系統軟、硬體均為自主設計。軟體演算組：可採用外購或現成的硬體開發板針對特定算法作設計開發，完成特定功能。

每個大組設特優獎1名，優等獎1名，佳作獎3名，每組之特優隊伍由TI補助至至中國大陸上海參賽（含機票、住宿費用，補助對象為老師1名、學生1～2名），每個獲獎隊伍將獲得獎金和證書。

獎金按獲獎等次按參賽隊為單位發放，不再按指導教授和參賽個人為單位發放，特優獎1名（五萬元），優等獎1名（三萬元），佳作獎3名（各五千元），而不考慮參賽隊由幾名隊員參加。

活動時程表：

1. 97年01月31日 構想書繳交截止
2. 97年03月01日 初審完畢，公布初審通過名單
3. 97年06年01日 繳交實作報告書
4. 97年06月     複賽_參與競賽隊伍成果展示並遴選優秀隊伍
5. 97年07月     頒獎典禮

主辦單位：美商德州儀器股份有限公司
承辦單位：中華民國自動控制學會
國立交通大學 電機與控制工程學系
技術指導單位：教育部顧問室嵌入式軟體聯盟

法律源地
GPL2 相關產品釋出機制的自我評量
OSSF 法政組 林誠夏    2007/11/25

此文的適用客體是以 GPL2 釋出的商用產品，建議欲將 GPL2 軟體應用於商業販售的廠商，可依此於產品販售前進行簡單的自我評量。原始參考文字取自歐洲自由軟體基金會（Free Software Foundation Europe, 以下簡稱為 FSFE）編著之 Useful Compliance Tips for Vendors 一文。本譯文摻雜了許多原文所無的贅字以幫助讀者得其真意，所為者乃提供簡單的中文說明及引論，有意參訪原文者可參照註解所附連結（註一）。而承繼 FSFE 編撰此文的目的，所謂的自我評量清單僅為便利 GPL2 程式的商業利用者，快速掌握 GPL2 程式商用的相關釋出機制，若更有契約條款及法律上的疑問，則廠商應自行徵求法律從業人員尋求其進一步協助。

謝良奇/編譯    2007/11/13
紅帽（Red Hat）推出了紅帽企業 Linux 的 Amazon 彈性運算網雲版本（Red Hat Enterprise Linux on Amazon Elastic Compute Cloud），這是紅帽 Linux 自動化策略中的一步，希望給帶給用戶簡化應用軟體運作與管理的 Linux 基礎架構。在此一合作下，Amazon 彈性運算網雲（Amazon Elastic Compute Cloud，Amazon EC2）網路架構與資料中心的使用者，可以使用到紅帽企業版 Linux 的所有功能，如紅帽網路（Red Hat Network）管理服務、世界級技術支援與超過 3400 種認證應用軟體。

Everex 低價 Linux 個人電腦在 Wal-Mart 銷售一空
謝良奇/編譯    2007/11/15

萬聖節前夕 Wal-Mart 開始銷售 Everex 的低價 Linux 個人電腦 TC2502 Green gPC，售價為 199 美元。Wal-Mart 日前表示這批 Linux 電腦已經銷售一空，該零售商預計補貨後會重新上架。

Oracle Unbreakable Linux 支援計畫成果可觀
謝良奇/編譯    2007/11/20

Oracle 宣佈該公司一年前公開的 Unbreakable Linux 支援計畫，已經有 1500 個企業用戶加入。同時，該計畫在資格認證與 Oracle 驗證架構計畫（Oracle Validated Configurations Program）上也吸引了相當多的合作夥伴參與。

Oracle Unbreakable Linux 支援計畫由於針對 Linux 與全系列 Oracle 軟體堆疊，提供單一的企業級品質服務窗口，已經成為 1500 家企業用戶的選擇，並持續成長中。其中超過 1450 家企業用戶是在最近 6 個月中簽訂該服務，Oracle 執行長 Larry Ellison 在 OpenWorld 使用者大會上自豪表示，這個數字一點都沒有灌水。用戶數比起 6 個月前 Oracle 宣佈的 26 個用戶，有相當顯著的成長。

Funambol 為 Android 平台開發開放源碼行動傳訊系統 Sun 憂心 Android 平台將出現
謝良奇/編譯    2007/11/22

Google 的 Android 平台即將出現開放源碼的行動傳訊系統。發展開放源碼行動應用軟體伺服器的 Funambol，宣稱他們正在開發該平台上，第一套開放源碼行動傳訊及同步用戶軟體。不過 Android 平台有意推出自己的 Google Java 版本，卻讓 Sun 大感緊張。

根據 Funambol 的說法，這套應用軟體可以讓 Android 的手機裝置，與 Gmail、Yahoo 和 AOL 等電子郵件系統，以無線方式同步個人資訊管理（personal information management，PIM）資料。
