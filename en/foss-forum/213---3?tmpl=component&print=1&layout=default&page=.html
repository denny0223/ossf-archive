運用自由軟體 融入資訊教學 (3)

Created at Friday, 10 December 2004 08:00      Last Updated on Thursday, 21 July 2011 17:28 Written by 陳貴成
本週我們繼續聊下去，一項公開的秘密，以及自由軟體另外的特色跟 教育的關係。 第三個我們要聊到的是拷貝給親朋好友的自由，我想這大概是最最符 合人性的一項自由了吧！

沒人會在檯面上說出來的東西，卻是最最真實的事情，我們可以仔細 地一一到每個有使用 Windows 電腦系統的族群去詢問一下，看看有 購買合法授權的家庭使用者有幾位，想來這答案其實是非常公開而隱 密，讀者朋友可以去問問你周遭的 Windows 使用者就會知道答案了。

中國人常常說「好東西要跟好的朋友分享」，所以就會有很多很多的 人在非刻意的狀況下，將自己電腦安裝許多非運用合法管道取得的軟 體，也就是說其實有很多人在非刻意的狀況下不小心的違反了著作權 法。這樣的狀況或許在一般環境好像不是什麼大不了的事情，反正大 家都一樣，所以大家也就無所謂了（當然這樣應該是錯誤的）。若是 我們把場景轉移到教育現場，尤其是道德教育為主要傳承舞台的中小 學教育平台，問題就大囉！！

簡單講清楚，許許多多的老師在學校教學生使用商業軟體的時候，似 乎忘了一件最最基本的事情，就是學生的軟體取得管道為何？各位讀 者可以猜猜看，學生若想要回家後繼續使用老師在學校所教授軟體， 該如何取得？若是依照「正常」的程序，學生練習老師在學校教的商 業軟體，或是將老師所教授的商業軟體應用到日常生活中，大概要花 的費用，從作業系統到應用程式大約要花的費用可能要花上六位數！ 各位讀者或許不相信，但仔細算一算從作業系統、個人防火牆、防毒 軟體、辦公室應用軟體、美工編輯軟體、網頁美工軟體……這樣算一 算，再加上電腦的硬體，我們在接收義務教育的小朋友們學習電腦的 成本還真高耶。

有錢又有軟體合法授權觀念的家長，就會幫學生買這些對學生未來不 一定會用到的軟體，但是這樣畢竟只是少數吧，對大多數的家長或學 生而言，這些軟體取得的管道，就非常明顯是透過非「正常」的程序 取得。這樣說起來，好像是校園內倡導道德教育，傳道、授業、解惑 的講師變成引導學生犯罪的人了，但可憐的是學生、家長、老師們都 不覺得這樣有什麼大不了，或是說並不想正面的去接受這般事實，或 是根本沒去想過這個問題。這對現今大力倡導道德教育倡導智財權的 我們，真是情何以堪阿！

可是若我們將老師所教授的軟體全部用自由軟體取代，這些所謂道德 跟法律的問題將不存在，因為自由軟體基本上的精神就是「分享」與 「散佈」，可說是非常符合人性的一種授權跟發展模式，而且不用擔 心沒有適合的軟體可以教或可以學，在 windows 的系統中要運用那 些自由軟體來導入到校園中可以參考下列的網址 [1]，若是希望直接 用 Linux 教學，又不希望影響到學生家中的作業系統，其實也可以 運用所謂的 Live CD 來達成目的，像校園自由軟體應用諮詢中心與 大豐國小合作的 Linux Live CD [2] 就是一個例子。這樣不管是基礎 教育中的學生、家長、還是老師，散佈及安裝這些學校所教授的軟體 都不會有法律的相關問題，也不會跟現有的法律有任何的抵觸，其實 是一件非常完美的事情。但需要改變的環節有許許多多，就像推動環 保運動一樣，需要持之以恆才會有看得見的改進吧！
