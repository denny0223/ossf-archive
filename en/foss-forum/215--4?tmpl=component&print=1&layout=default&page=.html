運用自由軟體 融入資訊教學(4)

Created at Thursday, 17 February 2005 08:00      Last Updated on Thursday, 21 July 2011 17:28 Written by 陳貴成

各位朋友新年快樂！好久不見啦，經過了 ICOS2004 的洗禮，不知道 各位周遭的親朋好友有沒有愈來愈多人在使用自由軟體呢？記得我們 接續的是 Richard M. Stallma 所主張軟體的四大自由對教育領域的 正面意義作逐一的解說，先前我們提到的三個方向是：

  1. 任意使用的自由
  2. 研究及修改以符合自己用途的自由
  3. 拷貝給親朋好友的自由；
接下來，我們針對他所提出的第四個主張，來說明他對教育的正面意 義有哪些，第四個主張是散佈修改後軟體的自由。

我們最近就遇到一個堪稱之為典範的案子，就是全國中小學校園自由 軟體應用諮詢中心替台北縣力大豐國小製作了一片 Linux LiveCD， 所謂的 Live CD 有一個非常大的特色，就是他可以完全免安裝的方 式，直接就在光碟上執行作業系統及應用程式。這片 Live CD 的環 境，完全是仿造大豐國小的電腦教室 DRBL 的環境而設計的，也就是 說，學生若是拿了這片光碟回家練習，其實就跟在學校作練習的環境 沒什麼兩樣。因為是自由軟體的關係，大豐國小花一萬多塊的的費用 就可以讓全校四五六年級的學生每個人拿一片回家作課堂上的練習， 完全不用擔心在學校學的東西回到家之後沒辦法練習，這就是大豐國 小任意散佈校園自由軟體應用諮詢中心所修改的 LInux Live CD 的 模式。也因為這樣的模式，讓正在求學的小朋友可以用最低的學習成 本，去學到最適合他們的東西。

仔細想想，若是將那 2000 片的 LInux Live CD 換成Windows XP 的 作業系統，再加上 Office 及其他的應用軟體，這樣學生的學習成本 不知要提高多少？（當然在此我們得先暫時忽略台灣盜版嚴重及道德 觀念低落的的問題，不然可能要扯不完囉！）

我們其實可以非常清楚的知道，學生在求學階段，軟體的操作及使用 對他們而言並不是那樣子的重要，重要的是「樂趣」。若是學生為了 讓家中的電腦軟體跟校園中所教授的一致，而花費掉了過多無需要給 付的費用，而且花錢之後還對那些軟體沒有主宰權，你說這樣的學習 生活會有多少的樂趣？（下期待續）
