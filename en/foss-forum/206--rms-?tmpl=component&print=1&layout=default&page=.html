聆聽 RMS 演講心得

Created at Friday, 13 May 2005 08:00      Last Updated on Wednesday, 12 January 2011 18:45 Written by 陳貴成

各位讀者大家好，好久不見啦，哈哈哈 ！

今天就跟各位分享一下比較熱門的話題就是 Richard Stallman 來台 灣這件事情，為什麼要聊這件事情呢？就我們台灣人的習慣呢，這種 人物我們通常都會歸類為「神」級的人物，當然依照對神級崇拜的習 慣，自然小弟我就要在此跟各位報告一下小弟參與這位神級人物的演 講後所得到的一些心得。當然會圍繞在我聽得懂的地方啦，尤其是在 教育的環節上。

所以我就依據下面幾個大項來跟各位分享心得吧！

◎ OH！！這樣的一個大鬍子

第一次見到他的感覺，說實話他不像是一個 hacker，反而像是一個 思想家，一個哲學家的感覺。在他在台大的演講的過程中，我可以非 常清楚地感覺到他是一個對自由軟體非常堅持的人，因為他對 "Open Source" 這個字眼非常的敏感，對我們這種自由軟體的使用者而言， 或許覺得自由軟體跟 Open Source 是差不多的東西，但是對理想主 義者而言或許就有非常大的差異了。

在他演講的過程中，另外給我的感覺是，他是一個非常隨性的人，有 多隨性呢？他可以在演講的過程中無視於台下所有的聽眾就當眾的吃 藥、對重要的部位搔癢，好在在我的心目中他是神級的人物，作任何 的動作都充滿了帥氣的感覺，更充份反映出他隨性的自由頃向，簡單 地講可以完全的說他是一個表裡如一，不會作作的人，我想他可愛的 地方就是這樣吧。

◎ 我心中王牌天神對 Linux 的解釋

原本在 RMS 的演講場合中，我請公司的同事製作了一片從Fedora Core 3 所客製化出來的 Live CD 做為參與他演講來賓的贈品，可是 ……據台大物理系客座教授葉平先生代為詢問的結果是，RMS 先生因 為對自由軟體的堅持，只要 Linux 的光碟套件中有任何的軟體含驅 動程式所授權的方式不符合自由軟體的原則，那麼就不是 RMS 所認 可的 Linux，也因為這樣，不但是 Fedora 不太符合 RMS 的精神（ 因為不確定套件中是不是有加入商業公司所提供的 driver 或是應用 程式），具說就連 Debian 就不太合格（也是因為不確定套件中是不 是有加入商業公司所提供的 driver 或是應用程式）。

喔喔！！還有一點特為重要的，就是 Linux 不應該只稱為 Linux， 應該稱為 "GNU/Linux"，這也是 RMS 非常堅持的一點，因為非常簡 單一個關係，RMS 認為開發 Linux 核心的工具本身就是屬於 GNU， 當然 Linux 就不應只稱為 Linux 囉，所以請大家多多愛用 GNU/Linux。

◎ RMS 對於自由軟體在教育領域的看法

或許是因為演講的場地是在台灣大學的關係，RMS 特別針對為何在教 育市場中，應該要以自由軟體做為資訊教育的基礎？他指出在學校教 育中，若是可以針對自由軟體做為教育的基礎，不但可以讓我們的學 生有分享的精神，而且也因為自由軟體自由分享的特性，不會讓學校 的教育被封鎖在封閉的格式中，例如 Word、PowerPoint 等等軟體， 其格式就是封閉的，所以尤其是在中小學這樣的基礎教育中更是應該 以自由軟體為基礎。

當然神說的話我就是要聽囉，也希望各位可以替自由軟體傳傳福音， 讓大家知道，「自由的好處」也開始嘗試去傳遞使用開放環境的喜悅。
