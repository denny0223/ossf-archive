GitHub 贏得 1 億美元投資的背後

Created at Monday, 23 July 2012 15:17      Last Updated on Tuesday, 24 July 2012 11:14 Written by Kenneth Mak

◎ 本文原載 Linuxpilot 網站，原文章連結

熟悉開源產業的朋友大概都會聽過 GitHub，甚至是 GitHub 的用戶，最近 GitHub 得到著名創投公司 Andreessen Horowitz 的 1 億美元投資，究竟 GitHub 為何會這麼值錢？

GitHub 一如其名是 Git 的集散中心 (Hub)。Git 是 Linux 之父 Linus Torvalds 創立的專案，是版本管理系統 (Version Control System, VCS) 的一種，可以為軟體在開發的途中提供版本的管理。從前開發人員看到一個他想參加的開源項目時，首先就要到官方網站下載程式碼，修改後告知程式碼維護人員，維護人員認為沒有問題的話，才會把修正的部分整合到原來的程式碼中。

很多情況下維護人員與修正檔的提供者素未謀面，在細看修正內容前，往往難以得知每一位修正檔的提供者的專長，亦沒有辦法有系統地記錄這些人的貢獻。後來出現的版本管理系統 CVS 和 Subversion 就提供了機制，方便維護人員和專案擁有人和大量提供程式碼的開發人員進行交流，唯一的問題是程式碼只有一個中央的儲存庫 (Repository)，任何開發人員所作的變更，將會直接影響中央儲存庫的內容。於是 Git 便出現了。Git 是分散式的版本管理系統，開發人員全體都擁有程式碼的複本，可各自進行修正。當維護人員認為沒問題，便可對中央伺服器進行 Check In 的動作改變其內容。GitHub 則為 Git 提供了 Web 界面，方便一些不喜歡用文字命令的人。

GitHub 提供的是一個能整合不同地區、不同企業的完整開發平台，而且吸引了大量開發人員使用，從而令它身價暴漲。在這次投資中 GitHub 的估值達 7 億 5,000 萬美元，而且這只是現時的價值。隨著流動裝置和各種 Web 服務的普及，同時為開源社團提供免費服務和商業機構提供收費服務的 GitHub，其需要只會有增無減。誰說開源產業不賺錢？GitHub 又是另外一個好例子。
