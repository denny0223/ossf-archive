CVS and Subversion 升級公告 / Upgrade

Created at Friday, 06 June 2008 10:00      Last Updated on Friday, 23 July 2010 23:08 Written by ossfepaper

各位自由軟體鑄造場專案開發平台(OpenFoundry)的使用者您好：

我們已完成版本控制系統的升級。

本次升級主要的變更有：

升級 Subversion 至 1.4.3 (http(s)://svn.openfoundry.org)

升級 CVS 至 1.11.17-FreeBSD

將網頁瀏覽介面改為 ViewVC (https://viewvc.of.openfoundry.org)

將 svn.openfoundry.org 與 cvs.openfoundry.org 轉移到新的 IP 位址： 140.109.17.106

本版本暫不提供 VCP 服務

本版本暫不提供自動轉換 CVS 至 SVN 的服務

如有任何問題與建議，請不吝與我們聯絡，謝謝！

 

 

Dear OpenFoundry users:

We have upgraded our version control systems.

The major changes are:

Upgrading Subversion to version 1.4.3 (http(s)://svn.openfoundry.org)

Upgrading CVS to version 1.11.17-FreeBSD

Changing the browsing interface to ViewVC (https://viewvc.of.openfoundry.org)

Directing both svn.openfoundry.org and cvs.openfoundry.org to 140.109.17.106

Halting the VCP service in this version

Halting the automatic CVS-to-SVN migration service in the version

If you have any questions, suggestions or concerns, please do not hesitate to contact us
