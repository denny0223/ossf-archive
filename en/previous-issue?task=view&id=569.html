福【虎】生豐 如【虎】添翼！
李貞臻    2010/02/07

再過幾天就是除夕夜啦！電子報編輯小組先來跟大家拜年，希望大家－虎年行大運，萬事皆順心，也謝謝大家持續對於電子報的支持與鼓勵。

迎接全新的虎年，我們也投注了嶄新的作法，希望呈現給讀者不同風貌的電子報，例如：關鍵字的建立，便利您閱讀相關文章；規劃文章留言功能，方便您即時發表意見與想法。另外，除了持續招募各個面相的專欄作家外，我們還會廣納更多熱門部落客與技術新知文章於本報中。在這一期裡，我們邀請到知名部落客--永遠的真田幸村來分享他個人測試各大瀏覽器的JS效能。希望這一切的一切，都能讓電子報的內容更貼近您的需求與閱讀習慣。當然，對於本報的任何想法，我們非常歡迎您隨時來函指教，因為您寶貴的建議，都是讓電子報更臻完美的動力。

最後，祝福大家每一天的生活－福【虎】生豐，如【虎】添翼！

電子報編輯小組 敬賀

開放原力
電子書雜談
Jedi    2010/02/07

要說今年引人注目的科技發展，大概很難不提到電子書。在這麼多年的發展之後，電子書今年終於站上新秀舞台，主要的原因是來自於硬體的進步：電子紙、手機、平板電腦，這些產品現身展售會、發表會，冠上了時尚的妝扮，牽繫著代工訂單以及工作機會，屢屢受到矚目。

在硬體背後，還得有對應的運作模式來支撐：一旦這些可以閱讀電子書的裝置到了消費者手上，如何讓消費者花錢購買（或租賃）內容的平台。（當然，這也包括了，如何刪除消費者先前購得內容的邪惡勾當。）這類平台還會拿來養成社群，增加使用者黏滯力，作為取得個資及行銷的重要管道。

技術專欄
Vimana 簡介
林佑安 (Cornelius)    2010/02/07

一般來說，Vim Script 的安裝都是到 vim.org 上，搜尋 Script，然後參考該項目的安裝細節來手動安裝。但是每個 Script 的發行、打包的方式又不一樣，所以大部分 Script 的安裝都還是得手動進行。 有的是直接將檔案複製到 Vim 的 Runtime 目錄裡頭即可，有的是得先解壓縮，然後使用 cp 或是 rsync 命令依序照目錄結構複製到 Vim 的 Runtime 目錄裡。Vim 雖然也提供了 Vimball 格式的打包檔，但是也不夠普及。

然而 Perl 有 cpan 命令、Ruby 有 gem 命令、Debian 有 apt-get 命令，Vim 卻因為發行方式不統一而一直沒有一個可用、堪用的工具來做這樣的事情。為了解決這個問題，我的想法是，針對每一個發行的 Script (該 Script 有可能用是 tarball 包裝也可能只是純文字檔案) 來做安裝策略的偵測，於是 Vimana 就誕生了。

紅帽成立線上社群網站 Opensource.com
謝良奇/編譯    2010/02/02

為了促進開放源碼在不同產業的運用，紅帽 (Red Hat) 日前成立了新的線上社群。雖然在開放源碼領域早已充斥許多類似的線上社群，紅帽似乎認為網路還有空間容納新的社群。

新的線上社群稱為 Opensource.com，網頁上掛有紅帽社群服務的標籤。該網站宣稱將作為許多開放源碼內容的匯聚地，分享的內容包括文章、音訊、網路簡報、影像或公開討論。開放源碼技術理所當然是焦點之一，除此之外，網站內容也報導開放源碼文化如何傳播到社會的其他領域，引用的例子包括 Wikipedia、美國總統 Barack Obama 與麻省理工學院公開在網路上的免費教材。
