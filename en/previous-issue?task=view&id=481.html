GPL 授權的圖示
葛冬梅    2009/02/20
提到 GPL 這個授權條款通常都與軟體程式碼相關，但是近年來也看到採用 GPL 來授權圖示的例子，就筆者所接觸到的例子，都是開發者為正在開發的軟體上網尋找的圖示，因而發現這些採用 GPL 授權的圖示（註一）。其中最令人疑惑的是：散布這些圖示的時候，只要將圖示的圖檔連同原始碼一起提供給他人，這樣就符合 GPL 的規定嗎？採用 GPL 圖示的軟體是否也必須採用 GPL 授權呢？光看 GPL 的文字，並無法直接得出問題的答案，因此筆者從探究四大自由開始，來尋找這個問題的答案。

2/27 台灣社群企業 BoF - 自由軟體於隨身裝置的應用
呂健宇    2009/02/11

自 Netbook 於 2007 年底快速火紅以來，隨身裝置就一直是產業高度關切的話題。諸如智慧型手機、行動上網裝置(MID)…等，被視為是下階段的明星產品。而在此硬體規格受限前提下，軟體的開發水準決定了隨身裝置的成敗！因此，企業已意識到須擺脫軟體「閉門造車」的舊開發思維，而積極採自由軟體開放共享的做法，如 Google 就是一個全球成功的案例。

自由軟體鑄造場(OSSF)秉持著促進『台灣社群與業界的交流互動』，邀請和碩聯合科技以同好聚會(BoF)方式就自由軟體於嵌入式隨身裝置的應用進行討論與分享，歡迎有興趣的軟體開發者共同參與！

大型多人線上遊戲 EVE Online 終止 Linux 版用戶端支援
謝良奇/編譯    2009/02/10

在大型科幻多人線上遊戲 EVE Online 即將推出最新升級版前夕，CCP Games 宣佈由於使用者人數不如預期，將停止 EVE Online 的 Linux 用戶端開發和支援。CCP Games 製作人 Arend Stührmann 在網誌上證實了此事。

EVE Online 從 2007 年底推出 Revelations 2.3 升級版開始，提供 Mac、Linux 的原生用戶端支援。這兩個版本的原生用戶端，來自於和 TransGaming Technologies 的開發合作，該公司在 Mac 用戶端上運用其 Cider 技術，而在 Linux 作業系統上，則採用了類似的 Cedega 技術。然而根據 Stührmann 表示，同時推出在 Mac 與 Linux 平台上的原生用戶端卻有不同結果。

紅帽推出 Enterprise MRG 1.1 完整提供即時、傳訊與網格能力
謝良奇/編譯    2009/02/12

紅帽 (Red Hat) 宣佈推出 Enterprise MRG 即時 Linux 的第二次釋出版本，這也是 MRG 首次的全功能版本。而紅帽所贊助的 Fedora 專案，也推出 Fedora 11 開發釋出的 alpha 版本。

Enterprise MRG 是 Red Hat Enterprise Linux 堆疊的分支，和 Enterprise Linux 支援一般資料庫與應用軟體伺服功能不同，MRG 乃專為支援傳訊 (messaging)、即時與網格 (grid) 工作而設計調整。根據紅帽原定計畫，新版 Enterprise MRG 1.1 預計在 2008 底推出，如今已延遲數月。

Sun 釋出整合式開放源碼平台 GlassFish Portfolio
謝良奇/編譯    2009/02/17

Sun 宣佈釋出基於 GlassFish Java 應用軟體伺服器的開放源碼 Web 應用軟體平台 Glassfish Portfolio， 提供 JavaEE 應用軟體伺服器、AMP (Apache、MySQL 與 PHP) 堆疊以及入口軟體和管理工具。此一整合平台可作為企業更具吸引力的替代方案。如今開放源碼應用伴隨著整合不同元件的時間與成本，以及支援。根據 Sun 表示，較之私有方案，GlassFish 平台可節省 90% 的價格效能比。
