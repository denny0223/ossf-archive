課程簡介

Ruby on Rails 是一套高生產力的 Web framework，本課程將由淺入深講解 Rails 最具特色的 RESTful 架構、如何使用 RJS 做出 Ajax 效果、以及使用 Plugins 撰寫使用者認證、資料分頁等功能，協助您用 DRY(Don't repeat yourself) 的方式打造網站應用程式。

本此課程分一天舉行共三小時 ，詳細的內容時間描述與報名資訊如下：

  • 對象：大專院校資訊相關系所的學生，或是對自由軟體有興趣者。
    (本課程要求具備基本程式設計能力，如果從未接觸過 Ruby，建議可以造訪 Ruby Programming Language 網站學習基本語法。如果從未使用過 Ruby on Rails，建議可以一起報名參加另一堂 Ruby on Rails 導覽)
  • 人數：每班約 20 人 ( 實機操作 )。
  • 平台：目前平台為 Windows XP。
  • 費用：全程免費，但無供餐。
時間 內容
18:40-19:50 Rails RESTful
19:50-20:10 休息
20:10-21:30 RJS 及 Plugin 介紹與使用
