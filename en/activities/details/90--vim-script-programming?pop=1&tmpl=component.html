Topic Information 議程簡介

Vim 是一針對程式開發所撰寫的編輯器，其可程式化功能為 Vim 最大的特色之一，此一課程將介紹 Vim Script 程式設計以及 Vim Plugin 設計，學習量身訂做輔助開發用插件。

詳細的內容時間描述與報名資訊如下：

  • 對象：Vim 重度患者以及初學 Vim 的朋友們。
    （本課程要求具備基本程式設計能力。）
  • 人數：約 28 人。
  • 費用：全程免費。
  • 平台：提供 28 台電腦實機操作，使用 Windows 模擬 Ubuntu 9.10 系統。
時間議程
13:30-13:35 單位介紹
13:35-14:45 資料型別、內建變數、內建函式、使用者定義函式、變數及函數範圍、條件式
14:45-15:00 中場休息
15:00-16:10 編輯緩衝區操作、常用選項、撰寫自動補齊函式、撰寫命令補齊函式、撰寫延伸插件、延伸插件開發工具
16:10-16:30 Q & A
