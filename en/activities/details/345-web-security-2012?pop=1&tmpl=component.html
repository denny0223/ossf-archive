議程簡介

本課程將會利用淺顯易懂的教學，讓一般使用者與網站開發者了解 Web Security 的重要。透過對攻擊手法的了解以及對網站的實際模擬測試，學習修補其中的弱點，達到網站安全的基本需求。
  • 對象：
    • 大專院校資訊相關系所的學生，或是對資訊安全有興趣者。
    ( 本課程要求具備基本程式設計能力，例如 PHP、MySQL、JSP、ASP 等動態網頁開發經驗。)
  • 人數：28人。
  • 費用：免費參加。
  • 平台：Windows 作業系統。
