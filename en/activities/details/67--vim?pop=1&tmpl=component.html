課程簡介

這是個文字編輯 2.0 的年代。有效率的文字編輯以及各項作業的自動化，在這個資訊快速發展的年代相當的重要。時間寶貴，那麼就需更有效率的編輯文字，Vim 便是一個能夠為你量身訂做自動化需求的好工具之一。本次課程將簡單介紹基礎的 Vim 使用，進階指令集以及 Vim Script 的撰寫及部屬。

本課程共三小時，詳細的內容時間描述與報名資訊如下：

  • 對象：大專院校資訊相關系所的學生，或是對自由軟體有興趣者。
    ( 本課程要求具備基本程式設計能力，以及 Unix-Like System 基本操作能力)
  • 人數：305 教室 - 33座位 (實機操作)。
    ( 現場只提供 33 台電腦供參與伙伴實機操作，報名超過成為侯補名單伙伴亦歡迎自行攜帶筆電到場旁聽)
  • 平台：Windows 上使用 Sun VirtualBox 虛擬 Ubuntu 9.04 。( 依實際狀況再進行調整 )
  • 費用：全程免費。
時間內容
19:00-19:05 單位介紹
19:05-20:15 基本操作：編輯模式，介紹移動技巧，文字插入技巧，搜尋及取代，編碼，基本設置，快捷鍵基本設置，Buffer 基本操作，分割視窗基本操作
20:15-20:35 中堂休息
20:35-21:45 進階操作： fold 折疊 ，syntax 設置，autocmd 自動指令，completion 自動補完，ctags，變數定義，函數定義，條件式，例外 (exception)處理，built-in functions 內建函數，feature-list，gVim 的基本調效，plugin之撰寫，plugin ，ftplugin ，autoload ，doc ，misc ，vimdiff ，helpers ，VIM::Packager module ，Vimana module
21:45-22:00 Q & A
