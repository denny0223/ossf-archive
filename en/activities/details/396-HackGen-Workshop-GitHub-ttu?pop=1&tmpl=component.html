議程簡介

版本控制是現代攻城屍打怪（程式開發）不可或缺的一環，而眾多的武器（版本控制系統）中 Git 更是不二人選。因 Git 而生的 GitHub 帝國聚集了各路的英雄好漢在此組隊練功解任務，不知道怎麼起步的初心者們快來新手村吧！

課堂前請先註冊 GitHub 帳號成為 GitHub 帝國中的一員。若您使用自己的筆電請向 NPC 領取新手裝備完成 Git 相關環境安裝。
