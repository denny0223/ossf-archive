議程簡介

學會寫 PHP 很容易，但如何寫出好的 PHP 程式就是個很大的課題。現在有許多 PHP 開發者都是透過 PHP Framework 來學習 PHP，雖然能很快上手，但也因此忽略掉很多核心觀念。
另外也有很多開發者對 PHP 有所誤解，持續地以對舊版本的認知來評判 PHP ；事實上 PHP 現在已經有了長足的進步，如果能多去瞭解這些新的特色的話，相信一定能對 PHP 改觀。
因此這次的課程，將會帶領大家從更深一層的角度來認識 PHP。 主題包含有：

  1. PHP 5 新特色 (含 5.3 / 5.4)
  2. 除錯及錯誤管理
  3. 程式碼品質
  4. 物件導向與設計模式
  5. 套件管理
  6. 測試與重構
  7. 程式架構
  8. 程式安全
  9. 效能原則

  • 對象：有 2~3 年經驗的 PHP 開發者。
    （本次課程將不會有初級語法教學或系統安裝方面的內容，學員需要具備相關的經驗。）
  • 人數：50 人。
  • 費用：全程免費。
  • 平台：Fedora 14

講者簡介

朱建志

一個喜歡 PHP 及其他網站開發技術的程式工作者，並沈迷在這個世界裡，進而去探索這些技術有趣的地方。 分享技術的理念是：取之於網路，回饋於網路；並且讓大家能感受到自己學習到新東西的快樂。
  • 主要專長：Web Development
  • 部落格：網站製作學習誌
