議程簡介

版本控制系統是當代軟體開發所不可或缺的工具，而 Git 是其中最先進和熱門、且為開放原始碼的版本控制系統，Git 是由 Linux Torvalds 所發明，因其分散式、效能好、本地存取、無痛分支的特性，而普遍適合各種開發流程，受到多數人喜愛。
使用 Git 的專案包括：Linux Kernel, Apache, Debian, Drupal, Eclipse, Fedora, Gnome, KDE, Perl, PHP, PostgreSQL, Ruby on Rails, Node.js, JQuery, YUI... 等等。
諸如 Google, facebook, Microsoft, Twitter, Linkedin, NetFlix 等公司皆有使用 Git 作為版本控制系統。
本課程將介紹 Git 的原理、指令操作以及各種開發流程。
  • 對象：具備程式開發能力、對於版本控制系統有興趣者。
  • 人數：28 人。
  • 費用：免費參加。
  • 平台：Ubuntu。
