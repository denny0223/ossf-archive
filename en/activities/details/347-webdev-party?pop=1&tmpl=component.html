議程簡介

許多網站開發者常需要跟視覺設計人員合作，但是雙方常常會因為套版的問題而浪費許多時間在溝通上。究竟，視覺設計該怎麼切版，才能讓大家在合作上更為順利？
本課程將更深入地介紹 HTML 及 CSS 等技術的原理到實作，讓視覺設計人員可以瞭解它們並非只是為了網頁表象而存在。另外課程中也會介紹如何透過 jQuery 來操作網頁元素，讓網頁有更強的互動性。
  • 對象：常需要與網站工程師配合的視覺設計師，或對現代化網站前端技術有興趣者、需對於 HTML / CSS / JavaScript 有基本認知與實作能力。
  • 人數：28 人。
  • 費用：免費參加。
  • 平台：Windows XP。

    Day 1 (11/17)

  • OSSF 單位介紹 (5 分鐘)
  • HTML5 很有「意思」 - 語意化 HTML5 (50 分鐘)
  • 摩登 CSS 排版術 (上) - Position / Box Model / Float (50 分鐘)
  • 摩登 CSS 排版術 (下) - 多欄排版 (50 分鐘)
  • 課程練習 - 版型該怎麼切？(120 分鐘)

    • Day 2 (11/18)

  • 人生因為有 CSS3 而豐富 - CSS3 入門 (50 分鐘)
  • 讓網頁在手機上也好看 - Responsive Web Design 入門 (50 分鐘)
  • 媽呀！網頁活起來了！ - jQuery 入門 (50 分鐘)
  • 課程練習 - 讓網站更有互動性。 (120 分鐘)

講者簡介

朱建志

一個喜歡 PHP 及其他網站開發技術的程式工作者，並沈迷在這個世界裡，進而去探索這些技術有趣的地方。 分享技術的理念是：取之於網路，回饋於網路；並且讓大家能感受到自己學習到新東西的快樂。
  • 主要專長：Web Development
  • 部落格：網站製作學習誌
