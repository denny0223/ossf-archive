議程簡介

大家都用 Google Analytics；作為技術人，也都覺得自己會網站分析。我本來也是如此。但其實認真下去看，還有些東西或許忘了。猛然回頭，才發現以前自以為會的東西，其實連皮毛都不到。本次分享就是要來談一些其實基礎得很，但「以為自己會網站分析」的技術人經常會忘記的事情。

課程內容包含：

  • 網站的目標設定
  • 網站分析重要數據內涵與應用
  • 埋設追蹤碼
  • 目標與價值
  • 區隔，與進階區隔
  • GA 自定報表
  • 自動通知

目標對象：

    因為如果只是學而沒有用，那都是無謂的，所以對象必須是：
  • 擁有任一志工所經營之 FLOSS 網站的管理權限，可以實際下去修改程式碼，或者
  • 任一擁有上面那項條件的人，願意幫你背書，願意按照你上完課後所設計的東西下去改網站，或者
  • 你已經擁有任一志工所經營之 FLOSS 的網站 GA 管理權限。
  • 如果你有一台筆電，請記得可以帶著過來（場地有插座及無線網路可使用）

報名方式：

  • 本活動僅開放持 vip code 者報名。
  • 為確保本工作坊目標及成效，您必須先填寫「本 Google 表單」，講師將會判斷報名者是否符合目標對象的內容。
  • 經確認後，將會寄發報名本活動之 vip code，請您再持 vip code 至本報名頁面進行報名；本 vip code 限本人報名。
  • 若發放之 vip code 已達報名人數上限（30人），則將關閉本 Google 表單。

講者簡介

趙柏強

趙柏強，網路上常用的 ID 是 BobChao，一個住在台北的高雄人。曾經以為自己會當科學家、程式設計師、電腦老師、教材作家、網頁設計師，但結果全不是那麼回事。喜歡與社群經營、資訊科技、使用體驗與訊息溝通相關的各項議題。身為人稱「數位原生代」的一份子，從資訊管理學系畢業後，就從事各種與網路相關的工作。過去推廣公眾授權、做使用體驗設計、協助社群發展也辦行銷活動，看起來毫無關聯，於我還是一以貫之。
