於 Ubuntu 12.04 安裝 indicator-virtualbox
作者  凍仁翔

自從 Ubuntu 11.04 改用 Unity 以後，原先 Gnome 2 狀態通知區就有了大變動，從原先的 Applet 改為 Indicator 了！而 Indicator 在字面上可以翻成指示者、信號...... 但在這裡則是代表右上角狀態通知區的小工具 。

當中凍仁最喜愛的是 indicator-virtualbox，這對需要常開虛擬機器 (Virtual machine) 測試各個作業系統 (Operating system) 的凍仁而言是個好物，相信重度使用者應該都會喜愛它！

OIN 將專利保護傘擴展至 Android 核心元件！
作者  黃郁文／編譯

Open Invention Network (OIN) 日前擴張了該組織對於 Linux 作業系統的定義，增加了 18 項運作在 Linux 行動裝置上的重要套件到保護範圍裡，其中包括 Dalvik runtime，以及其他 Android 作業系統下的核心元件。這個新的保護範圍重新定義了 OIN 防禦性專利組合所要保護與授權的專利為何，以避免 Linux 作業系統及其相關應用程式受到專利蟑螂之攻擊。OIN 的 CEO - Keith Bergelt 表示：「智慧型手機及其他行動裝置，是近年讓市場玩家持續採用 Open Source 解決方案的催化劑，定期更新 Linux 系統的定義範圍，能讓 OIN 的 Linux 保護任務更為完善，我們很榮幸能將開發與改良 Android，與其他 Linux 行動裝置之貢獻者加入被授權人及保護範圍的行列。」

美國國會研究報告提出對抗專利蟑螂方案
作者  黃郁文／編譯

美國國會研究單位 (Congressional Research Service) 近來透過書面報告提供建言，指出透過某些舉措的施行，將能有助於釐清軟體專利的模糊界限，並進一步阻止專利蟑螂 (Patent Troll) 的橫行肆虐。根據許多客觀資料顯示，專利的公示功能在資訊科學領域，已接近徹底崩壞。因為，在這個資訊爆炸的時代，要求因專利侵權而可能被告者去檢索全體現存專利，以避免侵權，根本不可能、也不合理，所以現行的法律機制，正是讓專利蟑螂肆無忌憚勒贖專利侵權金的源頭！
