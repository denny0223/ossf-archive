美國上訴法院肯定自由／開放源碼的授權模式
黃雪雁    2008/11/20
今年 8 月 13 日美國聯邦巡迴上訴法院 (CAFC) 針對 Jacobsen v. Katzer 一案判決，原加州北區地方法院關於本案之判決撤銷，發回更審。因該上訴法院認定 Artistic 1.0版（簡稱 Artistic 1.0）授權中關於著作權的聲明及標示等約定乃該授權成立之前提要件（屬停止條件，註一），所以被告 Katzer 及其屬公司 Kamind 未遵守 Artistic 1.0 所要求的義務—隨產品有完整的著作權聲明及標示，則被告的行為不僅違反契約也構成著作權侵權。地方法院必須依據上訴法院就本案所認定之法律關係重新做出是否准許核發禁制令給原告 Jacobsen 之決議（註二）。

OSSF新聞
『與大師相遇－PHP教父談理想網路環境』與會心得
thkuo    2008/11/20

此次 Yahoo! 邀請 PHP 教父 Rasmus Lerdorf 來台，我們很高興參與了這次精彩的演講。會中 Rasmus 簡介了 Web 發展的歷史，與最早 CGI 模式下的限制與效能問題。接而傳達 PHP 設計理念－效能導向，並且讓我們看到了 1995 年的 PHP 語法歷來的演變，到今天已有相當成熟的
物件導向架構。

而 Rasmus 亦分析了各種狀況下 PHP 的效能，也闡述 Framework 所造成大量運算消耗的問題，這是效能與便利的兩難，Rasmus 的觀點則是贊同不要任何的 Framework 來達到最佳效能。當然他也說了用 C/C++ 撰寫會更快，不過這就不是 Scripting Language 所追求的目標了。

最後 Rasmus 帶來了現行熱門 XSS 攻擊的防治策略，分析各種未經過濾 (filter) 的變數內容會造成甚麼樣的問題，甚而洩漏寶貴的隱私資料。

此次演講的精神可歸納為三個重點：延展性(Scalability)、效能(Performance)與安全性(Security)，希望開發者們都能發展出三者兼顧的網路系統。

12/15 如何運用軟體協同開發網站進行軟體開發
李貞臻    2008/11/20

OSSF 將在 12/15 假中央研究院計算機中心電腦教室，舉行【如何運用軟體協同開發網站進行軟體開發】工作坊，課堂中將介紹軟體協同開發網站，說明其對軟體開發的協助、關係及合作，其中有什麼功能？有什麼好處？另外也將提到和軟體開發最密不可分的版本控制，讓學員了解版本控制的操作與運用，並可現場實際操作及體驗版本控制帶來的好處，提供軟體開發者在剛接觸此類網站的一個入門學習。

本次課程講師為王再國 (wangaguo)，是自由軟體鑄造場的成員之一，目前主要負責開發 OpenFoundry 專案開發平台。其專長在網站程式開發、ASP.NET、Ruby on Rails等領域。

本課程共三小時，詳細的內容時間描述與報名資訊如下：

  • 對象：任何軟體開發者，想學習軟體協同開發網站、軟體發展、與版本控制。
  • 人數：每班約 15 人 (實機操作)。
  • 平台：Windows。
  • 費用：全程免費，但無供餐。
  • 網站：https://whoswho.openfoundry.org/workshop/details/16-openfoundry.html

開放源碼走入儲存市場 Sun 公開開放儲存產品線 Sun Storage 7000
謝良奇/編譯    2008/11/11

Sun 日前公開了 Sun Storage 7000 Unified Storage System，打算擴大在資料儲存市場的版圖。該項產品也反映出 Sun 持續利用開放源碼挽救其業務的策略。

Sun 推出了架構於開放儲存此一概念之上的最新硬體產品線。開放儲存避免私有技術，強調標準元件與用戶可修改的開放源碼軟體。Sun 表示此一開放路線能打造比競爭對手更低成本且更具彈性的產品。

Canonical、ARM 合作 Ubuntu Desktop Linux 移植 ARMv7 架構
謝良奇/編譯    2008/11/18

Ubuntu Linux 商業贊助廠商 Canonical 宣佈 Ubuntu Desktop Linux 將移植到 ARMv7 架構上。鎖定 Netbook 的 Ubuntu ARM 散佈套件，可能將會讓 Intel 失去過去 30 年來 x86 一路甩開其他架構所依賴的軟體優勢。
