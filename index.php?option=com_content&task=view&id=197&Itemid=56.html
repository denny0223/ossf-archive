台灣也可以這樣做嗎？（下）

Created at Wednesday, 26 October 2005 08:00      Last Updated on Thursday, 21 July 2011 17:23 Written by 陳貴成
4.自由軟體真的是超級弱勢：
社會上的弱勢團體一堆，首要是處理民生問題的弱勢團體，包含健康醫療、老人、孩童、家暴、智能障礙、外籍配偶、救災……等都幾乎都處理民生問題，這種公益 團體大家多樂意慷慨援助，畢竟救人一命勝造七級浮屠阿。可是反觀自由軟體的領域，不用軟體又不會死人，真的盜版又不會出什麼事情，三五年一次的 MP3 事件又跟自由軟體的關聯性不大，在這樣的情形就可以知道要人家重視自由軟體的難度有多高啦。對了！！又因為自由軟體給人的刻版印象是免費的廉價的，所以政 治人物對這樣的議題興趣應該也是不大吧。所以可以將它稱之為超級弱勢。 5.自由軟體是否足以放眼大陸：
這是一個非常有趣的事情，台灣的市場規模在跟中國大陸一比簡直就是……。所以搞軟體這件事情，商人的想法是要搞英語系會在印度搞，要搞中語系，這……應該就會在大陸搞吧。

綜合上述幾點事情，可以知道的是在台灣搞自由軟體，搞得跟 Ubuntu 一樣幾乎是不太可能了，那還要不要搞？

若問我，我一定會說當然要。只是有幾件事情似乎是一定要克服的，當然啦這也是我的想法：

 

  1. 破除門戶之見：
    台灣其實有非常多的 Linux 使用者社群，或是愛好者社群，可是說實話似乎沒有一件事情是大家願意一起共同完成的，因為大家都深深的相信 opensource 的精神就是 just for fun，所以咧，似乎團結就是力量的老生常談就相對而言沒這樣的重要了，可惜的是台灣的自由軟體愛好（開發）社群朋友就相對少了，若是沒有一個可以團結的 共同目標，其實自由軟體在台灣的發展其實是會非常的困頓。
  2. 政府單位應該要更加專業一點：
    1. 承辦人員的專業需要討論一下……在下先說明，搞自由軟體絕對不是政府的全部責任，但是保護跟扶植一個必要存在的超級弱勢就絕對是政府的責任。說實 話政府在滿足產業的需求時都是依據民間的商業組織、學術機構來進行專案的規劃以及審理，好笑的是目前所有在公部門眼中夠份量的商業組織或是學術機構就算是 真的夠專業提供公部門全面性的推廣策略或是計劃，公部門的官員們也不見得真的有心來作這樣的一件事情，由於目前台灣的公職任用制度的問題，有心的人不一定 有力，有力的不一定有心，所以有很多的事情就在多一事不如少一事，少一事不如沒事的狀況下，日子一天天的就過去了，反正只要在年度結束前把經費用到一定的 標準就沒事啦。說實話，有多少的公部門資訊幕僚在推動自由軟體時，真的用心的去了解自由軟體。說實話！機率真的很低，不信各位讀者真的可以打電話去問問看 有多少公部門的相關承辦人員真的了解自由軟體就知道了。
    2. 統合的力量：社群有門戶之見，公部門其實也有許多的問題。一年有好幾億的自由軟體的產業發展預算在經濟部及其他的公部門如教育部……等，然後再分 包給許多不同的單位如資策會……等。當然在下要突顯的不是層層分包的問題（雖然這件事好像有很明顯的進步空間），而是分包出去後的成果是不是真可以作有效 的統合，這件事我們應該可以好好的思考一下。
    3. 公部門自己的目標：這是很簡單的問題，但是我想沒有一個公部門的負責人、或稱為老大可以告訴我們這句話「是的，我們目前不會去準備買下一版本的 windows」或是「我們預計在幾年內，完成單位內的文件格式的開放格式標準的移轉」，所以我們或許可以問問公部門，他們的目標是……？
  3. 結合社會運動的力量：
    1. 宗教團體：我想台灣的宗教團體應該都是用最高的道德標準作自我檢視的，所以他們應該是不會去浪費信眾的任何的一分錢，若是讓他們去了解自由軟體可 以帶給他們的好處，我想應該就是可以踏出成功的第一步。而且據我所知目前至少已經有一個以上的大型宗教團體在內部進行 MS office 跟 OpenOffice.org 的移轉策略。
    2. 社區大學：這是台灣社會另外一股在野的力量，社區大學是最貼近社區族群的社會力量，只有社區的群眾可以接受的電腦環境，才有可能在台灣生存，若是可以讓愈來愈多的社區大學接受自由軟體，這樣自由軟體在台灣的機會才會更加有優勢。
  4. 向產業學習：
    1. 台北市電腦公會有一個非常出名的組織叫作 Linux 促進會，我想這是目前在業界對 Linux 產業發展最為了解的組織之一了，或許有些社群朋友將台北市電腦公會視為洪水猛獸，但是若是他可以提升自由軟體的產業價值，其實我想套用一句廣告辭「唯有團結才能找到生路」。
    2. 自由軟體是一個比較技術面的層次，若不能提供給各個產業應有的價值，那它永遠也只是一個技術層次的東西。所以若是可以與產業合作，那種價值的顯現將會是自由軟體壯大的另一步必要的棋。
我想只要上述的四件事情都有著落，台灣要搞一個像 Ubuntu 這樣的東西並不是什麼困難的事情，但是我們其實「不需要」搞一個台灣的 Ubuntu，我心中想的是 Ubuntu 的正體中文版會跟英文版一樣的好用，而且從宗教組織、社會團體跟公部門都可以放心地訂定移轉的計劃。當然會舉 Ubuntu 作例子不是因為我愛 Ubuntu，事實上我是 fedora 的使用者，只是因為目前依統計最受歡迎的 linux distribution 是 Ubuntu，資料來源（註一）。

發展一個跟英文 Linux 環境一樣友善的中文 Linux 環境、且在套件的維護與更新上也是跟全球同步的，如此的境界其實是需要長時間環境培養而成的，但也因為這樣，所以可以乎應出另外一個廣告辭「人類因有夢想 而偉大」若在加上另外一句話 impossible is nothing，我相信漸漸有一天他會一步步的實現，不一定是像 Ubuntu 基金會的模式，也不一定是像 RedHat、SuSE 的模式，但一個跟英文 Linux 環境一樣友善的中文 Linux 環境終會顯現。

 

***** 以上絕對只是個人意見，決不代表任何單位立場 *********
