中國移動加入 Linux 基金會

建立日期 2010-11-18 08:00      最近更新在 2010-11-23 01:11 作者是 謝良奇／編譯
旨在加速 Linux 成長的非營利組織 Linux 基金會 (Linux Foundation) 宣佈，中國移動通訊集團公司 (China Mobile Communications Corporation) 成為了該基金會的金卡會員 (Gold member)，同時也是第一個加入 Linux 基金會的中國企業。中國移動通訊集團公司是中國移動有限公司的最大持股者。中國移動有限公司以市值而論，是全球最大的電信營運商，也擁有全球電信業者中最大的客戶群與網路規模。中國移動近來透過其 OPhone 行動作業系統，以及發展基於開放源碼的雲端運算系統，而在 Linux 上進行投資。加入 Linux 基金會顯示該公司對於 Linux 平台的進一步承諾。

Linux 基金會參與全球 Linux 推廣、訓練、標準化，並促進運算產業在技術和市場上的合作。該基金會會員分為白金、金卡、銀卡 3 種等級。目前近 100 名會員中，包括了 7 名白金會員，金卡部份包括中國移動在內，尚有 AMD、Cisco、HP、Google、Motorola、Nokia、Novell 等共 10 名會員。

Linux 基金會執行董事 Jim Zemlin 表示，中國移動加入 Linux 基金會的決定，以及對 Linux 的承諾，代表中國和電信通訊產業的各家作業系統版圖即將重新洗牌。對於 Linux 基金會而言，讓 Linux 成為 5 億 6000 萬名使用者的選擇，是個充滿可能性的絕佳機會。從增加使用者接觸 Linux 產品，特別是 Linux 行動作業系統的機會，到吸引更多開發者貢獻這套不斷成長的平台，中國移動加入 Linux 基金會的動作，可說是該基金會的一大勝利。

中國移動研究院 (China Mobile Research Institute) 總經理 Bill Huang 指出，中國移動認定 Linux 基金會為全球最重要的非營利 Linux 組織，也是 Linux 技術、標準化、創新的焦點。他們從 Linux 系統的開放架構中看見極大的價值，並希望在全球規模上對 Linux 做出貢獻。近年來中國在 Linux 上扮演了日漸吃重的角色。中國開發者不斷對 Linux 核心開發做出貢獻，Linux 也在中國商業與教育領域受到廣泛採納。根據 Springboard Research 與 Spiceworks 共同發佈的研究報告指出，亞太地區中小企業的 Linux 伺服器使用率已經超過 25%，高於世界平均水準。許多大型中國網際網路公司，例如 Sina、Shanda、Baidu、Taobao、Netease 等，因為開放架構、穩定性、效能而選擇使用 Linux。更不用說某些硬體製造商如 Huawei 和 Lenovo，也推出以 Linux 為基礎的路由器、伺服器、筆電、手機和其他產品。
