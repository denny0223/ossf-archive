Linux 基金會將舉行全新的 Linux 會議 LinuxCon

建立日期 2008-10-02 08:00      作者是 謝良奇/編譯
以加速 Linux 成長為目標的非營利組織 Linux 基金會 (The Linux Foundation)，宣佈將舉行一個新的年度技術會議 LinuxCon，會議的召開將依循開放源碼發展的精神，也就是以社群之力，行社群之事。此外，該基金會也同時宣佈其他重要 Linux 活動，例如年度協同合作大會 (Collaboration Summit ) 與核心大會 (Kernel Summit) 的相關細節。

Linux 基金會在舉辦過數次邀請制的活動後，將舉辦首次的 Linux 社群會議。Linux 開發者網絡社群經理 Brian Proffitt 說，他們口中的社群，代表了每一個人，如一般使用者、開發者與廠商。不管你在個人或專業領域上投注於 Linux 中，都歡迎參加。

LinuxCon 2009 預計將於北美的開放源碼聖地，俄勒岡州 (Oregon) 的波特蘭 (Portland) 舉行，時間訂於 2009 年的 9 月。LinuxCon 2009 將與一年一度的 LPC 大會 (Linux Plumbers Conference) 共同舉辦，該會議吸引了許多知名的技術天才參加。日前於 9 月舉行的第一屆 LPC，聚集來自開發社群的 300 位領導人，共聚一堂進行面對面討論並解決技術問題。
 
LinuxCon 將包括以論文為基礎的技術會議議程、教學與主題演講、技術展示與針對行動、桌面與嵌入式等多項主題的小型會議。

Linux 基金會執行董事 Jim Zemlin 表示，LinuxCon 透過以論文為主的大會議程、熱烈討論與實作的技術議程，將成為 Linux 社群最佳與最出色人才共聚分享知識的地方。同時LinuxCon 並非一般由某商業機構控制的營利會議，而是由社群為社群所舉辦的。LPC 發起人同時也是核心開發者的 Kristen Accardi 就認為，LinuxCon 將是所有關心 Linux 的人，彼此學習、連接與合作的大好機會。

openSUSE 社群經理 Joe 'Zonker' Brockmeier 表示，由於目前並沒有一個單一論壇，讓 Linux 貢獻者與使用者能夠從各方面的實際議題進行合作。LinuxCon 將可因應社群需要。Linux 基金會處於一個獨特地位，能結合正確對象與各種因素，進行實地合作與構築生產力網路。而他不會錯過這樣的一個盛會。

Linux 基金會宣佈 LinuxCon 當天，也公佈了包括 Linux 基金會協同合作大會 (Linux Foundation Collaboration Summit) 在內等年度盛事的時間與地點詳情。協同合作大會此次已是第三屆，將於 2009 年 4 月 8 到 10 日在舊金山舉辦。該地也是 Linux 基金會總部所在。屆時，也會同時舉辦嵌入式 Linux 大會 (CELF Embedded Linux Conference) 與 Linux 儲存與檔案系統研討會 (Linux Storage and Filesystem Workshop)。

另一個由 Linux 基金會公佈舉辦時間與地點的重要會議是核心大會。該會將於 2009 年 10 月 18 到 20 日，於日本東京與新的年度 Linux 基金會東京論壇 (Linux Foundation Tokyo Symposium) 共同舉行。該會除了基於現有的東京論壇，將擴展成大規模的技術會議，邀集全日本與亞洲有志於提高本國 Linux 教育與發展的開發者。

對於 LinuxCon，社群的意義相當廣泛，包括了所有和 Linux 有關連的人。在此 Linux 開發者可與同儕相互學習與指導，Linux 用戶免於置身展覽會場，即可了解到 Linux 平台的最新技術發展。Linux 基金會為了支援未來數量漸長的各種活動，已聘請 Ottawa Linux 論壇 ( Linux Symposium) 的共同創辦人 Craig Ross， Ross 將負責組織 LinuxCon，他也將加入基金會擔任社群關係經理。
