德國保險公司轉換系統為 Ubuntu Canonical 宣佈加入車載資訊娛樂聯盟 GENIVI

建立日期 2011-06-02 00:00      最近更新在 2011-06-13 11:26 作者是 謝良奇／編譯

Ubuntu 背後的商業 Linux 散佈商 Canonical 和德國保險公司 LVM Versicherungen 日前達成一項交易，將在該公司 1 萬部電腦上安裝 Ubuntu。在該交易下，Canonical 將在 LVM 位於德國敏斯特總部的 3000 部桌上電腦與筆電中安裝 Ubuntu Linux 作業系統。此外，LVM 位於德國其他據點共 7000 部電腦設備，也將由 Canonical 部署該開放源碼作業系統。

LVM 主要採用的核心軟體是一套基於 Java 的索賠處理應用程式 LAS。在最小的努力下，LVM 可在 Ubuntu 上使用該應用軟體，並確保成功轉換。官方並未提到被取代的作業系統，外界估計應該是較老舊的 Windows 版本。根據 Canonical 的說法，Lotus Notes、Open Office.org、Adobe Reader 也是 LVM 使用的標準應用程式，同樣也獲得 Ubuntu 完全的支援。

Canonical 業務開發副總裁 Steve George 表示，許多公司正逐漸理解到，在耗費數百萬元，永無止盡的授權費用循環之外，還有其他選擇。他們相信 LVM 透過與 Canonical 合作而轉換至 Ubuntu 所花的投資，將可收到數倍於此的成效。

事實上此一系統轉換並非完全在意料之外，因為 LVM 使用 Ubuntu 已經有一段時間。但將該公司所有系統全數轉換為該作業系統，其轉變仍相當可觀。LVM 對於作業系統的要求頗為嚴苛。因為該公司勞動力是由自僱和行動銷售代表所組成，他們會在街頭和客廳銷售保險。銷售團隊是透過永遠在線 (always-on) 的方式使用 LAS 系統。Canonical 表示 Ubuntu 支援了此一行動使用模式。

Canonical 近日宣佈加入 GENIVI 聯盟，這是個以建立並推動開放源碼車載資訊娛樂 (In-Vehicle Infotainment，IVI) 參考平台的非營利產業聯盟。Canonical 並且將開發符合 GENIVI 的 Ubuntu IVI Remix。

基於適用於下一代網際網路嵌入式設備的 Ubuntu 技術子集 Ubuntu Core，Ubuntu IVI Remix 是 Ubuntu 作業系統的特殊版本，將協助 IVI 製造商為客戶建造特定設備。Ubuntu 將為嵌入式市場帶來更豐富連網體驗的能力。

Canonical OEM 服務副總裁 Chris Kenyon 指出，汽車零部件供應商這 18 個月以來不斷地向 Canonical 在 IVI 領域提出需求。這些供應商已經在開發系統中採用 Ubuntu，並希望能在交付的產品中運用同樣技術。此外，公司希望他們的產品更快上市，並且尋求基本上跨架構的已驗證平台。Ubuntu 的 ARM 支援，加上 Canonical 和 Linaro 還有 Qt 的合作，使得 Ubuntu IVI Remix 在作為 GENIVI 成員現成解決方案上，已站穩有利位置。

Kenyon 表示 IVI Remix 的概念驗證約在 7 或 8 月釋出，並以 Ubuntu 11.04 「Natty Narwhal」 為基礎。最終與 Ubuntu 11.10 一同在 10 月釋出，可讓汽車零部件供應商在 2012 年上半年提供基於 Ubuntu 的 IVI 系統給汽車製造商。

他也提到 IVI Remix 會是新 Ubuntu Core 合作關係的第一個產品。儘管 Ubuntu Core 目前仍處於概念階段居多，Kenyon 認為未來會有更多新的 Ubuntu Core 夥伴，Ubuntu IVI Remix 將可為這些夥伴提供參考。

GENIVI 是由 ARM、Intel、BMW、Jaguar、Delphi、Magneti Maelli、Citroen、Visteon、XSe 在 2009 年成立，在 IT 與汽車產業擁有許多會員。該聯盟之前選擇 MeeGo 作為參考平台基礎，並且於 2010 年 8 月釋出了 MeeGo for IVI 版本 1.0。
