2007 Taiwan Linux Forum 8/1, 8/2 世貿一館盛大展開，歡迎蒞臨！

Created at Monday, 30 July 2007 08:02      Last Updated on Monday, 14 June 2010 18:48 Written by 台北市電腦公會/提供

● 日 期：2007年8月1日～8月2日
● 地 點：台北世貿展覽中心 二樓 第二、三、四會議室
● 主辦單位：經濟部工業局
● 執行單位：台北市電腦公會、財團法人資訊工業策進會、財團法人工業技術研究院
● 協辦單位：台灣Linux促進會、中華民國開放系統協會
● 報名網址：https://www.tca.org.tw/seminar/D10f00197.php

開放原始碼 (Open Source) 的應用與發展在學術界已行之多年，近年來隨著網際網路及數位家電的蓬勃發展才漸受重視，Linux 的商業化應用日益普及，彰顯出開放原始碼不僅是資訊技術研發的重要資源，同時 Linux 也是資訊產品的作業平台及發展 3C 產品的重要工具。

為了凝聚政府、學界及工業界投入發展 Linux 的力量，五年前開始的 Taiwan Linux Forum，邀請學術界來發動 Linux 校園研發創新論壇，政府行政機構則籌設 Linux 政府應用論壇，工業界則設立 Embedded Linux 論壇，企業界則是 Linux 企業應用論壇，經過幾年來的努力，現在 Taiwan Linux Forum 已經是台灣在自由軟體領域中，一年一度的盛大活動。

為了協助業者進行業務交易，在舉辦本項活動的同時，也會舉行 Linux Expo 的展示活動，及 Linux 黃金企鵝獎的頒獎典禮，期能吸引更多的人才與企業投入 Linux 的發展，往年也在會議中邀請國外知名學者或企業代表在論壇發表演說，以提供國際 Linux 發展經驗及最新的趨勢。今年 KeyNote speech 更將邀請到 Mr. Jim Zemlin (Executive Director, Linux Foundation) 及幾位重要成員擔任本次會議重要佳賓，將帶來有關 Linux Standardization、 Linux Kernel 等重要議題。

而今年 Taiwan Linux Forum 的子論壇包括有：社群論壇、政府應用論壇、校園研發創新論壇、驅動程式論壇、企業應用論壇等，另外，還有黃金企鵝獎優秀得獎作品發表；在8月1日上午開幕儀式中，將舉辦自由軟體週邊驅動程式網站啟動儀式，該網站除匯集眾多的 Linux/OSS 驅動程式之外，更將持續蒐集相關技術資源，提供開發驅動程式時所需資訊，預期將成為台灣自由軟體廠商及使用者重要的資源庫。

為配合今年 Linux Forum 的舉辦，Linux Expo 將在 8/2~8/6 在台北市貿展一館展出，展出內容包括 Linux 相關軟硬體產品或技術、自由軟體計畫示範成果、2007 黃金企鵝獎得獎作品等。

2007 Taiwan Linux Forum 不僅將是台灣自由軟體的年度盛事，希望藉由您的參與，成為匯集亞太地區 Linux 力量的重要活動。

(本文由台北市電腦公會提供)
