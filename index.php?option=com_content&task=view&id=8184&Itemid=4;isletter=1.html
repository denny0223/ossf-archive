防止小筆電採用 Android 微軟將向 Acer、Asustek 收取權利金

建立日期 2010-10-26 08:00      最近更新在 2010-11-09 00:27 作者是 謝良奇／編譯
微軟計畫針對該公司在電子郵件、多媒體和其他功能上的專利，向台灣的 Android 手機廠商收取權利金。Acer 和 Asustek Computer 將會是這波行動中鎖定的目標，而外界認為這項動作實際上是為了防止這兩家公司在小筆電和平板電腦上，採用 Google 的 Android 和 Chrome OS。

在數家台灣手機廠商中，僅有 HTC 獲得微軟專利的授權使用，使得 Acer 和 Asustek 成為這次收取權利金的目標。外界指出，由於 Acer 和 Asustek 的手機出貨規模並不大，因此微軟收取權利金並非為了獲利。消息來源表示，因為 Acer 和 Asustek 都是小筆電的國際大廠，微軟收取權利金的真正用意，是希望這兩家公司在其小筆電和平板電腦上，不要採用 Google Android 或 Chrome OS 來取代微軟的 Windows Mobile。Asustek 在發給臺灣證券交易所的聲明中，否認了此一新聞報導，並表示該公司尊重智慧財產權。Acer 則拒絕做出評論。稍早微軟才向美國聯邦法院提出訴訟，同時向國際貿易委員會 (International Trade Commission) 提出抗議，認為 Motorola 與其 Android 手機侵害了微軟 9 項專利。

Android 近來也飽受其他公司的攻擊。Apple 控告智慧型手機製造商 HTC，宣稱 HTC 的 Android 手機侵害部份 iPhone 專利。Oracle 也針對 Android 控告 Google，表示該智慧型手機軟體使用了來自 Java 的部份技術。回頭檢視微軟針對 Android 的策略，或許要更歸功於美國首席大法官 (Chief Justice) John Roberts，而非微軟的軟體架構長 Ray Ozzie。由於在 Bilski 一案中拒絕裁定軟體或商業方法專利是否合於憲法，Roberts 所帶領的最高法院等於是針對企業的創新活動課了一筆重稅。和藥物與醫療設備不同，軟體專利的問題，在於這些專利並非針對你採用的方法，而是你的點子。於是乎，微軟宣稱握有在網路和行動裝置之間進行電子郵件同步的專利，你就無法圍繞此一專利進行創新。這一點使得軟體專利格外具有殺傷力。創新所必須付出的代價越來越高，因為創新不免要以過去的成果為基礎。如果你沒辦法繞過某個想法進行創新，那麼你就得付錢再付錢。微軟正在向 Android 手機製造商傳達一個訊息，他們仍必須支付使用其專利的費用，而這麼一來採用 Windows 所支付的智財權費用或許比 Linux 還要少。此一情況下，Windows 不必比競爭對手更好，只需要還可以接受就好了。於是專利權像 1990 年早期的隨機版軟體一樣，成了該公司免於競爭的工具。

Google 將手機軟體 Android 發展為一開放平台，供手機製造商依照需求使用或加以修改。這套軟體也連接到許多 Google 網路服務，Google 則從該平台賺取廣告收益。Google 高層在某會議中曾提到，Android 如今每年營收可達 10 億美元之譜。然而，在其專利組合 (patent portfolio) 和競爭對手比較起來相對弱勢的情況下，外界等著看 Google 將如何做出回應。
