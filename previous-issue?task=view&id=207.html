松福禮的資服經驗分享－導入 Compiere ERP + CRM
松福禮股份有限公司

“Compiere ERP+CRM”是一套 Open Source 軟體，任何企業均可直 接將之自網站上下載，直接使用，它的設計也是符合了大多數企業的 工作流程，並且具有相當的彈性，甚至在上線後仍可持續更改。這對 中小企業來說，是一個很大的優勢。

Compiere ERP+CRM 具備了報價收款（報價自動轉訂單及收款條件）， 申購付款（採購及自動比價、採購建議、付款條件及分配），客戶關 係管理、供應連管理、績效分析（總帳、財務、管理會計分析）、網 上商店等功能，對於一般流通、加盟連鎖、批發及服務業而言，已足 以符合其所需的功能。同時，也提供了工作流程的設計，可針對企業 內部特殊的流程加以設計，或提供客戶更佳的作業流程規劃。

第二屆台灣自由軟體社群調查活動熱烈進行中
OSSF工作小組

活動網址：https://swan.iis.sinica.edu.tw/opensource2005

Firefox 分頁套件－Tab Mix 使用感想
lancetw 2007/01/23

◎ 我從使用 Firefox 以來，便有使用分頁套件的習慣。Firefox 基本 上以精簡為原則，額外的功能便交由不同的擴充套件來實現，每個人 都可以依喜好做出選擇。所以預設的 Firefox tab 是不能滿足我的。 剛開始我是使用 Tabbrowser Extensions（TBE）這個套件，但 TBE 的各式設定愈趨複雜，可以說是強大，卻也容易影響到 Firefox 的穩定。幸虧有 hemiola 做了 TBP Lite repackaged （也就是後來的 Tab mix，根據這個網頁記載，TBP Lite repackaged 在 2005 年 1 月 20 日正名為 Tab Mix。），整合了 Tabbrowser Preferences、TBP lite、Minit、Tab X 各家的優點，使用上比 TBE 穩定，便用到現在了。
