JBoss 第一季釋出多項工具 擴大中介軟體系統擔保

Created at Friday, 20 May 2005 08:00      Last Updated on Friday, 09 March 2007 01:29 Written by 謝良奇
開放源碼軟體供應商 JBoss 日前發表本年首季成果，包括多項工具 釋出，與擴展 JBoss 企業中介軟體系統（JBoss Enterprise Middle- ware System，JEMS）全產品擔保範圍（indemnification coverage） 在內的多項服務。

JBoss 近來擴大與 Novell、Unisys 及 Sun Microsystems 間的合作 關係。Novell 宣佈將擴大對於 JEMS 的支援及程式碼與資源貢獻， 並首先將採用 Web Services for Remote Portlets 標準的 portlet 容器（portlet container）與程式庫提送至 JEMS 關鍵元件 JBoss Portal 之下。Unisys 則將於全系列伺服器產品上支援 JBoss 應用 伺服器。在 Sun 方面，JBoss 與該公司簽訂協議將持續支援 J2EE 平台。 JBoss 執行長兼主席 Marc Fleury 表示，JBoss 會持續為解決用戶 與夥伴需要，提供具先進服務支援的技術，並推動可化簡 Java 開發的技術創新。為回應社群需求，JBoss 成立 JBoss Open Source Federation，將該公司所謂專業開放源碼（Professional Open Source）方法論推展至個人與公司，讓現有與新成立的開放源碼商品化並符合企業所需。 JBoss 資深開發者 Anil Saldhana 表示，專業開放源碼為交付低價但具高品質軟體及服務的方法論，要求開放源碼軟體應提供可與私有 軟體匹敵的專業服務及責任。JBoss 於 2003 年首度提供擔保範圍， 可視為此一方法論的具體實現。 同時該公司也加強其支援服務 JBoss 專業支援服務（JBoss Professional Support Services），包括成立 JBoss Network 作為 JBoss 支援訂購服務（JBoss Support Subscription Service） 的主要構件，為 JBoss 客戶支援的標準途徑。JBoss 也將全線 JEMS 產品的擔保範圍擴大，原僅提供同等於合約價格的擔保，如今延伸至最多四倍於支援合約價格。 另外，本年第一季中 JBoss 也舉辦使用者會議 JBoss World 2005、 推出 JBoss Open Source Federation 與 JBoss Network、釋出 Hibernate 3.0 版，此外針對 JEMS 產品釋出多項更新，包括物件關 連映對技術 Hibernate 3.0 版、EJB 3.0 Preview 5、JBoss Eclipse 1.4 Alpha，目標旨於簡化 Java 開發，降低 J2EE 平台使用門檻。
