廣東省政府採購擬定《Linux 軟硬體產品推薦目錄》
Linuxpilot 2007/06/10

廣東省政府於 2005 年舉辦「廣東省 Linux 軟體推進計劃」的《Linux 軟硬體產品推薦目錄》，共有 200 多個產品申請列入該目錄之內，其中約有 110 個產品能獲得批准。由於廣東省政府各單位均以此目錄為採購軟硬件優先考慮名單，令獲得批准的產品/公司生意大增。去年廣東省政府宣布此項目再次接受申請報名，不少香港公司更趕在期限前作最後申請。

IDC：開放源碼軟體 2011 年前年成長率達 26%
謝良奇 2007/06/10
IDC 新的研究報告指出，開放源碼軟體未來 5 年將有強勢成長，成長不僅限於 Linux 作業系統，更延伸至開發工具、應用軟體與企業系統等領域。開放源碼軟體成長同時，也將影響到私有軟體市場，其成長的主因，除了用戶逐漸熟悉開放源碼軟體之外，創投業者認識到收取後續支援服務費用帶來的收益的確是可行的商業模式，也逐步支持開放源碼軟體。

GPL 3.0 版最終草案釋出 Novell、微軟專利協議不受影響
謝良奇 2007/06/10
自由軟體基金會 (Free Software Foundation) 釋出了 GNU 通用公眾授權 (General Public License，GPL) 3.0 版的第 4 次草案。一反外界預料自由軟體基金會將會修訂 GPLv3，禁止 Novell 與微軟合作發展與散佈以新的 GPLv3 授權釋出的軟體，GPLv3 最新草案條款允許 Novell 在 SUSE Linux 企業版、OpenSUSE 與其它 Novell 產品中，繼續納入以 GPLv3 授權的軟體。

Palm 推出 Linux 小型筆電 Foleo
謝良奇 2007/06/10
