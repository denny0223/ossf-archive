李日貴（松凌科技技術總監）    2007/06/24

Java Opensources for Web Development Part I：
Chapter 5 檢測程式碼
Lession 19 : PMD

往往我們在撰寫程式碼的時候，會忽略一些小細節，可能是當初貪圖方便，未來將造成系統的負擔。例如，使用 System.print 方式來進行除錯的作業，雖然小小的一行輸出的動作，卻容易造成系統頻繁的 IO ，而且輸出一些沒有等級的 logging 資訊，更會造成除錯上的負擔。


PMD 就是專門協助工程師來判斷程式之中是否有一些小錯誤出現，提供了許多 IDE 工具的整合，在 ant、QALab、XRadar與 maven 等都有支援 PMD，當然你也可以利用命令列的模式來進行 PMD 的驗證。在 Eclipse 僅需要利用 HELP 之中的 Software Update，將下載網址指到 https://pmd.sourceforge.net/eclipse 就會自動進行安裝作業。

自由軟體的商業應用模式（下）－雙重授權篇
林誠夏    2007/06/24
除了前述的服務收費及硬體加值模式外，自由軟體商業應用模式中最足以開創新局當屬「雙重授權模式 (Dual Licensing Mode)」。所謂的雙重授權模式意指，軟體程式在散布時兼採自由軟體授權條款及傳統的商業授權條款併行釋出，使用者可視需求選擇無償釋出的自由軟體授權版本，或是付費使用商業授權版本。因自由軟體授權條款係屬非專屬授權，與類似權利賣斷的專屬授權（註二）模式不同，是以權利人可以同時對不同授權人約定不同的授權條款而並行不悖，而雙重授權模式在自由軟體商業模式發展中最引人注目的特點，在於其一方面採用了 GPL2 此種具感染性且約制權利義務關係詳密的授權條款，但另一方面卻透過商業授權模式讓商業付費使用者不受 GPL2 所拘束全身而退。

校園共筆應用推廣課程－維基百科 & Mediawiki
OSSACC    2007/06/24
◎ 軟體簡介
MediaWiki 是一套以 GPL 授權發行的 Wiki 引擎。它具有很豐富的功能，並被維基百科及其他危機媒體基金會的計畫所採用，亦為世界上很多不同的維基所採用。MediaWiki 採用 PHP 語言寫成，並以 MySQL 作為其資料庫。它的徽標圖像化了它如何運用標記語言即雙括號 ("[["，"]]") 來連結不同的文章。MediaWiki 具有很高的相容性和可塑性，現在的版本支持 memcached 及 Squid cache 系統，亦支持 TeX 數學公式。其擴展系統允許用戶自行添加他們需要的特性及客製化程式介面。

◎ 課程簡介
此次研習對象主要以學校老師的校園應用為主，從入門開始導引您深入探索 MediaWiki 之整體結構，並以實際範例帶您學習編輯內容、安裝軟體以及管理系統，讓您輕鬆的使用 MediaWiki 系統來進行寫作。

課程說明網址：https://news.ossacc.org/wiki/index.html

校園部落應用推廣課程－部落格 & LifeType
OSSACC    2007/06/24
◎ 軟體簡介
LifeType 是一個基於開放源碼授權的 Blog 平台，支援多 Blog 與多使用者。必須要在 PHP 與 MySQL 的平台才能安裝與執行。它的前身 pLog，因為和 Amazon 在美國的商標權 Plog 雷同，因此在 2005 年 11 月 1 號正式改名為 LifeType。

LifeType 擁有親切友善的介面、整合多媒體的檔案管理、多樣化的模板、眾多的外掛程式、並且支援多國語言、產出的網頁符合 XHTML 1.0 規範、極高效能的資料存取等強大功能，讓它成為相當受歡迎的部落格網路應用軟體。

◎ 課程簡介
此次研習對象主要以學校老師的校園應用為主，從零開始引導您深入探索部落格的彩色世界，並以實際應用帶領您學習部落格的內容編輯，進而延伸到 LifeType 的建置與管理，讓您輕鬆的擁有屬於自己的部落格。

課程說明網址：https://news.ossacc.org/LifeType/index.html
