Canonical 加入 Linux Foundation 成為正式會員

建立日期 2008-08-19 08:00      作者是 謝良奇/編譯

以加速 Linux 成長為目標的非營利組織，Linux 基金會 (Linux Foundation)，宣佈 Ubuntu Linux 散佈套件的商業贊助者 Canonical 成為該組織的新會員。

隨著 Ubuntu 逐漸成為伺服器與桌面領域作業系統的熱門選擇，Canonical 做為該散佈套件的贊助商，能見度也逐漸增加。但 Canonical 不僅是 Ubuntu 的商業贊助者，更支持了多個開放源碼專案，包括 Bazaar、Storm 與 Upstart。

Canonical 與 Linux 基金會高層認為，Canonical 加入該基金會後，可對該基金會做出貢獻，由於該基金會成員包括眾多 Linux 社群的領導廠商，例如 IBM、Novell、Oracle、Intel、紅帽 (Red Hat) 以及 VMware，Canonical 亦將因此而獲益。

Canonical 行銷主任 Andrew Rodaway 表示，Ubuntu 擁有廣大的使用者基礎與來自全球可觀的社群支援，他們認為這些用戶對於 Linux 以及開放源碼運動的熱情，將可協助 Linux 基金會加速 Linux 在各類型組織內的運用。

此外，Rodaway 認為，會員身分讓他們有機會在有影響力的關鍵組織內發聲，並且獲得在該基金會擴增中的組織會員之間的能見度。

Rodaway 指出，加入 Linux 基金會顯示出他們在 Linux 與開放源碼社群，特別是商業領域中的能見度提升，他們相信 Ubuntu 正逐漸在商業的伺服器部署中，獲得和桌面應用相同的影響力，擁有對此的宣傳管道，對於 Canonical 來說是一件好事。

Canonical 的 Ubuntu 專案技術長 Matt Zimmerman 表示，Linux 基金會在全球 Linux 使用與推廣上，扮演關鍵且非商業性的角色。他們重視 Linux 基金會的價值，如今更樂於成為其正式會員並支持基金會的活動。Zimmerman 表示，他們期望與該基金會合作，持續在各個計算領域推廣 Linux。

Linux 基金會行銷與開發者計畫副總裁 Amanda McPherson 表示，Canonical 加入 Linux 基金會此事具有相當重要性。Canonical 自該基金會成立之初，一直以來與該基金會關係密切，參與 LSB (Linux Standard Base)，並出席該基金會的協同合作大會 (Collaboration Summits)，而 Canonical 的創辦人兼執行長 Mark Shuttleworth 也是該基金會的個人會員。這次正式加入 Linux 基金會，基金會的行動將獲得他們的資源支持，同時也象徵他們重視基金會的工作成果。

Rodaway 指出，由於 Linux 基金會在企業市場的影響力逐漸增加，Canonical 也正在逐步擴大在同一領域的版圖，因此對 Canonical 來說，與其他主要廠商聯手支援 Linux 基金會是相當合理的動作。

儘管 Ubuntu 是相當熱門普遍的 Linux 散佈套件，Canonical 過去卻未曾直接參與 Linux 基金會。McPherson 指出，雖然 Canonical 過去幾年也參與過基金會的活動，如今 Canonical  的直接加入，意味著 Linux 基金會與 Canonical 將會有更為緊密的合作。

Ubuntu 社群成員過去一直活躍在 Linux 基金會各工作小組中，包括 LSB、Desktop Architects 與 Driver Backport 小組。藉由 Canonical 的支援，不論 Ubuntu 的商業或社群版本用戶的權益，將不會遭到漠視。

Linux 基金會執行董事 Jim Zemlin 在聲明中指出，Canonical 是該基金會重要的新成員，Matt 與其團隊創造了令人興奮的散佈套件，並迅速地擄獲全球用戶。他們這幾年來整合了跨產業與跨社群的協同合作。該基金會非常樂於與 Canonical 進一步合作，將 Linux 推向另一波成長高峰。
