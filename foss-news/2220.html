以 Ubuntu 9.10 為基礎 Linux Mint 8 釋出

建立日期 2009-12-01 08:00      最近更新在 2009-12-13 22:51 作者是 謝良奇/編譯
Linux Mint 開發團隊宣佈釋出代號 Helena 的 Linux Mint 第 8 版。Linux Mint 是一套以使用者親和力，以及透過支援  DVD 播放、Java、plug-ins 與眾多媒體 codecs，提供立即可用體驗為訴求的散佈套件。在 DistroWatch.com 網站上，Linux Mint 僅落後 Ubuntu、Fedora 與 openSUSE，位居最受歡迎散佈套件第 4 名。

Linux Mint 8 是以 Ubuntu 9.10 (Karmic Koala) 為基礎，主要特色為採用 Linux 核心 2.6.31 版、X.org 7.4 與 GNOME 桌面環境 2.28 版。最新釋出版本修正了眾多程式臭蟲，提供了 相對於 Ubuntu 更好的視覺效果，並且包含許多預先安裝套件。雖然對於有經驗的用戶，安裝 Ubuntu 然後透過 Synaptic 安裝這些軟體套件，也可以達到 Linux Mint 的效果。但這一點就是 Linux Mint 的主要優勢之一。使用者安裝 Linux Mint 後，就擁有包含大多數常用套件的系統。

除此之外，Linux Mint 8 包含了 OEM 安裝，供檔案上傳使用的新 Upload Manager，以及 Software Manager 上的改進，例如可選取多個應用軟體進行安裝或移除。Update Manager 允許使用者決定軟體套件是否要接受更新。

Linux Mint 因為包含許多非自由與開放源碼軟體，一直以來遭受到批評。這次 Linux Mint 第 8 版的釋出則以提供 Universal Edition 作為因應。Linux Mint 8 提供 Main 與 Universal 兩種版本。Main Edition 包含了所有 plug-ins，以及為了達成使用者親和力所需的私有軟體。Universal Edition 提供了與 Main Edition 相同的功能，但並不包含私有軟體、專利技術以及對限制格式的支援。 Universal Edition 的使用者透過 Sound & Video 選單中的額外選項，可以自行安裝缺少的 codecs。 Main 與 Universal 版本都提供 LiveDVD。

在標準桌面佈局上，Linux Mint 在螢幕下方只有 1 個 panel，而非 Ubuntu 的 2 個 panel。Linux Mint 主選單也只有 1 個，而非 Applications/Places/System 3 個選單。此外類似 openSuSE 所有的 Favorites 區域的加入，也增加了系統易用性。

詳細的釋出細節包含新功能列表，可以在 Linux Mint 8 的釋出說明中找到。另外也有 Linux Mint 8 的官方使用者指南可供參考。根據開發者的說法，Linux Mint 8 的 64 位元版本還沒準備好推出。
