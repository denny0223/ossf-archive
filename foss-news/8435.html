Microsoft 與中標軟件共同開發雲端解決方案

建立日期 2011-09-05 13:56      最近更新在 2011-09-06 17:21 作者是 謝良奇／編譯

Microsoft 宣佈將與中標軟件 (China Standard Software，CS2C) 合作，共同在中國開發行銷雲端運算解決方案。根據合作協議，Microsoft 將更新虛擬化堆疊，以求能夠支援中國國防與政府其他單位的客製化 Linux 散佈套件。Microsoft 的聲明指出，此一協議將提供以 Microsoft Hyper-V Open Cloud 架構為基礎的混合原始碼 (mixed source) 雲端解決方案，並且支援 CS2C NeoKylin Linux 伺服器的產品。


瞄準中國雲端市場，全球最大軟體製造商 Microsoft 日前與 CS2C 簽署開發與行銷協議。此舉將有助 Microsoft 對中國政府機構銷售產品。

廠商與分析師皆預測中國雲端運算市場將快速成長。一項來自 Gartner 研究公司的調查發現，55% 的中國受訪者願意花費 10% 以上的 IT 預算在雲端運算上。相較之下，歐洲的數據為 42% 而美國則為 49%。

Microsoft 這次和 CS2C 的合作焦點，是讓 NeoKylin 作業系統能作為第一級 (first-class) 客戶虛擬機 (guest)，運作於Microsoft的 Hyper-V 和 Windows Server 2008 R2 環境，且使用者可以更輕易得透過 Microsoft 全面性的 Systems Center 套件加以管理。

根據報導，NeoKylin 作業系統由 CS2C 與中國國防科學技術大學共同開發，是以 Red Hat Enterprise Linux 為藍本修改出來的 Linux 散佈套件。最初開發目的是希望運用於個人電腦上。CS2C 發言人曾表示，該散佈套件的發展是為加強中國本土作業系統環境鋪路。這間大學由中國國防部與教育部管理。

針對 Microsoft 將參與 NeoKylin Linux 在中國的部署，一些分析師認為，這意謂 Microsoft 承認了 Windows 無法打進許多中國企業的軟體部署，然而事實並非如此。Microsoft 了解到企業希望透過虛擬化技術，在伺服器層級運行 Linux 與 Windows，這正是 Microsoft 接下來所要推動的方向。

ReadWriteWeb 指出，據信 Internet Explorer 9 瀏覽器與 Bing 搜尋引擎在中國的市占率高於美國，而且 Microsoft 參與 NeoKylin Linux 可以讓中國的伺服器營運者更容易在執行 Windows 的資料中心部署 NeoKylin 的虛擬機器，因為許多中國的資料中心已經在使用 Windows Server，但是營運者也希望透過虛擬化技術執行 Linux。

IT 部門的管理者希望採用異質性與虛擬化環境，以利用多重作業系統的優勢，而不是受限於 Windows Server。許多人特別希望共同執行 Windows Server 與 Linux，這意味著 Linux 散佈套件、Linux 核心、Windows Server 要能夠同時順暢運行。之前曾有報導指出 Microsoft 是 Linux 3.0 核心的主要貢獻者之一，這些貢獻大部分是關於 Microsoft 的 Hyper-V 虛擬化驅動程式。

Microsoft 和 CS2C 將針對新技術與雲端運算解決方案共同贊助一間北京的技術實驗室，該實驗室將運行在 Hyper-V 與 Windows Server 2008 R2 上的 NeoKylin 虛擬化作業系統進行認證。
