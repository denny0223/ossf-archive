AVM 控訴 Cybits 遭德國法院駁回 Barnes & Noble 要求美國司法部調查 Microsoft

建立日期 2011-11-21 14:13      最近更新在 2011-11-24 15:02 作者是 謝良奇／編譯

德國地區法院日前揮去了對於自由軟體原則的一大挑戰，駁回了德國 DSL 路由器廠商 AVM 針對 Cybits AG 的著作權侵害控訴。而捲入 Microsoft 所提出的專利訴訟的書商 Barnes & Noble，則向美國司法部申訴，要求調查 Microsoft 的專利流氓 (patent trolling) 行為。

針對德國 DSL 路由器廠商 AVM 控告 Cybits AG 的案子，柏林地區法院日前駁回 AVM 所提出，Cybits 的 Surf-Sitter 軟體侵害其著作權與商標法的主張。法院裁決 AVM 必須允許其他公司提供的軟體，修改該公司 Fritz!Box 路由器中以 GPL 授權、基於 Linux 上的韌體。

AVM 之前嘗試要禁止 Cybits 修改該公司路由器中所使用的韌體，其中包含了基於 Linux 自由軟體的關鍵元件。這兩家公司都使用到以 GPL 授權的 Linux 核心。

Cybits AG 的 Surf-Sitter 軟體是一套家長控制軟體，允許家長設定上網時間與網路過濾功能。這套軟體可以被安裝到像 Fritz!Box 這類的路由器之中，此時，該軟體會跟路由器連線，下載、修改並重新上傳路由器的韌體到該裝置中。

AVM 宣稱修改該核心的行為已經侵害該公司的著作權，並於去年 1 月取得初步禁制令，禁止 Cybits AG 散佈修改該路由器韌體的軟體，並於同時針對 Cybits 提出告訴。

從日前的裁決看來，法院顯然站在 Cybits 這一方，法院表示預先安裝自由軟體的嵌入式設備，其用戶有合法自由以製作、安裝、執行，並且散佈對該自由軟體的修改。

在此案件中，gpl-violations.org 創辦人與 Linux 開發者 Harald Welte 的涉入成為格外令人關注的部份，因為握有 AVM 與 Cybits 都使用到的 iptables/netfilter 的智財權。他代表 Cybits 以第三方干預者 (intervener) 參與這次的訴訟。

Welte 認為 AVM 嘗試阻止第三方散佈軟體修改版本的行為，已經違反了 GPL。因為來自自由軟體基金會 (Free Software Foundation) 的 GPL 授權明確允許針對 GPL 授權軟體的修改， 並允許散佈修改後的形式。

在聽證會上，法院聚焦在路由器應該被視為由軟硬體構成的封閉系統，例如透過 SIM卡鎖，製造商可禁止修改行為的手機；或是擁有者可安裝額外軟體的電腦。在裁決中，法院拒絕了 AVM 禁止 Cybits 軟體銷售的要求，不管是 Cybits 或其他第三方都可以修改 Fritz!Box 韌體。

Cybits 執行長 Stefan Pattberg 表示，他們還沒看到判決，但已從律師聽到 AVM 多數主張被駁回的消息。對他們而言，最重要的莫過於該法院判決 Fritz!Box 的確是電腦，他們可以為其開發出用戶能夠安裝並使用的軟體。

Welte 表示，他很樂見法院拒絕 AVM 為了控制對 Fritz!Box 韌體 GPL 授權元件的修改，所提出的要求。這鼓勵所有人在現有軟體與產品上進行創新，是自由軟體運動的關鍵面向。

不過，法院方面也做出有利於 AVM 的裁定，禁止 Cybit 散佈會讓該路由器顯示錯誤或故障狀態資訊的軟體。AVM 曾宣稱，使用 Cybits 的軟體會讓用戶誤以為該路由器發生故障。對此裁決，AVM 發言人表示這正是他們的目的。

此外，由於拒絕支付授權費給 Microsoft，因而捲入專利訴訟中的書商與平板電腦製造商 Barnes & Noble，日前正式向美國司法部提出控訴，指控 Microsoft 正在從事反競爭行為。該公司提交給美國司法部一份長達 29 頁的投影片，在此投影片中該書商詳列了反對 Microsoft 向 Android 設備製造商收取專利授權費的意見。
