Firefox 3 正式版推出

Created at Tuesday, 24 June 2008 08:00      Written by 謝良奇/編譯
首次預先釋出版本後歷時 18 個月，Firefox 3 日前終於正式推出。在此同時，Mozilla 基金會 (Mozilla Foundation) 發起下載日活動 (Download Day)，希望締造 24 小時內下載的世界記錄。而基於 Firefox 所開發的社群網路瀏覽器 Flock，日前也隨著 Firefox 新版的釋出，推出的新的測試版本。 Firefox 3 的安裝程式已經可自 Mozilla 的 FTP 伺服器上下載，Firefox 3 是 2006 年以來，Mozilla 瀏覽器最重要的釋出版本，對 Windows、Mac OS X 與 Linux 作業系統的用戶，提供多達 46 種不同的地區化版本。不過 Firefox 3 不再支援 Windows 98、Me 與 NT，Mac OS X 10.3 以下的用戶。

該瀏覽器的 Gecko 呈現引擎 (rendering engine) 升級後，使得 Firefox 3 更為快速、記憶體使用更有效率，在網路標準相容性也有更好的表現，特別是 CSS 與 SVG 的處理。Mozilla 針對使用者介面與安全性方面，在新版瀏覽器加入許多全新功能。該基金會發表了一份 Firefox 3 對 Firefox 2 所做出的修改清單，根據發表 Firefox 3 釋出的新聞稿指出，Firefox 3 改善項目超過 15000 個。

Mac 用戶大概會是最先感受到 Firefox 3 不同之處的使用群。Firefox 3 的外觀如今與 Mac 作業系統有更高的整合度，採用了許多 Mac OS widgets。

新版瀏覽器的書簽管理功能有完全不同的面貌。位址欄的星狀圖示點選後，可將網址儲存為未分類書簽，留待稍後分類。書簽管理功能也能處理標籤 (tag) 了。編輯時，瀏覽器會提示已經指派的標籤，以減少重複的標籤。

Firefox 3 的網址自動完成功能，如今也能辨識網頁標題，以及書簽中網址的標籤，並且會隨著使用者的使用習慣調整排列順序。智慧型書簽夾 (intelligent bookmark folders) 可用來分類網頁，底層運用關聯式資料庫 SQLite 加以管理。該資料庫也讓網路應用軟體得以離線運作，等到網路連線重新建立時，再與線上服務進行同步。

新的下載管理器有更好的綜覽功能，可以重新完成先前中斷的下載動作。對於各種協定，例如 mailto:，Firefox 3 允許使用者指派網路服務而非本地應用軟體，來處理該協定。

Firefox 延伸模組功能儘管強大，使用者使用日久候經常要面對模組過多的管理問題，這次 Mozilla 也提供協助用戶的選擇器，而且 Firefox 還會自動地取消過時套件的功能。

在安全性方面，Firefox 3 不會顯示出已知的假網站。點選網址前的網站小圖示 (favicon) 就可以了解網站的安全相關資訊。瀏覽器會定期檢查 SSL 認證是否過期。此外，與防毒軟體以及 Vista 的兒童保護 (child protection) 功能的整合，該新版瀏覽器有更好的表現。

不過，Safari 瀏覽器所具備，能夠不讓瀏覽器在使用者電腦中留下任何資料的隱私上網模式 (private surfing mode)，未能及時完成加入新版釋出中。

Firefox 3 這次提供了完整縮放功能，在縮放時不會破壞頁面的排版。在文字呈現與圖形色彩管理上都有所改善。此外 Mozilla 還宣稱網頁下載與顯示速度都加快了。

據美聯社 (Associated Press) 報導，Flock 強調網路的社交層面功能，針對 Digg、Facebook 以及 Twitter 等網站提供了更好的整合能力。今年 3 月，Flock 宣佈已有近 3 百萬次下載，今年前 2 個月的主動用戶有 135% 的成長。

根據 Download.Com，Flock 瀏覽器納入 Firefox 3 技術的新測試版本，日前推出後下載次數正逐漸攀升。
