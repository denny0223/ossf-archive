行動版 Firefox 瀏覽器 Fennec 釋出有限功能 beta 版本

Created at Thursday, 26 March 2009 08:00      Written by 謝良奇/編譯
Mozilla 釋出了代號 Fennec 的新行動瀏覽器有限功能 beta 版本。此一初期測試僅局限於執行 OS2008 或稱之為 Maemo 軟體平台的 Nokia N810 網際網路 Tablet。不過 Mozilla 也準備了模擬軟體，供 Windows、Mac OS 與 Linux 平台用戶搶先在桌面上測試該應用程式。

Mozilla 行動架構師 Stuart Parmenter 在自己的部落格上寫到，對於 Fennec 在 Maemo 平台的首次 beta 釋出感到相當興奮。Parmenter 表示他們在前端程式碼做了相當大幅的最佳化，對於該平台也做了若干最佳化，加快了縮放 (zooming) 速度並在平移 (panning) 上更為流暢。藉由減少不必要的作業，他們也改善了啟動效能。

根據 Mozilla 的釋出文件，其他改進之處包括分頁瀏覽 (tabbed browsing)、整合式網路搜尋、存取多個搜尋引擎，以及書籤和標籤 (tags)。

此次測試主要目標是了解使用者是否能正確流暢地運用螢幕外 (off-screen) 的控制元件，以及相關的使用問題。沒有 Nokia N810 的使用者，也有基於 Javascript 而建構的 Fennec 版本，供 Windows、Mac 與 Linux 平台執行。

另一項 Fennec 引人關注之處，就是納入 Javascript 引擎 TraceMonkey 的支援。Parmenter 表示，納入 TraceMonkey 將使 Fennec 得以運用 Firefox 3.1 上所展現的 Javascript  速度提升。在 Fennec 中提供預覽的 TraceMonkey 引擎，將隨著 Firefox 3.5 正式推出。

正如桌面版本的 Firefox，Fennec 允許安裝附加元件 (add-ons)，藉此延伸該行動瀏覽器的功能。雖然目前僅提供少數附加元件支援，測試者已經可以運用附加元件存取 Twitter、修正 URLs 錯誤，以及加強安全性。

在選單、書籤、URLs 自動完成，以及若干從桌面瀏覽器 Firefox 延伸而來的功能上， 評論者也對於 Fennec 直觀的處理表示讚賞。Interpret 副總裁 Michael Gartenberg 就認為，Fennec 此一釋出相當出色。

但是 Gartenberg 也指出，即使這的確是一項里程碑，然而和所有瀏覽器一樣，Fennec 的競爭力來自在各個關鍵行動平台上的廣泛運用。由於 Apple 以 WebKit 為基礎的技術在行動業界相當普及，獲得重生的微軟 IE 也不容忽視，目前看來 Fennec 只能算是一次技術展示而已。

顯然的問題是 Fennec 是否能出現在其他行動裝置上。Mozilla 據報已經計畫開發其他行動作業系統的版本，其中包括 Windows Mobile、Symbian，甚至是 Android。

以上獨缺 Apple 的 iPhone。根據先前的報導，iPhone 應用軟體被加諸的限制將阻礙 iPhone 上的 Fennec 官方釋出。Mozilla 執行長 John Lilly 一年前曾經表示，Apple 的授權方案不允許 Firefox 在 iPhone 上執行。只是在 Apple 日前發表新的軟體開發套件後，此一情況是否改變則不得而知。
