Broadcom 等四家公司加入 Linux 基金會

Created at Tuesday, 11 January 2011 08:00      Last Updated on Tuesday, 25 January 2011 01:56 Written by 謝良奇/編譯
Linux 普及率成長的同時，Linux 基金會 (Linux Foundation) 也隨之持續地增添新的成員。挾帶著 2010 年 Linux 基金會迅速地擴展成員名單的氣勢，2011 年剛開始，該基金會已經增加了 Broadcom、Timesys、Protecode、GoAhead Software 共四家新的成員企業。

2010 年 Linux 基金會增加了至少 23 個新成員，相比之下，2009 年僅有 8 個組織公開宣佈加入該基金會。如今 Linux 基金會約有 78 家成員企業。

Linux 基金會行銷與開發者計畫副總裁 Amanda McPherson 表示，他們慶幸能處於穩定狀態，來自新產業的新舊公司都有意加入 Linux 基金會。自 2007 年開放源碼開發實驗室 (Open Source Development Labs) 和自由標準團體 (Free Standards Group) 合併成立 Linux 基金會後，該基金會在 3 年內即成長至此一規模。

Linux 基金會從一開始具備的其中一項能力，就是吸引原先不屬於開放源碼開發實驗室的新成員企業。日前宣佈加入該基金會的 Broadcom，就是一個例子。

Linux 桌面用戶多年來一直受到缺乏 Wi-Fi 晶片支援所困擾。儘管此一情況近年來有所改善，例如 Atheros 和 Intel 都為 Linux 提供 Wi-FI 驅動程式和程式碼，但是 Broadcom 卻一直沒有對 Linux 用戶提供多少支援。

該公司從 2007 年開始轉變並為 Linux 提供越來越多支援。2010 年 9 月，Broadcom 針對旗下若干 802.11n 無線晶片，釋出了完全開放且相容 Linux 的驅動程式 brcm80211。之後，該驅動程式已經被整合到最新的 Linux 核心 2.6.37 釋出中，由 Linux 社群加以改進。如今該公司透過加入 Linux 基金會，將其 Linux 支援帶往下一階段。

Broadcom 將會和 Linux 驅動程式專案 (Linux Driver Project) 以及 Linux 基金會產業 & 社群合作高峰會 (Linux Foundation Collaboration Summit) 合作，在此該公司將與社群開發者、其他產業成員和供應商一同運作。Broadcom WLAN 產品線資深副總裁兼總經理 Michael Hurlston 指出，Linux 無疑已是通訊設備與技術的主要平台。該公司 802.11 晶片驅動程式開放源碼的決定，是為了回應該公司 Linux 用戶的成長，預期將成為後續開放開發成功經驗的第一步。

McPherson 表示，Broadcom 了解幾乎今天所有主要技術公司都知道的一件事，合作、開放開發會帶來包括支援硬體到節省開發成本在內，所有可能的優勢。他們對 Broadcom 近來與 Linux 社群更為的緊密合作表示讚許，而該公司加入 Linux 基金會則展現其決心。

其他新加入 Linux 基金會的公司包括嵌入式 Linux 公司 Timesys、針對網路設備製造商的商業化成熟產品 (commercial off-the-shelf，COTS) 解決方案專家 GoAhead Software、智財權管理公司 Protecode。

成立於 2006 年，針對開放源碼軟體授權、著作權管理、軟體智財權管理等，提供產品與服務的 Protecode，將加入 Linux 基金會名為開放遵循計畫 (Open Compliance Program) 的泛產業方案。該計畫於去年 8 月啟動，透過工具使用、訓練、自我評估和授權資訊報告的標準格式，希望協助企業遵循開放源碼授權。

Protecode 產品管理主任 Kamal Hassin 指出，今日企業使用來自不同來源的程式碼來建構產品，以便創造上市時間與成本上的優勢，但同時也使授權問題變得複雜。Linux 基金會的開放遵循計畫提供關鍵且合作性的資源，以因應此一成熟的軟體開發環境。他們期待成為此一計畫的其中一份子。

GoAhead 銷售與行銷資深副總裁 Bill Yaman 則表示，Linux 已是電信市場主要作業系統，該公司用戶在其網路設備與系統中使用 Linux 的數量持續增加。該公司加入 Linux 基金會有助他們主動參與此一協同開發模型，同時推動該公司策略與 Linux。
