Nokia 推出新款 Linux 網際網路平板裝置

Created at Thursday, 25 October 2007 16:00      Written by 謝良奇/編譯
N810 迴異於 Nokia 核心的手機事業，是個以 Linux 為基礎的網際網路平板裝置（Internet tablet）。由 Nokia 發表，比前代 N800 更小、重量稍重而更快速的 N810，整合了全功能鍵盤與 GPS。這款新裝置反映出 Nokia 跳脫單純手機市場框架，企圖探索無線網際網路各式應用的策略。

和前代產品 N800 相比，N810 體積較小重量稍重，根據 Nokia 產品管理主任 Olavi Toivainen 的說法，將帶給使用者更好的質感。隱藏在底下的，是更新更快的處理器。

N810 採用和 N800 相同的德州儀器（Texas Instruments） OMAP2420 SoC。時脈由 320MHz 提升到 400MHz。上面搭載了 4.13 吋的 800x480 彩色觸控螢幕，周遭光感知器會調整螢幕亮度，改善在各類型環境下的螢幕可讀性。

除了螢幕上的軟體鍵盤，N810 更搭載了滑動式的 QWERTY 鍵盤。N810 配有 128MB DDR 記憶體與 256MB 快閃記憶體，有 2GB 內部記憶體作為儲存媒介，可擴充最高 8GB 的快閃記憶卡。售價為 479 美元，預計 11 月會在部份地區上市。

整合了 GPS 接收器使 N810 成為 Nokia 第一款內建此一功能的網際網路平板裝置。過去的平板裝置得額外安裝 GPS 附加套件。N810 預先安裝了地圖軟體，Toivainen 表示隨附地圖將依發售地區有所不同。

Nokia 也更新 N810 的 Skype 支援，Toivanien 認為 N810 是最好的 Skype 手機之一。雖然 Nokia 曾承諾明年起將推出 WiMax 產品，N 系列的 Linux 平板裝置也會在 WiMax 產品線之列，可惜 N810 未如市場預期，並不提供 WiMax 支援。

雖不具備一般手機通話能力，N810 卻可藉由 Skype 等 VoIP 服務撥打電話。N810 支援 802.11b/g WiFi，可連線網際網路，經由內建藍芽支援則可與手機連線。

Nokia 第一次在 N810 這類的 Linux 平板裝置上，使用了以 Mozilla 為基礎的完整瀏覽器。過去在 N800 上，使用者必須自行安裝類似的瀏覽器。Nokia 宣稱使用者可以在新裝置的瀏覽器上，享受更為複雜的 Web 2.0 社會網路網站，帶來有如桌面上 Firefox 的相似感受。

N810 使用了 Nokia 的 Maemo Linux 作業系統。Nokia 多媒體融合產品副總裁 Ari Virtanen 表示，N810 將傳統電腦的功能整合到口袋大小的尺寸，多重功能與服務因為運用的開放標準技術，得以加速整合到單一裝置中。

Nokia 並非熱衷於 Linux 的行動裝置市場的唯一廠商，包括 Mozilla、Arm 與 MontaVista Software 在內，多家公司近期共同投入 Linux 平台的研發，預料這類介於手機與筆記型電腦之間的裝置，開發門檻將因此而大大降低。Nokia 多媒體、企業部門在最新財報中成長各達 23%、105%，相對主流手機部門的 3% 淨利成長，加上這款納入先進功能的新裝置，反映出行動設備開發的產業趨勢。

基於以 Linux 為基礎所帶來的開放開發環境，Nokia 宣佈在 Nokia 論壇（Forum Nokia）開發者社群上提供支援服務，給該平台的開發者。
