Linus Torvalds 已經贏得與 Microsoft 之間的長久戰役？

Created at Monday, 12 August 2013 00:00      Last Updated on Tuesday, 13 August 2013 12:07 Written by 謝良奇／編譯

Microsoft 有兩大搖錢樹 Windows 和 Office。這兩套軟體讓 Microsoft 在計算領域占有壟斷地位，該領域中幾乎 80-90% 的人都會在桌面電腦上使用這兩套軟體。然而，Microsoft 的核心市場已經在動搖，現在越來越少人購買個人電腦，因為他們大多數的計算需求都可以透過平板和智慧型手機加以滿足。

事實上，一般人一開始並不需要全功能的個人電腦，大部份人們用不到現代個人電腦的 5% 計算能力。人們大多使用瀏覽器看網站、與家人朋友聯絡，或許還需要用文字編輯器編寫文件，以及播放音樂和電影的應用程式。僅此而已。在 Android 或 iOS 出現之前，他們唯一的選擇是購買昂貴且笨重的個人電腦或筆電。

情況不再是如此。

他們可以在平板與智慧型手機上，完成所有的一切，而且不像筆電或桌機，他們可以帶著這些極度可攜的裝置到處跑。

平板與智慧型手機的到來，使得桌機對於大多數用戶來說無關緊要。搭載 Android 的裝置打敗了 Windows 個人電腦的出貨量。Microsoft 和其核心商業伙伴如 Dell、HP、Lenovo、Intel、AMD，沒有預見這一切到來（Intel 其實有，但他們與 Microsoft 間的緊密關係蒙蔽了他們的眼光）。由於被 Microsoft 對行動未來模糊的視野所限，這些公司今天正在市場上掙扎，由於未能拋棄過去擁抱未來而處在滅絕的邊緣。和 Microsoft 的接觸就像瘟疫，它讓一度稱霸市場的 Nokia，變成了製造 Windows 手機的小芬蘭公司，就像某些不知名的中國 ODM 一樣。

Android 為市場帶來新的公司，讓 Samsung、HTC、LG、ASUS 成為了後個人電腦時代的市場領導者。

今天，Android 可說是現代（在道德與技術上都更好的） Windows，而且 Android 跑的是 Linux 核心。Linux 之父 Linus Torvalds 曾經說過，當 Microsoft 開始為 Linux 發展應用程式時，就代表我贏了。

Microsoft 日前釋出其搖錢樹之一 Microsoft Office 的 Android 版本。由於 Microsoft 不明白用戶所需，而且仍舊陷在廠商依賴性的思維中，這套應用程式只是 Office 365 的 Android 前端，而且僅供付費用戶使用。Android 上已經充斥著許多辦公室套件，如 Office Pro、QuickOffice、KingSoft，所以 Microsoft 想必得和這些既定對手進行競爭。

不過 Microsoft 的核心應用程式的確來到了 Linux。

所以，看來 Linus 已經勝出。

恭喜。
