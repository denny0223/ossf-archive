中國基礎軟體發展趨勢

Created at Saturday, 15 December 2007 08:00      Last Updated on Wednesday, 12 January 2011 19:32 Written by Linuxpilot
中國「2007 IT年會」在北京閉幕，北京中科紅旗軟件公司獲評為「2006-2007 中國 Linux 軟件市場年度成功企業」，有關人士認為來年中國國產基礎軟體將有很大發展趨勢。首先是中國國產 Linux 平台應用已日趨成熟，逐步向高端行業應用發展，來年這一發展速度將繼續加快。遠程教育、農村市場、醫療衛生是突破口，目前中國國產基礎軟體在這些領域已經開展了一些試點應用，在 2008 年中國大陸基礎軟體會得到更大範圍的應用，走向全面普及。基礎軟體同渠道的合作將更加緊密，在行業內的應用範圍和深度將加速，當中又以中間件發展最好，能在各行業同外國中間件廠商展開競爭。

◎ Linux 在手機市場穩居第二

今年中國智能手機銷售突破 1500 萬，有關人士預測明年還將有較大的增長。其中手機作業系統 Symbian 獨佔鰲頭，保持市場份額第一。今年微軟加大了其 Windows Mobile的推廣力度，吸引了更多的終端廠商加入微軟陣營。但同時由於諾基亞手機擁有絕對的優勢，導致微軟作業系統未能在智能手機市場，實現市場佔有率上的大幅增長。伴隨著摩托羅拉智能手機的銷量，Linux 依然保持作業系統份額第二的位置。

◎ 自主研製的 64 位元作業系統

較早前四川國芯科技公布了全系列四款電腦產品的預售價格，並宣布國芯 Future Alpha的 64 位元核心作業系統內核測試成功。國芯稱這是其自主設計的一款新作業系統，並非修改 Linux 核心而完成的，並聲稱要做中國未來的微軟。由於 2006 年發生過類似誇大宣傳，對此業界目前持保留態度。

◎ Linux 移動商務之旅推出

近日，中國短信推出了一款新型短信插件，該插件旨在幫助使用 Linux 作業系統的企業實現移動商務應用。中國短訊技術人員稱將在近日推出內測成功的中國第一款真正意義上的 Linux 插件。該插件適用紅旗、Red Hat、CentOS 4.4 等市場上主流的 Linux 作業系統。成功安裝後即可實現四網合一的短訊發送、短訊接收、查詢餘額等操作。

◎ 中標軟件力推藏文版 Office

日前上海中標軟件有限公司向北京西藏中學贈送了辦公軟體，在贈送儀式現場，中國信息產業部產品司陳英副司長親自體驗了中標普華 Linux 及藏文版 Office 軟體，表示希望中國大陸 Linux 廠商能夠研推出愈來愈多適合中國特色的軟體產品，促進資訊產業的蓬勃發展。中標軟件常務副總經理秦勇表示，力推藏文版 Office 是看到了這一市場的良好前景。
