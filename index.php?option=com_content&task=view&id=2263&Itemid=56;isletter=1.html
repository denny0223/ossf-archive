Oracle 結束購併 Sun 外界觀望後續效應

建立日期 2010-02-04 08:00      作者是 謝良奇/編譯
擔心 Oracle 壟斷資料庫市場，一度讓 MySQL 社群嘗試阻擋此購併案，歐盟執行委員會 (European Commission) 也曾在去年 11 月表示反對，不過 Oracle 還是在日前完成了對 Sun 的購併。換句話說，Oracle 如今擁有了 Java、OpenOffice、Solaris 以及 MySQL。這對開放源碼領域會發生什麼影響，各方都在猜測與觀察。

Oracle 則是在一份長達 5 小時的網路直播中承諾，將透過對 MySQL、OpenOffice 與  Solaris 等技術投注金錢與支援，持續 Sun Microsystems 對開放源碼的承諾，並且改善這些專案。對此，開放源碼使用者則一如預期地抱持懷疑態度。

開放源碼顧問公司 Freelocke Computing 的 John Locke，對於 Oracle 的 MySQL 長期計畫表示懷疑。Locke 認為，關鍵問題在於他們會如何運作 MySQL，是作為擁有龐大市場佔有率，且有利於 Oracle 的獨立資產，或者是加以邊緣化並逐漸銷毀掉的競爭對手？除了承諾，更重要的是行動。

由於 MySQL 的所有權是敏感議題，Oracle 小心地強調 MySQL 將獨立於 Oracle 的主要產品之外。Illuminata 分析師 Gordon Haff 表示，然而大多數的 Sun 產品與技術將會被納入其中。

Oracle 企業架構長 Edward Screven 試圖解釋 Oracle 的 MySQL 策略，他說，Oracle 的工程程序與專業知識將伴隨 Oracle 的支援，一起運用在 MySQL 上。他認為 MySQL 是 Oracle 在 2005 年購併的 InnoDB 的自然延伸。

MySQL 將納入 Oracle 堆疊，並與 Enterprise Manager、Secure Backup 以及 Audit Vault 整合。MySQL 將成為 Oracle 開放源碼全球事業單位 (Global Business Unit) 的一部份，但仍維持獨立銷售與開發組織。

Oracle 執行長 Larry Ellison 表示，Oracle 計畫維持 MySQL 存續與獨立，正如該公司其他資料庫產品一樣。Ellison 指出，MySQL 並非最大的資料庫，Berkeley DB 才是，該公司擁有並改善了這套資料庫。他們擁有市場各區段的不同資料庫，並且一直嘗試改進這些產品。

儘管如此，Locke 表示直到 Oracle 用實際行動證明，例如主動行銷並且銷售 MySQL 授權以增加其市佔率，否則他不會相信 Oracle 的承諾。他也希望 Oracle 能維持所有公開開發程序，例如公開存取的原始碼倉儲，並且主動與社群針對未來開發佈局進行對話。

Oracle-Sun 的購併案為每個開放源碼專家，上了幾堂重要的課。首先，開放源碼無法置身於市場之外，也無法免於市場的合縱連橫力量影響。

的確，開放源碼有替代選項，MySQL 已經有了分支專案，SpringSource、JBOSS 等公司都提供了 Java 支援，IBM 也散佈基於 Openoffice.org 的 Symphony 套件。然而這次的購併是一個轉捩點，個人或小公司藉由組織社群，就能戰勝產業巨人的想法，到此或許要告一段落。因為巨人一轉眼就可能將你買下。

Interarbor Solutions 首席分析師 Dana Gardner 認為，Java 與 MySQL 的用戶對於 Oracle 的意圖有同樣的疑慮，Oracle 擁有巨大的影響力，用戶們疑慮的是，該公司會運用這股影響力來培植或抑制這些產品。Haff 表示，Oracle 的開放源碼貢獻與私有延伸之間有著灰色地帶。例如，MySQL 已經有提供商業授權版本的模式，而該版本並不在 GPL 授權之內，問題就出在 Oracle 是否會逐漸著重在非 GPL 的這部份。

MySQL 在 Oracle 所有權下產生的不確定性，在此購併案結束前就已經開始對開放源碼市場產生影響。使用者表示他們會增加 PostgreSQL 的使用，同時漸漸放棄 MySQL。根據技術研究公司 The 451 Group 去年底針對 347 位開放源碼用戶所作的調查，2011 年 MySQL 在開放資料庫市場的市佔率將從目前的 82.1%，下降到 78.7%，在 2014 年更將跌至 72.3%。

在 Solaris 方面，Oracle 計畫要大力投資 Solaris 與 Linux，並針對 Oracle 完整堆疊為兩套作業系統進行最佳化。Oracle 也表示將持續開發、推廣並支援 OpenOffice 以及 OpenOffice.org 社群版本。該產品將成為獨立的全球事業單位，並運用 Sun 的開發與支援團隊。
