台灣也可以這樣做嗎？

Created at Wednesday, 26 October 2005 08:00      Last Updated on Thursday, 21 July 2011 17:23 Written by 陳貴成
前幾天去基隆的深美國小教課，主題呢當然是圍繞在自由軟體的推廣 上面，這其實一點也不希奇，因為現在身邊的朋友只要是有接觸電腦 的，而且不是在自由軟體社群的人，每個看到我的第一個反應就是 Linux、就是自由軟體。當然啦！我不是什麼大師，也不是什麼自由 軟體的超級駭客（雖然我把駭客當偶像 ^_^），只是覺得有些東西不 是一定要用 Microsoft 的方案才叫作正確方案、或是正確格式。 講到這個東西我又要念念我們公部門了，不知道還打算每年花幾億給 比爾蓋茲換他的法拉利輪胎，然後呢又因為我們所有的日常生活都跟 公部門有關係，所以公部門在不知覺中都成了 Microsoft 的最大業 務推廣單位，我真想建議微軟頒給中華民國政府個「惠我良多」匾額， 不知道 Microsoft 有沒有這個預算，中華民國政府敢不敢收咧……。

算了，不要炒冷飯了，這樣吵下去大概就只有一個結論，就是 kc 你 已經腦袋空空無話可說了，我本來是要用「無Ｘ可放」這四個字的， 只是想到這個一打出來對可能刊出本篇的國家最高知識殿堂之一而言 未免有辱斯文！所以就作罷囉，雖然這四個字用來精彩，總還是要顧 及主人家的面子問題吧！

在我跟深美國小聽我課的老師們互動時，我突然靈機一動。國外的有 錢人成立 ubuntu 基金會作了一個專門給教育用的 Linux Distribution 就叫作 ubuntu 或是更加教育取向的 edubuntu（註一），我們台灣 的有錢人這樣多，是不是也可以搞基金會來作一個符合國際規範又符 合台灣 K-12 教育需求的電腦系統呢？尤其是台灣是個資訊王國，很 多的有錢人都是搞電腦起家的，這樣子作應該更是合乎常理吧！

但是，事情沒有想像的這樣簡單，我想至少會有幾個可能的問題，不 然以台灣企業家關心社會不落人後，捐款避稅總衝第一的個性，應該 早就已經作了吧！我現在把我所猜到的問題列出來，看看大家認不認 同喔！

 

  1. 沒有人出來進行遊說及整合的工作：這件事情或許中華民國軟體自 由協會可以作，但說實話，有能力的人沒時間，有時間的人沒能力， 還有就是出嘴巴的人沒能力也沒時間……，我 kc 應該是算第三種啦！ 哈哈哈。
  2. 作慈濟有立即好效果也有長遠的好結果，但自由軟體只能放眼中長 期的未來：不可否認這是宗教的力量，他的好結果甚至可以到來世耶， 我想自由軟體在基礎環境的推廣要有立即的好效果難！只能放遠在未 來若干年後，也因為這樣的現象，很難讓有心人士去推估到底要投入 多少資源才可以有起碼的效果。
  3. 來自社會的壓力遠小於來自微軟的壓力：其實若要現在的台灣 IT 大廠來作這樣的一件事情，共同成立一個基金會來協助 K-12 的小朋 友用自由軟體或開放的標準資訊架構來學習對他們而言好像應該不難， 搞不好跟桌上拿柑一樣的簡單。只是若微軟因為這樣而施加了壓力給 這些 IT 的大廠例如降低授權的折扣，取消技術的移轉等等，那我想 這些大廠應該也不會跟自己的荷包過不去，並竟要作公益事業管道多 得是，何必選一個對自己公司弊多於利的呢？
  4. 自由軟體真的是超級弱勢：社會上的弱勢團體一堆，首要是處理民 生問題的弱勢團體，包含健康醫療、老人、孩童、家暴、智能障礙、 外籍配偶、救災等都幾乎都處理民生問題，這種公益團體大家多樂意 慷慨援助，畢竟救人一命勝造七級浮屠阿。
