十個必用的 Vim Plugin
Cornelius    2010/01/22

◎ The NERD Tree

操作 Vim 時，通常都在 Terminal 底下作用，無法像一般的 GUI
應用程式可以以樹狀目錄來瀏覽檔案。 The NERD Tree
是一將檔案目錄以文字模式方式繪製出來的 Plugin。讓使用者可以不必跳出 Vim
編輯器 而可進行檔案目錄的瀏覽、開啟等動作。新版的 NERD Tree
還另外支援了書籤 (Bookmark) 功能，使用者可將路徑節點存成書籤，以便下次載入時
直接回到該路徑上。

自由軟體授權條款的附加限制─從被 Google 拒絕的「善行條款」談起
林珈宏    2010/01/20
去年 (2009) 年底，企業精神標榜「 莫作惡 (Don't be evil) 」的 Google，對於在其開放源碼開發平台 Google Code 的 JSMin-PHP 專案，表示不再繼續提供專案託管的相關服務，因為 JSMin-PHP 的授權條款中包含了「本軟體必須用於善行，不得為惡 (The Software shall be used for Good, not Evil.)」的這段文字（以下簡稱為「善行條款」），不符合 Google Code 的專案授權政策。

國科會資訊安全技術研發專案計畫 — 自由軟體鑄造場作業暨智財權說明會
李貞臻    2010/01/22

國科會為增進國內資訊安全關鍵技術及創新應用的研發，促進產學研界在此領域的合作，培育產業所需的相關人才，特別於去年下半年成立了【國科會資訊安全技術研發專案計畫】。為了提昇研發品質，國科會希望此專案的執行過程，能使用中央研究院資訊科技創新研發中心 自由軟體鑄造場的專案開發平台來開發相關的專案計畫，並善用其專案管理工具來詳實記錄研發歷程。


有鑑於此，自由軟體鑄造場特別規劃兩場說明會，協助相關計畫的主持人、助教與學生瞭解專案進駐的操作方式、介紹版本控制系統、分享程式開發經驗、剖析自由軟體授權條款等，進而對自由軟體的本質與實際的運用有更深入的了解。其時間、議題如下：

● 99/01/13 (三) 13:30 - 15:30 交通大學資通安全中心 3樓電腦教室

講題：OpenFoundry 專案開發平台簡介（檔案格式： PDF/ODP )

講師：陳富川

● 99/01/22 (五) 09:00 - 12:00 台北科技大樓 2 樓 第 13 會議室

講題(a) OSSF 服務介紹   講師：陳飛亨 (檔案格式：PDF/ODP)
講題(b) OpenFoundry 工具介紹與版本控制系統操作  講師：陳富川 (檔案格式：PDF/ODP)
講題(c) 自由軟體授權分析  講師：林誠夏 (檔案格式：PDF/ODP)

Linux 基金會技術長任期屆滿 加入 Google
謝良奇/編譯    2010/01/21

知名 Linux 核心開發者 Ted Ts'o 日前宣佈，他在離開 Linux 基金會 (Linux Foundation) 的技術長職務後，將加入 Google。Ts'o 是一位檔案系統開發的專家，他在最新一代 Linux 檔案系統 Ext4 的發展上，扮演了主要角色。

在個人部落格上，Ts'o 表達了對於新工作的期待，文章中提到某位 Google 工程師在郵件論壇的發言，其中指出該搜尋龍頭正在更新其儲存架構，由 Ext2 更新至 Ext4。Ts'o 表示在加入 Google 後，將持續投入 Ext4 與其他 Linux 核心部份的開發。
