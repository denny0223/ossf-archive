搶進小筆電、手機等裝置 Intel 展示 MeeGo

建立日期 2010-04-20 08:00      最近更新在 2010-06-07 22:38 作者是 謝良奇/編譯

距離宣佈合併 Maemo 與 Moblin 兩套作業系統，為單一開放源碼作業系統 MeeGo 之後 2 個月，Intel 在北京舉行的年度 Intel 開發者論壇 (Intel Developer Forum) 上，展示在一系列裝置上運行的 MeeGo。

MeeGo 專案結合了 Intel、Nokia 兩家公司的行動 Linux 技術，目標是提供適用於不同行動外型尺寸和硬體架構的廠商中立平台。其關鍵參與者邀請了 Linux 基金會 (Linux Foundation) 管理該平台之協同合作體系。同時該計畫與上游核心社群與 Linux 軟體體系都有緊密的合作。

MeeGo 吸引了各方面的注意，並且正邁向行動與嵌入式 Linux 計算的產業標準平台，在 Linux 協同合作大會 (Linux Collaboration Summit) 上，其重要性十分顯著。

在 Intel 開發者論壇上，首先展示的是採用新 Atom Pinetrail 平台建構的 Acer 小筆電，可以看出其使用者介面和 Moblin 大致相同，包含一個位於螢幕上方的圖像式全系統選單。在使用者介面上方有許多標籤 (tabs) 可以在不同區域間切換，例如檢視連絡人的人群區域，以及播放音樂、電影的媒體區域。使用介面有一個應用軟體的啟動 dock，供用戶執行常用的程式，另外右方則顯示來自社群網路例如 Facebook、Twitter 的更新資訊，以及即時訊息等。關於使用者介面，在 YouTube 上可觀看 Intel 發佈的展示影片。

MeeGo 的一項主要改變是揚棄了 Firefox 作為預設網路瀏覽器，而改用 Google Chrome。開發者也增加了對多點觸控手勢的支援，使得 MeeGo 更適於應用在平板電腦。然而，MeeGo 不僅僅針對小筆電與平板電腦，還包括針對智慧型手機最佳化的版本。此一版本的使用者介面更為緊密以適應小尺寸顯示，但與小筆電使用者介面共享了許多關鍵元件，例如螢幕上方切換區域之用的標籤。

MeeGo 的智慧型手機版本顯然採用了 Fennec 網路瀏覽器 (Firefox Mobile) 而非 Google Chrome。這是個耐人尋味的選擇，因為 Google Android 智慧型手機的預設瀏覽器基本上是經修改過的 Google Chrome 版本。因此該瀏覽器很明顯地可供基於 Linux 核心的行動裝置使用。

Intel 展示了從智慧型手機到平面電視等裝置上執行的 MeeGo，並示範如何在裝置間進行影片等內容的同步，於是用戶在不同地點間移動或更換裝置時，仍可持續觀看影片。這只是 MeeGo 功能的一個範例。Intel 軟體與服務資深副總裁 Renee James 表示，MeeGo 目標是推動包括小筆電、3G smartbooks、平板電腦、媒體電話 (Media Phone)、行動娛樂系統等多樣化裝置。

MeeGo 不僅僅是一套作業系統，也是一套框架，可供其他作業系統開發者整合到其產品。換句話說，你可以執行 Ubuntu、Xandros、Linpus、Mandriva Linux，同時使用 MeeGo 的使用者介面和針對 Intel Atom 晶片的 MeeGo 加速功能。這也是為何許多 Linux 開發者與軟硬體廠商支持 MeeGo 專案的原因之一。

MeeGo 的最終 1.0 版本預計今年中釋出。Linux 基金會提供一份報告列舉支援 MeeGo 的所有公司，共有 28 家不同的企業，其中包含 BMW、EA Mobile、Novell、Asus 等。
