各路精英看好中國Linux發展

Created at Friday, 09 November 2007 16:00      Last Updated on Tuesday, 27 December 2011 14:17 Written by Linuxpilot

較早前中國 Linux 用戶大會在北京新大都飯店召開。大會透露的消息顯示，中國 Linux 保持年 80% 以上的增長率，中國紅旗本土公司現在佔市場佔有率第一位，Novell、Red Hat 分別佔第二和第三位，Turbolinux 佔第四位，其次是中標軟件等企業。種種跡象表明，2007 年 Linux 在桌面平台、手機嵌入、企業應用等諸多領域將會有長足的進步，中國國內不少機構在政府推動下，採用 Linux 作為解決方案的勢頭不會放緩，Linux 在金融、銀行等行業將繼續吞噬 UNIX 的市場份額。

 


◎ 中國Linux用戶大會觀點匯萃

開源界頗具影響力的陸首群教授發言中指出作業系統有三個衝擊波，其一是資料庫、伺服器，第二是桌面，第三是行動裝置。中國有很多優秀的程式工程師，像 C++ 等等，但中國的 PHP 應用方面較差，全世界倒數第四；另據 Nexen.net 統計，中國應用 Apache 情況較差，佔中國 Web 伺服器市場的 17.65%，居全球排行倒數第二，中國國內開源資料庫的應用剛剛起步。北京中科紅旗軟件技術有限公司副總裁鄭忠源在演講中表示「目前 Linux 的發展已經很成熟，但應用仍然是一個瓶頸，在我們碰到的很多項目當中，用戶原本用的是 Windows，實際應用時也應在 Windows 平台上跑得到，要取代 Windows 必須解決這些應用在 Linux 上兼容的問題。」

北京市平谷區信息中心電子政務辦公室主任張士良先生表示，希望由政府出面制訂桌面 Linux 和辦公套件統一的開發標準，組織桌面 Linux 和辦公套件廠商們共同攻克像電子表格等共有難題，開發桌面 Linux 下的應用程式，提高開源桌面 Linux 和辦公套件的競爭力。中標軟件公司技術總經理鄭嵩認為，中國 Linux 大概有將近 600 萬台的裝機量是個好消息，中國軟體體系建設為中國國產伺服器作業系統提供巨大的市場空間，關鍵是有否核心的自主的技術來佔領市場。Linux 遷移等技術，若缺少真正核心業務的應用，用戶對 Linux 的信心就會不足。賽迪顧問分析師胡珂女士透露，中國 Linux 在伺服器、嵌入式系統、Internet 和信息安全等領域取得了突破性進展，政府、金融、電信、教育是 Linux 的主要應用行業。華北、華東、華南是 Linux 的主要應用區域。據賽迪顧問預測，2006 至 2010 年，中國 Linux 市場銷售額年複合增長率為 28.1%。

◎ 銀行轉用 Linux

中國工商銀行宣布，從原來的 SCO UNIX 逐步轉向 Turbolinux 7 DataServer，此次轉移包括所有 20,000 家分支機構的前端伺服器，為期 3 年。四大銀行中的其他三家也都有同類計畫或者已經在小範圍試用。數月前，中國郵政業務發展平台在政府及公共事業 Linux 應用論壇上登場，中國郵政局、31 個省、自治區、直轄市的郵政局、236 個郵區中心局及各郵區中的地、市、縣局以及全部電子化區網中有超過 2 萬個 CPU 運行紅旗 Linux。之前中國建設銀行項目開標，Red Hat、中科紅旗、中標軟件等三家廠商入圍。

◎ 手機平台

手機平台在今年愈來愈多地看到了 Linux 的活躍身影。值得一提的是已在亞洲銷售了近千萬部 Linux 手機的摩托羅拉極為熱心，在年初更收購了 Linux 電視機頂盒廠商 Kreatel，顯然目的是將 Linux 手機跟家電系統連成一氣。至於 Linux 的桌面，聯想在其一款筆記本電腦上與 Novell 產品進行了認證，而且有數家廠商表示會根據客戶需求提供預裝 Linux 桌面系統的服務。微軟也高調贊助了北京八月召開的中國 LinuxWorld，但本次大會中 Red Hat、中科紅旗、中標軟件、Turbolinux 等大陸主要 Linux 廠商集體缺席。儘管如此，Novell SUSE LINUX 10 仍在會上發布，其採用了最新的 XGL 技術讓 Linux 桌面效果相當絢爛。

◎ 2008 年的 Linux 桌面

分析人士指出，目前中國國內許多項目的計畫在節省經費、提高效率及安全穩定的考量上，都會考慮應用 Linux。但有消息顯示，微軟以超低價位給 OEM 廠商 Windows XP 與 Windows Vista，2008 年 Linux 肯定會遭到重壓，桌面版 Linux 從某種意義上而言其象徵意義更大於實際意義。儘管如此，出於政治因素中國國內企業也紛紛預裝 Linux，四家公司分別是聯想、TCL、同方和方正。專家建言 2008 年 Linux 廠商們需要下大功夫，依靠 Linux 特性進駐企業級應用，努力作到提供優質而滿意的服務來鞏固市場、才可贏得更多客戶的認同。
