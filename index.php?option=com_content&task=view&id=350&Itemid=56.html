蘇格蘭開放源碼認知待提升

Created at Friday, 26 August 2005 08:00      Last Updated on Friday, 09 March 2007 01:05 Written by 謝良奇
蘇格蘭議員近日聲稱，儘管蘇格蘭對於開放源碼軟體支持日增，蘇格 蘭行政院（Scottish Executive）與議會對於開放源碼的認知仍舊有 限。中蘇格蘭（Central Scotland）警方日前即決定將以微軟 Windows 作為 IT 平台，替換過去採用的開放源碼解決方案。

格拉斯哥（Glasgow）的蘇格蘭議員 Harvie 日前宣稱，雖然蘇格蘭 公共領域對於開放源碼的支持日增，但蘇格蘭行政院與議會對於開放源碼的認知程度仍然不高。Harvie 表示，蘇格蘭行政院近期並不計畫在近期推展任何與開放源碼有關之政策。任何有關傾向開放源碼的動作，也都會引起微軟的強烈反對。 依據與微軟的合約，該警方服務 1100 名警員與支援人員的 IT 部門，將以 Windows Server 2003、Windows XP、Microsoft Office 取代 開放源碼系統，以支援其現代化計畫、彈性工作安排，並加強與其他公共領域部門的合作。 中蘇格蘭警方與微軟將於一系列資訊與通訊技術專案上緊密合作，其 中包括因應資訊自由法案（Freedom of Information Act）的電子文 件管理系統。 IT 部門主管 David Stirling 表示，該警方一份內部研究是最後選擇改用 Windows 的主因。該研究指出中蘇格蘭警方為了提高效率必須使用立即可用的套裝軟體以降低客製需求，並進行提高與其他組織資訊通訊系統間的相容性等多項改善。今年初另一份延伸研究進一步促成了中蘇格蘭警方與微軟簽訂為期三年的企業授權合約（Enterprise Agreement）。 Harvie 在 2003 年 9 月曾對蘇格蘭行政院與議會發放問卷，調查他們對於開放源碼的意見。很快地 Harvie 就收到來自微軟的晚餐邀約。在與微軟代表的接觸中，Harvie 發現微軟對於開放源碼的積極態度。 Harvie 表示，微軟對於其問卷的迅速反應，顯示了該公司如何從政治層面處理開放源碼軟體對於其商業模式的威脅。Harvie 也表示，未來他將積極尋找機會向蘇格蘭行政院引介開放源碼軟體。 微軟過去即曾多次迅速回應其他政府單位改用開放源碼的動作。當幕 尼黑傳出考慮改用 Linux，微軟執行長 Steve Ballmer 便曾一度中 止其休假親訪幕尼黑市長，儘管該次遊說最後並未奏效。
