消費性電子產品市場興起 嵌入式 Linux 系統受矚目

Created at Tuesday, 27 April 2004 08:00      Last Updated on Friday, 09 March 2007 00:04 Written by 謝良奇
嵌入式 Linux 正要在消費性電子產品市場中崛起！傳聞指出韓國消費電子產品大廠 Samsung，有意採用 Linux 作為產品主要的作業系統。根據 Korea Times 報導，Samsung 內部人員表示該公司將增加基於 Linux 的產品數量及開發人員，將其作為該公司的策略性作業系統。 2003 年 8 月，Gartner Inc. 報告指出嵌入式軟體工具市場已逼近10 億美元。此炙手可熱的市場大餅中，MontaVista Software 佔有頂尖嵌入式作業系統廠的地位。Venture Development Corporation認為以市佔率而言，該公司可說是全球第一的商業嵌入式 Linux 公司。

Siemens Venture Capital、Samsung Ventures America、Infineon Ventures，和臺灣的中華開發工業銀行，看準嵌入式 Linux 的未來可期，成為 MontaVista 最新的投資者。過去曾投資 MontaVista 的其它公司尚包括 IBM、Intel、Sony、Toshiba America 和 Yamaha Corporation。

MontaVista Software 執行長兼總裁 Jim Ready 表示，此一投資案象徵來自三大國際技術廠及臺灣金融機構對該公司的強烈信心。

Siemens Venture Capital 投資夥伴，也是本次投資案的負責人 Luis Llovera指出，他們體認到 MontaVista 在嵌入式軟體方案重要領域中取得的巨大動能，而正在尋求可結合 MontaVista Software 長處及 Siemens在全球電子產品領先地位的策略關係。

Samsung 已於去年 12 月在中國推出採用 Linux 的智慧型手機。據稱該公司將於一個月內推出基於 Linux 的數位錄影機。Samsung Ventures America總經理 Jay Eum 認為，他們的投資可在 Samsung 及 MontaVista 之間建立有力的互惠關係，他們相信轉向 Linux 的風潮加速之際， MontaVista將可在嵌入式軟體產業扮演主導的角色。該公司將盡可能與 MontaVista緊密攜手合作，以確保雙方達成雙贏。
