TnLUG 群英會現場目擊

Created at Thursday, 06 November 2003 16:00      Last Updated on Tuesday, 11 January 2011 21:48 Written by 林鈺維
十月二十五日由 TnLUG (Tainan Linux User Group) 在南台科技大學主辦的 2003 群英會吸引了近半百的 Linux 社群成員參加盛會，主辦單位希望透過舉辦這個大型對外活動，能在南部地區推廣 Linux 的使用，並且讓大眾了解開放原始碼 (OpenSource) 的精神及其所帶來的附加價值，降低使用者因為不夠瞭解所產生的高門坎好處，更進一步可以毫無障礙的使用。 上午的演講時段為〝Linux & Open Source社群現況座談會〞，主辦單位邀請了台灣開放原碼軟體社群中的先輩到場演講。梁楓除了談 TnLUG 與南部 OpenSource 之現況，並且也點出台灣自由軟體社群普遍之發展瓶頸。Spin Interactive Tech 的王盛立 (wolflord) 先生應邀從嵌入式科技產品談無所不在的 Linux 科技。早上的時段則由台灣自由軟體社群的名嘴洪朝貴老師壓軸談自由軟體文化所鼓勵的價值觀。下午的時段則分為技術場次和商業場次兩部分，前者分別請國家高 速網路與計算中心的黃國連、蕭志榥兩位先生談 DBRL 無硬碟工作站，以及藝立協的簡信昌先生談 RT 電子文件流程管理系統。後者則分別由IBM，SUN，和亞太智通談 Open Source 在各商業公司之發展與展望。

群英會現場主要的互動在於認識平常久仰大名但卻未曾相識的群英，但是如同[群英]一詞所暗示的，現場都是[群英]但無[群雌]，連講者舉例的時候亦以當兵 時的男性經驗為主，這個現象反映出台灣 (雖然也是全球的現況，但是再歐洲國家似乎稍微平衡一點，比例懸殊沒有那麼大)自由軟體社群成員性別比例懸殊的跛腳現象，更深入探討其成因則可歸咎於國內 資訊科技教育的性別參與問題。

群英會現場亦販售〝鳥哥的Linux私房菜伺服器篇〞一書，這是 TnLUG 團隊中的〝鳥哥〞再次推出的書藉，可以提供初學者對 Linux 有伺服器的了解。

雖然場地和流程有些小瑕疵，與會者給予這次的群英會相當高的評價，尤其是能夠將大夥兒在百忙中聚在一起搏感情，新手也有機會認識社群內的老大哥並且互相討教，有什麼這個難得的相聚會更值得的呢？
