2007 國際企業運算研討會 (ICESA 2007) 報名 享有優惠價格喔！
OSSF 編輯小組    2007/10/26

◎ 活動時間：2007/12/12-13
◎ 活動地點：中央研究院 學術活動中心 二樓
◎ 報名網址：https://icesa.openfoundry.org/

中研院資訊所自由軟體鑄造場擬於 12/12、12/13 在院內學術活動中心舉辦 2007 國際企業運算研討會 (2007 International Conference on Enterprise Systems and Applications；ICESA 2007)。兩天的研討會中，包含企業運算原理與實務的論文發表及海報展示，此外，主辦單位特別邀請了國立清華大學榮譽講座教授劉炯朗院士及資策會黃彥男副執行長擔綱主題演講者，分別從學術及產業的角度，親身闡述企業應用軟體的未來趨勢。另外自由軟體鑄造場亦安排國際大廠 IBM 及國內 JAVA 工具軟體領導廠商普奇科技分別進行兩場寶貴的技術教學交流。在面對面的座談會上，延攬了來自韓國、香港、新加坡等地的專家，跟與會來賓針對在地的經驗與亞洲區域軟體發展的機會與挑戰進行討論。

歡迎各界先進一同來共襄盛舉，機會難得，席次有限，11/30 前完成報名繳費者，還享有優惠價格喔！

各路精英看好中國Linux發展
Linuxpilot    2007/11/09

較早前中國 Linux 用戶大會在北京新大都飯店召開。大會透露的消息顯示，中國 Linux 保持年 80% 以上的增長率，中國紅旗本土公司現在佔市場佔有率第一位，Novell、Red Hat 分別佔第二和第三位，Turbolinux 佔第四位，其次是中標軟件等企業。種種跡象表明，2007 年 Linux 在桌面平台、手機嵌入、企業應用等諸多領域將會有長足的進步，中國國內不少機構在政府推動下，採用 Linux 作為解決方案的勢頭不會放緩，Linux 在金融、銀行等行業將繼續吞噬 UNIX 的市場份額。

Vyatta 釋出新版開放源碼路由器軟體
謝良奇/編譯    2007/10/30
以 Cisco 用戶為目標，企圖瓜分數十億路由器市場，開放源碼網路商 Vyatta 釋出新版開放源碼路由軟體。一般的 x86 硬體，經由這套社群支援以 Debian 為基礎的 Linux 散佈套件，搖身一變成為企業路由器、防火牆與 VPN 設備。Vyatta 社群版 3（Vyatta Community Edition 3）新功能包括 IPSec VPN、多重鏈結 PPP（multilink PPP）、邊界閘道協定（border gateway protocol，BGP）。

Google 手機即將問世 行動 Linux 與開放源碼前景可期
謝良奇/編譯    2007/11/01

隨著即將問世的 Google 手機，加上其他手機廠商的新產品推出，如 Motorola 首支基於 Linux 的 Razr2 裝置，開放源碼在行動市場的熱度正在逐漸升高中。

儘管在許多 Linux 組織與新興廠商的努力嘗試下，行動裝置與智慧型手機市場一直仍由 RIM、微軟、Nokia 與 Palm 等廠商的私有平台主導，Google 手機的開放源碼特色加上線上服務與廣告夾帶的龐大商機，預料將是行動 Linux 社群成功的重要助力。
