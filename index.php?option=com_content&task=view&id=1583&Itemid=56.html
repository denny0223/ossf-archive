微軟與 OLPC 合作 XO 筆電將安裝 Windows XP

建立日期 2008-05-22 08:00      作者是 謝良奇/編譯
微軟正打算將該公司的技術帶往貧窮區域，該公司日前達成一項合作案，將在 OLPC (One Laptop Per Child，一孩童一筆電) 的低價 XO 筆記型電腦中，提供 Windows XP 作業系統。這一項舉措有助於該公司讓其作業系統與相關技術，先一步深植在這些區域孩童的電腦使用習慣中，但也引來評論者譏諷。

儘管 OLPC 應該不至於全面放棄採用 Linux 的 XO 筆電，然而隨著該專案共同創辦人 Walter Bender 的離開，此專案的最初目標，希望開發以 Linux 為基礎的教育軟體，恐怕將跟著遭到此專案的逐漸棄守。

Bender 與 XO 筆電 Sugar 使用者介面的其他貢獻者，已經另外創辦了名為 SugarLabs 的專案，專責在開發開放源碼的教育軟體上。基於 GNU/Linux 並運用圖示導向的視覺化介面，Sugar 打算要提供孩童一套協同合作的創造性學習環境。

OLPC 的 XO 筆記型電腦，有來自麻省理工學院 (MIT) Nicholas Negroponte 的支持，交由台灣 Quanta 製造，去年底開始出貨。Nicholas Negroponte 與 MIT 媒體實驗室 (Media Lab) 成員所創立的 OLPC，希望設計、生產並散佈 100 美元的筆記型電腦。

XO 筆電搭載了 433MHz 的 AMD Geode LX-700，內建圖形控制晶片，搭配 7.5 吋 1200 x 900 的螢幕，有 256MB 的記憶體以及 1GB 的 flash 儲存設備。OLPC 的這款筆電搭載 Linux，並使用客製化的特殊圖形使用者介面 Sugar。

在 2006 年 Negroponte 公開表示有意在該筆記型電腦上安裝 Windows 後，引起開放源碼社群的不滿。而在今年 1 月傳出 OLPC 與微軟雙方合作的消息，更是惹來開放源碼社群的反彈聲浪。

如今此一合作已經底定。Negroponte 表示，從一開始 OLPC 的目標就是以技術改變教育。這項合作案的宣佈，加上未來 XO 筆電的雙重開機版本計畫，都在加強其實現此一目標的能力。此外，OLPC 將與第三方合作將 Sugar 移植到 Windows 上。而在紐約時報 (New York Times) 的訪談中，Negroponte 則指出，購買這些電腦的用戶，大多並非會去使用的孩童，而是政府機關，而這些人比較樂於接觸 Windows。

目前尚無法得知採用低階硬體、記憶體容量低的低價 XO 筆電，是否能像執行 Linux 一樣，執行微軟的 Windows。不過到時候 Linux 支持者肯定會來一場雙方的效能大比拼。

預計將安裝在 XO 上的 Windows XP 版本，將支援該筆電的所有功能，包括網路、揚聲器、麥克風與網路攝影機。此外，根據微軟表示，該作業系統也提供電子書 (e-book) 設定，可進入節省電力的日光下單色閱讀模式。微軟無限潛能 (Unlimited Potential) 團隊行銷與溝通總經理 James Utzschneider 在部落格上表示，移植到 XO 上的 Windows 版本將是足以在 XO 有限記憶體與儲存環境下運作的精簡版本，但在功能上不會有所刪減。

不過之前已經購買 XO 筆電的用戶，不管是從捐一得一 (Give One, Get One) 活動期間購買或自 eBay 所購得的 XO 筆電，都無法執行 Windows。根據無限潛能的 Bohdan Raciborski 的說法，Windows 無法在 XO 僅 1GB 的 flash 儲存空間上執行。微軟透過 OLPC、Quanta 共同協助，將為該裝置多增加一個 SD 擴充槽，加上 Insyde Software 新開發的 BIOS，XO 將可從預先安裝的 2GB SD 記憶卡上以 Windows 開機。

在名為貧窮孩童用不著再與 Linux 奮戰(Poor Children Of The World No Longer Will Have To Struggle With Linux) 的文章，知名部落格 TechCrunch 編輯 Michael Arrington 諷刺地寫到，從處理 Windows 當機中獲得的經驗，可能會將這些孩童造就成超強的技術支援專家。他們可能受夠了 Linux 總是這麼穩定。

相對於此，Utzschneider 認為開放源碼社群的心態，根本是要讓孩童自己修自己的電腦。他表示，在印度與非洲等地要找到 Windows 系統管理者的機會遠高於 Linux 系統管理者，而且由於 Windows 廣泛在全球商業環境中運用，在學校使用這些系統的孩童，未來學以致用的機會也比較大。

對此，Linux 支持者的回應可能會是，Linux 系統本身較不需要技術支援，而不同作業系統間的電腦技能多半可以輕易互通運用。不論如何 OLPC 散佈 Linux 筆記型電腦的概念，這幾年來已經接連著帶領出更多的 Linux 筆電，如 Everex 的 gOS Cloudbook、Van Der Led (VDL) Designs 的 "Jisus" UMPC，當然也包括了獲得不小成功的 Asus Eee。

微軟表示執行 Windows 的 XO，最快將於 6 月起在關鍵的新興市場進行試驗。但該公司並未透露全球推出的時間。
