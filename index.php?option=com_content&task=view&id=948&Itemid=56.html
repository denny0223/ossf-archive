香港開放源碼界齊響應綠色環保

Created at Thursday, 26 April 2007 08:00      Last Updated on Wednesday, 12 January 2011 19:39 Written by Linuxpilot

面對來自政府、環保人士、公眾、學術界的環境保護訴求，資訊及通訊科技 (Information and Communication Technologies, ICT)業界日益關注如何減低 ICT 設備對環境的污染，而 Linux 及開放源碼軟體更被視為對綠色環境有利的科技應用趨勢。為了令各界明白箇中好
處， Linuxpilot 訪問了香港 Linux 商會主席簡錦源及協辦單位香港工程師學會 IT 分部副主席譚偉豪博士。

◎ 再思電腦升級周期

香港 Linux 商會主席簡錦源表示：「環保人人有責，商會一直在推廣 Linux 及開放源碼軟體的應用，同時支持業界維護綠色環境，由於考慮到 Windows Vista 即將推出，加上 Windows Vista Capable 要求處理器速度最低為 800MHz，記憶體為 512MB，而功能較完整的 Windows Vista Premium Ready 要求處理器速度最低為 1GHz，記憶體為 1GB，大部分（估計有 80%）現正使用的電腦也未符規格，預料部分企業及個人用戶會因為升級新作業系統，而被迫將舊機大量棄置。如果處理不當，將會對全球的生態環境造成損害。」

大家在升級電腦前，應考慮以下問題：

1. 每三至四年一次將舊電腦更換是否合理？有否方法延長電腦的使用壽命及提高投資回報效益？
2. 當作業系統升級後，有些舊的應用軟體便無法兼容，需要再購買新版本，令軟體開支大增，這樣又是否值得升級呢？
3. 升級新的作業系統及應用程式意味用戶又要重新學習和培訓，影響機構生產力，隨時會引致業務停頓？如果轉用綠色軟體又是否可行？
4. 在棄置舊電腦到垃圾收集站前，可否考慮交給本地電腦回收組織，再捐贈給社會福利或教育機構？

◎ 舊機回收需解決軟體問題

香港工程師學會 IT 分部副主席譚偉豪博士指出：「全球各國已經關注到產品回收的問題，歐洲已經通過「危害物質限制指令」（簡稱 RoHS 指令），而中國的相關法律也即將實施，香港沒有相關的法例，處理電腦棄置問題是業界所關注的一大問題。」

「由教育統籌局及香港社會服務聯會主辦的「家家有腦」電腦循環促進學習計劃，便為舊電腦提供一條出路，香港工程師學會也曾派出義工協助將電腦翻新。過去如果電腦是由商業機構捐出，我們需要將裡面的軟體刪除，但是由於軟體版權問題，我們沒法安裝新的作業系統及應用軟體，用戶要自行解決，現在有了 Linux 及開放源碼軟體，我們可以在新一年的計劃內，送出有完整軟體的翻新電腦，造福更多家庭及孩子。」
