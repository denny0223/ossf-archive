Canonical 取悅軟硬體夥伴 推出 Ubuntu Advantage 計畫

建立日期 2011-08-04 00:00      最近更新在 2011-12-27 14:22 作者是 謝良奇／編譯

有為數不少的 Linux 用戶也是 Ubuntu 的愛用者，儘管對 Ubuntu 新的 Unity 桌面有意見，但是到最後他們還是在使用 Ubuntu。不過科技公司對 Ubuntu 贊助商 Canonical，並沒有太多好感。為了扭轉這一點，Canonical 正試圖採取行動取悅潛在的軟硬體夥伴。

 

Canonical 宣佈了一項名為 Ubuntu Advantage (UA) 的通路夥伴計畫，以改善與經銷商合作夥伴的關係。UA 的目的是幫助經銷商直接為企業帶來針對 Ubuntu 伺服器、桌面、雲端建置的新支援服務組合。Canonical 表示該計畫將與全球若干合作夥伴共同啟動，包括美國、亞洲、歐洲、中東及非洲地區的 CSS，以及日本的 Ashisuto、斯堪的納維亞的 RedPill Linpro、法國的 Alterway。

UA 將提供企業用戶最大化其 Ubuntu 基礎建設回收所需的工具和支援，包括 Ubuntu Landscape 管理與監控工具、知識庫、全天候支援、法律保護。UA 會協助最小化對關鍵任務服務的影響，並減少系統停機時間的成本。新的 UA 夥伴計畫能將此一服務擴展至 Canonical 之外，提供來自合作夥伴的當地資源給用戶。

對經銷商而言，他們可以從新服務中獲得額外的收益來源。此外，Canonical 承諾經銷商將可獲得行銷、技術、商業與售前支援，並且獲派一位客戶經理。

Canonical 過去曾經和 IBM 合作提供企業軟體堆疊，也曾經短暫嘗試過零售方式的開放源碼套裝軟體，並針對 Linux 伺服器市場，瞄準 Red Hat 和其他伺服器作業系統巨人。儘管獲得了部份成功，Red Hat 依然是伺服器 Linux 中的龐然大物。

根據 The VAR Guy 網站的觀察，Canonical 過去在合作夥伴上也有許多的想法，卻從未成功實現。這是因為 Canonical 經歷了多次的管理變動與產品發表，每一次都將該公司拉往不同的方向。除了每次 4 月和 10 月的 Ubuntu 定期釋出之外，Canonical 還建構了額外的平台，如 Ubuntu Enterprise Cloud、Ubuntu One、Landscape。

一路走來，Canonical 多位高層人士與管理者來來去去，其中包括前營運長 Matt Asay－掌管獨立軟件開發商關係與 Landscape 業務發展的關鍵領導者。然而，不論經銷商或企業用戶，都不希望看到一家公司經常發生變動。他們希望看到的是穩定的通路計畫和產品線。對於 Linux 愛好者，經常變動以及小小改進或許令人興奮，但企業卻偏好穩定。

此外，Canonical 也正在簡化其硬體認證計畫。過去製造商在系統背書上有兩種不同選擇，也就是 Ubuntu Certified 與 Ubuntu Ready。Canonical 了解到這兩種選擇可能造成混淆，因此從 10 月份的 Ubuntu 11.10 釋出開始，只會有 Ubuntu Certified 這一種由 Canonical 背書的硬體認證。

假如 Canonical 能夠取得成功並且穩定其管理結構，那麼 Canonical、Ubuntu 將更有機會從 Linux 愛好者的桌面，進入組織辦公室與伺服器機房中。其次 Canonical 也必須將其夥伴計畫，與 Red Hat 或 SUSE 的行動做出區隔。該公司得大力宣傳 UA 計畫的通路夥伴收益，因為其競爭 Linux 散佈套件特別是 Red Hat Enterprise Linux，已經取得了足夠數量的通路夥伴。
