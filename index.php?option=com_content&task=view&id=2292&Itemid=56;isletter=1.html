MeeGo 原始碼釋出 Intel、Nokia 攜手共推新 Linux 行動作業系統

建立日期 2010-03-30 08:00      最近更新在 2010-06-07 22:39 作者是 謝良奇/編譯

Intel 和 Nokia 日前將 MeeGo 行動作業系統的原始程式碼，開放給外部開發者。Intel 開放源碼技術中心 (Intel Open Source Technology Center) 主任 Imad Sousou 宣佈，MeeGo 原始碼和首批可安裝的映像檔已可供下載。儘管 MeeGo 平台目前仍不完整且處於有待積極開發階段，這次釋出的目的正是希望廣邀第三方開發者參與該專案。

MeeGo 結合了兩套以 Linux 為基礎的作業系統，包括 Intel 的 Moblin 和 Nokia 的 Maemo。MeeGo Linux 平台是由 Intel 和 Nokia 在 2 月時宣佈合併 Moblin 與 Maemo 時所成立。藉由與開放源碼社群的合作，這兩家廠商希望 MeeGo 能成為行動 Linux 平台的產業標準，支援多種外觀尺寸 (form factors) 和處理器架構，並減少市場分裂和鼓勵協同合作。

為了讓 MeeGo 更具兼容性和廠商中立性，Intel 和 Nokia 找來了 Linux 基金會 (Linux Foundation) 作為合作對象。

根據 Sousou 表示，初次的程式碼釋出包含了 Linux 核心與作業系統架構的關鍵部份，例如系統程式庫和通訊框架。這些程式碼已置於公開給開放源碼開發者的線上倉儲。可供下載的可安裝映像檔，包含了支援 Nokia N900 智慧型手機和以 Intel Atom 處理器為基礎的小筆電 (netbooks)。不過這些映像檔並不包含 MeeGo 的使用者介面和標準的應用軟體堆疊，因此並不適合一般用戶使用。

Intel 和 Nokia 希望 MeeGo 最終能與來自 Apple、Google、Research in Motion 的智慧型手機平台相抗衡。然而這兩家廠商也有意讓 MeeGo 應用在智慧型手機以外的各種裝置。他們與 Linux 基金會合作，開發一套基於共同核心的架構，其使用可跨越多種模式，例如小筆電、手持裝置、車內裝置、聯網電視 (connected TVs)。

此一核心包含了許多關鍵子系統，像是核心作業系統程式庫、通訊與電話服務、網際網路與社群網路服務、視頻服務、媒體服務、資料管理、裝置服務、個人服務。

Intel 與其合作夥伴計畫於 5 月釋出 MeeGo 的完整版本。供 MeeGo 裝置使用的第三方應用軟體將透過 Intel 的 AppUp Center 提供。該線上商店也將開放給消費者電子產品製造商。Acer、Asus、Dell、Samsung 是 Intel 在該線上商店的第一批合作對象。這些廠商計畫打造基於 AppUp Center 用戶軟體的自有品牌商店。用戶需要透過此用戶軟體來存取 AppUp Center。

MeeGo 與 Android 和其他競爭平台之間有若干關鍵差異點。其中之一是 MeeGo 與上游 Linux 核心及桌面堆疊的合作更為密切。此外，MeeGo 採取的是參與式的開發模式，其程式碼將在公開倉儲中開發。因此新技術採用者與貢獻者可以在開發的最初期階段，甚至是首批產品釋出前，就參與其中。

Sosou 表示，全球 Nokia 和 Intel 團隊向開放源碼社群分享 MeeGo 作業系統程式碼，在今天達到了此一努力的高峰。這是 Maemo 和 Moblin 完全合併的最新階段，他們很高興開放程式碼倉儲，並正如一開始所計畫般，開放此一持續進行的開發工作。

分析師指出，供行動裝置取用的線上應用軟體商店是此一平台的成功關鍵。因為第三方應用軟體可延伸裝置的能力。在 iPhone、Google 的 Android、RIM 等環伺之下，MeeGo 如何吸引開發者的興趣，將是一大挑戰。
