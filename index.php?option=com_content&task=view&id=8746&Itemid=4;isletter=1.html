Android 版 Firefox 14 正式推出

建立日期 2012-07-06 15:01      最近更新在 2012-07-09 11:29 作者是 黃湘婷／編譯

Mozilla 於 6 月 26 日發布了 Android 版 Firefox 14 的正式版本，在行動裝置上開啟網頁能夠更加快速。其實早在 2011 年，Firefox 就曾出現在行動裝置上，但以往的 Firefox 執行速度較為緩慢，而 Android 除了有內建的瀏覽器外，還有 Dolphin、Google 及 Opera Mini 等多種不同的瀏覽器軟體，因此開啟網頁的速度若太過緩慢，可能會令使用者選擇別種軟體作為預設的瀏覽器。

為此，Firefox 14 的更新就著重於執行效率，主打快速與個性化的升級，使得啟動、載入網頁以及縮放頁面更加快速，Mozilla 也透過自行開發的工具 Eideticker，測得 Firefox 的瀏覽器執行速度為 Android 內建 Stock 瀏覽器的兩倍。

除了加快的效能外，新的功能還有個人化的起始頁面，能夠節省前往常用網站花費的時間。Firefox Sync 能夠將電腦中的瀏覽紀錄、書籤、密碼，以及表單數據同步至 Android 手機，對於原本在桌面版就使用 Firefox 瀏覽器的使用者更加方便，在手機端及桌面端便能同步使用，希望能提供使用者最佳的移動瀏覽體驗。

另外還有在桌面版受歡迎的分頁瀏覽以及附加元件，都包含在此次的正式版當中，同時也支援 Flash 及 HTML5，令開發人員能夠使用 HTML5, JavaScript, CSS 以及其他開放 Web 標準來創建豐富的 Web 應用程式與網頁。作為 Web 的提倡者，Mozilla 也公開了新的 Web API，希望能讓網路成為最大的平台。
