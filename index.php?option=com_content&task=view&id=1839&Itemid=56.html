微軟否認支付款項給承包商 要求放棄 Linux

建立日期 2008-11-13 08:00      作者是 謝良奇/編譯

據媒體報導，微軟為了因應政府單位中的 Linux 運動，曾有意支付一家奈及利亞 (Nigeria) 承包商 40 萬美元，將學校電腦的 Linux 改換為 Windows。微軟否認曾支付該承包商此一款項。

據報微軟曾在去年簽訂的共同行銷協議中，提案以 40 萬美元金額支付某政府承包商，要求該承包商在數以千計的學校筆記型電腦上，將 Linux 作業系統更換為 Windows 作業系統。

微軟西東與中非地區經理 Thomas Hansen 表示，雖然曾經草擬了一份有關教育領域技術應用最佳實務的共同行銷協議，但該協議並未實際實行。原因很簡單，因為有用戶想要 Linux 作業系統。

Hansen 指出，有鑑於此，該共同行銷協議於是作廢，也就沒有所謂的行銷協議存在，更不用說支付什麼款項了。

由於 Linux 免費散佈的特性，其功能性、穩固性與適應性使 Linux 成為私有 Unix 與微軟作業系統的主要替代選項。迦納 (Ghana)、納米比亞 (Namibia)、奈及利亞與南非 (South Africa) 都已經在政府部門與學校部署 Linux。然而，Hansen 表示微軟與這些國家政府都有相當穩固的關係。

他表示，從他們的觀點，這些政府，以及每一位用戶，應該決定何種軟體解決方案最合乎其需求。他們強烈地認為，這些政府必須仔細考慮個人電腦採購與使用的所有成本，以及廣大可用應用軟體、維護與訓練所帶來的利益。

Hansen 強調有研究指出，當總持有成本納入考量時，Windows 平台的成本通常與 Linux 相同或是更低。此外，當全面性地考慮對用戶帶來的益處，例如廣泛的可用應用軟體、熟悉度以及易用性，Windows 通常會是更好的選擇。
