台法自由軟體教育經驗交流圓滿落幕
葉名軒、李貞臻 2006/07/31

2006 台法自由軟體教育經驗交流研討會 (2006 Libre Software Meeting) 於七月七日在中中央研究院資訊科學研究所圓滿落幕。透過自由軟體鑄造場 (OSSF) 的安排，讓台法雙方熱衷自由軟體教育者，以跨國視訊會議的方式進行互動。與會國家除了台灣、法國之外，還有西班牙及南非，互相分享如何將自由軟體推廣在學 校、教育領域的經驗。

謝良奇 2006/07/28
