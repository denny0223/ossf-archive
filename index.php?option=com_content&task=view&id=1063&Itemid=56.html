Linux 基金會合作高峰會展開

Created at Monday, 25 June 2007 07:45      Last Updated on Wednesday, 04 July 2007 17:46 Written by 謝良奇
倡導自由軟體的社群人士日前群聚在 Google 總部，討論此一運動的未來，以及近來微軟向Linux 拋出的專利威脅。這群 Linux 社群人士認為，應回頭專注在開放源碼作業系統 Linux 的發展，諸如擴展系統能力等，而非花費過多心力在微軟的專利協議上。 於 Google 園區舉辦的 Linux 基金會合作高峰會 (Linux Foundation Collaboration Summit) 由 Linux 基金會 (Linux Foundation) 主辦，該會成員與支持廠商包括 AMD、美國銀行 (Bank of America)、EMC、Intel、Hewlett-Packard、IBM、Oracle 與 Novell 等企業。會議吸引了 200 位開放源碼開發者、使用者與軟體廠商參加，包含了演講、專題討論，以及與 6 大 Linux 核心開發者的問答時間。

近來微軟藉著宣告 Linux 等開放源碼軟體侵害其 235 項專利權，同時與 Novell、Xandros 兩大 Linux 廠商簽署合作協議，免除其專利訴訟，已在社群內引發爭論。在 Wal-Mart 等零售商販售 Linux 個人電腦的 Linspire 也在最近成為第三家與微軟簽訂協議的廠商。

在一股反微軟的氣氛下，不少與會者公開稱該公司為“敵人”。然而大多數人仍認為最終決定權在 Linux 使用者手中，微軟的專利威脅只是散佈 FUD 的最新手段。Steeleye Technologies 技術長 James Bottomley 認為，由於 Linux 用戶多半也是微軟的用戶，使用者重疊的情況下，微軟不可能控告 Linux 用戶。他也說，這種做法只是讓人覺得很煩、很討厭。

Zend Technologies 的態度則較為中立，未參加此次會議的該公司執行長 Harold Goldberg 表示，他認為微軟因為一方面想捍衛其企業版圖，一方面該公司內確有一批人視開放源碼為未來，微軟仍在考慮其下一步。

回到 Linux 作業系統的發展議題上，IBM 開放系統發展副總裁 Dan Frye 表示，IBM 贊助的開發者正著手於供國防與金融服務用戶使用的即時 Linux。Novell 的 SUSE Linux 已自去年 10 月起提供即時版本，MontaVista 與 Concurrent 也有即時 Linux 散佈套件產品。Frye 表示華爾街希望看到能在不到一秒內完成多次交易決策的即時 Linux 系統，製造業與醫療系統也有即時 Linux 發揮的空間。

從 2.4 到 2.6 版 Linux 核心，Linus Torvalds 等人每 2 至 3 個月就會發出更新，核心開發者 Greg Kroah-Hartman 表示，從未看過 Linux 展現如此的進展步調。但也因此產生新功能無法完整測試的問題。Bottomley 就表示，引進新功能與穩定性之間的確有所衝突，因此他們很需要使用者社群協助找出 bugs。

Motorola 市場開發副總裁 Christy Wyatt 認為 Linux 應該在支援行動裝置上加快腳步。她預測 Linux 可以在手機作業系統市場爭取到優勢地位。她更表示 Motorola 預估短期內該公司 60% 的手機都會使用嵌入式 Linux。
