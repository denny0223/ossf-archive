2009 自由軟體暨開放原始碼校園巡迴種子研討會
呂卓青    2009/04/23

想要瞭解什麼是自由軟體嗎？利用自由軟體的技術與觀念，可以為我們帶來什麼樣的機會？嘗試自由軟體程式開發會不會很困難？要如何跨出自由軟體程式開發的第一步？為推廣國內自由軟體應用及培養相關技術開發人才，由工業局指導、資策會主辦的自由軟體校園巡迴種子研討會將於 4 月 28 日開始播種！議程包含國內外自由軟體最新動態、Linux 桌面應用程式開發、Linux 多媒體技術分享等豐富資訊，此次活動亦會邀請在台灣活躍的自由軟體社群分享開發經驗，以及說明第一屆自由軟體百萬大獎的競賽辦法。不論您是想要瞭解自由軟體的新手，還是已經善用自由軟體的老手，或是想要挑戰比賽獎金的高手，都請不要錯過此次研討會的精彩內容！

本次研討會的議程內容包括，自由軟體競賽與自由軟體介紹，使用 QT4 與 GTK+ 開發自由軟體桌面程式，GStreamer 多媒體應用框架，Joomla 整合系統介紹，Plone 網頁開發框架，自由軟體價值論述等精彩講題，報名網址為：https://www.oss.org.tw/seminar/schools/index.php，或是造訪活動網頁，以獲得更多訊息。

新進專案列表 04/25
OSSF 工作小組    2009/04/25

1. Easy Screen Switch Mode
專案摘要：簡便螢幕模式切換軟體。提供 Linux 筆記型電腦的使用者可以容易地切換外接/投影螢幕。此軟體提供圖形化介面，不再需要文字模式進行操作，也不再需要重新啟動 Xwindow 或重新開機才可使用外接/投影螢幕。

2. 可攜式USB病毒砍殺程式
專案摘要：1. 透過隱藏檔與非隱藏檔交叉比對技術提升偵測的命中率 2. 透過檔頭與副檔名的檢查進行偽裝的病毒找尋 3. 透過查表方式進行指定路徑及檔案刪除 4. 允許自訂腳本設定

3. web bot
專案摘要：An interactive assistant to handle requests for services from Dr Bean's "Course Management System"

4. WindowsFish - Cloud Computing Utility Tools
專案摘要：Apache Hadoop - Visual Manager and Relation Researc

5. OpenLegal–OpenFoundry 平台的法律授權層面
專案摘要：OpenLegal 一詞代表自由軟體授權方面的二個重要意涵，Open 代表自由軟體使用上的最大特徵是程式原始碼的公開共享，而 Legal 代表的是自由軟體使用者只要遵守它預設的遊戲規則，則授權範圍內的一切利用行為都是合法合份的。OpenLegal 為 OpenFoundry 平台的衍生專案，其代表 OpenFoundry 法律授權研究的諮商面及其它服務層面，專案管理人員會不定期的透過文章書寫、研討會舉辦，以及授權議題的討論及回覆來分享自由軟體法律授權相關的研究心得，也非常歡迎其他對於自由軟體法律授權有興趣的朋友，透過問題提問或是資料上傳的方式一同參與這個專案，幫助我們將專案的相關內容擴充的更為實用及活潑。
