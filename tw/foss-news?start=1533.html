ICOS 2004-- 開放源碼：社群、產業與政府綜合座談

建立日期 2004-12-28 08:00      最近更新在 2011-07-21 17:43 作者是 涂敏怡

大會第一天下午有「開放源碼：社群、產業與政府」的綜合座談，邀 請各界代表一起探討產、官、學、社群如何就自由/開放源碼軟體進 行合作。

關於政府近年來推行自由軟體的政策規劃，行政院科技顧問組副執行 秘書蔡志宏表示主要方向有三，一是讓政府採購成為自由軟體主要力 量之一；二是自由軟體的支援不限 Linux、也可延伸到商業軟體上； 三是產業發展跟使用者權益的平衡。蔡志宏並表示，行政院國家資訊 通訊發展推動小組（NICI）在落實自由軟體相關政策上有三大方向： 一是促成自由軟體產業可以與其他產品結合，例如網路設備、智慧型 手機等；二是教育推廣自由軟體以縮減數位落差；三是與資訊安全的 結合。

運用自由軟體策略 創造產業雙贏

建立日期 2004-12-17 16:00      最近更新在 2011-01-11 17:58 作者是 webmaster
【2004年12月17日台北訊】
由於開放源碼所帶來的軟體開發效益和成本之優勢，一股自由創作分享的風氣正快速瀰漫於世界各角落，使得軟體世界更有活力。許多知名的商業公司也不斷的透過開放源碼的方式，與全球的開放源碼社群進行合作，改良其產品，或作為另外一種提高普及率與滲透率的行銷方式。而「鑄造場」（forge/foundry）則提供自由軟體開發者所需之服務，匯集軟體專案與開發人員，讓企業主與軟體開發者能透過這樣的開發平台進行經驗交流與分享資訊，進而創造商業機會。


中央研究院資訊科學研究所自由軟體鑄造場（OSSF）於2004年12月17日特別邀請了享譽國際的 SourceForge.net總監Patrick McGovern，和國內知名的威盛電子(VIA)與網威國際(Novell)與會，分享他們如何運用鑄造場，與開發者社群的研發能量創造雙贏。並介紹自由軟體鑄造場因應自由軟體策略，為國內開發社群所提供的嶄新功能與服務。


開放源碼軟體開發網站的典範

Sourceforge.net 是世界上最大的開放源碼軟體開發網站，提供許多軟體開發所需的工具及專案管理資源。藉由此開放源碼中心，不僅活絡了開放源碼的開發社群，同時，很多資訊大廠也將其軟體專案放置於SourceForge.net上與開放源碼社群互動，並且創造了許多產業與社群合作的成功經驗。目前，Sourceforge.net進駐專案有超過9萬，會員則超過90萬人，每天有約250萬網頁被觀看，每天有約140萬次的下載紀錄。創造了開放源碼軟體開發網站的奇蹟。


台灣的自由軟體鑄造場

中央研究院資訊科學研究所自由軟體鑄造場專案管理平台（Openfoundry；http：//rt.openfoundry.org），是一個為自由軟體專案開發人員、管理人員以及一般使用者而設置之共同平台。自由軟體鑄造場團隊依照本地自由軟體開發者之工作模式，整合既有的自由軟體開發工具模組，建立符合軟體開發者需求的工作平台，並蒐集使用者意見，不斷進行改良與最佳化，滿足軟體開發的需求，減輕專案管理負擔。專案管理平台服務包括：(1) 提供共同開發軟體專案所需之工具及服務，(2) 提供專案管理所需之相關工具及服務，以及(3)彙整一般使用者所需之自由軟體、其說明文件與使用手冊等資源。

整個專案管理平台是由許多開放源始碼套件所組成，包括RT, Subversion, Sympa, Kwiki，可以獨立拆解或只取所需之部分，進行彈性客製化修改；專案管理平台的每個服務都可各自獨立安裝於不同主機，所以，當一個伺服器當機或某一個服務停擺，並不會影響到其它服務的運行。此外，也具備支援多國語言的架構，目前已支援了繁體中文、簡體中文以及英文。若日後需要其它語言支援，只要進行文字翻譯即可，供更多不同語言之開發者使用。期望站在專業的角度，協助台灣的業界及政府，利用開放源碼軟體，創造商機，強化台灣在自由軟體發展的的優勢，培養及深化開放源碼的人才與專業能力。


其他企業在自由軟體發展與鑄造場的經驗

Novell Forge

Novell Forge (https://forge.novell.com)是Novell積極參與開放源碼社群的具體展現。其特色之一，是提供了諸多知名軟體的Novell版本，例如MySQL for NetWare，Apache for NetWare等等。這些分支版的開放源碼軟體，可透過Novell Forge取得，而這些專案在Novell Forge上的許多開發成果，也經常併入自由軟體的主幹（trunk）中，成為對開發社群重要的貢獻。Novell Forge並提供CVS版本管理伺服器，並提供郵遞論壇、線上瀏覽程式碼、使用者身份管理等服務。Novell透過Forge的建置，成功整合了自家所開發的技術與開放源碼界的重要工具。

Via Arena

VIA Arena (https://www.viaarena.com/)是威盛電子公司的行銷服務網站，藉此與產品客戶及使用者進行多向交流，並創造出一個平台應用社群。該網站提供產品新聞、使用者討論區、應用專題文章、應用創意競賽、下載服務、常見疑問解答等。使用者可以透過 VIA Support Forum 和服務人員進行交流，整個論壇目前有超過六萬名註冊使用者，討論主題包括有主機板、作業系統、多媒體及遊戲等。針對 VIA 的晶片組及處理器，網站上提供有各式產品的資料庫、產品新聞、主機板製造商資訊等。在檔案下載的服務目錄裡，提供 Windows 及多種 Linux 環境下的驅動程式、安裝說明、工具程式等。VIA Arena 扮演了威盛電子的網路服務窗口，同時也吸引使用者社群貢獻改進建議及分享使用經驗。


運用自由軟體社群創造產業雙贏

近年來自由軟體專案的崛起令人驚奇，成功的自由軟體專案往往都是以全球、分散式的協同開發樣貌呈現。讓來自各地的開發者共同對某一軟體專案的原始碼進行開發與修改，在廣大的社群網絡上，透過公開傳播，使得各類成員得以積極（如修改原始碼及文件）或消極（如使用自由軟體）地參與軟體開發的活動。政治經濟學者 Steven Weber認為此種「自願參與、各取所需」的軟體開發流程，是自由軟體成功的關鍵。

而軟硬體公司基於不同的策略動機考量，發起或參與開放原始碼軟體專案，和外部的開發社群協同來開發產品，這些技術遍及作業系統、驅動程式、開發工具、語言/標準/平台、多媒體技術、遊戲、甚至是研發成果等。這些公司無非是希望藉助原始碼軟體社群的力量，再造如Linux般成功的傳奇。

雖然商業應用與軟體授權還有若干問題仍待解決，但自由軟體在「價格」、「功能」、「後續維護」與「品質」上的優勢，對於資訊產業的衝擊，具備相當的影響與貢獻。隨著開放源碼風潮的興起，鑄造場儼然已經成為新一代資訊軟體產品的孵化器，企業如何運用社群豐沛而源源不絕的研發能量，將是取得科技研發優勢的關鍵。

關於網頁瀏覽器 你有更好的選擇

建立日期 2004-11-25 16:00      最近更新在 2011-01-07 00:16 作者是 webmaster
台灣自由軟體社群發表Firefox 1.0 中文版
提供更安全舒適的上網經驗

【2004/11/19 台北訊】在沈寂一段時間後，瀏覽器產品市場又再度掀起波瀾。日前 Mozilla 基金會於發表新一代瀏覽器 Firefox 1.0，引起熱烈反應，每日下載人數超過百萬人。Firefox 1.0也推出了繁體中文版，即（19）日起，只要連上https://moztw.org就可以免費下載最新的Firefox。這是台灣的自由軟體社群Mozilla Taiwan與美國Mozilla基金會合作的成果。台灣主要的自由軟體社群也將發起系列名為「這下『火』大了」的推廣活動。


台灣的 Firefox 推廣活動
軟體自由協會常務理事台大物理系客座助理教授葉平表示，這一系列的「放火行動」，將從校園出發。國內熱心推廣自由軟體的大專院校的學生，將一起在校園中點燃Firefox 之火的行動，目標要讓最多的學生接觸到 Firefox。在 11 月 21日則將在台灣大學凝態中心舉辦「飆風再起」Firefox 研討會，介紹 Firefox 的入門和進階功能，並於同日晚上於台大體育館內的迴廊咖啡舉辦盛大的慶祝派對。

參與Firefox推廣活動的單位除了各大專院校電腦社團以外，還包括軟體自由協會（SLAT）、校園自由軟體應用諮詢中心（OSSACC）、中研院資訊所自由軟體鑄造場、中華民國網路消費協會、亞矽氏資訊顧問公司、晟鑫科技等。

備受好評、獲獎無數的瀏覽器

負責Mozilla Taiwan中文化統籌事宜的林弘德表示，Firefox備受專業媒體好評，今年9月16日華爾街日報的個人科技專欄中作者 WALTER S. MOSSBERG針對 Windows 使用者提出許多保全上的建議，其中一項就是建議「以 Firefox 上網」。

此外，Firefox在媒體和各項 IT 展覽中也獲得許多大獎。國外的 Laptop Magazine 的Editor's Choice Award、Linux World Fall 2004 的 Best in Show 獎和Best Open Source Solution 獎，並在 Maximum PC 的 Softy Awards for 2004 中勇奪首獎。

林弘德指出，Firefox的特色是有更安全的功能，也提供很多貼心的功能，為了提供更好的品質，Firefox也建立更方便的版本檢查功能，可以隨時檢查是否有最新版本，還有瑕疵通報系統協助Firefox研發團隊修正瑕疵。

不斷推陳出新的功能

目前Firefox提供的主要功能包括：
* 阻擋彈出廣告視窗：未經使用者的同意，網站自作主張要彈出的視窗一律會被 Firefox 擋下，除非使用者建立起白名單，Firefox 才會放行。

* 保護使用者不受線上詐欺之害：Firefox 會清楚地顯示出連上的網站的真實身份，使用者不會在不知情下把重要的信用卡號或密碼洩漏出去。

* 更快更簡單的搜尋：Firefox 將常用的搜尋服務網站整合起來，包括Google、Yahoo!、eBay、Amazon、Dictionary.com、IMDB（電影資料庫） 等等。而最新的 Firefox啟動首頁更提供了 Google 搜尋和 Firefox 相關的資訊和小技巧等。

*提供「分頁瀏覽」：工作列不再擁擠，而「清新書籤」讓使用者隨時可以閱讀從多個網站收集而來的最新新聞標題。

*更貼近每個使用者獨一無二的上網習慣：上百個擴充模組，包括A9.com、Google 和 Yahoo! 的工具條以及自動繁簡轉換的模組。還有許多社群開發者提供為各種用途開發的小工具與擴充模組，供使用者依照不同需求下載。

* IE使用者轉換無障礙：Firefox 的匯入功能讓使用者可以馬上換用 Firefox 並保有之前的書籤、密碼、cookie等。

多語言、多作業系統支援

中研院資訊所研究員何建明指出，包括繁體中文在內，Firefox 目前已經有十九種語言的版本，在 Windows、Mac OS 和 Linux 三種作業系統都能使用。一個軟體能同時推出這麼多版本是項令人驚奇的巨大工程。

這要歸功於其開發模式--開放源碼，使得各種平台的程式師和全世界說各種語言的社群成員可以協同合作，貢獻到大家共同的興趣上，讓這個瀏覽器更安全，使用上更貼近個別使用者的需求。Firefox 是從 Mozilla 這個整合瀏覽器、郵件軟體、線上聊天室等功能的軟體專案中把瀏覽器的部分切割出來優化而成的。

來自全球自由軟體社群的支援

全世界上千名程式師和更多的使用者社群已然形成一完善的軟體生態系統。若使用中文版有任何問題，在 https://moztw.org/ 可以找到 Mozilla Taiwan 社群的中文討論區，與社群開發者進行討論。熟悉英文的使用者則可以在
https://www.mozilla.org/ 找到技術文件和支援廠商，
在 https://www.mozillazine.org/ 找到相關新聞﹑評論和社群對彼此的支援。

Creative Commons 與 Creative Commons Taiwan 之源起

建立日期 2004-11-23 00:00      最近更新在 2011-08-17 15:40 作者是 OSSF法律與政策組

◎ Creative Commons 源起

Creative Commons 是一個非營利組織，2001 年在公共領域中心 （Center for the Public Domain）的大力支持下得以創建，目前設 於史丹福法學院。Creative Commons 由董事會所領導，董事包括網 路法律與智慧財產權專家 James Boyle、Michael Carroll、Molly Shaffer Van Houweling 和 Lawrence Lessig、MIT 資訊科學教授 Hal Abelson、由律師轉任紀錄片製作，再轉而為網路法律專家的 Eric Saltzman、知名的紀錄片工作者 Davis Guggenheim、知名的 日本企業家 Joi Ito、以及公共領域網路出版者 Eric Eldred 等人。

Firefox 1.0 中文版發表記者會

建立日期 2004-11-20 16:00      最近更新在 2011-01-07 00:16 作者是 ossfepaper

●活動時間：2004/11/19 14:00~16:30

●活動地點：遠東國際大飯店 怡東園

在沈寂一段時間後，瀏覽器產品市場又掀起了一個波瀾。隨著網頁瀏覽器Firefox 1.0版推出，目前資訊界最熱門的話題便是「已經稱霸瀏覽器市場多時的微軟，是否遇到了夠份量的對手？」

日前Mozilla Firefox 網頁瀏覽器開放下載以來，已經創下800多萬人下載的紀錄。更難能可貴的是，這個優良的瀏覽器是由開放源碼社群合力開發完成的成果。台灣的社群開發者，也參與了FireFox的中文化工作，預料將在近期內與其它國家版本的FireFox一起推出。

Mozilla Taiwan社群與中研院資訊所自由軟體鑄造場，在2004年11月19日於遠東國際大飯店舉辦『Firefox 1.0 中文版發表會』，讓媒體朋友瞭解Firefox的功能。會中由參與開發的台灣社群開發者趙柏強介紹Firefox的功能，以及資訊專家中研院資訊所研究員何建明博士、台大物理系教授葉平、交大資工系副教授黃世昆，從軟體開發、資訊安全等角度，為現場來賓剖析此一新世代瀏覽器出現的意義。
