戲說自由軟體文化—The showbusiness of freesoftware
林鈺維 2004/06/14
自由軟體或科技或社會的實踐所創造出的獨特社會科技文化，讓使用者不再被侷限在微軟視窗產品獨大的單一選擇裡，有更多元的選擇，而且能夠為自己量身打造一 套作業軟體，或甚至可能親身參與軟體的開發。自由軟體的實踐除去了傳統科技創新黑箱化的現象，讓一般人有機會打開軟體科技這個黑箱子，把裡面裝的東西拿出 來看一看、摸一摸，這個自由軟體衍生出的將科技去黑箱化的過程可以從許多不同的面向來解釋，例如，一些社會學者即嘗試從科技民主的角度來分析自由軟體文 化，但是，作為一篇即席的小品文，我想跳脫出硬梆梆的象牙塔知識圈，用比較輕鬆的態度和大眾化的語言來談這個將軟體科技去黑箱化的自由軟體文化。最近英國 由觀眾票選的流行偶像 (Pop Idol) (註1) 正風靡，突然有個靈感想用演藝圈文化來比擬自由軟體文化，而 Linux 雜誌舉辦的票選活動就好像演藝歌唱界經常舉辦的票選活動一樣，如果再配上 Linux 頒獎會議，就更具娛樂效果了，且讓我緩緩道來這一套科技娛樂化的理論。

新版 Opera 支援嵌入式系統 加入 IRC、RSS 等功能
謝良奇 2004/05/25
Opera Software 為該公司新款 Opera 瀏覽器加入 IRC，以及使用者介面改善。新版 Opera 在不大幅提高執行記憶體、軟體容量需求的前提下，如今除了瀏覽器，更包含了電郵軟體、新聞閱覽器（newsreader）、IRC、整合式 Google 搜尋等功能。新的 Opera 支援 Linux、嵌入式 Linux，以及其他普遍的嵌入式和一般平台。

自由軟體授權模式眾多 OSSF自由軟體授權指引提供建議
ossf 2007/01/29

自由軟體授權模式的選取，攸關軟體開發專案的命運。然而，目前的授權模式眾多，光是自由軟體組織OSI（Open Source Initiative）認可的授權方式，就有54種之多。對此，目前並無相關網站提供相關的協助。為協助專案開發者選擇授權方式，自由軟體鑄造場日前推出法律授權指引（License Wizard），讓駐站的專案開發者在幾分鐘內，就可以從自由軟體社群最常用的十數種授權條款中，取得適合專案性質的授權模式。
