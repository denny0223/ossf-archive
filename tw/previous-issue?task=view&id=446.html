一招半式解遍壓縮檔 - 7-Zip
Mischa Hsu    2008/12/24
不想到處去找註冊碼，卻又想擁用支援多種格式的壓縮軟體嗎？7-zip 就是你要的答案了。7-zip 輕巧、方便，支援解壓縮各種常用格式，並能夠壓縮 ZIP、7z、GZIP、BZIP2 和 TAR檔。除此之外，7-zip 還是個免費的壓縮軟體，任何人不需任何費用，都可以取得 7-zip 程式和原始碼，並能自由修改和散布。也因為如此，除了官方放出的 Windows 和 Linux/Unix Unix 版之外，你幾乎在各種常見平台都可以看到支援 7-zip 格式的壓縮程式，即使想將壓縮檔傳給不同平台的使用者，也不用擔心沒有程式支援。

我國法下自由軟體契約成立時點
葛冬梅    2008/12/24
「自由軟體授權契約究竟在甚麼時候成立？」，這是一個法律人常會問的問題，但是因為自由軟體是一種新型態的授權模式，至今沒有法院判決或學術文章針對這個問題做深入探討，所以對於這個問題至今還沒有一個統一的見解。因為工作上也會被問及，自己本來對於這個問題也不敢有甚麼樣的定見，但是被問多，對於這個問題也形成了一套自我邏輯，因此今天就來談談筆者個人對於自由軟體契約成立時點的看法。不過必須先說明的是，這個問題涉及諸多法條背景資訊，而電子報篇幅有限，因此筆者將不在正文裡說明這些法條背景資訊，而是直接進入問題核心討論，對於法條規定有興趣的讀者，請自行依照附註中的條號查閱我國的規定。

自由軟體基金會控告 Cisco 違反 GPL 授權
謝良奇/編譯    2008/12/16
自由軟體基金會 (Free Software Foundation) 在美國紐約區域法庭控告知名網路設備商 Cisco Systems，指控 Cisco 未將其 Linksys 路由器中使用的開放源碼軟體原始程式碼公開。Cisco 隨後發表聲明，表示該公司相當支持開放源碼軟體，並且相信該公司在實質上已經符合自由軟體基金會 GPL (General Public License) 的規定。對於自由軟體基金會指控該公司違反 GPL 並且提起訴訟的決定，Cisco 表示感到失望。

Sun 更新桌面虛擬化軟體 VirtualBox 至 2.1 版
謝良奇/編譯    2008/12/18

Sun Microsystems 日前更新開放源碼桌面虛擬化軟體 xVM VirtualBox。這次釋出的 xVM VirtualBox 2.1 版提供更好的 64-bit 支援、3D 加速、簡化的 Windows 與 Linux 網路、Mac 上的硬體虛擬化支援，以及包括快照 (snapshots) 在內的完整 VMDK/VHD 支援。

Sun 於 2008 年 2 月購併 Innotek 之後，開放源碼專案 VirtualBox 便獲得 Sun 的贊助。 xVM VirtualBox 軟體可以讓用戶產生虛擬機器，並安裝用戶選擇的作業系統。因此用戶得以在各種作業系統上使用喜愛的應用軟體，而開發人員也可以在一部筆記型或桌面電腦上，跨平台建立、測試與執行應用軟體。

Palm 新平台 Nova 作業系統可望於 CES 現身
謝良奇/編譯    2008/12/23

Palm 剛剛發表了 2009 會計年度第二季的營運報告。在一場與財務分析師的多方通話會議中，對於 Palm 下個月在 CES 的產品發表，將圍繞該公司以 Linux 為基礎的 Nova 作業系統，Palm 高層做出近乎肯定的證實。
