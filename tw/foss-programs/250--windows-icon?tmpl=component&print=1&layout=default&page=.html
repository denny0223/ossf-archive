製作 Windows 的圖示（Icon）

建立日期 2005-07-22 08:00      最近更新在 2011-01-11 18:23 作者是 吳國倫
相信大部分電腦使用者都知道什麼叫作圖示（英文：Icon）。在 Windows 的作業系統下，一個圖示可以用來代表系統裡的某一個檔案 或應用程式，使用者很快便能辨認出該檔案的用途或是該應用程式的功能。圖像化的表示方式是現代視窗式作業系統不可或缺的一部份。然而，當你設計及撰寫出一個新的 Windows 應用程式後，如何製作一個屬於自己的圖示？本期為大家介紹如何使用開放源碼工具來創作 出適合在 Windows 上使用的圖示檔案。 Windows 的圖示檔案都有（.ico）副檔名，你可以在 C:\Program Files 的資料夾下搜尋 *.ico 來找出很多範例。 如果你以為可以使 用一些圖片處理程式（如 Photoshop、GIMP、Illustrator 等）便能 建立新的圖示檔案，你會發覺它們並沒有把圖片儲存成為 ico 檔案 的功能，其原因是一個 ico 檔案實際上同時包含了超過一種圖案， 它會因作業系統的需要而顯示出大小不同或顏色深度不一的圖案。要 說明這一點，大家可以作一個簡單測試：首先開啟 Windows 檔案總 管，然後從功能列上點選「檢視」，並且改變其檢視模式為「清單」 或「圖示」。你會發現右邊視窗內的檔案圖示在清單模式中是小型的 圖示；然而在圖示模式中會是大型的圖示。

假設我們已經有了一個不錯的圖案，儲存成 Gif、JPG 或是 PNG 的 格式。 我們可以使用 GenIconXP 來產生 ico 檔案。 它是一個開放 源碼軟體，雖然最新版本是 0.2，但已經能夠把多種不同格式的圖案， 製作成為適合多種不同版本上的 Windows 系統下使用的 ico 圖示。 安裝 GenIconXP 前，必須先安裝 ImageMagick dll（同樣也是一套 開放源碼軟體），並且勾選它所有的安裝選項。 安裝後必須重新啟 動 Windows，才不會在使用 GenIconXP 的時候產生錯誤訊息。筆者 的過往經驗是先利用繪圖工具設計出自已喜好的圖案（不要大於 128x128 像素；以及可以使用透明的背景），儲存成 PNG 檔案，然 後再使用 GenIconXP 來產生 ico 檔案。

在許多的開放源碼軟體中，也有不少圖象處理或繪圖軟體。 例如 GIMP 便有類似 Adobe Photoshop 的功能，新的 Inkscape 也希望創 造出功能酷似 Adobe Illustrator 的 Vector 繪圖工具。 筆者不是 一位專業的美工人員，沒法詳盡的比較這些開放源碼工具是否能滿足 工作上的實際需要。 但有一點可以肯定的是，如果大家下一次需要 軟體來解決或輔助工作上的需要，很多適合的軟體可以從開放源碼的 龐大資源中取得。
