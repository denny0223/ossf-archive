撰寫專業說明文件的好用工具

建立日期 2005-07-06 08:00      最近更新在 2011-01-11 18:24 作者是 吳國倫

上一期介紹如何使用開放源碼軟體來製作出專業的軟體安裝程式，讓 使用者可以快速安裝新的軟體。 現在大部分的軟體裡（包括開放源碼軟體）都附有不同種類的說明文件如:使用手冊、安裝手冊、README、 授權文件等。這些文件通常採用純文字、HTML 或 PDF 的格式儲存， 因此方便、快速及容易使用的文件製作軟體是必需的，以下將介紹三 套開放源碼工具來幫助製作專業的說明文件。

Notepad++ 是一套純文字的編輯器，有完整的中文化介面及支援多國 語言撰寫的功能（採用萬國碼 UTF8 技術）。它的功能比 Windows 中的 Notepad（記事簿）強大，除了可以用來製作一般的純文字說明文件，也十分適合當作撰寫電腦程式（如 Java、C/C++、HTML、PHP 等）的編輯器。安裝時記得勾選「Context Menu」的安裝選項，以便 安裝後用滑鼠右擊 Windows 上任何一個檔案時，可以選擇「Notepad++」 來顯示及編輯檔案中的文字內容。 NVU 在今年的 6 月 29 日發佈了 1.0 的版本，可以在 moztw.org 下載安裝中文繁體的版本。它是一套所見即所得（WYSIWYG）的網頁 編輯器，適合在 Linux、Windows 或 Macintosh 作業系統下使用。由於 NVU 的使用介面簡單易用，以及可以快速增加表格、修改儲存 格屬性、插入圖片、拼字查尋、編輯 CSS 屬性、撰寫 Javascript 等等，因此十分適合非技術性的人士使用。 很多人會把 Adobe 和 PDF 聯想在一起。Adobe 是研發 PDF 技術的 公司，使用者可以下載它所免費提供的 Acrobat Reader 來檢閱 PDF 文件。在製作 PDF 文件方面，Adobe 主要售賣 Acrobat 來滿足企業 的需求，但是很多人卻不知道開放源碼有提供不錯的解決方案。 PDFCreator 即是一套適合在 Windows 上使用的 PDF 製作程式，它的操作方式有如使用一台印表機。當 PDFCreator 安裝完成後， Windows 上便會多出一台 PDFCreator 印表機。使用者可以使用一般 列印的方式，把文章列印，最後儲存成 PDF 檔案格式。筆者的使用 經驗是產出來的結果相當不錯，但如果文件內容有豐富的圖片，製作 出來的效果會比原來的版本有較大的差異。有興趣的人士，可以參考 OpenSourceInstall.org 內的安裝教學，除了能夠學習如何一步一步 地安裝 PDFCreator 外，更可以參考製作 PDF 檔案的簡短示範。 （本篇文章由百加資通公司提供，不代表鑄造場電子報立場。） 相關網站：
