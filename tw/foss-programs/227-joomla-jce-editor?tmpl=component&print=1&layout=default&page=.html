Joomla 的外掛模組介紹（一）JCE Editor

建立日期 2006-03-24 08:00      最近更新在 2011-10-12 11:13 作者是 林崇仁
如果一個出名的產品要更換名字，相信需要一段時間讓舊有使用者適應新的產品形像。然而針對去年獲得 Best Open Source Solution, Linux World 2005 的 Mambo 軟體，因開發團隊與 Mambo 原身的軟體公司發生意見衝突，從 Mambo 4.5.2.3 版本開始衍生出 Joomla 1.0 版本。 網路上很多提供 Mambo 技術支援或是相關服務的網站，有部份把名字改成 Mambo/Joomla，也有不少網站直接把名字換成 Joomla。 以筆者在過去的觀察，Joomla 開發團隊獲得不少支持，在短時間內便建立了他們在 Open Source 世界的名聲。

有關 Mambo 或 Joomla 的介紹已經出現不少，因此筆者想針對一些Joomla 的外掛模組為大家作出介紹。今期是 JCE Editor，它是一個適合在 Internet Explorer 或是 Firefox 執行的 WYSIWYG（所見即所得）的網頁編輯器。 雖然 Joomla 本身有提供了 Tinymce 的網頁編輯器，但缺乏了如檔案上傳（圖片、影音 AVI MPEG FLASH）的功能模組。JCE Editor 除了延伸了 Tinymce 的基本功能外，透過 JCE Editor 的系統管理元件 (Component) 可以設定 JCE Editor 的各種屬性、切換介面語言、開啟/關閉/新增/移除 外掛功能、功能按鈕的排位（在網頁上直接以拖拉方式來改變順序）等。

另外，我們習慣按滑鼠右鍵來設定物件的屬性，在 JCE Editor 內也可以照樣使用，大大提高了它的使用性。 在 JCE Editor 的官方網站，提供了不少外掛模組的下載，以及教你如何安裝的動畫教學。相較於 Joomla 現有的 HTML 編輯，相信它能幫助更多對 HTML 語法不熟悉的使用者。
