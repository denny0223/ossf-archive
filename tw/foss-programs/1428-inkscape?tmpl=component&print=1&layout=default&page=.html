強力的自由軟體繪圖工具－Inkscape

建立日期 2008-02-24 08:00      最近更新在 2012-01-06 11:58 作者是 張凱慶

Inkscape 是一款基於 GNU 通用公共授權（GPL）下的自由軟體，性質與功能上與 Illustrator、CorelDRAW 相似，採用 W3C 制定的 SVG 開放標準，SVG 為 Scalable Vector Graphics 的縮寫，意即可縮放式向量圖形。

概略來說，電腦所顯示出的圖分為點陣圖與向量圖。點陣圖是把圖用一格一格的顏色來表示，常見的螢幕解析度 1024×768，就是說螢幕橫向分成 1024 個光點，縱向則是 768 個光點。雖然說點陣圖是很常見的圖像格式，它的缺點是當我們想要放大的時候，當放大倍數超過原有的解析度，該圖便會出現鋸齒狀的痕跡，如果無限放大，點陣圖就會裂解成一格一格的顏色。

向量圖便是為了改善點陣圖的縮放問題而相應而生的技術，其定義基本的直線、曲線，然後將顯示的細部工作交由顯示設備來處理。也就是說，向量圖把圖用直線、曲線來表示，當放大縮小時，我們所見到的向量圖形都是緊密連續的直線和曲線，向量圖形不會因為縮放比例而造成圖形顯示上的差異。實際常見的應用有 Flash 動畫、TrueType 字型等。

與 Flash 和其他商業向量繪圖軟體最大的不同是，SVG 嚴格遵守可擴展標記語言（XML），使得圖形文件可讀，容易與其他技術互相融合，支援濾鏡及效果製作，並且能夠產生動畫，設計互動網頁等。Inkscape 採用 SVG 開放標準，現今的功能已經相當齊備，可媲美重量級的商業向量繪圖軟體，多國語言介面，跨平台可在 Windows、GNU/Linux、Mac OS X 等作業系統上執行，未來發展將支援完整的標準。

這篇關於 Inkscape 的教學文章，將從介面開始，然後說明向量繪圖中基本的繪圖工具，藉由繪圖工具繪製出複雜圖形中的元件，如何以這些元件組合成容易操控變化的向量圖形，最後蒐集整理一些網路上利用SVG或Inkscape的網站或教學資料，擴展讀者對用 SVG 或 Inkscape 的認識。
