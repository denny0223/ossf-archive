NVU 網頁編輯器使用經驗分享

建立日期 2006-06-09 08:00      最近更新在 2011-01-11 18:21 作者是 吳國倫
NVU（發音 N-View）是一個網頁的編輯工具，功能像如 Microsoft FrontPage 或是 Dreamweaver 的商業軟體。 它的軟體設計有幾個重點：(1)該軟體適合非技術人士使用，操作介面簡單。(2)連接遠端檔案系統，讓使用者可以直接存取在網頁伺服器上的網頁。(3)可 以加入其它網頁相關元素，如使用 javascript 產生的日期選擇小幫手、PHP 程式插入、CSS 編輯器等外掛功能。
