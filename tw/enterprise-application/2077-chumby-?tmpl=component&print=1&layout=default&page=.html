Chumby 新一代嵌入式產品的開發模式

建立日期 2009-05-20 08:00      最近更新在 2011-01-11 22:35 作者是 黃雪雁

◎ Chumby 採軟硬體均開放的商業模式，深獲消費者喜愛

自從開放源碼軟體開始被廣泛運用到商業產品上進行販售之後，連帶影響許多人也想將這樣的開放思維運用到硬體設計上，後續便有許多改採開放規格的硬體專案出現，早期較有名的如 OpenMoko 開放源碼手機計畫。

OpenMoko 將軟體內容及硬體規格都開放，讓使用者可以自行修改手機的軟硬體設定。這種軟硬體均開放的模式，也啟發了美國的 Chumby 公司，其將嵌入式電子產品以開放模式進行商業化，與過往的封閉開發模式相比，在銷售成績方面毫不遜色，而且在消費族群方面深獲美國青少年的青睞。


◎ 以提供 Web2.0 服務為設計理念

Chumby 是造型特別的無線網路個人資訊顯示器（PID），外型就像一台迷你小電視，有柔軟的皮質外觀讓人忍不住想捏它，設計輕巧、移動方便可當鬧鐘使用，並可用來播放網際網路上的音樂、相片、影片、網站內容，亦可接收網路廣播節目以及執行簡單的電腦遊戲。

Chumby 透過無線網路接收網路上的各種資訊，比如最新新聞、交通狀況、天氣預報、網路影音、電視節目以及 eBay 拍賣訊息等等。它甚至允許用戶可以自己編輯 Widget 來強化產品的功能，編輯者也可以透過網路平台與其他 Chumby 用戶分享這些 Widget. 目前在 Chumby 平台上可供下載的 Widget 已超過 1000 個，大半出自於消費者自發式的創意表現，這些 Widget 包括可以檢查 Facebook 朋友狀態的 Facebook Status、讓你追蹤及發布微網誌的 Twitter 和 FriendFeed Widget 以及能夠將電腦中的影片串流到 Chumby 上的 ORB Streamer 等。

因為 Chumby 提供讓使用者自行客製化產品並進行分享的 Web2.0 共工平台，除了成功的吸引年輕人購買它的產品，並且有效的鼓勵消費者自主的加入改善產品的貢獻族群，從結果來看、它不僅獲得了消費者的熱烈參與，同時在銷售數量方面也獲得了傲人的成就，可說是近期以 Web2.0 設計理念進行產銷的商業化典範。


◎ 與傳統封閉式開發不同的嶄新產銷模式

Chumby 最初在構想的時候，決定揚棄科技公司傳統的生產模式，改以開放創新的作法來生產 Chumby 這個嶄新作品。

一般科技公司的傳統生產模式，是由其行銷及業務團隊就市場及價格的角度提出產品的銷售原型，然後再多方評估產品的成本、品質與功能，經過反覆內部討論之後確定新產品的基本規劃。

接著再僱用高知名度的產品造型設計公司，針對不同的使用族群設計出造型不一的昂貴樣品，以這些樣本進行全國性的市場偏好度調查。

然而這樣的流程常常淪於形式化，而不能真正徵詢到商品買家的消費需求，最後的結果往往製造出許多不符合消費者真正需要的多餘產品。這是因為整個新產品的規劃、設計、測試與生產流程完全由該公司主導而沒有消費使用者主動且熱忱的參與協助。


◎ 藉由開放軟硬體吸取廣大使用社群的創意和智慧

Chumby 所採取的嶄新產銷模式，就是將其設計好雛型的新產品架構與大眾分享，並與關心電子科技產品的使用社群保持密切連繫，首先其以低成本的物料快速產出可供修改的一代產品，再將這個測試產品交給對使用 Chumby 有興趣的社群朋友，待社群朋友實際試用過後，發表對該產品的使用經驗、改善意見，以及新功能方面的需求建議。

透過這種方式，很多非專業人士的創意和智慧都可以匯入產品的開發流程，這些參與者其實都不是 Chumby 公司的聘僱員工，而是外界對於電子科技產品有實際使用興趣的玩家。

Chumby 公司透過提供測試產品的方式，順利收納了測試玩家對於產品的創意想法以及需求回報，因為好的科技產品該有哪些功能，光靠業者的自我想像並不充份，有時需要消費者的實際建議才能創造出真正符合需求的熱門商品，這一點、Chumby 做到了，它透過開放軟硬體的共享模式，成功的吸納了廣大使用社群的創意和智慧。

◎ 產品開發的過程獲得許多方面的支持

在 2006 年 8 月 O'Reilly 出版社所舉辦的 FOO Camp 活動裡，Chumby 設計公司將第一代的原型機發給 500 名關心電子科技產品的愛好社群，這個活動吸引了許多熱情的使用者開始試用 Chumby，後續更因此成立了 Chumby 的使用者論壇，讓試用者能持續地將產品的改善意見提供給 Chumby 公司。

隔年7月，Chumby 成功的生產出功能表現上更為成熟的二代機，並接續從社群朋友中選出 50 名較活躍的試用者進行第二階段的測試，這 50 名的試用者有的從產品的使用角度提供精闢的修改意見，甚至發揮創意自行修改原產品的軟硬體設定，以及手動增設原測試機所沒有的新功能。

50 名的進階參與者當中，不乏具程式開發能力、創意設計概念，以及有影像處理能力的人才，所以整體測試流程完成之後，Chumby 公司得到了第一手、豐碩、實用，而且具體的改善建議。

當 2008 年 2 月充滿眾人創意的 Chumby 正式上市進行販售之時，短短時間內馬上一躍成為美國青少年在耶誕節及生日宴會中最想收到的祝賀禮物。


◎ Chumby 的開放式開發模式值得國內業者參考及仿效

Chumby 有它優於同類型科技產品的獨賣特點，首先，它的造型可愛吸引人又方便移動，容易吸引消費者的目光。

再者，Chumby 的經營團隊不乏具有多年販售經驗的行銷專員，懂得如何妥善包裝將產品賣給最感興趣的消費族群。

最後，Chumby 執行長 Stephen Tomlin 於網路傳輸業擁有 15 年以上的投資經驗，其為 Chumby 籌措了穩定的開發基金以供市場開拓以及未來發展所需，亦為造就 Chumby 銷售成功的重要幕後推手。

由此可知，妥善的產品向性規劃、消費群眾的創意吸納、事先縝密的行銷策略布局、以及背後穩固的資金備援，均是產品銷售成功不可或缺的重要因素。

然而，Chumby 的銷售數字之所以能夠得到始無前例的高速增長，主要的關鍵因素仍在於開放程式原始碼及開放硬體規格的開發模式，這讓許多嵌入式電子產品的愛好者及使用社群，得以實際參與產品的開發及設計流程，從而催生出真正廣受消費者需求與歡迎的熱門產品。

Chumby 所運作的開放式創意吸納模式，由 Web 2.0 的群眾參與觀點分析之，是透過網路意見分享平台，鼓勵終端使用者主動地參與產品開發，讓產品的製造商能夠獲得消費者最真實直接的使用需求以及改善意見，這個分享平台能讓生產者與消費者熱烈討論、各抒己見，最後生產出符合雙方期待與需求的熱銷產品。

這樣嶄新的開發模式，過去也許難以想像，但未來靠著網際網路影響層面的持續發展，其所能吸納技術的前瞻性以及商機的未來性不可忽視，值得國內的軟硬體生產廠商善加思考，是否能夠將這樣的開發流程轉植至國內進行仿效，一同邁進 Web2.0 生產者與使用者創意分享、共工共榮的嶄新時代。
