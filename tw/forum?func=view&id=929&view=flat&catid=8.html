你好
首先想請問的是
貴網站的
使用者注意事項
以上皆屬於開源授權
若以此上述授權之軟體是否皆可帶至公司使用而無版權問題
(帶去公司僅使用無散佈與修改原程式碼之問題)

目前我有帶去公司使用的共有下列軟體與我覺得可能有爭議的部分請協助解答

首先為應該無爭議的部分，請幫我看一下是否完全無爭議

AUTOHOTKEY
網站最下方有寫上他的授權為GNU這個應該沒有問題，且寫好的腳本僅為自行使用，並無散佈之可能
Copyright © 2013-2014 AutoHotkey Foundation LLC - AutoHotkey Software License: GNU General Public License


===================================================
Greenshot
此軟體首頁亦有提到使用GPL授權
Greenshot was published under GPL, i.e. this software can be downloaded and used free of charge, even in a commercial environment.

===================================================
7-ZIP
License

7-Zip is open source software. Most of the source code is under the GNU LGPL license. The unRAR code is under a mixed license: GNU LGPL + unRAR restrictions. Check license information here: 7-Zip license.

You can use 7-Zip on any computer, including a computer in a commercial organization. You don't need to register or pay for 7-Zip.

===================================================
DM2
DM2 is free for use. Source is released under the terms of the GPL license.

===================================================
Notepad++
Notepad++ is a free (as in "free speech" and also as in "free beer" source code editor and Notepad replacement that supports several languages. Running in the MS Windows environment, its use is governed by GPL License.

===================================================
GIMP

Introduction to GIMP

GIMP is an acronym for GNU Image Manipulation Program. It is a freely distributed program for such tasks as photo retouching, image composition and image authoring.

It has many capabilities. It can be used as a simple paint program, an expert quality photo retouching program, an online batch processing system, a mass production image renderer, an image format converter, etc.

GIMP is expandable and extensible. It is designed to be augmented with plug-ins and extensions to do just about anything. The advanced scripting interface allows everything from the simplest task to the most complex image manipulation procedures to be easily scripted.

GIMP is written and developed under X11 on UNIX platforms. But basically the same code also runs on MS Windows and Mac OS X.

Features and Capabilities

There are some free program I wrote, if you have any question, feature suggestion or bug report, welcome to post comment on blog, or e-mail to me.
而且麻煩的是作者消失很久了

===================================================
PotPlayer
維基百科內有寫道
由於PotPlayer與KMPlayer同屬一個開發者的產品，因此無法擺脫侵害其GPL授權條款的身份。[3][4][5][6]然而大多數使用者只圖方便，對於這一點並不在意。但在最新版已有改善，安裝程式以安裝額外解碼器的方式來安裝受GPL保護的開源解碼器。
但我沒在官網找到他屬於何種授權，下方為維基百科連結
維基百科(PotPlayer)

===================================================
strokesplus
我找不到他的授權，只可以從首頁看到他好像是用"LUA"所寫的
於是我轉到了"LUA"的頁面
發現到"LUA"是以MIT授權
LUA
此授權是否會"感染"用其寫出來的程式?
使得其可於公司使用(不做修改原程式之程式碼與販售)

===================================================
思源黑體
抱歉我找不到他的官網，故用維基的取代
思源黑體（英語：Source Han Sans）是Adobe與Google所領導開發的開源字型家族，以Apache 2.0授權條款授權，屬於無襯線黑體。思源黑體於2014年7月16日首次發佈，支援繁體中文、簡體中文、日文及韓文，並且各有7種字體粗細。公開之時為當時涵蓋字元數量最多的字型，44,666個字元分屬於65,535個字形中，此為OpenType字型技術的極限。

其中使用了Apache 2.0授權條款授權是否就代表可以帶去公司使用呢?

===================================================
Everything
Everything License
授權內文並無提到為何種授權
但看起來像是不做修改軟體、散佈與販賣就可以無償使用?
這樣這個軟體是否可以在公司作使用呢?

===================================================
Paint.NET
Paint.NET License
他有寫到下列，所以公司應該是可以使用的
Can I use Paint.NET for business, commercial, school, or government use? Or is it only free for "personal" use?
Yes, you may use it for business, commercial, school, and government use -- the Paint.NET application is free for all of these purposes (and also for personal use). You may install it on as many systems as you'd like, although we do request and appreciate a donation to support our ongoing development costs. The main restriction is that you may not modify the Paint.NET software itself (e.g. change the name, add/remove features, "don't modify the DLL or EXE files", and as a corollary you may not distribute modified copies of Paint.NET.
但問題來了，他寫的是安裝，但公司因為系統權限問題，故我軟體都是使用原本的免安裝版，或他人做好的免安裝版
這樣會造成問題嗎?
