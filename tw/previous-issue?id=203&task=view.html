製作 Windows 的圖示（Icon）
吳國倫 2005/07/22
相信大部分電腦使用者都知道什麼叫作圖示（英文：Icon）。在 Windows 的作業系統下，一個圖示可以用來代表系統裡的某一個檔案 或應用程式，使用者很快便能辨認出該檔案的用途或是該應用程式的功能。圖像化的表示方式是現代視窗式作業系統不可或缺的一部份。然而，當你設計及撰寫出一個新的 Windows 應用程式後，如何製作一個屬於自己的圖示？本期為大家介紹如何使用開放源碼工具來創作 出適合在 Windows 上使用的圖示檔案。

Firefox 竄上 CNET.com 10 週年慶下載軟體排行
謝良奇 2005/07/12

CNET.com 為慶祝成立 10 週年，發表過去 10 年內的 10 大下載排 行，開放源碼瀏覽器 Firefox 高踞第 4 名。Spread Firefox 表示， 該網站所有人都應為此一成果深表自豪。 CNET.com 的 Download.com 於 1996 年成立，為早期下載網站，曾是 3000 個小型共享軟體、免費軟體的匯集地。該排行依次為 ICQ、Winamp、Napster、Firefox、WinZip、iTunes、Ad-aware、Skype、RealPlayer 與 Adobe Acrobat Reader。 為了進一步推動 Firefox 下載，Spread Firefox 還希望用戶相邀親朋好友，一起到 Download.com 發表評論提升 Firefox 人氣。

從 SCO 控告 IBM 一案看源碼的混入
葛冬梅 2005/07/22
2003 年 3 月美國一家電腦公司 SCO 控告電腦界的藍色巨人 IBM， 在未經 SCO 的授權下將 Unix 的源碼使用在 Linux 的程式核心中。 這項指控的基礎在於，SCO 擁有 Unix 的著作權，SCO 並未授權 IBM 可以將 Unix 的源碼用在 Linux 中，因此 SCO 向 IBM 提出控訴， 並要求賠償其所受到的損害。IBM 也於同年八月初向 SCO 提出反訴。 目前全案審理仍處於膠著狀態（註一）。
