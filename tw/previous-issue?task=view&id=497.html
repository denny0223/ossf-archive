GPL2的授權承繼性：以CMS程式與版型為例
葛冬梅    2009/03/25

最常見的 GPL2 授權拘束性討論，就是利用 GPL2 程式碼所因此產生的新程式是否必須採用 GPL2 授權，此外就是 Linux 系統上的應用程式是否也必須採用 GPL2，後者的討論因為 Linux Kernel 主要開發者兼精神領袖 Linus Torvalds 表態，寬鬆地允許應用程式可以不採用 GPL2 授權而塵埃落定。而筆者最近與一些朋友討論到另外一類的授權拘束性問題：GPL2 授權的 CMS（content management system） 程式，其上的版型（template）是否也必須採用 GPL2 授權。

令人驚豔的 Java Annotation
李貞臻    2009/03/27

Java 歷經十幾年的演化，已經是一個非常成熟的程式語言，但隨著時間的變化與 .NET 的挑戰，還有 Script Language 的簡單開發，都讓 Java 必須符合時代需求，因此，簡化開發最令人驚艷的方式就是 Java Annotation。

4/18~4/19 OSDC.TW 邀請技術愛好者一同來共襄盛舉！
簡信昌    2009/03/19

今年的 OSDC.TW 預計在4/18~19假台灣微軟盛大揭幕，是台灣第四屆開放源碼開發者研討會，目前已經吸引了超過兩百五十位的技術愛好者報名。每年持續快速成長的參加人數，可以發現台灣相關的程式開發或網站設計，對於開放源碼的使用量越來越大，也讓每年的 OSDC.TW 成為台灣開放源碼最重要的研討會之一。

 

這次的贊助廠商包括了 Yahoo!奇摩、台灣微軟、自由軟體鑄造場 (OSSF)、痞客邦 (Pixnet) 等等國內知名推動自由軟體的單位。微軟近幾年在某些技術開發的工具，積極與開放源碼的程式語言進行銜接，例如 Python，Perl 等等，因此今年主動贊助會議場地，期待能投入台灣的開放源碼開發者社群。
