Learning Z SQL Method with GadflyDBhot! 04/07/2008 Hits: 2996
作者：胡崇偉 ~ 於20080407撰寫
Z SQL Method 是 Zope / Plone 世界裡，連結關連式資料庫 (relational database) 的傳統方法，它具有操作介面一致的優點，已熟悉關連式資料庫的工程人員，很快就能上手存取既有資料。常見的 open source 關連式資料庫包括 MySQL 或 PostgreSQL 等，而 Gadfly 是 Zope 內建的關連式資料庫，無需額外安裝 database adapter (可視為 database 的驅動程式) 即可使用，其 SQL 語法易學易懂，非常適合在教學練習的場合。
