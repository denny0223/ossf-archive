我們正在改變世界
TonyBlk    2008/12/10

最近發現，我們正在改變這個世界，而且任何人都可以加入改變這個世界的行列。

兩年前誰會想到教育部的自由軟體的推廣光碟會一年壓了5萬片？誰會想到宜蘭縣政府開始全面導入 OpenOffice.org？ 誰會想到 ASUS 的 Eeepc 成功的帶領了 Netbook 的風潮？誰會想到一直都是 Wintel 不分家的 Intel 喊出了 Moblin 的口號？

是的，這些事情在兩年前講出來沒有人會相信，但是再這短短幾個月內都發生了。只要你是自由軟體的開發者、使用者、推廣者，你都是改變世界的一員，不會因為你只是一個小學生、不會因為你只是一個家管、不會因為你只是一個只會上網的 Netbook 的使用者，而有所不同。我們都可以一起加入改變世界的行列。

天阿～這是多麼令人振奮的事情，即使再全球經濟不景氣的現在，仍然覺得這世界是充滿希望的，因為自由軟體給大家一個平等的機會去使用無限的資源，因為自由軟體給大家平等的機會去享受資訊的樂趣。不會因為你買不起軟體而被歧視，不用因為學校教的軟體家裡買不起而違反法律的在家盜版使用、不用擔心沒有了微軟的 Office 套裝軟體而沒辦法開啟及編輯文件，還有太多太多的好處…。

如果你也想要改變世界，請加入自由軟體的行列，不管你是開發者、使用者或是 NOBODY，我們可以一起改變世界。

Yes ~ We can !!

把檔案抹除乾淨的Eraser （上）
Attila    2008/12/10

在 Windows 底下，千萬不要以為執行清理資源回收筒或刪除檔案的同時按住 Shift，就可以讓檔案完全從電腦中消失。要是徹底清除檔案是這麼容易的一件事，那麼救援軟體就沒有存在的價值了。

當然，要刪除的檔案，不見得都很重要，如果要刪除的只是一些無關痛癢的檔案，當然就不需要使用 Eraser 這一類的安全軟體，若檔案的內容牽涉到個人隱私、個人與企業的機密、網頁瀏覽紀錄、網路服務使用紀錄…等等，那麼還是要小心一點的好，以免洩漏出去後，為自己惹來無窮盡的麻煩。

因此，送修電腦、丟棄硬碟、遺失了隨身碟等，不希望洩漏了不該洩漏的祕密，那麼平常最好養成徹底抹除重要資料的習慣，此時此刻，Eraser 絕對會是一個必備的工具。因為 Eraser 不但提供有效的方法可以徹底抹除檔案，連相關的緩衝區（Buffer），都會一併抹除，絕對不留下任何「殘黨餘孽」！

7.1 複合資料型態

到目前為止我們已經看過五種型態：int、float、bool、NoneType 及 str。字串性質上不同於其他四個，因為他們是由更小的片段所構成的－字元。

由小片段組成的型態叫做複合資料型態。依據我們所做的事，我們可能會想要將複合資料型態視為單一的整體，或者我們也可能想要存取它的一部分。這種模稜兩可的情況是有用的。

Oracle、 Emulex 聯手貢獻資料完整性技術 Linux 核心用戶獲得更多資料保障
謝良奇/編譯    2008/12/04

Oracle 為 Linux 核心貢獻出與硬體公司 Emulex 聯手開發的資料完整性保護程式碼。根據其聲明指出，該程式碼可在資訊由應用軟體傳輸到資料庫，以及自 Linux 作業系統到磁碟儲存設備時，維護全面的資料完整性，有助降低錯誤資料被寫入磁碟的機會。

當用戶儲存檔案時，預期檔案總是會無誤地寫入磁碟中。然而，事實上檔案的資料很可能發生毀損。如今有了 Oracle 貢獻到主要核心的 I/O 資料完整性基礎架構程式碼， Linux 的用戶，比起其他作業系統，要多了一份保障。

Linux 散佈套件已相容美國國防部 IPv6 標準
謝良奇/編譯    2008/12/09

Linux 基金會 (Linux Foundation) 宣佈所有主要 Linux 散佈套件目前皆已支援 IPv6。該非營利組織表示，這些散佈套件商已符合美國國防部 (Department of Defense) 訂定的採購標準，並已可供部署於 IPv6 網路之中。

2005 年美國政府要求所有單位採購的電腦與網路設備，都必須符合下一代的網際網路協定需求。美國國防部為求加速進階到 IPv6 技術，將 2008 設定為完成此一程序的目標年度。主要軟體廠商為避免被排除於美國國防部採購案之外，必須實現規格需求，並驗證其產品符合國防資訊系統局 (Defense Information Systems Agency) 的 IPv6 特別相容運行認證 (IPv6 Special Interoperability Certification)。美國國防部並於今年 6 月強制實施此一認證。
