Subversion模組升級及新功能

建立日期 2008-10-16 17:04      作者是 ossfepaper

OpenFoundry將在台北時間 2008/10/22(星期三) 上午10:00 升級Subversion模組。

 

此次的升級有兩項任務。

1.讓專案可選擇subversion不開放給無權限的使用者來讀取資料，只有專案成員並且有給予權限者才可存取。

可在專案的基本資料中選擇＂版本控制系統＂做設定。

2.同時於此次將subversion從版本1.4.6升級至1.5.3版 這個 E-mail 地址已經被防止灌水惡意程式保護，您需要啟用 Java Script 才能觀看
