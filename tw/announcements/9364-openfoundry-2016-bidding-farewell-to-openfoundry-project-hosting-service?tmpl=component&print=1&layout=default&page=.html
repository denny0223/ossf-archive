[OpenFoundry]自由軟體鑄造場專案託管服務將於 2016 年底終止 / Bidding farewell to OpenFoundry project hosting service

建立日期 2016-08-17 13:36      最近更新在 2016-08-17 13:39 作者是 webmaster

[Please see the English version below]

各位使用者您好，2016 年 12 月 1 日將停止以下專案管理的服務：  


  • 使用者帳號服務
  • 版本控制服務，包含 Subversion, Git, CVS
  • 待辦事項
  • 通訊論壇
  • FTP 上傳

此外，您在版本控制系統中的程式碼及資料，將會被打包成壓縮檔並存放在伺服器上。我們建議您儘速將版本控制系統中的資料轉移至 GitHub 等其他開放源碼託管平台上。  

 

感謝您一直以來對我們的支持。 

 

OSSF 團隊敬上 2016.08.17  

 

 

Dear OpenFoundry Users,  

We will be shutting down the OpenFoundry Project Hosting service about 3 months from now on Dec. 1, 2016.

The following service listed here will not be available till then:


  • User Account
  • Version Control System (Subversion, Git, CVS)
  • Issue Tracker
  • Forum
  • FTP

Besides that, all the files and data you submitted in the Version Control System will be archived and well stored in the OpenFoundry server. However we strongly recommend that you migrate these materials off OpenFoundry to other source code hosting platforms such as GitHub quickly.  

 

We thank you for the support of staying with us all the time.  

 

Sincerely,  

OSSF Team https://www.openfoundry.org 2016.08.17
