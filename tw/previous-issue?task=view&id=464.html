有效率的管理帳號與密碼 - KeePass Password Safe
許耕榜    2009/02/03

拜科技發達之賜，以帳號密碼作為身份認證的方式越來越常見，舉凡作業系統登入到各大大小小的網站、論壇、網路銀行，無不用到帳號密碼。

但帳號與密碼的管理一直是件很困擾人的事，早期我習慣用同一組密碼通吃所有帳號，但後來發現有時候會用 Email 去申請一些名不見經傳的小論壇或服務，這時如果還用同一組密碼，等於把 Email 的帳號與密碼同時告知對方，但如果臨時起意建立其他密碼，這個密碼也很快就忘了。

何苦砸錢拋售到手的點閱率？
朝陽科技大學資管系 洪朝貴老師    2009/02/03
網路時代的經濟模式已經變成 attention economy。用白話文來說，點閱率是珍貴的資產。對成功的媒體及網路公司來說，點閱率商品，可以賣錢的。不過這帖的標題有點不合邏輯：既是 「拋售」 (就算是「賤售」或「賠售」)，應該是交出商品，收回金錢，卻為何又說是「砸錢」 呢？我也不懂。但的確有公司做這樣的事：辛苦地提高自己的點閱率，卻又砸錢請人將已經到手的珍貴點閱率送給微軟。我真的辭窮，不知該用「買」還是「賣」來形容這種奇特的商業行為。奇特，但是並不少見。事實上，在臺灣，這種花錢自殘的行為到處可見。新聞媒體朋友們，請您千萬不可忽略這篇；觀光旅遊業、外貿業、還有任何與外國客戶有來往的企業/機關/組織/學校，您也是權益最可能因此大受影響的單位。

KDE 4.2 正式釋出 加強 Plasma 桌面與跨平台相容性
謝良奇/編譯    2009/01/27

KDE 4.2 日前正式釋出，此一新版開放源碼桌面環境包含更為豐富完整的 Plasma shell，改進 3D 視覺效果支援。如果要問 Linux 桌面是否已經準備好推向主流用戶，最新 KDE 桌面背後的開發者可能會給予肯定的答案。

開放源碼 KDE 桌面環境背後的開發者已宣佈正式釋出 4.2 版。在 4.0 版本釋出後一年推出的此一最新版本，帶來更為優雅與完整的使用者體驗。KDE 4.2 是 KDE 4.x 桌面第二次主要釋出，其代號為 "The Answer"，透露出此一版本所設定的目標。此一釋出奠基於首次於 KDE 4.0 版問世的若干功能。

微軟釋出 Web Sandbox 源碼 提高 JavaScript 安全性
謝良奇/編譯    2009/02/03

微軟宣佈了將 Web Sandbox 加以開放源碼的計畫，該項專案是一套虛擬化技術，用以解決JavaScript 的安全問題。微軟將在 Apache Software License (ASL) 釋出 Web Sandbox 程式碼，而非微軟自己的共享源碼授權。

微軟宣佈將採行常用的開放源碼授權 ASL，釋出其 Web Sandbox 專案的程式碼。此舉反映微軟對以互通性標準為基礎的 Web 技術貢獻，其關注持續增長，此外也展現該公司願意在自家專案上，採用為大眾所接受的開放源碼授權。
