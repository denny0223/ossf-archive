思科推出開放式網路環境迎戰 OpenFlow

建立日期 2012-07-06 10:44      最近更新在 2012-07-23 15:26 作者是 Kenneth Mak

◎ 本文原載 Linuxpilot 網站，原文章連結

展示最新網路技術的 Interop Tokyo 2012，在會場一角首次設置了名為 OpenFlow ShowCase 的展區，首次將全球不同廠商的 OpenFlow 產品作相互的兼容測試。面對 OpenFlow 來勢洶洶，思科最近終於推出開放式網路環境 (Open Network Environment) 還擊。

一直以來思科雄霸全球的網路設備市場，但由於採用的是思科自家的制式，難以與其他品牌的網路設備兼容。為此數年前業界開始積極推廣開放網路標準 OpenFlow，現時 Juniper、IBM、HP、NTT 等廠商都推出了使用 OpenFlow 的網路產品。以往各家廠商都會在各自的攤位中展示自家的 OpenFlow 產品， OpenFlow ShowCase 則是全球首次將不同廠商的 OpenFlow 網路產品連起來的大規模測試。

面對來自 OpenFlow 的挑戰，思科終於推出 Cisco ONE (Open Network Environment) 還擊。Cisco ONE 將提供工具套裝 One Platform Kit，為開發人員於思科不同的作業系統包括 IOS、IOS-XR 和 NX-OS 提供應用程式界面 (API)。並為軟體定義網絡 (Software-Defined Networking, SDN) 研究提供概念驗證控制器軟體及概念驗證的 OpenFlow 代理程式。

另外思科亦推出了 Nexus 1000V 虛擬交換器，為多租戶雲端部署提供可擴展虛擬覆蓋網路支援，包括 OpenStack 支援、可程式化、多虛擬機管理程式 (multi-hypervisor) 能力，以及虛擬擴展區域網 (VXLAN) 網關功能，產品將於 2012 年第 4 季開始提供試用版和分階段推出市場。OpenFlow 迫使思科改變以往採用自家封閉網路技術的策略，標誌著開放網路標準的勝利。2012 年底將會是網路設備戰國時代的開始，且看合縱連橫勝者誰屬。
