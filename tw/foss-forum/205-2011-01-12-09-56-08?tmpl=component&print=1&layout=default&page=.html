現在連電腦軟體都搞全配這一套

建立日期 2005-06-24 08:00      最近更新在 2011-01-12 17:56 作者是 陳貴成
前一陣子電視上有一個廣告非常的轟動，他問說：沒有預裝 Microsoft Office 的 Notebook，你還能作什麼？這真是大哉問阿！我在想這是 Microsoft 給全民的有獎徵答嗎？如果是的話，那我真是要好好的給 他投投稿了，真搞不懂 Microsoft 是不是就真的把台灣的電腦使用 者給給看扁了？如果真是這樣，我還是跳出來回答一下 Microsoft 的問題好了，免得 Microsoft 真的以為他把台灣的相關單位採購決 策者跟硬體廠商搞定之後台灣的電腦使用者就都一定要用他的產品了。 ◎ 天阿！現在居然連電腦軟體都搞全配這一套

各位看倌，想想鴻海、廣達、宏碁、華碩、微星、技嘉、英業達、趨 勢……等等的世界級電腦軟硬體相關廠商，再想想台灣的網路普及率， 若把台灣稱之為資訊大國，這應該是沒有人會反對的吧！有趣的是， 在這個泱泱大國中個人如果想要買沒有預載 Microsoft 作業系統的 Notebook，其難度之高，不會亞於在上海看到中華民國的國旗，若是 想要買一台全新而且沒有預載任何一種作業系統的 Notebook，其難 度就更是跟讓大陸領導人喊中華民國萬歲一樣。

尤其有趣的是，我們自稱是民主國家，結果連買台 Notebook 都如此 不民主，好像規定只要花錢買 Notebook 的硬體一定要買微軟的作業 系統一樣，然而全中華民國的消保官也沒有人去處理這種壟斷行為， 天阿，資訊大國的消費者居然沒有軟體的購買選擇權！這樣已經很悲 慘了，或許是中華民國的消保官被看扁了，現在居然連 Microsoft 的 Office 軟體都要借由 Microsoft 的壟斷優勢強迫推銷給消費者，我 就不知道政府養這些消保官要作什麼，唯一的好處是，在台灣罵政府 不會被抓去關，但是政府也不一定會檢討就是了。

◎ 告訴你「沒有 Microsoft 的 Office 你的 Notebook 能作什麼」

呵呵，不知道會不會有許多人被 Microsoft 跟許多的硬體廠商所作 的這則聯合廣告給嚇到了，其實沒有 Microsoft 的 Notebook 還是 可以做很多事的。因為 Notebook 又不是只可以作文書處理，你可以 打電動、可以用來寫網頁、可以用來作你想作的任何的事情……，當 然你需要有能力可以選擇你想要安裝的軟體，因為本篇文章第一次出 現的地方是開放鑄造場電子報，相信大部分的讀者都已經知道如何去 尋找跟處理自己所需要的軟體了。其實若是自由作家或是 SOHO 要用 文書處理的工具其實也不一定要用到 Microsoft Office吧！這些相 關的資訊，其實都可以在下列的網站中找到相關的資訊，請各位朋友 們多多造訪，多多指教。

（本專欄文章僅代表個人意見，不代表鑄造場電子報立場。）
