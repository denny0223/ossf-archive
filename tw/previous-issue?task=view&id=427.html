開放原力
Freedom..0
Jedi 2008/06/14
「Freedom..0」要念成「Freedom 二點零」，是 Web 2.0 時代的關鍵自由。

自由軟體基金會為自由軟體訂定了四種屬於軟體使用者的自由：

第零自由（Freedom 0）：為任何目的執行軟體的自由。
第一自由（Freedom 1）：學習程式如何運作，並將其原理納入自己作品的自由。此自由建立在能取用源碼的前提上。
第二自由（Freedom 2）：重新散布副本的自由。
第三自由（Freedom 3）：改善程式、並將此改進釋出予公眾，使社群獲益的自由。此自由建立在能取用源碼的前提上。

自由軟體平台標準化研討會
李貞臻 2008/06/12
  • 活動時間： 97年6月20日(五) 10：00- 12：00
  • 活動地點： 台北市八德路三段2號地下1 樓 B101 會議室
  • 報名網站： https://www.tca.org.tw/seminar/D10f00221.asp

在國際 Linux 基金會領導下，已經建立了完善的 Linux Standard Base(LSB) 自由軟體平台標準規範及驗證體系。主要的 Linux 發行版皆已遵循 LSB 標準，為應用軟體及週邊提供了更好的 Linux 發展環境。

台北市電腦公會承辦經濟部標準檢驗局「自由軟體平台標準研訂與推廣」專案，除了研擬中文標準外，亦受邀參加美國 Linux 基金會於 2008 年4月舉辦的 Linux 合作高峰會，並蒐集最新的 Linux 相關進展。本研討會將介紹自由軟體平台標準化，並分享參加 Linux 合作高峰會之心得。歡迎 Linux 研究開發者參加，共同討論相關議題。
