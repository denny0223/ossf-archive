十一月份自由軟體技術分享工作坊，即日起開放報名!
陳飛亨    2009/11/05

十一月底在台北與高雄，自由軟體鑄造場分別規劃了三場不同類型的分享工作坊。第一場技術分享工作坊邀請到素有台灣開放源碼 Java 教父之稱的李日貴，移師到南台灣的高雄，跟當地的朋友分享令人驚豔的 Java Annatation；而兩個月固定舉辦的社群技術分享工作坊，則是透過社群朋友引薦講師 gasolin ，他將帶領參與學員瞭解近年最受關注的開放原始碼手機/手持設備作業系統 Android；第三場的技術充電站，則由服務於台灣網通設備代工廠的林智斌先生，跟大家分析 OpenWRT 的商業應用與其開發的實務整合經驗。希望透過活動的舉辦，鼓勵國內更多朋友加入自由軟體的研究開發世界。

報名網址資訊如下：

● 11/21(六) 自由軟體技術分享工作坊-Java Annotation 的時代  (高雄場次)
● 11/22(日) KaLUG 社群技術分享工作坊-Android 應用程式設計入門  (高雄場次)
● 11/28(六) 自由軟體技術充電站-OpenWrt (台北場次)

即日起開放報名，席次有限，歡迎有興趣的朋友踴躍參加 !!

美國白宮網站翻新 改採開放源碼技術
謝良奇/編譯    2009/10/27

美國白宮網站最近的翻新動作在技術圈引起不小的騷動，因為，這個熟悉網際網路的當政團隊，採用的是開放源碼。這表示白宮網站的程式將受到公眾檢視，提供公開使用與編輯修改。

競選時運用網際網路工具，對於線上生態並不陌生的 Obama 團隊，日前對白宮網站 Whitehouse.gov 軟體堆疊做了更動，改而採用開放源碼的 Drupal 內容管理系統與 Apache Solr 搜尋伺服器。根據 Obama 政府的說法，改變的主要原因之一，是為了對 WhiteHouse.gov 打造更多協同合作的功能。Drupal 除了提供內容管理外，還支援部落格、評論留言、投票以及建立使用者檔案的功能。

美國防部釋出人力資源系統程式碼 新備忘錄支持開放源碼
謝良奇/編譯    2009/11/03

開放源碼應用於政府與國防已經不是新鮮事，甚至可以說政府單位就是開放源碼最大的目標市場之一。然而，最近美國國防部表態強力支持開放源碼，以及進一步從使用者變成源碼貢獻者，釋出企業人力資源應用軟體程式碼，仍然讓開放源碼社群或支持者感到振奮。

這並非美國國防部第一次公開釋出程式碼。6 月時，Georgia Tech Research Institute 為美國國防部開發的地圖應用軟體 FalconView，就曾經以開放源碼方式釋出。簡單來說，美國國防部如今不只是開放源碼使用者，也是貢獻者，可以期待未來看到更多美國國防部贊助的開放源碼軟體出現。

4. 基於四相交握通訊協定的非同步控制器最佳化技術

專案摘要：本計畫我們針對四相式爆發/延伸爆發模式控制器，提出一個歸零區間移除技術，我們命名為"四相式爆發/延伸爆發模式控制訊號重組"。在不改變原控制器的主要控制順序，透過本計畫所提出的歸零區間移除技術，最佳化處理後的控制器將能有效剔除歸零區間所造成的效能損耗。

5. 運用資料隱藏技術於照片定位認證之行動地圖日誌

專案摘要：在現今資訊發展迅速的時代裡，照片分享與管理服務已頗為盛行，像是現行遠傳相片分享服務、Locr 照片分享網頁、地圖日記網頁…等，連帶興起了一種結合網路地圖(Google Map) 和相片管理的服務叫做 GeoTagging 也備受使用者歡迎，其實想法很簡單： 替相片加上 GPS 的座標在網路地圖上直接看照片。但由於缺乏檢驗的機制，檔案中 EXIF 資料也很容易篡改變造等問題，造成網頁照片與上傳地點不符，管理與瀏覽不易。本研究便以現今最熱門的 GeoTagging 來做為研究的目標；目前多數新上市的 PDA 手機都具有 GPS 定位及照相之功能，未來這兩項功能也會越來越普及；本研究中我們提出一個行動地圖日誌的架構，在行動端開發一套軟體，可在照相時，將這類手機所取得的 GPS 定位資訊及相關的驗證資料藏入所拍攝到的照片中，以確保拍攝地點與照片內容之一致性。使用者可將已嵌入之照片上傳至照片管理網站。照片上傳後，網站將抽出所嵌入之定位資訊，若定位資料格式正確，責允許上傳並連同定位資訊等建入資料庫中，否則不允許上傳。此網站結合  google map 界面讓使用者搜尋瀏覽與分享照片，定位資訊可供查詢照片使用。透過所提出之架構，將可改善現行的各著名地圖日誌網站許多照片內容與 GPS 資訊不一致之混亂情形(例如將士林夜的照片放到阿里山的地圖位置上)，同時所完成地圖日誌中的照片與定位資訊可以作為遊客「到此一遊」的見證。本年度國科會計畫編號為 NSC98-2220-E-309-001-
