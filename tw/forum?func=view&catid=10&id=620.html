Hi syman888，

以下簡單回覆你的問題。

一、關於Xming

(1)從SourceForge上(https://sourceforge.net/projects/xming/files/)下載到純粹軟體的部份，是依其個別元件的自由軟體授權條款來運作；(2)從Xming的專題網站(https://www.straightrunning.com/XmingNotes/)下載的技術文件與程式，則依特別向其恰談商業化的授權方式。

這個專案的運作方式較為複雜：(1)基礎的元件有MIT、BSD授權的軟體，但(2)Xming專題網站的架設者將這些元件重新整理，也加工使其最佳化，所以從Xming專題網站所下載的程式元件與技術文件，其要求商業使用者必須另外向其取得授權，而非商業使用者，亦僅只有一年的使用期間，並且須要做某程度的捐獻動作。

這些聲明公布於此：https://www.straightrunning.com/XmingNotes/terms.php#head-14

1. Xming installers and components are Public Domain if put on [WWW]SourceForge, by me. You can do what you like with Public Domain releases including freely distributing them; they still retain all original authors' licenses appropriately, but I have relinquished my rights over them. Xming專案個別元件的授權說明，則可參加右列連結：https://www.straightrunning.com/licenses/

小結論是、Xming這個專案並不完全符合你所提出，完全免費在商業公司內部使用的期待。

二、關於XWinLogon Windows X Server

(1)可以在商業公司內部使用；(2)再散布的話須要提供程式原始碼。

這個專案比較單純，就同你所找的資料，XWinLogon Windows X Server(https://www.calcmaster.net/visual-c++/xwinlogon/)這個專案是一個橋接Windows使用環境到Unix/Linux Ｓerver的Interface，這個Interface是用GNU GPL來授權，而其使用的Cygwin(https://www.cygwin.com/)也是以GNU GPL來授權的，這代表你可以將這些程式拿來做不限目的的利用，但是如果再散布程式時，就肩負提供後手程式原始碼的義務。

所以依你的規劃，純粹在公司內部使用而不涉及產品製作以及再散布，是可以的。

希望以上的回覆對你有所幫助。
