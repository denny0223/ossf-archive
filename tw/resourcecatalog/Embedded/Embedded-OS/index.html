ART-Linux

ART-Linux是一個硬即時(hard real-time)作業系統核心，旨在機器人應用程式的開發。即時(real-time)不需要特殊驅動裝置，即可自使用者層級直接存取。

適用平台:All POSIX (Linux/BSD/UNIX-like OSes), Linux ;原始碼授權類型: GNU General Public License (GPL)

資料來源: https://art-linux.sourceforge.net/

Pinguy OS

Pinguy OS是一個創新運作的作業系統，適用於所有對象，不僅限於行家或技客們。

適用平台:不適用 ;原始碼授權類型: GNU General Public License (GPL)

資料來源: https://pinguy-os.sourceforge.net/

LGTV embedded OS (douglas)

這是由LG電子提供，以47LG70 LCD HDTV為對象，滿足GPL驗證的官方源碼副本。

適用平台:不適用 ;原始碼授權類型: GNU General Public License (GPL)

資料來源: https://douglas.sourceforge.net/

Osnofian Embedded Linux (x86)

OsnofianLinux是一組小型作業系統(distributions)，旨在載入1.44Mb軟碟機(floppy disk)，CDROM啟動磁區(CDROM boot sector)，硬碟驅動器(hard drive)或隨身碟(USB stick)。

適用平台:不適用 ;原始碼授權類型: GNU General Public License (GPL)

資料來源: https://osnofianlinux.sourceforge.net/

Femto OS

Femto OS 是一個非常簡約、可攜且優先排程的作業系統，適用於記憶體(RAM)與快閃記憶體(FLASH)極小的嵌入式裝置，最低限度可至16K flash，1K ram。主要適用對象為AVR architecture，例如ATtiny861/461/261系列。

適用平台:不適用 ;原始碼授權類型: GNU General Public License (GPL)

資料來源: https://www.femtoos.org/

µOS++

µOS++是一個開放源碼、使用免費的即時(real-time)作業系統，適用對象為以ARM Cortex M3為核心的嵌入式系統或AVR裝置，具有優先排程的性能。

適用平台:不適用 ;原始碼授權類型: MIT License

資料來源: https://sourceforge.net/projects/micro-os-plus/

HybridOS

HybridOS是基於 Linux 核心及飛漫公司產品的軟體平台和作業系統。該作業系統有別於 Google Android，是飛漫軟體專為中低端嵌入式設備打造的作業系統。 如同汽車有混合動力技術一樣，飛漫軟體發展HybridOS之目的，就是為了融合多種主流的嵌入式應用開發技術（如 C/C++、Java、瀏覽器 Widget），為嵌入式設備提供一個統一且具有靈活性的和便於二次開發等方面獲得良好平衡的作業系統平台。

適用平台:All POSIX (Linux/BSD/UNIX-like OSes) ;原始碼授權類型: GNU General Public License (GPL)

資料來源: https://sourceforge.net/projects/hybirdos/

Punix

Punix是一個與Unix相仿的作業系統，適用對象為以MC6800中央處理器作基礎的T1繪圖計算機系列(始於TI-92)。此系統目前仍在開發中，尚未能廣泛應用，以下是系統核心的主要特色：

1.先佔式多工(Preemptive multitasking )

2.虛擬終端(Virtual terminal on the console)

3.可經由I/O 連接埠連上其他電腦或計算機

4.可經由連接埠播放音效

適用平台:不適用 ;原始碼授權類型: BSD License, GNU General Public License (GPL), GNU Library or Lesser General Public License (LGPL)

資料來源: https://punix-os.blogspot.com/

Prex Embedded Real-time OS

Prex Embedded Real-time OS是一個嵌入式可移植的即時作業系統，具備可靠性、低耗能的特性，採用C語言開發、微核心設計的概念。文件系統、process以及網路部份則透過通過user mode mission來提供。

適用平台:不適用 ;原始碼授權類型: BSD License

資料來源: https://prex.sourceforge.net/

AstLinux

AstLinux是一個開源PBX軟體Asterisk的Linux發行版。雖然核心焦點仍然是Asterisk，不過其他VoIP軟體如OpenSIPS和FreeSWITCH也可以用。AstLinux有許多新功能，是嵌入式和商業Asterisk的理想solution。

適用平台:不適用 ;原始碼授權類型: GNU General Public License (GPL)

資料來源: https://www.astlinux.org/

FlowScreen OS

FlowScreen OS是一個以FlowScreen Router為主要對象的軟體系統，能執行高效能的路由器系統(1.5M pps網路流量)，採開放式體系結構以開發第三方網路應用。

適用平台:uClinux ;原始碼授權類型:GNU General Public License (GPL)

資料來源: https://sourceforge.net/apps/trac/fsos/

Linux on Toradex Colibri PXA270 Module

Vanilla Kernel僅支援嵌入式快閃記憶體(flash)與DM9000乙太控制器(Ethernet controller)，本方案將支援範圍擴大至Toradex、Orchild等的周邊，或者其他嵌入式應用。

適用平台:Linux  ;原始碼授權類型:GNU General Public License (GPL)

資料來源: https://sourceforge.net/projects/tocolinux/

FunkOS

FunkOS是一種可攜式的即時作業系統，適用於低資源的處理器，例如Atmel的AVR微控制器，德州儀器MSP430等等。

適用平台:不適用 ;原始碼授權類型:Sleepycat License

資料來源: https://funkos.sourceforge.net/

TinyOS 8051 working group

TinyOS 8051 working group的主要特色如下：

跨平台支援8051-variants

支援多重編譯器

適用Windows、MacOS X與Linux的開發環境

適用平台:不適用 ;原始碼授權類型: BSD License

資料來源: https://tinyos8051wg.sourceforge.net/

FreeOSEK RTOS

FreeOSEK是一個以嵌入式系統為對象，可延展的解決方法，建基於OSEK RTOS的規格之上。

適用平台: All BSD Platforms   (FreeBSD/NetBSD/OpenBSD/Apple Mac OS X),All POSIX (Linux/BSD/UNIX-like OSes),Cygwin (MS Windows);原始碼授權類型: GNU General Public License version 3.0 (GPLv3), GNU General Public License with Classpath exception (Classpath::License)

資料來源: https://opensek.sourceforge.net/

MaRTE OS

MaRTE OS是一個以嵌入式應用設備為主的硬即時(hard real-time)作業系統，提供了一個易用的受控環境以開發多重即時應用程式(Real-Time applications)。

適用平台:不適用 ;原始碼授權類型:GNU General Public License 2 (GPL 2)

資料來源: https://marte.unican.es/

DSP/BIOS Real-Time Kernel

DSP/BIOS kernel是一個可延展的即時(real-time)多工(multi-tasking)作業核心，以符合TMS320C6000™、TMS320C5000™與TMS320C28x™ DSP平台的規格要求而設計的，更多細節可參考網頁的介紹。

適用平台:Linux, Windows ;原始碼授權類型: TSPA-licensed

資料來源: https://focus.ti.com/docs/toolsw/folders/print/dspbios.html

ChibiOS/RT

ChibiOS/RT is designed for deeply embedded real time applications where execution efficiency and tiny code size are desiderable features. The system is implemented as a library to be linked with the application code, the OS and the application code together form the image to be loaded on the target device.

RTEMS(Real Time Executive for Multiprocessor Systems)

RTEMS(Real Time Executive for Multiprocessor Systems)，是一個為嵌入式系統而設計的自由的開源即時作業系統。 它最早用於美國國防系統，早期的名稱為即時導彈系統（Real Time Executive for Missile Systems），後來改名為即時軍用系統（Real Time Executive for Military Systems），現在由OAR公司負責版本的升級與維護。目前無論是航空、軍事，還是一般領域RTEMS都有著極為廣泛的應用。RTEMS是在GPL授權條款下分發，但它允許將RTEMS的目的檔鏈結至其它檔案而不需將整個檔案以GPL授權覆蓋，這方面有點像LGPL。

適用平台:不適用 ;原始碼授權類型: Primary RTEMS License which covers most of the source code in RTEMS、TCP/IP Stack License 、RPC/XDR License 、GoAhead Webserver License (requires graphic to be displayed on website hosted) 、Stanford University Permissive License

資料來源: https://www.rtems.com/

QNX Operating System

QNX是一種遵從POSIX規範的類Unix即時操作系統，目標市場主要是嵌入式系統。它可能是最成功的微內核操作系統之一。且能讓Linux/UNIX程式方便地移植到QNX系統上來，擴展了QNX系統的可用資源。QNX開放性還表現在網路聯結性上。QNX不僅有QNX機器之間專用網路，還支援與不同類型之間網路通訊的協定，如TCP/IP等的各種協定。

適用平台:不適用 ;原始碼授權類型:無相關資料

資料來源: https://zh.wikipedia.org/zh-hant/QNX
