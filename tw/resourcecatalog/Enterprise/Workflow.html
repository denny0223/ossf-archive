webERP

webERP是個完整的網路會計/ERP 系統,僅僅要求一個網路瀏覽器和pdf 閱讀器. 它廣泛適用於許多商業,尤其是分散式商務的批發與分配. 它是作為一個開源軟體被開發並開放下載. 其功能隨著新的商業形勢與開發者持續擴充.

適用平台：Linux ;原始碼授權類型: GNU GPL

資料來源: https://www.weberp.org/HomePage

Imixs Workflow

Imixs開放源碼的設立是為了促進已開放軟體為基礎的工作流程技術的發展。在範圍內的各種技術和方法可以支援被開放或流程導向的工作流程系統的發展。該計劃包括制定一個框架來創建工作流程系統，以及實現 JEE的標準。此外，Imixs也發展創造和管理圖形編輯器的WorkflowModels Eclipse框架，還有客戶端框架，已建立用戶者界面。

適用平台： Linux;原始碼授權類型: GNU GPL

資料來源: https://www.imixs.org/

Opentap

Opentaps Open Source ERP + CRM是一個帶來更效率的管理的頂級開源專案完整整合應用程式包

適用平台：Linux ;原始碼授權類型: GPL

資料來源: https://www.opentaps.org/

ADempiere ERP

Adempiere專案創建於2006年九月, 在一場與ComPiere Inc. 長久的爭議之後(Compiere的開發團隊與其相關之社群). 社群認為Compiere Inc.放入過多的側重於開源的封鎖/專案的商業性質, 不同於社群分享/強化專案的初衷,  在一場討論以後決定離開Compiere創建Adempiere.

適用平台：Linux ;原始碼授權類型: GPL

資料來源: https://www.adempiere.com/index.php/ADempiere_ERP

JAllInOne

JAllInOne是一個ERP/CRM 基於SOA,有著Swing UI的應用程式.它管理單件,倉庫, 銷售, ＰＯＳ, 供應, 訂單追蹤, BOM, 製造, 會計, 呼叫, 議程, 文件, 多語言/企業/貨幣.

適用平台：Linux ;原始碼授權類型: GPL

資料來源: https://jallinone.sourceforge.net/

OpenERP

OpenERP 於2002年創始於比利時，此開源專案已歷經十年的持續開發，並已發展出其配套的雙重授權商業模式，除了商業授權的服務加值版本外，其不同的方案的程式源碼分別以GPL-3.0、AGPL-3.0，以及自訂的OpenERP Web Client License授權釋出。OpenERP提供了企業管理上的必要解決方案，目前已佈署超過60個國家，300多個合作夥伴。由於國內產業環境以中小企業為主，在內部管理系統上，如果能採用具有持續授權延展性與客製化自主性的自由開源軟體專案，將可減輕企業主的負擔，進而客製化獨特的管理模組來增加企業競爭力，故評估為具有未來發展性的重要專案。

JFire

JFire is entirely free/open-source software, uses the latest technologies (EJB 3, JDO 2, Eclipse RCP 3.3) and is designed to be highly customizable.

JFire是一個完全免費/開源的軟體, 使用最新的科技(EJB 3, JDO 2, Eclipse RCP 3.3) 並被設計成高度客製化

適用平台: Windows, Linux, Mac OS X;原始碼授權類型: GNU LGPL

資料來源: https://www.jfire.org/modules/content/

xTuple

免費開源ERP, 會計, CRM程式包設計給中小企業.

適用平台: Windows, Linux, Mac OS X;原始碼授權類型: GNU LGPL

資料來源: https://www.xtuple.com/solutions/enterprise

Openbravo

Openbravo 提供世界領先的網頁開源ERP, 利用讓現代ERP給全世界使用來改善世界的效率

適用平台: Linux, Windows, Mac OS X, FreeBSD, Solaris;原始碼授權類型: Openbravo Public License (OBPL)

資料來源: https://www.openbravo.com/product/erp/

Intalio BPMS

Intalio BPMS 是世界上最被廣泛部屬的商業流程管理系統(BPMS). 設計給Intalio所開發的Eclipse BPMN Modeler, Apache ODE BPEL engine, 和 Tempo WS-Human Task service, 它可以支援任何程序, 大或小.

適用平台: Linux, Windows, Solaris;原始碼授權類型: 請參考以下網址: https://community.intalio.com/faq/dispQlay-2.html#FAQ36

資料來源: https://www.intalio.com/bpms

ProcessMaker

ProcessMaker是第一個真正開源的商務程序管理(BPM)和流程軟體。它是設計給中小型企業(SMBs)以及組織。

適用平台： Linux;原始碼授權類型: Affero GNU Public License

資料來源: https://www.processmaker.com/

Uengine

uEngine BPM是一個2003年註冊於SourceForge.net至今的開源商業流程管理系統。 uEngine BPM 套件分為三個主要要件: uEngine BPM Foundation（含模型化工具與流程引擎), uEngine Process Portal (含儀表與單一登錄功能), uEngine BP Analyzer（OLAP為基礎之流程分析工具),uEngine BMP foundation的建構借助了在其他優異的開源軟體如 Liferay Enterprise Portal, Mondrian OLAP Server, JBoss Drools BRE 和 Apache Axis II。

適用平台： Windows;原始碼授權類型: GNU GPL

資料來源: https://www.uengine.org/web/uengineen/test

Galaxia

Galaxia 是一套以Openflow為基礎的工作流程系統開放源代碼。它原始是為了 Tiki CMS/Groupware撰寫。它曾經被匯入許多非開放源的專案。Galaxia是一個活動為基礎的工作流程應用程式，許多程序和工作流程組成動態活動，為了完成一個事件必須被完成。與此相反，一套流程系統的每一步都必須按照順序完成，並且這個應用程式的功能會更加清晰。

適用平台： Linux;原始碼授權類型: LGPL

資料來源: https://workflow.tikiwiki.org/tiki-index.php

Wilos

Wilos是一個編排程式的軟體。它讓使用者可以用一套從Eclipse Process Framework Composer匯出的XML檔案裡頭的程序工程，以管理一個專案。

適用平台： Linux;原始碼授權類型: GNU GPL

資料來源: https://sourceforge.net/projects/wilos/

Imixs

Imixs開放源碼的設立是為了促進已開放軟體為基礎的工作流程技術的發展。在範圍內的各種技術和方法可以支援被開放或流程導向的工作流程系統的發展。該計劃包括制定一個框架來創建工作流程系統，以及實現 JEE的標準。此外，Imixs也發展創造和管理圖形編輯器的WorkflowModels Eclipse框架，還有客戶端框架，已建立用戶者界面。

Zebra

Zebra是一個工作流程引擎─原先是開發用來彌補一些商用和開源工作流程引擎的不足。它與其他工作流程系統的主要不同是

可以模型化用工作流程規律敘述的所有工作流程

有一個簡意好用的GUI設計工具

介面化的持續層

物件導向設計

Turbine網路應用的整合元件

適用平台： Linux;原始碼授權類型: GNU GPL

資料來源: https://zebra.berlios.de/

OpenWFE

Ruote是一個用Ruby寫成的工作流程引擎。Ruote不是一個狀態機函式庫。它可以被描述成一個商業流程用的作業系統。Route引擎可以同時執行多個程序。程序被以Ruby DSL或XML(或直接JSON)寫成的流程定義啟始。流程定義描述參與者之間的工作流程。參與者可以是使用者、一群使用者、服務、legacy systems等等。

適用平台： Linux;原始碼授權類型: MIT

資料來源: https://ruote.rubyforge.org/configuration.html

OFBiz

Apache Open For Business Project是個Apache2.0授權的開源企業自動化軟體專案。開源企業自動化指的是開源ERP, CRM, E-Business/E-Commerce, SCM, MRP, CMMS/EAM等等。

適用平台： Windows, Linux;原始碼授權類型: Apache

資料來源: https://ofbiz.apache.org/

Taverna

Taverna是一套用於設計和執行工作流程的開源工具。

適用平台： Windows, Mac OS X, Linux;原始碼授權類型: LGPL

資料來源: https://www.taverna.org.uk/

Bonita

Bonita開源解決方案結合三個解決方案於一：創新流程設計studio、強大BPM引擎和突破性終端使用者介面。

適用平台： Linux;原始碼授權類型: GNU GPL

資料來源: https://www.bonitasoft.com/
