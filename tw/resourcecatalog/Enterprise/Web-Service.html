Apache Axis2

Apache Axis2 is a toolkit for creating and using Web Services, including SOAP, MTOM, XML/HTTP and advanced WS-* standards such as WSRM and WSSecurity. Axis2 includes a very fast runtime engine, together with tooling support for WSDL and WS-Policy, and plugin support for WS-Addressing, WS-ReliableMessaging, WS-Security, WS-Eventing, WS-Transactions, WS-Trust and WS-SecureConversation. Axis2 runs either standalone or hosted in Tomcat or other servlet containers.

Open Source FUSE

FUSE 服務框架是個彈性的開源SOAP和RESTful服務平台基於Apache CXF 並產品化和支援用於企業SOA基礎建設.

適用平台： Linux, Windows, Mac OS X;原始碼授權類型: Apache

資料來源: https://fusesource.com/products/enterprise-cxf/

Open Source OSGi ESB

FUSE ESB 4 帶有 JBI, OSGi and BPEL 支援是個開源整合軟體平台基於Apache ServiceMix4 並且完全基於標準, 產品化, 和支援企業SOA基礎建設支使用.

適用平台： Linux, Windows, Mac OS X;原始碼授權類型: Apache

資料來源: https://fusesource.com/products/enterprise-servicemix/

Compass

Compass框架是提供搜尋引擎文法到應用程式堆中的一流開源JAVA框架。建構在優異的Lucene搜尋引擎，Compass無縫整合到普及的開發框架如Hibernate, Spring。它提供搜尋功能給你的應用程式資料模型並與資料源同步。有Compass:寫更少代碼,更快找到資料。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://www.compass-project.org/

Cofax

Cofax是一個以網頁為底的文字和多媒體內容發布系統。它設計用來簡化報紙在網頁上的顯示及促進即時的網頁發布。

適用平台： Windows;原始碼授權類型: GNU LGPL

資料來源: https://www.cofax.org/content/cofax/home/

DotGNU

DotGNU致力於成為網路服務與C#軟體的自由軟體解決方案提供者和產業領導者，就如同GNU/Linux目前快速在桌面與伺服器應用領域成長的地位一樣。

適用平台： Windows, Linux;原始碼授權類型: GNU GPL

資料來源: https://dotgnu.org/

Apache Tomcat

Apache Tomcat是用在JAVA Servlet官方執行參考和JavaServer網頁技術的servlet container。這個Java Servlet和JavaServer頁面規格，是由Java Community Process之下的Sun所研發的。

適用平台： Linux;原始碼授權類型: Apahce

資料來源: https://tomcat.apache.org/

Apache CXF

Apache CXF是一個開源服務框架 CXF助你建造和開放運用frontend programming API像JAX-WS和JAX-RS。這些服務可能用到不同的協定如SOAP, XML/HTTP, RESTful, 和CORBA 並依循不同的傳輸 如HTTP, JMS 或JBL。CFX包含一組極ˋ廣的功能表但主要聚焦於網路服務標準支援, Frontend, 易用性, 和二進位與老舊協定的支援。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://cxf.apache.org/

 

Apache JUDDI

JUDDI是一個以UDDI v3為規範的網路服務JAVA實作，是一套開放源代碼。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://juddi.apache.org/

Apache Muse

這套Apache Muse專案，是以JAVA為基礎來實行的WSRF,WSN,WSDM規範。它是一個讓使用者不用實行所有前面提及的規範，所建立的網路服務介面的頁框。以Muse建立的應用程式，可以同時部署在Apache Axis2 和OSGi環境下，並且這套專案包含了一組命令列工具，它可以產生依自行安排分配的適當人為結果。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://ws.apache.org/muse/

Apache Tuscany

Tuscany社群是用來建立一個可以簡化網路服務和處理被貼在SOA上實質商業問題的耐用且易用的基本架構。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://tuscany.apache.org/

Apache Synapse

Apache Synapse是一款始於Apache並被設計成簡易，輕量，高表現的企業服務匯流排(ESB)。基於一個微小異步的核心，Apache Synapse具有對XML和其他網路服務的絕佳支援，同時Synapse也支援二進位與文本格式。Synapse引擎被設定作一個簡易XML格式並有一組可用傳輸與中間層。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://synapse.apache.org/

Crispy

這個project的目的是使用一個非常簡易使用的API來呼叫不同的服務。Crispy的目的在於提供一個可以遠端使用許多不同傳輸的單一入口點，像是RMI，EJB，JAX-RPC，XML-RPC。它利用許多特性去設置一個可以遠端控制API的服務管理員。這些遠端的呼叫都是一些簡易的JAVA物件，舉例來說，可以將Crispy

SOA或RCP結合，並且這樣的通訊方式，可以很容易地修改其屬性。

適用平台： Linux;原始碼授權類型: GNU GPL

資料來源: https://crispy.sourceforge.net/

Gomba

Gomba是個JAVA服務程序聚集，能讓開發者加速開發RESTful網頁服務也讓資料存取連接到JDBC API後再到網頁。Gomba能夠利用中間層的動態網站，來當做企業聯合組織化的系統或者資料庫實習報告的工具。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://daubau.it/gomba/

Metro

Metro是個高效能，擴張性高，且簡易使用的網頁服務庫。它能一次滿足你對網頁服務的所有需求，從最簡單的hello world到依賴性，安全性，交易性甚至包含.NET服務的網頁服務都能滿足你的需要。

適用平台： Linux;原始碼授權類型: CDDL 1.1 + GNU GPL

資料來源: https://metro.java.net/

XINS

Xins是支援POX-RPC, SOAP，和XML-RPC以及更多的網頁服務技術。主要包含XML基本規格和JAVA實做框架，可產生HTML， WSDL，客戶端和伺服器端的程式碼和測試檔。

適用平台： Windows, Linux;原始碼授權類型: BSD

資料來源: https://xins.sourceforge.net/

Hessian binary web service protocol

Hessian是一個輕便的binary RPC通訊協定。這套 Hessian binary 網頁服務通訊協定使網路服務不需大量的頁框也可以很方便使用。因為它是一個binary通訊協定，它可以不需要附加擴大的通訊協定，就很方便地送binary資料。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://hessian.caucho.com/

WSIF

WSIF是一個無論任何實做和平台皆能產生網頁服務的簡易JAVA應用軟體 。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://ws.apache.org/wsif/

Axis

Axis是第三代Apache SOAP(其始於IBM並被稱為SOAP4J)，它包含了

一套簡易又獨立的伺服器 像Tomcat一樣，是一個連結了servlet engines的伺服器

WSDL有額外的支援

有工具可從WSDL生成Java classes

一些範例程式檔

有一套監控TCP/IP封包的工具

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://ws.apache.org/axis/

Apache ODE

Apache ODE執行遵循WS-BPEL規範寫成的程序(processes)，它會通知網頁服務，傳送及接收訊息，處理資料的控制與依照你的程序定義來回復錯誤。另外，它同時支援了執行中的長或短程序(process)，以編排所有應用程式的服務。

適用平台： Linux;原始碼授權類型: Apache

資料來源: https://ode.apache.org/
