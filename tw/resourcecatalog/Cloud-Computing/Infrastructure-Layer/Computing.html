OpenStack

OpenStack 是 IaaS(基礎設施即服務)軟體，採用 Apache-2.0 授權釋出，任何人都可以下載其程式源碼自行建立和提供雲端運算服務。此外，OpenStack也用作建立防火牆內的「私有雲(Private Cloud)」，提供機構或企業內各部門共享資源，此一開放平台已獲得多數跨國性指標公司的支持，並經國內工研院、資策會、中華電信等重要法人/組織的支援而續行引進。

Amazon Elastic Compute Cloud ( Amazon EC2 ) Featured

亞馬遜彈性雲端運算(英语：Amazon Elastic Compute Cloud，Amazon EC2) ，由亞馬遜公司提供的web服務，是一個讓使用者可以租用雲端電腦運行所需應用的系統。EC2藉由提供web服務的方式讓使用者可以彈性地運行自己的Amazon機器映像檔，使用者將可以在這個虛擬機器上運行任何自己想要的軟體或應用程式。 使用者可以隨時創建、執行、終止自己的虛擬伺服器，使用多少時間算多少錢，也因此這個系統是"彈性"使用的。EC2讓使用者可以控制執行虛擬伺服器 的主機地理位置，這可以讓延遲還有備援性最高。例如，為了讓系統維護時間最短，用戶可以在每個時區都運行自己的虛擬伺服器。Amazon.com以Amazon Web Services (AWS)的品牌提供EC2的服務。
以上介紹取自 維基百科: https://zh.wikipedia.org/wiki/Amazon_EC2

Enomaly

Enomaly的Elastic Computing Platform (ECP)讓IT人員能很輕易的將企業內的Data Center與外部的Data Center做整合，讓你在一個管理介面中就能夠管理內外的資源，你也可以很輕易將某個VM自一個Data Center移到另一個去.

資料來源:https://www.enomaly.com/

Nimbus

Nimbus 是open source 針對 IaaS 的工具組, 提供 建立私有雲或是社群雲的服務.

資料來源：https://www.nimbusproject.org/

FlexiScale

flexiScale 提供虛擬主機服務, 允許使用者在線上使用租用的方式建立他們自己的虛擬機器.

資料來源：https://www.flexiant.com/products/flexiscale/

GoGrid

GoGrid 提供虛擬主機服務, 使用者採取自動租用的方式建立他們自己的虛擬機器.

資料來源：https://www.gogrid.com/

Eucalyptus

Eucalyptus 以 IaaS 型態，提供企業與使用者建立屬於自己私有雲的架設系統，採用 GPL-3.0 授權釋出(https://github.com/eucalyptus/)，此專案於美國已進行大規模的商業化利用並進行相關服務，其於 2012 年第三輪融資中募得 3000 萬美元的營運資金，相關的技術說明文件齊全，重點特色在於相容於 Amazon EC2 與 S3 標準，對於中小企業的雲端資訊服務系統建置具有相當誘因，所以未來商業應用延展性獲得保障，並能吸引投資者的目光。 

OpenNebula

OpenNebula為資料中心虛擬化和雲端解決方案的引擎；OpenNEbula 是開放原始碼的虛擬基礎設備引擎，用來動態佈署虛擬機器在一群實體資源上，OpenNEbula最大的特色在於將虛擬平台從單一實體機器到一群實體資源；可將一群實體 cluster 轉換彈性的虛擬基礎設備，且可動態調適 service workload 的改變， OpenNebula 在 service 和實體機處設備間產生新的虛擬層 ，這個 layer 可支援叢集的 service 執行和加強 VMMs的效益。

資料來源:https://trac.nchc.org.tw/grid/wiki/GTD_Cloud_One_intr

RightScale

RightScale 是管理多個線上雲端服務系統的網站, 提供延伸及易用的雲端系統管理平台.

資料來源： https://www.rightscale.com/
