Slitaz Aircrack-ng Distribution

Slitaz Aircrack-ng Distribution 是基於 Slitaz cooking 的版本、Aircrack-ng SVN 版本，以及修正無線網路的驅動程式，以利封包的竄改。目前定制化版本是針對 Acer Aspire One 筆記型電腦，但仍然可以適用於其它的環境，包括虛擬環境。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://www.slitaz.org/

Matriux

Matriux 是一款全功能的安全性分析組，包括了一堆強大、開源和免費的工具。可用於各種用途，包括但不限於滲透測試、系統與網路管理、網路蒐證調察、安全性分析、弱點分析等。目前 Matriux 仍在開發中，而且第一個測試版本已經可以下載，未來團隊將持續更新與維護。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://www.matriux.com/

Pentoo

Pentoo 是一款支援 Live CD 或 Live USB 設計的滲透測試和安全評估工具組。以 Gentoo Linux 為基礎，包含 Nessus 和 Metasploit 等滲透測試和安全評估軟體。視窗管理介面使用 Englightenment，且針對 Pentium III 架構特定優化。本專案的 QEMU 使用 virt-manager，所以可以移植到虛擬環境中。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://trac.pentoo.ch/

Frhack OS

Frhack OS 是一款基於 BackTrack 4 的更新及修改版本。本專案更新了一堆程式，包括 spoonwep-wpa, fakeap, airgraph-ng, airoscript, WEPcrack, ssidsnif, aphopper, fierce, wfuzz, zulu, airsnort, mdk3, wepbuster, bluemaho, bluescan, bluesnarfer, ghettotooth.pl, bt_audit, fatback, pasco, unhide, memdump, allin, tor, privoxy, ophcrack, vncrack, TFTP-bruteforce, DNSBruteforce, kalgecin, fuzzgrind, origami, MetaScan.rb, complemento and middler 等。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://www.securinfos.info/frhack/

Frenzy

Frenzy 是一款 "可攜式系統管理工具組"，以 FreeBSD OS 為基礎，允許管理員從系統啟動到所有系統功能的軟體調試、網路測試與分析，以及硬體測試等。目前本專案支援兩個不同的模式：(1)標準版：特別針對系統管理員，ISO 大小僅為 200MB，包含了 500 多個應用程式；(2)擴展版：針對進階的系統管理員及高級使用者，ISO 為 250 MB，包含了 600 多個應用程式。

適用平台: FreeBSD;原始碼授權類型: New BSD License

資料來源: https://frenzy.org.ua/eng/

OWASP Live CD

目前 OWASP Live CD 是基於 OWASP Live CD 2007 版本的更新。此外，維護者建立了一連串的論壇和操作手冊。目前，OWASP Live CD 可以支援虛擬環境，如 VMWare，所以可以製作成 VM 及 Live USB，並執行在 Asus Eee PC 上。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://www.owasp.org/index.php/Category:OWASP_Live_CD_Project

Samurai Web Testing Framework

Samurai Web Testing Framework 是一款基於 Linux 的 Live CD 環境，已包含事先的配置，目的是為了滲透環境的測試。本專案包含了最好的開源與免費工具，重點是測試和攻擊網站。為了這樣的環境需求，在開始偵察前，可以使用 Fierce 和 Maltego 的工具。然後，可以使用 WebScarab 和 ratproxy。到了最後階段，可以使用 BeEF, AJAXShell 等工具。以上這些功能都包含在專案中。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://samurai.inguardians.com/

Network Security Toolkit

本專案的主要目的是提供一個完備的開放源碼網路及系統安全工具組。大部分搜集的工具是 insecure.org 公佈的前 100 個著名安全工具。本專案提供先進的操作介面，含系統管理、導覽、自動化和配置等設定。亦可安裝於虛擬系統中，方便在大型網路架構中進行部署。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://networksecuritytoolkit.org/nst/index.html

DEFT Linux

DEFT (Digital Evidence & Forensic Toolkit) 是一款基於 Kubuntu Live CD 定製化的套件組。因為基於 Kubunto 所以這是一個非常容易使用的系統，包括優先的硬體環境檢測，和最佳的開源應用程式組，以及包含本專案新增的系統與網路安全應用程式。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://code.google.com/p/deftlinux/

MultiISO LiveDVD

MultiISO LiveDVD 是一款整合時下非常流行的 Live CD ISO 技術，這些 Live CD ISO 都可以從網路上取得。MultiISO 可以用於安全性偵察、弱點分析、滲透測試，系統還原、多媒體中心等。這是一個 all-in-one 的 Live DVD，所以對於每個人來說，裡面絕對有每個人所需要的東西。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://badfoo.net/

Ophcrack

Ophcrack是一個使用Rainbow table來破解視窗作業系統下的LAN Manager雜湊的程式，它是基於GPL下發佈的開放原始碼程式。它接受不同格式的雜湊，包括視窗作業系統的SAM檔案。使用作者免費提供的Rainbow table，可以在短至幾秒內破解最多14個英文字母的密碼，有99.9%的成功率。包含更多字母（包括符號和空格）的Rainbow table可以在Objectif Securité購買。從2.3版開始可以破解 NT 雜湊，這功能對已經關閉 LAN Manager 雜湊的系統（Windows Vista 的預設設定）或是長於14個字母的密碼特別有用。 它另有一個自生系統的版本，可以自動破解視窗作業系統的密碼。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://zh.wikipedia.org/zh-tw/Ophcrack

Protech

與 nUbuntu 同樣是基於 Ubuntu。這是由 techm4sters 團隊打造的 Protech。Protech 的特點是非常輕巧，視窗管理使用 Fluxbox，除了 Live CD 外還可安裝在電腦中。雖然看起來與 nUbuntu 類似，但不同的是，為了使操作更易於使用，團隊在工具選擇上特別講究，所以靈活性與易用性變成了本專案的特點。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://ubuntulook.com/ubuntu-based-distributions/protech/

Kcpentrix

Kcpentrix 是基於 SLAX 5 (一款 Slackware Live DVD) 基礎客製而來。Kcpentrix 使用了強大的模組化能力，使得我們能夠更輕集地客製化屬於自己的版本。Kcpentrix 2.0 更是革命性的創舉，它切換到 Linux 2.6 的核心，也將 Zisofs 改為 squashfs 的壓縮，以提供更快更小的壓縮檔。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://kcpentrix.com/

DamnVulnerableLinux

每個發行版的套件都希望安裝最新的應用程式，但 DamnVulnerableLinux 不一樣，它反其道而行，故意將內建的應用程式使用有著大量的弱點的版本，應用程式的安裝配置也故意地留下了錯誤，因此存在著大量的安全問題，這可以稱為有史以來最容易被入侵的系統。本專案的目的是為了讓學生學習計算機安全課程。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://www.damnvulnerablelinux.org/

BackTrack

BackTrack 是基於 SLAX (一款 Slackware Live DVD) 可開機光盤客製而來的。它包含了一套系統與網路的安全取證工具。本專案其實是融合 Auditor Security Linux 和 WHAX (先前的 Whoppix) 而創立的。到了 BackTrack 4 以後，改用了 Ubuntu 為基礎客製。BackTrack 是一種專門用來破解無線網路密碼的工具，可以用來破解 WEP 的加解密，甚至 WPA 加解密的方式亦可以順利破解，但前提是需要足夠多的字典檔。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://www.backtrack-linux.org/

Operator

Operator 是一款完整的 Linux (基於 Debian Linux) 的發行版。可獨立開機成功，並將己身置於記憶體中運行，而不需安裝。Operator 包含了一堆開源與免費的系統與網路安全審查工具，可用於監測需求。此外，還包了資料還原與環境取證的工具。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://www.ussysadmin.com/operator/

The Penguin Sleuth Kit

The Penguin Sleuth Kit 是一個可啟動 CD 和 VMware 的虛擬環境平台。這是取用了社群中廣大的開源與免費工具，包括非常有用的工具，以執行系統與網路取證分析和安全審查。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://sourceforge.net/projects/psk/

nUbuntu

nUbuntu 的主要目標是創建一個基於 Ubuntu 的安全發行版，其中加入了許多安全測試工具，而且還將不必要的應用程式給予移除。使用方法同於一般 Ubuntu 的操作，所以易於擴充與維護。

適用平台: Linux;原始碼授權類型: GNU Genearal Public License v2

資料來源: https://nubuntu.org/
