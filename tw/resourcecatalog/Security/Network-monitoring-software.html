什麼是資源表列?

OpenFoundry 蒐集與自由軟體相關的連結資訊並將其分類，供您依此資源目錄架構，循序深入自由軟體豐富的應用世界。若此分類下發現有任何問題及建議，非常歡迎您協助我們改進，請寄至contact@openfoundry.org。

網路監控軟體

提供監控主機、服務、應用程式以及網路狀態。

Listings

結果 1 - 14 共 14

raven-monitoring

Raven-monitoring 是一款程式執行狀態與設備監控服務的工具，可以監測網路伺服器的行為。本專案提供開放源碼的服務檢測工具，而且為了方便操作，並不覆加難以配置的安裝流程。

適用平台: Windows/Linux/BSD;原始碼授權類型: Apache License 2.0

資料來源: https://code.google.com/p/raven-monitoring/

armazd

Armazd 是一款軟體式系統，主要設計來監控網路元件所使用。本專案可以區分為兩部份: 實體與邏輯。實體部份包含路由機、交換器或主機等。邏輯上包括系統上的服務，如 SOA 或網頁伺服器等。

適用平台: Linux/BSD;原始碼授權類型: GNU General Public License v2

資料來源: https://code.google.com/p/armazd/

pymon

Pymon 是一款開放源碼的網路及流程監控解決方案，並使用 Python 程式語言撰寫而成。本專案無論是界面操作或是安裝置配都可以很容易的快速建置與部署。

適用平台: Windows/Linux/BSD/MacOS;原始碼授權類型: New BSD License

資料來源: https://code.google.com/p/pymon/

nscand

Nscand 全名是 Network Scanning Daemon Web Interface，是一款基於網頁型式的監控工具。本專案支持網路掃描及系統服務狀態檢查等。

適用平台: Windows/Linux/BSD/MacOS;原始碼授權類型: GNU General Public License v3

資料來源: https://code.google.com/p/n

phpMonitoring

phpMonitoring 是 100% 基於 PHP 程式語言撰寫而成的，後端採用 MySQL 資料庫。本專案的主要目標是可以快速建立一套易於使用的網頁式監控平台。第二個目標是方便製作出基於 PHP 開發出來的任何套件，以擴展其功能。

適用平台: Windows/Linux/BSD/MacOS;原始碼授權類型: GNU Lesser General Public License v2.1

資料來源: https://code.google.com/p/phpmonitoring/

keywatch

Keywatch 是一款自由的開放源碼軟體的網路監控系統平台，提供了簡易容易上手的安裝方式。本專案伺服器主要用 Java 程式語言來運行，加上前端以 Ajax 來搭配，所以是一套基於標準的網頁型態監控系統。

適用平台: Windows/Linux/BSD;原始碼授權類型: Apache License 2.0

資料來源: https://code.google.com/p/keywatch/

Smokeping

Smokeping 是一款測試延遲性的評估工具。本專案得以送出網路封包至網路端或主機端，並且測量其處理的速度。從這些數據中可以得知目標是否正常運行，且運行的效率為何。

適用平台: Linux/BSD;原始碼授權類型: GNU General Public License v2

資料來源: https://oss.oetiker.ch/smokeping/

Open-AudIT

Open-AudIT 是一個應用程式層級的網路狀態偵測系統。本專案可以讓您管理網路上主機的配置情形、以及它的變化通知，對象主機支援 Windows 及 Linux 平台等。

適用平台: Linux/BSD;原始碼授權類型: GNU General Public License v2

資料來源: https://www.open-audit.org/

OpenNMS

OpenNMS 是世界上第一款基於開源碼的企業級網路管理平台。本專案除了開放源碼專案外，亦提供商業級的支援。

適用平台: Linux/BSD;原始碼授權類型: GNU General Public License v2

資料來源: https://www.opennms.org/

Sysmon

Sysmon 是一款網路監控管理工具，主要目的是提供高性能且穩定的網路監控服務。目前支持的協議包括 SMTP, IMAP, HTTP, TCP, UDP, NNTP 及 ICMP 等。

適用平台: Linux/BSD;原始碼授權類型: Public Domain

資料來源: https://www.sysmon.org/

Mon

Mon 是一款通知與測試目標服務是否正常運作的工具，並且可以針對特定狀態回報訊息給系統管理者。服務的定義可以是一個正常回傳的字串，或僅是一個 ICMP 回傳封包，因此可以依系統回傳的不同型態來定義服務目前的狀態行為。

適用平台: Linux/BSD;原始碼授權類型: GNU General Public License v2

資料來源: https://mon.wiki.kernel.org/index.php/Main_Page

Munin

Munin 意指 "memory"。Munin 工具可以輔助你調查目前網路伺服器的狀態，並且記下其目前狀態。此外本專案還提供網頁介面瀏覽與操作。

適用平台: Linux/BSD;原始碼授權類型: GNU General Public License v2

資料來源: https://munin-monitoring.org/

Zabbix

Zabbix 是一款企業級開放源碼分佈式監控的解決方案。本專案採用彈性的機制，允許使用者定義不同的警示媒介，例如 Email 或其它型式，此方式得以讓系統管理員快速的收到問題的主機及情形通知。

適用平台: Windows/Linux/BSD;原始碼授權類型: GNU General Public License v2

資料來源: https://www.zabbix.com/

Nagios

Nagios 是一款非常流程的開放源碼伺服器與網路健康狀態的監控平台。本專案可以定期偵測服務的狀態，並且在狀態異常時提供通知功能。為了擴展 Nagios 的功能，可以利用本專案的模組 API 來設計出自己客製化的延伸套件。

適用平台: Linux/BSD;原始碼授權類型: GNU General Public License v2

資料來源: https://www.nagios.org/
