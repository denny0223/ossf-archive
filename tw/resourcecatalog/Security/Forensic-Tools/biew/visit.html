biew Binary file viewer and editor

Maintainer: danfe@FreeBSD.org
Port Added: unknown
Last Update: 2017-06-25 01:29:09

License: GPLv2
Description:
BIEW is multiplatform portable viewer of binary files with built-in editor with binary, hexadecimal, and disassembler modes. It uses native Intel syntax for disassembly and offers many useful features such as highlighting for AVR/Java/x86-AMD64/ARM-XScale/PPC-64 code, Russian codepage converter, full preview of formats MZ, NE, PE, NLM, COFF32, ELF (and partially a.out, LE, LX, PharLap), code navigator, and much more. WWW: http://beye.sourceforge.net/
