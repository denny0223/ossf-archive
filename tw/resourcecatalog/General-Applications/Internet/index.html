Evolution

Evolution 是 Ubuntu預設的郵件管理軟體。它親切的介面和熟悉的功能會讓人聯想到 Windows 下的郵件管理軟體︰Outlook。Evolution 具備了郵件、行事曆、通訊錄和工作排程的功能，方便使用者管理自己繁雜的事務。另外，它也提供了 Windows 的版本。

smb4k

smb4k 是一個在 KDE 桌面環境下執行的網路共享軟體，它就好像 Windows 下的 "網路上的芳鄰" 一樣，可以在不同電腦之間做檔案的網路傳輸。不論辦公室的電腦系統是 SMB（Linux）或是 CIFS（Windows），都可以輕鬆地透過這個軟體來達到共享檔案資源的目的。

PCManX

PCManX 是 Windows 下的知名 BBS 連線軟體 "PCMan" 的 Linux 版本。它不是從 Windows 移植到 Linux，而是一個重新開發的軟體，比起 Windows 下的 PCMan 功能短少許多，但是該有的基本功能也都有。比較特別的是，它有類似 MSN 的訊息彈出視窗功能，使用起來更覺得親切。

PuTTY

PuTTY 是一個免費、開放源碼，而且體積非常小的客戶端連線軟體。它支援 Telnet 與 SSH1、SSH2 連線協定，可以在連上支援 telnet、SSH 的電腦之後，以安全的加密通訊方式進行遠端存取，交換在防火牆後的資源，還可以設定安全金鑰，更有效地防堵資料外洩的危險。

PieTTY

PieTTY 是國人以 PuTTY 為基底自行開發的免費、開放源碼，而且體積非常小的客戶端連線軟體。它除了完全支援中文以外，也更加簡化了繁瑣的操作介面，讓使用者可以更簡單、以安全的加密通訊方式進行遠端存取，交換在防火牆後的資源，還可以設定安全金鑰，更有效地防堵資料外洩的危險。

Free Download Manager

Free Download Manager 是一套在 Windows 環境下執行的軟體，如同它的名字一樣，它是免費而且開放源碼的。它可以直接瀏覽 FTP站台的目錄，使用 HTTP、HTTPS、FTP 的傳輸協定來傳送檔案。它還可以直接下載網站上面的 flv 檔案，並將其轉成 WMV 或 MPEG 格式的影片，使用者還可以透過遠端控制來操作本軟體。另外，它還提供了輕量化的 Lite 版本。

gFTP

在 X Window 下，也有類似 CuteFTP 一般好用的 FTP 工具。 gFTP 是一個在 X Window 下運作的多線程客戶端 FTP 軟體。它支援 FTP、HTTP 、HTTPS、SSL 和 SSH 協定，也支援斷點續傳以及 Drag & Drop ，可以說是功能相當齊全的 FTP 軟體。 gFTP 的使用率相當普遍，目前已經有四十多種語言的版本。

KompoZer

KompoZer 是一套所見即所得網頁編輯器，它是在 NVU 於 2005 年停止開發後，由一些開發者接續下去的非官方分支版本。KompoZer 具有一個簡易使用的網頁編輯器，即使你完全不了解 HTML 語言，也可以使用 KompoZer 做出一個看起來專業美觀的網頁來。它還包含了一個 CSS 編輯器，你可以輕易地用它來更改網頁的 CSS。

Firefox

Firefox 是由 Mozilla 基金會的一個專案計畫所發展出來的瀏覽器，操作介面非常具有親和力，還有很多很好用的擴充套件來無限增加其功能，也能任意的變換佈景主題，它是一個跨平台軟體，可以在 Mac OSX、Windows 及 Linux 上正常執行。

FileZilla

FileZilla 和一般商用的 FTP 軟體相比，功能一點都不遜色，支援多線傳輸及檔案續傳功能，能夠穩定的傳輸大量資料。FTP 站台管理員可快速設定常用的伺服器，還支援 Proxy 及防火牆、SSL 加密及 SFTP 加密協定，目前還有 Linux 和 Mac 版，相同介面，操作起來一樣順手。

PCMan

PCMan是一個由國人自行研發的 Windows 環境下的老牌免費 BBS 連線軟體。有支援瀏覽器的 Web/BBS 兩用版本，以及純 BBS 版本，內建 Unicode 補完、繁簡體中文輸入及顯示轉換功能、全球中文 BBS 站台列表等，可分頁瀏覽，所有操作熱鍵、工具列按鈕、圖示皆可自訂，以符合使用者習慣。
