Unifont.org

這個網站提供有關於 Unicode 字體的資訊、 Unicode 的軟件、國際化和可用 Unicode 操作系統的信息問題於免費、自由、開放源碼 (FLOSS) 上。相關的資源工具： Unicode 的字體指南、國際文本的佈局和排版、設計一個更好的字體選擇工具。

Gentoo Linux Taiwan

GOT (gentoo.org.tw) 成立於 2003 年 4 月，為台灣 Gentoo Linux 推廣組織，由一群熱愛 Gentoo Linux 的成員組成。GOT 分享中文翻譯文件、中文化軟體整合、郵件論壇、網路更新服務以及技術團隊的各種技術支援等等。GOT 並不是一個推廣 Gentoo Linux 中文化為單一目的的網路組織，而是集合 Gentoo 玩家，藉由有組織性的分工與拓展，讓每一個成員都可以在 Gentoo Linux 的領域共同學習成長並且樂於分享與奉獻的網路社群。

B2D 中文 Linux 計劃

B2D 的本意是 Back to Debian，取字首 B，to，D 的諧音三者而成，B2D 的子計劃有：B2D pureKDE，B2D pureGNOME，B2D Desktop (IceWM)，B2D Server，Mini Server，B2D usb，B2D 無碟電腦教室，B2D 主要在於嘗試實證以下構想雛型：安裝容易、中文環境、先採用 debian 套件 (apt-get)，它日再發展成自己的套件庫、彈性大，採用模組化架構，可任意拆裝變形加入創意。B2D 多用途：桌面、工作站、伺服器、無碟電腦教室應用。

中文資訊處理相關聯結彙整

OpenVanilla 框架顯示，輸入法可不僅僅是一個文字輸入工具：它提供了一個便捷的方式開發各種文字服務和語言工具，OpenVanilla 是一套為 Mac 設計的輸入法套件。內建的輸入法包括了倉頡、簡易（速成）、大易、行列、符號與日文假名共五種。使用者可透過偏好設定，匯入其他輸入法的表格，來建立符合自己需要的輸入法。

中研院_許氏鍵盤

許氏鍵盤提供注音符號與英文字母的對照關係，分成字音、字型、對稱、手順等規則。充分利用注音符號和英文字母的自然對應關係，減輕記憶鍵盤的負擔。對於熟悉英文鍵盤的使用者，只需極短的時間就能學會許氏鍵盤，並且能將英文打字的速度轉移到中文打字上，免除因不熟悉標準注音鍵盤而必須二次學習的困擾。由許聞廉研究員領導開發，並以 GPL 授權條款釋出，程式以 C++ 寫成。

中研院_PAT-Tree

PAT-Tree 專案是一個中英文抽詞工具，PAT Tree 是一個廣泛應用在資訊萃取的索引方式，利用在搜尋字串方面上非常有效率， PAT Tree 是利用一個文件之所有半無窮長字串 (Semi-infinite strings) 所建立的特殊 PATricia Tree ，可應用於中英文辭庫的建立、資訊的檢索、語意的查詢、知識的抽取等。由簡立峰研究員領導網路知識擷取實驗室開發，並以 GPL 授權條款釋出，目前版本以 Java 實作。

中研院_漢字構形資料庫字型

中央研究院資訊科學研究所中由謝清俊院士及莊德明助技師所帶領的文獻處理實驗室沿用「交大字根系統」的方法，經年整理各代古文，提出構字式（基於Big5）的作法，整理出各代文字的部件資料庫，分析所有漢字在文字學上的合理組成，進行重作改造，進而建立漢字構形資料庫。漢字構形資料庫附有楷體、明體、小篆、金文、甲骨文、楚系文字等字型，使用者可由本網址下載楷體、明體兩套 True Type Font ，或安裝完整系統以取得所有字型檔案。
