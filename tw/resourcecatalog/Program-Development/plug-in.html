外掛模組

外掛（Plug-in，又稱addin、add-in、addon或add-on）是一種電腦程式，透過和應用程式（例如網頁瀏覽器，電子郵件伺服器）的互動，用來替應用程式增加一些所需要的特定的功能。最常見的有遊戲、網頁瀏覽器的外掛程式和媒體播放器的外掛程式。應用程式之所以支援外掛程式的使用原因很多，主要包括：使得第三方的開發者可以對應用程式進行擴充、精簡，或者將原始碼從應用程式中分離出來，去除因軟體使用許可權而產生的不相容。
資料來源:https://zh.wikipedia.org/zh-tw/外掛

Apache Lucene Featured

Apache Lucene 是一個高效能，全功能，跨平台的文字搜尋引擎函式庫，完全以Java寫成，適用於全文搜尋方面各面向的應用需求。Lucene 原本由 Doug Cutting 在1999年寫成，其後在2001年加入Apache Software Foundation，並於2005年成為 Apache Software Foundation轄下的重要專案之一。現在該專案包含的元件除了Apache Lucene Core之外，還有以Apache Lucene Core為基礎的 Apache Solr 搜尋伺服器。由於Web 2.0的網路共工模式興起，全球的使用者日積月累地在網路上製造了大量的資料，而如何搜尋、處理這些相關資料，也就成為了資料探勘方面重要的技術，並且在應用面，也可以與各資料儲存網站配合，搭建出新興的資訊服務文創產業。Apache Lucene提供資訊搜索與彙整方面的解決方案，所以現階段已被許多具有龐大資料庫的知名網站採用，例如Twitter、LinkedIn；並且在學術研究方面，亦不乏採用Apache Lucene做為資料探勘研究平台的專案。就如同Apache Software Foundation其他專案一般，Apache Lucene以Apache License 2.0授權釋出，任何人皆可任意使用，並以此作為基礎續行開發。

Enhydra Director

Enhydra Director是一套開放源碼網頁伺服器的外掛模組集，提供了負載平衡、clustering以及在不同網頁伺服器及不同作業系統間提供單一連接方法。Enhydra Director支援多種普及的網頁伺服器軟體及作業系統，包括了Apache (在Linux/Unix及Windows平台上)、Netscape Enterprise Server (在Linux/UNIX 及 Windows平台上)，以及微軟的IIS (Windows)。Director同時支援多種應用程式伺服器：Tomcat (v5.xx and v5.5x)、Jetty，以及 Enhydra應用程式伺服器。

適用平台： 跨平台;原始碼授權類型：GNU General Public License (GPL)

資料來源： https://forge.ow2.org/projects/director

OSCache

是一套頁面快取機制，可以大幅提高網站的效能。本工具可以預存整個頁面或是特定區塊，預存機制是可以調整的。本工具同時允許高容錯性（例如，若是新聞資料庫當掉了，則顯示最後的幾篇文章，而非告訴使用者有錯誤）。OSCache已經在下列應用程式和網頁容器中測試過：OrionServer(version 1.4.0 或更高), Macromedia JRun (version 3.0 或更高), BEA Weblogic (應該能在version 7.x 或更高版本上工作), IBM Websphere (在 version 5.0 測試過), Silverstream (在 version 3.7.4 上測試過), Caucho Resin (version 1.2.3 或更高), Tomcat (version 4.0 或更高), iPlanet (在 version 6.0 上測試過；附帶說明，快取過濾機制並未在此軟體上測試過，只測試過 taglibs)

適用平台：跨平台;原始碼授權類型：OpenSymphony Software License

資料來源：https://www.opensymphony.com/oscache/
