FDO Data Access Technology - Project Home

Submitted by rieksp on Wed, 2007-09-12 07:23.

Welcome !

FDO Data Access Technology is an API for manipulating, defining and analyzing geospatial information regardless of where it is stored. FDO uses a provider-based model for supporting a variety of geospatial data sources, where each provider typically supports a particular data format or data store. FDO (“Feature Data Object”) is free, open source software licensed under the LGPL.  Read more...

Also, for a bit of History, FDO Providers, Requirements, Products using FDO and other good stuff, go to the sections from the Essentials panel on the left.

Learn more, get involved or download the software today. There are several ways to participate either as a user or contributor.
