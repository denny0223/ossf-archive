What is Sahana Eden?

  • Sahana Eden is an Open Source Humanitarian Platform which can be used to provide solutions for Disaster Management, Development, and Environmental Management sectors.  Read more
  • Open Source means that it is easily Customisable, Extensible and Free.
  • It is supported by the  Sahana Software Foundation.
  • Used in many Deployments
  • Try it now on our  Demo Site

We can always use help from:

  • Developers
    • Blueprints - functionality that we would like to see implemented

  • Bug Marshals

  • Designers

Contact

You can get in touch with the Sahana Eden Community immediately though our Slack Channel or via the Mailing List.

We will normally get back to you within 24 hours.
