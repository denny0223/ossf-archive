台灣自由軟體民間社群

建立日期 2009-01-20 18:04      最近更新在 2009-01-20 22:53 作者是 OSSF

在台灣開放源碼界中，民間社群一向是最具活力與行動力的推手，許多有志推廣自由軟體的好手群聚在各個開放源碼社群中，憑藉著自己的熱情，為自由軟體的推動盡一份力。這些社群中，除了較具知名度的Tossug、OrzLab之外，還有許多使用者社群，也默默地在各地推動各類自由/開放源碼軟體，例如：地區性社群、程式語言社群和各個Linux散布版本的使用者社群等。以下社群介紹按照字母排序，如有遺珠之憾，歡迎自薦提供相關資訊至 這個 E-mail 地址已經被防止灌水惡意程式保護，您需要啟用 Java Script 才能觀看

Ubuntu 正體中文站

建立日期 2008-12-26 23:26      最近更新在 2009-02-14 00:16 作者是 OSSF

●網址：https://www.ubuntu-tw.org/
●討論區：https://www.ubuntu.org.tw/modules/newbb/

 

Ubuntu 是一套由社群開發的作業系統，適用於筆記型電腦，桌上型電腦和伺服器。無論您在家中、學校或是在工作中使用，Ubuntu 提供了您所需要的程式，從文字處理，電子郵件程式到網頁伺服器，程式設計工具，應有盡有。

Ubuntu 永久免費，您無需支付授權費用；您可以自由下載、使用、與您的同伴分享 Ubuntu，而無須任何代價。台灣地區也有一群熱心的社群經營，每年都會有聚會並分享 Ubuntu 上優秀、有趣的功能。

Tossug

建立日期 2008-12-26 23:25      最近更新在 2009-01-20 23:07 作者是 OSSF

●網址：https://wiki.tossug.org/
●IRC頻道：irc://irc.freenode.net/tossug（utf8）

Tossug 是台灣最活躍的開放源碼社群之一，其核心成員每三個禮拜就會舉辦一場開放源碼的分享聚會，提供了許多關於開放源碼軟體的知識和發表場合，今年就提出了關於 Turbo Gears、OpenMoko、Plone 等各種開放源碼的議題，為開放源碼社群提供許多相關知識。

Study-Area & TnLUG

建立日期 2008-12-26 23:23      最近更新在 2009-01-20 23:06 作者是 OSSF

●網址：https://www.study-area.org/linux/linuxfr.htm（Study-Area）
●聚會地點：台南社大（TnLUG）

Study-Area 底下所屬的「酷！學園」論壇可是不少人入門自由軟體世界的敲門磚；而 Study-Area 的網頁上，也留下過去許多開放源碼作業系統的教學，供有心學習使用自由軟體的使用者挖寶。Study-Area 的創立者 netman 同時也是 TnLUG（台南 Linux 使用者社群）的發起人，因此 Study-Area 和 TnLUG 的活動多有重疊。但 TnLUG 的活動主要是服務台南地區的開放源碼愛好者，而 Study-Area 則擴及各地，在北、中、南三大都會區中皆常舉辦開放源碼的分享活動和其他聚會活動。

OSGeo Taiwan

建立日期 2008-12-26 23:20      最近更新在 2009-01-20 23:06 作者是 OSSF

●OSGeo：https://www.osgeo.org/ 
●FOSS4G：https://www.foss4g2007.org/

鄧東波先生和一群對於臺灣 GIS 有使命的朋友於今年 3 月成立 OSGeo Taiwan(Open Source Geospatial Foundation Taiwan Chapter)，其使命是促進地理空間資訊開放源碼在台灣的發展，提升開放源碼地理資訊在公私營部門的收集與分享，扮演地理空間資訊開放源碼開發者和使用者的橋樑。

OSGeo Taiwan 於今年五月協同文化大學數位地球研究中心為 Free / Open Source Software for Geoinformatics （FOSS4G）於第五屆數位地球國際研討會（Taipei International Digital Earth Symposium, TIDES 2007）組織一個議程，主題為 FOSS4G 之介紹及 FOSS4G 如何運用。另外，於最後座談時則是針對「促進開發台灣開放源碼地理空間資訊活動」討論臺灣地理資訊之現況及未來應著重的方向。

OrzLab

建立日期 2008-12-26 23:19      最近更新在 2009-01-20 23:05 作者是 OSSF

●網址：https://orzlab.blogspot.com/ 
●IRC頻道：irc://irc.freenode.net/orzlab（utf8）
●討論區：https://groups.google.com/group/orzlab

OrzLab（Open RazzmatazZ Laboratory）是自由軟體推廣志工jserv於今年四月一日正式成立的一個實驗室。OrzLab專注於將自由軟體應用在嵌入式系統中，讓自由軟體能夠在開放的硬體架構中，發揮最具彈性的功能。OrzLab 致力於嵌入式系統的開發、教學與推廣，也提供許多與嵌入式系統相關的教學與文章，讓有意學習嵌入式系統者獲得更多相關知識。

Open Street Map

建立日期 2008-12-26 23:17      最近更新在 2011-12-29 14:14 作者是 OSSF
●網址https://www.openstreetmap.tw/
Open Street Map 是一個全球草根性的地圖繒製運動，目的是要建立一套自由開放的地圖。地理資訊都是由全球志工收集 GPS 軌跡然後彙集而成，且所有的地理資訊都是使用開放自由的 Creative Commons by SA 授權。
讓每個人都可以利用、繒製、修改地圖上的資訊。目前台灣地圖部分是由熱心的社群發起，歡迎有熱情的伙伴一起參與。

Mozilla Taiwan

建立日期 2008-12-26 23:10      最近更新在 2009-01-20 23:05 作者是 OSSF

●網址https://www.moztw.org/
●討論區https://forum.moztw.org/

Mozilla Taiwan 是台灣專屬於 Mozilla 相關資訊的網站，其中最出名的成果當屬目前的熱門瀏覽器 Firefox 和 Email 收件軟體 ThunderBird 。此外，您也可以在這裡找到其他的 Mozilla 中文版程式，例如 Camino、SeaMonkey 和 NVU 等。除了能在 Mozilla Taiwan 的網站上找到上述軟體的載點之外，Mozilla Taiwan 也提供討論區供使用者交流使用心得和意見。Mozilla Taiwan 目前由中研院資訊所資助機房和網路頻寬，提供 Mozilla Taiwan 推廣所需的資源。

KaLUG

建立日期 2008-12-26 23:09      最近更新在 2009-01-20 23:14 作者是 OSSF

●網址https://kalug.linux.org.tw/
●聚會地點摩卡摩娜咖啡（高雄市三民區明賢街69號）
●IRC頻道irc://irc.freenode.net/kalug（utf8）
●討論區https://groups.google.com/group/Kalug

KaLUG（大高雄Linux使用者協會）是台灣南部推廣自由軟體的首要社群之一。KaLUG 官網採用部落格聯播的方式提供官網資訊，主題範圍廣泛，而且更新快速，使用者常常可以在此處發現許多有用的知識。KaLUG 每個月都會固定舉辦一場技術心得分享，由參與活動者或各界開放源碼人士發表關於開放源碼的技術或心得，是台灣南部交流開放源碼資訊的主要平台。
