「2003夏季源碼營」之二
林鈺維 2004/01/31
由於與會人士來自不同的背景，想了解的東西也不同，因此議程分成兩大類，分別是安排給 NGOs 人士和安排給軟體工程師的。安排給 NGOs 人士的課程主要是技術上的如何操作 Linux 作業軟體，從依照自己的需求選擇不同的 Linux 套裝軟體開始，然後一步一步教如何安裝、操作，談論安全性問題的時候也是以基本的軟體知識為主，大部分的課程都是直接上機操作的 hands-on sessions，自己動了手之後就比較有經驗、知道如何操作、可能會出什麼問題、及如何解決，事實上這也和一般自由軟體界的社群成員解決問題的方式很 像，自己下載安裝了不同的程式之後再看會發生什麼問題，然後再解決問題，畢竟，要給 NGOs 組織的人魚吃，不如教他們如何釣魚，這樣的受益才長久。給軟體工程師的課程就比較偏社會工作學，因為工程師一般沒有這類的知識背景，因此他們在和 NGOs 組織的工作者共識的時候無法明確知道她們的需求，因此灌輸工程師們一點點這類的社工背景知識可以幫助他們和 NGOs 使用者溝通和了解使用者的需求，但是也有進階的技術課程讓這些工程師們交流經驗互相學習，最受歡迎的技術課程是討論內容管理系統 (CMS—Content Management System)，網路安全，和如何架站用廣播傳輸資訊等課程，當然，讓 NGOs 工作者和軟體工程師共同討論的時程亦很多，讓大家互相溝通不同的理念。

Open Studio 開放社群工作坊成立
Hopenet 2004/03/22
HOPENET 科技月刊為推動開放性社群活動及推廣自由軟體的使用，專為開放社群的朋友免費提供了一個專業、開放的活動聚會空間「Open Studio 開放社群工作坊」正式成立了。為促進及積極參與開放社群之技術及文化交流且更進一步希望能夠與社群朋友有密切的互動，HOPENET 特別為社群朋友開放了一個專屬的活動空間，讓社群朋友可以在一個專業、開放、無負擔的空間裡交流技術與文化，分享彼此的心得與經驗，使開放社群的組織活動 更加熱絡，促使開放性自由軟體的使用更加的廣泛。
