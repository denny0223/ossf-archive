How to Think Like a Computer Scientist (中譯版)

這是一本值得推薦的 Python 工具書，目前正由 OSSF 同仁著手進行內容的翻譯工作，適合 Python 初學者閱讀的入門書，亦是 Open Book Project 一系列的電子書之一。這系列的書不只是 Freely Distributable ，還是 GNU Free Documentation License，代表您可以免費閱讀、散布和修改該書。原文位址。

OSSF 將中文書名譯為 －『如何像電腦科學家一樣思考』，希望對有興趣的朋友帶來實質的幫助。中文編譯內容採 GNU自由文件授權1.2版授權，詳細授權資訊如下：
