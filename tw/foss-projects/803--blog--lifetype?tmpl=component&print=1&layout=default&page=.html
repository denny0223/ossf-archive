多人 Blog 平台首選－ LifeType

建立日期 2007-02-09 08:00      最近更新在 2011-01-11 18:39 作者是 Emmi

在目前主流的 Blog 平台中，LifeType 可以算是最知名的一套 Web 應用程式，由於採用入口網站廣泛使用的 PHP+MySQL 架構，使她顯得平易近人，國內知名的 BSP 如蕃薯藤 HerCafe、燦星旅行社的星遊記、公司學校……等，都採用 LifeType 作為 Blog 平台或入口網站用途。但您也許不知道，這套舉世知名的 Blog 開發團隊中包含著來自台灣的成員，並為 LifeType 開發貢獻一份心力，讓台灣與國際開放源碼社群之間的交流又添一樁佳話。

讓中文 LifeType 社群與國際開發團隊連線的最大推手，就是 LifeType 中文開發支援社群的負責人 Mark Wu。他回憶起 2003 年開始接觸 Blog，看了 Jedi 的 Blog 後開始用 MT 來架生平的第一個 Blog，但某天不小心砍掉 MT 的資料庫後，不熟 perl 的他只好開始尋找其他的替代方案。後來在 zonble 的 Blog 上看到 pLog（LifeType前身）介紹，熟悉 PHP 的他因此開始改用 pLog。

後來 MarkWu 玩上癮，直接與國外 pLog 的創始人 Oscar Renalias與開發團隊認識，後成為 LifeType 核心成員，並在另一位中文社群成員 lss 的鼓動下成立了現在的中文 LifeType 社群。現在的 Mark Wu 已從原先任職的企業顧問服務公司離開，專心做個 Soho 族，在各地中小學與大學社團進行 LifeType 教學推廣、用 LifeType 幫客戶架設網站。

◎ LifeType 中文開發支援社群現況

與其他以支援或推廣為主的中文開放源碼計劃不同，LifeType 中文社群因為是由 LifeType 的核心開發成員建立，所以社群屬性也包含了開發的性質。中文部分也由這個中文社群統一 LifeTpye 翻譯與訂定，不僅在詞語上沒有翻譯分歧的問題，也因為這個中文社群同時支援簡體與繁體，所以也沒有一般中文開放原始碼在社群上簡繁資源無法共享的問題。

Mark Wu 表示，目前中文 LifeType 網站論壇的版主包括台灣人與大陸人，台灣版主主要包括了學校老師與研究生等，在剛創立的初期，一些知名的部落客如老貓等有參與 LifeType 的發展，現在以技術社群為主。

擔任重要溝通橋樑的社群主機，原本為了成本考量放在國外，但是為了能對社群有更好的服務，所以在智邦生活館的贊助下，目前搬回台灣並且放在 Hinet 的機房中。Mark Wu 也表示，其實台灣企業應該多考慮贊助開放原始碼，這樣會讓台灣的開放原始碼更蓬勃發展。

◎ LifeType 近期發展規劃

今年度 LifeType 的程式發展，Mark Wu 透露 1.2 版即將推出，包括新增個人的權限、不再只有Admin 和 Blogger 兩種權限；新增 Live Journal 所使用的 memcache 功能等。

此外 Mark Wu 還有“Instant LifeType”的個人計畫，在 Windows 下經過簡單的安裝後會出現 tray icon，即可輕鬆進行各項設定。目前是教學性質大於實用性質，但會往實用性努力，將來希望可做成其他 Web Application（如：Word Press、Joomla! 等）均能套用的架構。

至於 LifeType 中文發展社群的發展，Mark Wu 說未來主要會放在簡體中文網站的組織建置。

◎ 兩岸三地造成迴響 台灣 BSP 的最愛

LifeType 在兩岸三地都有許多商業網站使用，如新浪香港、博客中國等，同時也是台灣最多 BSP 使用的 Blog 平台，如台灣 Yam-HerCafe、ITHome blog、著名旅遊網站等。在 OpenFoundry.org 上也有下載前 10 名的好成績。因為其簡單易上手的優點，在台灣中小學與大學也受到熱烈歡迎。包括：台北市、宜蘭縣、中壢市、台中縣、彰化市教育局（班級網頁製作比賽）、高 雄市許仲佑老師（帶領學生參加網際博覽會獲得多次金牌獎）等中小學皆有使用。

◎ 教育應用成功案例：台北市內湖區東湖國中

台北市內湖區東湖國中就是採用 LifeType 的最佳範例之一。在該校擔任美術老師的 James，同時也是 LifeType 中文支援社群的成員兼開發者，現在為 Template 討論版版主，曾和核心開發團隊的 Jon Daley負責 1.0 版的樣板設計。當初因為有架多人 Blog 的需要，接觸到 pLog 0.3 版後成為使用者，並加入開發的行列，完成多語系的中文翻譯。

James 利用 LifeType 架設東湖國中 Blog 平台，包括：2004 年的老師部落格，2006 年開始的班級部落格、以及規劃中的教學資源 Blog 等等。其中，班級部落格已經有 1300 個 Blog，都是在同一個 Blog 平台，由此可以看出 LifeType 的威力！東湖國中亦以電子繪本 Blog 參加宏碁舉辦的比賽並獲得名次，今年預計將把行政系統（薪資、會計、人事等）透過 Blog 發布，省卻繁瑣的上網流程。

◎ 從 LifeType 開發經驗看台灣 FS/OSS 社群發展

Mark 除了致力於 LifeType 開發工作以及中文社群的推廣，在台灣的部落圈亦是一位知名的部落客，他的 Blog“Mark’s Place”根據「部落格觀察」的統計，在全台灣的部落格中名列前 30 名的位置。同時，他對於台灣自由軟體/開放源碼社群的發展也十分關心。

他以自己的開發經驗提出建議，表示自由軟體/開放源碼計劃不應該侷限在特定的地區，將來應朝向世界性的發展，且透過多國語言架構開發，開發與使用的對象將不再限於台灣的社群，這樣的路才會更廣。

最後，對於未來台灣自由軟體/開放源碼社群的發展，Mark Wu 認為，目前使用者社群與技術社群之間關心的面向不太相同，應該建立一個「顧問型」的社群，負責整合協調使用者和技術社群之間的溝通，如此開發者的辛苦成果 較易為使用者社群了解、使用者社群的需求也能較快讓開發者得知。
