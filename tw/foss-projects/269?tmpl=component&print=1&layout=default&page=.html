WordPress 使用者心得

建立日期 2007-01-23 19:36      最近更新在 2007-03-08 23:31 作者是 Kirin Lin
隨著 MT 之後的商業化以及對 Perl 語言陌生，積極尋找新的部落格 系統。嚐試使用 pLog 以及 WordPress 之後，個人認為 pLog 很適 合作為多人多格的系統，而 WordPress 個人使用綽綽有餘。起初也 是透過搜尋引擎使用 Zonble 所製作的中文包，入門上手輕鬆愉快。 雖說 Zonble 之後或許忙碌無暇繼續更新中文語系檔，自己動手做做 看也很簡單。 書寫的高峰期是大學參加話劇社時，社團的留言本被 Kirin 當作日 記、隨筆塗鴉。對 Kirin 來說，書寫與閱讀本身就是某種目的與手 段；那時候不知道部落格。後來在明日報有了自己的新聞台，但是常 常當機。申請了固網也申請了網域名稱，在自己電腦上面架起網站； 那時用著文字編輯器書寫 HTML。

不知道在網路哪裡看到 Blog 的報導，藉由搜尋引擎連上 Jedi 的部 落格，開始學習書寫部落格。Jedi 將 MT 中文化、打包、翻譯手冊 最後甚至還出了一本書。那是 Kirin 的第一個部落格系統。

隨著 MT 之後的商業化以及對 Perl 語言陌生，積極尋找新的部落格 系統。嚐試使用 pLog 以及 WordPress 之後，個人認為 pLog 很適 合作為多人多格的系統，而 WordPress 個人使用綽綽有餘。起初也 是透過搜尋引擎使用 Zonble 所製作的中文包，入門上手輕鬆愉快。 雖說 Zonble 之後或許忙碌無暇繼續更新中文語系檔，自己動手做做 看也很簡單。

Kirin 除了自己的 Blog 之外，也幫忙維護幾位朋友的 WordPress 部落格系統以及幾個有趣外掛的中文語系檔。WordPress 系統是 PHP 語言寫成，並不難懂，發展歷史也很久，所以有豐富的外掛可以選擇。 可惜的是許多開發者因為是外國人，就無法兼顧到雙位元語言處理， 甚至 WordPress 本身對中文語言處理仍有些許不順暢的地方。

◎ 小橘（資訊顧問，小橘美食報導作者，使用半年）

電子報：小橘美食報導
Blog：橘子也有部落格

其實我用的 WordPress 就是 Kirin 架的，如果是技術方面，他比較 熟囉，所以我只能就一個使用者的角度來寫寫感想。

當初會使用 WordPress，完全是因為好友 Kirin 安裝了 WP，試用一 陣子之後覺得使用上還算方便，幾次的改版，也發現功能越來越完善， 有許多外掛程式可以使用、有專人負責中文化，讓使用介面比較親切 等等。用久了，也用得順手之後，就不太想再花時間換其他的 blog 軟體。

當初我用過許多免費提供的 blog，例如無名小站、msn space、 Xuite、blogger，其實像無名小站也持續推出新功能，不斷改進中， 只是我希望有較大的自主性，以及使用上的彈性，所以最後選擇自行 架設 WordPress，使用到目前為止，挺滿意的，符合我的使用需求， 真不知道改版之後，還會推出怎樣的新功能呢，真令人期待。

◎ jimmytp（學生，使用半年）

會使用 WordPress 是因為別人推薦加上自己試用後覺得好用，真正 可以達到像 zonble 所說的「只要專心在寫作上，其他的事情不用煩 惱」，因此開始使用 WordPress。

我使用 WordPress 的用途大部份是用來架團體組織的 blog，比如我 幫學生會弄了一個宣布活動的網站、幫學弟弄了一個班級 blog、幫 公司弄了一個 blog，全都是 WordPress。在建立了這些團體組織的 blog 以後，我才發現 WordPress 真的滿好用，我的使用者似乎很少 碰到困難。

跟我之前用過的 blog 程式比較，WordPress 在文字格式的設定上比 較方便，使用者介面清楚明確，可以架在 php+mysql 的組合上容易 普及，無論是現有網頁空間或是架在自己機器上都很方便，此外 WP 在會員設計上考量到了多人 blog 的環境，有分級可以管理也是優點。

WordPress 中文化真的方便不少，與其去「原廠」抓不如在有人整理 整合過的 openfoundry 抓，不過感覺最近開發者最近好像很忙，比較 沒時間弄 WordPress 中文化了，1.2 中文包裡面的 plugin 也沒在 1.5 出現。希望未來多人版本（WPMU）能夠早日能在 windows 上運作。
