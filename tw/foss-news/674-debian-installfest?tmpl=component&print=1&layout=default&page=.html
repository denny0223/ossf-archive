Debian Installfest

建立日期 2003-08-01 16:00      最近更新在 2011-01-11 20:10 作者是 藍雲紋

Debian Installfest 的發起人 Andrew Lee（李建秋）在 slat-talk 郵件論壇上發表一篇文稿，表示：有個構想是在暑假結束要開學的那段時間辦一次安裝大會，邀請想參加的學校及 LUG 的朋友們上網站討論。

目前是希望由各學校提供地點，由各地區的 Lug 人員來支援學校技術方面事項。所以地點要看各地區 Lug 找尋願意配合的學校而定，至於時間，目前暫訂 9/6 舉辦。

另外假設有些學校需要一些比較正式的公文來商借場地的話，SLAT 這邊可以提供這個服務，請有需要的人員直接在 wiki 上，或是在 slat-talk 郵件論壇版面回覆，或是寄信給李建秋（ 這個 E-mail 地址已經被防止灌水惡意程式保護，您需要啟用 Java Script 才能觀看 ）亦可。
