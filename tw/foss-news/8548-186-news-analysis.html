186 期電子報－重要新聞回顧

建立日期 2011-12-12 14:52      最近更新在 2011-12-13 16:26 作者是 陳瑞霖／編述

自由軟體鑄造場電子報從 2003 年創刊時就一直關注自由軟體界發生的大事。並且依商業性、技術性、發展趨勢等原則，為讀者挑選數則重要自由軟體新聞。經由綜合各方說法及比較不同觀點，讓我們回顧前陣子發生的重要自由軟體新聞吧。

世界各地吹起開放資料的風氣，在臺灣也有民間組織推廣開放資料的概念。儘管沒辦法讓開放資料的先驅滿意，前陣子臺北市政府架設其開放資料平台，釋出部分手上資料。這次英國政府釋出政府資料，是以促進經濟發展為主要考量，由財政大臣口中宣佈。

GNOME 3 讓許多用戶很不習慣，如今GNOME 計畫借鏡 Linux Mint 的作法，以延伸套件的方式讓使用者能自訂其桌面操作環境。親愛的讀者，您有試了哪些延伸套件呢？

提供開放源碼不稀奇，但是統合旗下開放源碼專案的產品組合呢？法國公司 Talend 日前發表第 5 版Talend v5 正提供上述服務。

id Software 延續釋出遊戲引擎原始碼的專統，這次是釋出 Doom 3 的原始碼。不知道會有多少開發者拿去修改後，開發出屬於自己的遊戲。

  1. 英國政府宣佈開放資料計畫
  2. GNOME Shell 延伸套件網站推出 降低 GNOME 3 客製化門檻
  3. Talend 發表開放源碼整合方案 擴展雲端運算與大規模資料技術支援
  4. id Software 釋出 Doom 3 原始碼
