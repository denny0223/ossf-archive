ICOS2003 會場花絮

建立日期 2003-08-01 16:00      最近更新在 2011-07-21 17:44 作者是 涂敏怡

ICOS 由前身 1999 年的 Open Source Workshop 開始，至今已經邁入第五年，今年與會的重要官員與商業人士比往年多，顯示政府與商業界已經愈來愈重視台灣開放源碼的議題，令人感動的是社群朋友參與的熱情 比起往年來絲毫未減，而且還見到許多新面孔。


在第一天以法政與商業為主的議程結束後，第二天緊接著登場的是學術與社群議程，學術議程分為 ICOS 論文發表與國科會 Linux 研究計畫成果報告，另一邊的社群議程則探討教育、中文編碼與軟體中文化、網路新興技術介紹與網路社群討論等，經由議程主持人馮正青的熱情帶動，讓社群議程 的氣氛炒熱到最高點。，ICOS2003 會場花絮即以照片集的方式為您完整重現會場的即景，請各位前往下列網頁觀賞：
