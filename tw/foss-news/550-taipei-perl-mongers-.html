Taipei Perl Mongers 介紹

建立日期 2004-02-29 08:00      最近更新在 2011-01-11 22:46 作者是 簡信昌

Perl 推廣組 (僧侶團)，這到底是甚麼樣的組織呢？簡單的說，這是全世界各地的 Perl 使用者社群，就跟全世界的 LUG (Linux User Group) 一樣。他們有著同樣的喜好，大多是一群技客 (Geek)。雖然是由各地自行組織的地方性組織，可是 Perl 推廣組還是有著中央的聯絡單位，作為各地 Perl 推廣組的支援單位，而台北 Perl 推廣組正是全世界幾十個 Perl 推廣組其中之一。

台北 Perl 推廣組申請成立於 1999 年，不過前幾年一直沒有太多的活動或是成果，主要的原因在於沒有足夠凝聚人氣的動力。到了 2002 年，第一本以「台北 Perl 推廣組」掛名翻譯的書「Perl 學習手冊第三版 (Learning Perl 3rd Edition)」正式出版，Taipei.pm 才得以確定能推動各式活動的核心成員。也在第一本翻譯成果完成之後，台北 Perl 推廣組又繼續和出版社合作，翻譯了「深入探索 Perl 程式設計」，「Perl 網路程式設計」等等 Perl 相關的書籍。


2003 年則是台北 Perl 推廣組非常重要的一年。三月的時候，Taipei.pm 獲得中央研究院資訊所的支持，承辦了「Perl, PHP, Python Party (P3P)」，並獲得了重要迴響，兩天的研討會共計約有超過八十人參加，這也是台北 Perl 推廣組所舉行的第一次大型活動。在 2003 年，台北 Perl 推廣組的主要核心成員也得到「自由軟體鑄造場」的肯定，參與「自由軟體鑄造場」的系統整合工作。這是一個政府計畫，主要在於推廣台灣地區的自由軟體發展， 鼓勵社群，企業與其他民間，政府單位把開發中的專案利用這個平台開放專案的原始碼。而平台主要是使用 Perl 社群中非常受到肯定的 RT，Kwiki，Sympa 整合，作為一個專案管理的整合方案，並且釋出 RTx::Foundry 套件。台北 Perl 推廣組並且本著持續推廣 Perl 的精神，在 2003 年十月舉辦了為期兩天，12 小時的 Perl 教育訓練。

延續 2003 年所舉行的 P3P，台北 Perl 推廣組在十月份的教育訓練課程結束之後，便開始著手籌備 YAPC::Taipei 2004。YAPC (Yet Another Perl Conference) 大多是由各地 Perl 推廣組舉行的 Perl 研討會，主要在於經驗交流跟社群的感情交流。由於 RTx::Foundry 受到國際 Perl 社群的好評，因此 YAPC::Taipei 2004 也讓許多 Perl 社群的重要人士熱情參與。其中包括RT的作者 Jesse Vincent，Mason 的作者 Dave Rolsky，Kwiki 的作者 Brain Ingerson，Encode 的作者 Dan Kogai，以及被譽為 Perl 社群最好的講者之一的 Mark Jason Dominus 等。

台北 Perl 推廣組並且開始著手計畫固定的聚會，我們希望可以舉行一個月一次的討論，可以讓社群的成員能有機會跟其他人分享自己在玩 Perl 的經驗。而且接下來的目標除了繼續維持 YAPC::Taipei 的舉辦之外，我們還希望能把 Perl 推廣到企業中，尤其目前 Unix/Linux 在台灣企業的使用越來越頻繁，可是發展 Perl 長才的環境也越來越好，確實是推廣Perl的時機。我們也會有人繼續寫書，寫專欄，翻譯，來作為推廣的方式，當然，有機會可能還會舉辦 Perl 的教育訓練課程。
