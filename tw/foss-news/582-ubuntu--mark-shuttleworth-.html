Ubuntu 創始人 Mark Shuttleworth 來台宣傳

建立日期 2006-02-26 08:00      最近更新在 2011-01-06 21:45 作者是 涂敏怡
在國外廣受好評的新興 Linux 套件 Ubuntu，其創始人 Mark Shuttleworth 於 2 月 16、17 日來台宣傳 Ubuntu，拜會政府單位與台灣硬體大廠，並與本地開放源碼及Ubuntu使用者社群交流互動。 Shuttleworth 此次為提升 Ubuntu 在亞洲能見度的巡迴之旅，足跡亞洲各國，其中有一週的時間都待在印度，可見 Shuttleworth 對印度軟體產業的重視程度。而Shuttleworth之所以將台灣列入亞洲行程中，據了解是因台灣的電腦硬體產業發達，來台行程中拜訪的均屬世界級的電 腦硬體廠商，如廣達 (Quanta)、威盛 (VIA) 與 LITEON 等，意圖與這些廠商尋求 Ubuntu 的合作機會。

Ubuntu 在發展之初曾經邀請 Debian 高手開發，Shuttleworth 本身在 90 年代也曾參與 Debian 的開發工作，與Debian 相比，Ubuntu 在使用者介面上下了很大的工夫，而且為了快速增加使用者，Ubuntu 推出 Live CD，無須安裝就可立即體驗，降低了 Linux 的使用門檻。此外，Ubuntu 稱不論是個人版或企業版均完全永久免費，此點也提升了使用者的數量。

不過，Shuttleworth 對於 Ubuntu 的成功感到相當謙虛，他認為 Ubuntu 可以在不到兩年的時間內崛起，一部分是因為時機成熟，以往Linux kernel 2.4 版較不適合桌上型電腦使用，使用上需要做許多系統調整，但現在 Linux kernel 已到 2.6 版，因此幸運地可以作為桌上型系統使用，且因為其免費的號召，獲得不少社群開發者的認同，進一步參與此套件的開發，讓 Ubuntu 的發展更為完善。

雖然 Ubuntu 套件免費釋出，但曾經將自己開設的網路安全公司賣給 VeriSign、目前有數百億台幣身價的Shuttleworth 具有精明的商業頭腦，他非常看好 Ubuntu 的獲利前景，甚至將之視為比自己於 2002 年成為全非洲第一個太空旅客的投資還要值得。為了持續發展 Ubuntu，他成立了 Canonical 公司，該公司以 Ubuntu 技術諮詢作為營利項目。為了確保 Ubuntu Linux 使用者可以有五年的安全與技術支援，他於 2005 年再接再厲，以一千萬美元成立 Ubuntu 基金會做為備援計畫，讓這個計畫可以順利運作下去。

除了 Ubuntu Linux，Shuttleworth 也進行數項具有創新性的開放源碼計劃，其中值得一提是 Launchpad.net，該網站提供包括 Ubuntu 等開放源碼軟體計畫的翻譯、瑕疵(bug) 通報、程式碼開發等功能。和 Sourceforge.net 不同的是，Launchpad.net 提供不同計劃間的協同開發與合作，而非單一計畫內的合作發展。Shuttleworth 認為，一個開放源碼軟體需要多個單位與社群的合作才有辦法發展，但若使用者在軟體使用上遇到問題時，卻苦無一個整合的窗口可以告知，Launchpad.net 便是以此為出發點而成立，讓使用者可以知道某一個瑕疵在不同平台計畫之間的解決情形，希望提供使用者更好的開放源碼軟體支援。

Shuttleworth 在台灣唯一的一場公開演講由自由軟體鑄造場主辦，於 2 月 16 日傍晚在台大資訊所舉行，Shuttleworth 為表達感謝之意，特地致贈親筆簽名海報一張，向在場聽眾宣揚 Ubuntu 的理念、並親切回答聽眾疑問。

由於 Shuttleworth 將自己的絕大部份財產進行各種創投，因此他能將全副精力放在 Ubuntu 上。現在的他是全時的Ubuntu 開發者，由於對 Ubuntu 的熱愛，即使在台灣的短暫兩天已經排滿各式拜會政府與廠商的行程，Shuttleworth 仍在 16 日傍晚於台大進行完公開演講後，風塵僕僕地到台大附近的一家咖啡簡餐店與台灣的開放源碼社群進行交流，討論社群最關切的 Ubuntu 中文化支援問題。

當天的活動由台灣開放源碼社群 (Tossug) 主辦，原本可容納十餘人的場地擠滿了社群朋友，許多後到的人只能站著聆聽。活動主辦人葉平教授一開始致贈了寫有 Ubuntu 中文字「烏邦土」的字聯贈送給 Shuttleworth，表達台灣社群的歡迎之意，在場的社群開發者更爭相請 Shuttleworth 在自己的筆記型電腦上簽名，並示範如何在 Ubuntu 上設定中文字型與輸入法，雖然遇到一些問題，Shuttleworth 表示 Ubuntu 中文支援需要再改進，但他對於這場社群活動感到滿意，台灣的社群也藉此機會見識到這位開放源碼界的國際巨星的風采，為這一夜活動劃下一個圓滿的句點。
