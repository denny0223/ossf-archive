Wikia Search alpha 測試上路
OSSF (Mischa Hsu)    2008/01/10

Wikia Search 網址：https://search.wikia.com/

一年前，Wikipedia 創辦人 Jimmy Wales 公佈了一個開放源碼搜尋引擎計畫，以提供 Google Search 外的另一種選擇，引起眾人討論。在 2008 年 1 月 7 日 Wikia Search 釋出了第一個 alpha 測試版，我們終於得以一窺這個全新搜尋引擎的真實面貌。Wikia Search 是個開放源碼的搜尋引擎，和Google Search 最不同的是，它並非採取精密的演算法改善自己的搜尋結果，而是如同 Wikipedia 一般集眾人之力，由受信任之使用者的回應，決定搜尋結果條目的適合度，這種眾人合力的判斷方式，能使得搜尋結果排列計算以一種開放且公開的方式運作。不過也由於搜尋結果必須經由使用者改善，因此在目前的 alpha 測試中，搜尋結果在可評分使用者不多的情況下，仍舊有進步的空間，而多語系的支援也有改善的必要。

廣受矚目的 gOS 2.0 Rocket 推出
OSSF (Mischa Hsu)    2008/01/10

gOS 網站：https://thinkgos.com/

如果你看過 Everex gPC 平價電腦，那你一定對它優雅的作業系統印象深刻，Everex gPC 內建的系統是一個名為 gOS 的 Linux 分支，gOS 主要是基於 Ubuntu 改成，並加入大量網路應用程式，融合了桌面應用程式與網路應用程式兩方的特長，再加上 Enlightment 視窗管理軟體，形成一個特殊而且性感的作業系統。

如果你迫不急待地想要玩玩這個新的 Linux Distribution，Good OS LLC. 已經釋出最新的 gOS 2.0 beta 版供使用者下載，別光顧著流口水，趕快下載來玩玩看吧！

自由軟體的故事－結語（上）
蘇孝恆 (Haggen)    2008/01/10

日前(https://www.openfoundry.org/index.php?option=com_content&task=view&id=1263&Itemid=335)我指出 Stallman 的堅持和自由軟體的原則是值得我們深思，究竟是什麼的意思？之前談到 Rebel Code 一書出版之後，開碼運動還繼續發展，其中一件事就是 BitKeeper。我可以由這件事件談起。

Rebel Code 中提到 Linus Torvalds 不能應付日益繁重的開發工作，甚至 Linux 企劃有被分支的危險。原來 Linus 由開始一直沒有使用版本控制系統去處理由社群貢獻而來的源碼，在開發社群中已有人自行採用 CVS 版本控制系統來增加效率，不過 Linus 始終不願意使用(註 1)。CVS 的單一中央儲存庫的架構的確不能完美地配合 Linux 企劃的分散式開發，但 Andrew Morton 等認為先採用 CVS，然後再開發新的版本控制系統是上策(註 2)。大家如果去 Internet Archive 找一找 Linux 企劃開發網站的歷史(https://web.archive.org/web/*/https://www.kernel.org)，2002 年 10 月在站上只有 FTP 和 HTTP 下載和電子郵件通信論壇。那時誰去 SourceForge 登記，都可免費使用版本控制和待辦事項等開發工具。不少其他的開碼企劃的開發網站都設計得比 https://www.kernel.org 好。如果不是局內人，只看網站，根本不會相信這就是那個最出名最厲害 Linux 企劃的家。Linux 企劃在使用開發工具上真的是落後。

補助國內社群推廣自由軟體活動
中華民國軟體自由協會    2008/01/10

好消息來囉！

中華民國軟體自由協會為促進國內自由軟體社群更加活絡，特編列 11.5 萬補助國內社群辦理推廣活動。

（一）活動期間：即日起至經費用完為止。

（二）補助規則：
a.補助對象：以推動自由軟體為目的之團隊。
b.活動申請：請至報名申請網頁填寫資料（含活動目的、活動內容、社群過去活動記錄），且在 SLAT 網站公告活動內容。(註一)
c.參與活動對象：需有公開活動，允許不特定對象參與。
d.每次活動以補助 2000 元為原則，一個團隊最多補助 6000 元。由本協會所補助之社群推廣自由軟體活動，需於活動公告處列明中華民國軟體自由協會為協辦單位，以利經費核銷。

（三）由協會指派理事初步審核以上 a.b.c 三項是否符合補助資格，如有不符合或不能判定者，則提交給常務理事會審核。
（四）可核銷費用之項目參考如下：
a.講師費以 1600 元/時，需檢附領據請款；
b.餐點茶水費實報實銷，憑發票請款；
c.工作人員費 500 元/人，需檢附領據請款；
d.出席費用以 1000 元/次，需檢附領據請款；
e.場地佈置費 1000 元/次(含布條、名牌、海報等)，憑發票請款；
f.影印費實報實銷，憑發票請款；
g.保險費實報實銷，使否為與會者加保意外保險，由主辦單位自行決定。

報名網頁請至：https://www.slat.org/slat/et_top.php
(請用 UTF-8 開啟，然後在此網頁右上角點選插入一筆，填妥申請表格後，公告活動內容於協會網站才算完成申請補助手續)

註一：在中華民國軟體自由協會網站上登錄帳號即可發佈新聞。
