Re:請教免費軟體於商業公司合法性 2010/09/09 19:02  (9 Years, 2 Months ago) Karma: 2  
Hi, syman888


1. PDF Split and Merge


我看了他的 About 這一頁，第一句話就寫："pdfsam is an open source tool (GPL license) designed to handle pdf files. It’s released in 2 versions, basic and enhanced." 總之，他是以 GPL 授權的，所以貴公司想在公司內部使用，完全合法，沒有任何負擔！



2. PDF-XChange Viewer

他的授權你已經有提到了："The FREE version of the PDF-XChange Viewer may be used without limitation for Private, Commercial, Goverment and all uses, provided it is not -: incorporated or distributed for profit/commerical gain with other software or media distribution of any type - without first gaining permission. Software Developers SDK available for licensing in 3rd party software products etc." 上面說，不管是私人、商業、政府及所有使用者，大家想用就用，但要符合以下條件：在沒有先獲得允許之前，不准拿這隻程式包到其他程式或媒體或是散布來賺錢。不過因為貴公司只是內部自用，所以也不會有侵權的問題。



3. SCREEN2EXE


這隻程式我還真的找不到任何授權的詳細內容，跟你一樣，只有看到 Free 的字樣，連安裝下去，都沒有看到相關的 License 視窗要選 I agree 的；灌完執行後點選 About 也一樣，沒有任何說明。不過我認為，他寫的那麼簡單，只有一個 Free 字，在我看來是隱涵著「你想用就用，不會跟你計較」的意思，因為如果真的介意什麼，那他會像 PDF-XChange Viewer 一樣，多少會講一下他不准的範圍。

無論如何，如果只是要選擇在公司內部使用的程式的話，用「自由軟體 (Free/ Open Source Software)」絕對是不用付任何一塊錢的，但如果是「免費軟體 (Freewawre)」的話，仍然可能有一些限制風險。例如，免費軟體如果給了使用的期間，那就會變成「免費試用軟體 (Shareware / Trial)」，等於未來要用正式版本還是要付錢，不付錢就會功能減少、停用等等，這樣就不符合公司的需求。以上三套，只有第一套是自由軟體，第二第三套是免費軟體，自由軟體保證永遠都不會要求你付給他授權金，但免費軟體的擁有者在法律上其實是有權利某一天開始終止你的使用權的（雖然一般的情況下，他們都不會這麼做），所以原則上，只是要做公司內部使用的話，用自由軟體是完全沒風險、用免費軟體則並非百分之百零風險就是了。
