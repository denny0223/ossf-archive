OpenID：以使用者為中心（User-centric）的單一網路身分認證
myID 團隊與 Arthur    2009/08/09

<專案源起>

隨著資訊科技快速發展與寬頻網路普及，網路使用者持續增長，使用習慣瀕臨轉變。在 Web2.0 發展熱潮下，您是否發覺自己擁有太多組的網路帳號密碼，造成登入時需要回想一下配對是否正確？或是有些使用頻率較低的網站，經一段時間就忘記帳號密碼，造成急用時的不便利？有鑑於此，Single Sign-On（單一簽入）的想法開始被提出，亦即使用者採同一帳號密碼，可登入多個網站，OpenID 就是單一簽入技術的其中一種作法。

自由軟體鑄造場 徐孟逵/翻譯    2009/08/06

§原文位址§

14.1 物件導向功能

Python 是一種物件導向程式語言，這表示它提供了支援 物件導向程式設計的功能。

要定義物件導向程式設計並不容易，不過我們已經看過它的一些特色：

◎ 程式是由物件定義和函數定義組成，而且大多數的計算以物件的操作來表示。
◎ 每一個物件定義都與一些真實世界中的物件或概念相應，而作用於該物件的函數則與真實物件的互動方式相應。

OSSF 將於 COSCUP 介紹 FOSSology 開放源碼授權分析工具
coolword    2009/08/10

一年一度的的開放源碼軟體社群研討會【開源人年會】(COSCUP) 將在 8 月 15 日 ~ 8 月 16 日，於台大應力所國際會議中心與台大電機博理館盛大展開。今年大會重點議題囊括台灣硬體產業與開放源碼的結合及行動作業系統等。值得一提的是，今年開源人年會一開放報名，短短四小時即吸引五百五十名業界人士、學生與開放源碼愛好者上網報名額滿，盛況空前，顯示自由軟體之推展已為現今潮流趨勢。

在善用開放源碼軟體寫程式的時代，愈來愈多的開發者或使用者因未能遵守軟體授權而面臨訴訟爭議之疑慮，而 FOSSology 提供了程式源碼授權狀態的分析工具，發掘未被發現的授權問題，減少開發者使用開放源碼軟體的疑慮。自由軟體鑄造場(OSSF)曾義峰軟體工程師於 COSCUP 第二天的議程中分享 FOSSology  的實機操作，並對其未來可強化的特性提出展望。另外，OSSF也在台大電機博理館設置攤位，提供現場來賓自由軟體相關簡介與諮詢。

微軟釋出驅動程式原始碼 外界懷疑違反 GPL
謝良奇/編譯    2009/07/28
微軟史無前例地以 GPL (GNU General Public Licence) 釋出兩套軟體。外界矚目的不單單只是釋出的軟體，此舉的重要性在於微軟多年來一直不願意擁抱 GPL 這個最重要的開放源碼授權。過去微軟從未採用 GPL 釋出程式碼，而是偏好將該公司的開放軟體，以自己的共享源碼方案釋出，MPL (Microsoft Public Licence) 即是其中之一，MPL 雖然取得開放源碼組織 (Open Source Initiative) 的認可，在開放源碼社群中並未受到廣泛採納。

Novell 推出 Linux 裝置計畫 降低獨立軟體商開發成本
謝良奇/編譯    2009/07/30

Novell 推出 SUSE Appliance Program，協助獨立軟體廠商開發基於 Linux 裝置，其中包含 Amazon Elastic Compute Cloud (EC2) 系統上的 SUSE Linux Enterprise 組態設定。Novell 高層表示，該計畫能降低獨立軟體商的開發上的支出，並減少其客戶支援成本。

Novell 資深副總裁兼任行銷長 John Dragoon 表示，獨立軟體廠商藉由該項計畫，將更容易透過解決方案供應商銷售 Linux 為基礎的裝置。他說，在此該公司的價值，是作為獨立軟體廠商與解決方案供應商之間的撮合者。Dragoon 表示，Novell 將收取裝置所得營收的一小部份，而非向獨立軟體廠商先行收取費用。

Adobe 修補產品漏洞 部份漏洞來自微軟 ATL 程式庫
謝良奇/編譯    2009/08/04

Adobe 日前修補了 Flash Player 中的 12 個漏洞，其中 3 個來自有問題的微軟開發程式碼，以及 1 個讓駭客破解至少 1 週以上的漏洞。

Adobe 先前發佈過安全警告，表示在  Flash Player 與 Acrobat Reader 中存在嚴重漏洞，可能讓攻擊者利用控制使用者電腦，或造成當機。影響範圍包括 Windows、Macintosh 與 Linux 作業系統上的 Flash Player v9.0.159.0 與 v10.0.22.87 版本，以及 Windows、Macintosh 與 UNIX 上 Adobe Reader 及 Acrobat v9.1.2 版本。

Linux 基金會推出 Visa 白金認同信用卡
謝良奇/編譯    2009/08/09

支援 Linux 核心發展的非營利組織 Linux 基金會 (Linux Foundation)，推出一張 Visa 白金認同卡，給希望為 Linux 作業系統盡一份心力的用戶。

認同卡早已行之有年，通常附有免稅團體如學校或慈善機構的標誌。這張已經在美國發行的 Linux 認同卡上，有 Linux 標誌兼吉祥物 Tux 企鵝的圖像，每張卡開卡後 Linux 基金會將會收到 50 美元，信用卡每筆消費的一部份也會捐給該基金會。Linux 基金會表示，該 Visa 信用卡計畫的所得，100% 將直接作為社群技術活動，以及開放源碼社群成員為推動 Linux 創新的旅費資助之用。
