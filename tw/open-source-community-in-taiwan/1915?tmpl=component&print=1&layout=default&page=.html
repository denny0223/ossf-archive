FTUG 指的是「Fedora Taiwan User Group」（Fedora台灣愛用者社群），FTUG 提供許多關於 Fedora 的教學文章，並開放論壇讓台灣Fedora使用者討論各種層面議題外，也常舉辦許多關於Fedora的研討活動，推行Fedora這個開放源碼的作業系統。如果您想開始試用 Fedora 的話，也可以在 FTUG 網站上找到 Fedora 的 Live CD 和相關的應用程式。




分類: 台灣自由軟體民間社群
