Study-Area & TnLUG

建立日期 2008-12-26 23:23      最近更新在 2009-01-20 23:06 作者是 OSSF

●網址：https://www.study-area.org/linux/linuxfr.htm（Study-Area）
●聚會地點：台南社大（TnLUG）

Study-Area 底下所屬的「酷！學園」論壇可是不少人入門自由軟體世界的敲門磚；而 Study-Area 的網頁上，也留下過去許多開放源碼作業系統的教學，供有心學習使用自由軟體的使用者挖寶。Study-Area 的創立者 netman 同時也是 TnLUG（台南 Linux 使用者社群）的發起人，因此 Study-Area 和 TnLUG 的活動多有重疊。但 TnLUG 的活動主要是服務台南地區的開放源碼愛好者，而 Study-Area 則擴及各地，在北、中、南三大都會區中皆常舉辦開放源碼的分享活動和其他聚會活動。




分類: 台灣自由軟體民間社群
