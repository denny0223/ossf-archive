Funambol 為 Android 平台開發開放源碼行動傳訊系統 Sun 憂心 Android 平台將出現

建立日期 2007-11-22 08:00      最近更新在 2007-11-26 02:29 作者是 謝良奇/編譯

Google 的 Android 平台即將出現開放源碼的行動傳訊系統。發展開放源碼行動應用軟體伺服器的 Funambol，宣稱他們正在開發該平台上，第一套開放源碼行動傳訊及同步用戶軟體。不過 Android 平台有意推出自己的 Google Java 版本，卻讓 Sun 大感緊張。

根據 Funambol 的說法，這套應用軟體可以讓 Android 的手機裝置，與 Gmail、Yahoo 和 AOL 等電子郵件系統，以無線方式同步個人資訊管理（personal information management，PIM）資料。

這家公司表示，這套應用軟體將是 Android 平台第一個主要的開放源碼行動傳訊專案。而且該公司會與其開放源碼社群共同合作進行開發。

預計這套 PIM 同步應用軟體的第一個版本，包含基本的通訊錄同步功能，可望於今年底前釋出。明年第一支 Android 手機上市時，Funambol 將隨之推出具備完整的電子郵件與行事曆功能的版本。

除了相容於 Gmail、Yahoo! Mail、AOL、微軟的 Outlook 等電子郵件系統外，這套軟體會與免費的 myFUNAMBOL 入口整合，用以儲存使用者最新的 PIM 資料。

Funambol 執行長 Fabrizio Capobianco 表示，Google Android 和 iPhone，兩者採取的方式雖然不同，但對行動市場都有巨大的影響。iPhone 擁有漂亮的介面，但屬於封閉環境並缺少公開的 SDK，其運作受限於現存行動營運商的企業模式。Android 卻不同。行動裝置有了 Android 開放的 SDK，其介面與功能將可隨開發者想像隨意延伸，以廣告為主的企業模式將會從根本撼動無線產業的發展。

Funambol 這套應用軟體是 Android 平台上第一套對外宣佈的開放源碼通訊同步軟體，不久前，Google 表示將提供一千萬獎金給開發出最佳 Android 應用軟體的開發者。隨後，Google 的 Android 開發者群組上，就湧現了許多開發構想，從最簡單的計算機到複雜的多人線上遊戲，其中最有趣的包括一個類似雷射槍戰（Laser Tag）的第一人稱射擊遊戲，讓玩家藉由 Wi-Fi 連線透過地圖技術來進行射擊遊戲。

Sun 執行長 Jonathan Schwartz 曾公開撰文祝賀 Android 的推出，將其稱為新的 Java/Linux 手機平台，並表示希望 Sun 成為對該平台推出第一個完整開發者環境的平台軟體公司。

意料之外的是，Android 平台上的 Java，可能是一個特別的 Google Java 版本。Java 引以為傲的跨平台能力（"write once, run anywhere"）可能因此毀於一旦，Sun 不由得大為緊張。該公司軟體執行副總裁 Rich Green 表示，Sun 已經跟 Google 連絡，希望以開發者的利益為重，避免傷害到 Java。

不過諷刺的卻是，這個自微軟以後對 Java 最大的威脅，竟是來自 Google 執行長 Eric Schmidt。Schmidt 當年曾在 Sun 帶領 Java 的發展，與微軟簽訂引發後續 Java 大戰的授權合約。
