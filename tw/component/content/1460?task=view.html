專訪廣州銘源公司總經理龍啟文

建立日期 2008-03-14 08:00      作者是 Linuxpilot
究竟開源軟體事業在中國的最新發展狀況如何？在中國利用開源軟體賺錢是否可行？這些都是海內外投資者及 IT 行業所關注的題目，較早前香港 IBM 用戶協會舉行一個名為「與時並進」的研討會，邀請了業界精英探討 IT 行業的發展趨勢，當中包括 Web 2.0、綠色運算及中國開源事業的發展，Linuxpilot 訪問了其中一位主講嘉賓：廣州銘源電子科技有限公司總經理龍啟文。

廣州銘源電子科技有限公司 (Power Red Company ) 是中國新冒起的開源事業新星，創業短短三年已經獲得可觀盈利，去年更繳稅超過 100 萬元人民幣，龍啟文作為公司的舵手，對發展中國開源軟體事業有獨特的見解，龍啟文從三方面說明「開放源碼在中國的發展」，分別為開放源碼於最終用戶的實際意義、開放源碼在中國的發展歷程及開放源碼在中國的未來展望。

◎ 中國的開源用戶分析

龍啟文指出，第一件要弄清楚的事情，是哪些人在使用 (或需要) 開源軟體，目前在中國，Windows 95 已經沒有技術支援，用戶要使用 Windows 2000、XP，開支將增加很多，因為原來的硬體規格跟不上，轉用開源軟體便沒有這個負擔。開源的優勢在於可降低程式更新的風險，提供更加穩定的基礎架構，易於轉換的系統平台，及更低的整體擁有成本，例如中國人民銀行便是由 SCO UNIX 轉到 Red Hat，而且不論是使用 Intel x86 或是 IBM POWER 架構，或是哪一個發行版本 (Red Hat、SUSE、Debian、Ubuntu 或是紅旗)，Linux 內核都是同一個，對用戶來說更加自由。更重要是開源已經滲透至我們的日常生活，當我們使用 Google 的服務、免費的電郵戶口、Linux 手機、電子政府及電子商務，我們已經在使用開源，中國的知名網站，包括阿里巴巴、淘寶網、網易、百度等都是在開源的作業平台、中介軟體及資料庫上運行。

◎ 開源在中國的發展歷程

龍啟文表示，開源已經在中國一步步的發展壯大，1997 年中國的第一個自由軟體庫組織正式成立，由國家信息產業部領導，開始了自由軟體的宣傳工作，1999 年，中國開始商業的開源生產，藍點 Linux 是第一個由中國人開發的 Linux 發行版本，2005 年，IBM 在上海 LinuxWorld 宣布推出 IBM OpenPower 系統，對推動國內商業開源應用產生很大影響，2006 年，中國正式推出針對大學生的「Linux on Power」學習營，成功點燃中國的開源教育之火。

為甚麼龍啟文樂意投身於中國的開源事業？龍啟文回答說：「約在 3 年前，當時我在 IBM 公司負責 pSeries eServer 的銷售，OpenPower 系統的出現使我注意到開源在中國有很大的機會，因為有了 IBM 的 Linux 基礎建設方案，可以大大降低 IT 的成本，提高穩定性及伸縮性。擺在中國眼前的並不是開源好不好的問題，而是懂得架馭開源的專業人士太少，我們的 Linux 人才培訓並不足以滿足市場需求。我們公司的定位很清楚，我們並不需要開發作業平台、或是資料庫，我們的業務是提供開源的服務，包括顧問諮詢、系統設計及規劃、部署及維護，我們利用 IBM OpenPower，配合 IBM 的 SOA 及 Tivoli 方案，成功為企業打造先進的開源 IT 基建架構，主要客戶包括銀行金融機構、流通業及其他企業。」

◎ 開源在中國的未來展望

對於開源在中國的發展，龍啟文認為：「中國的開發人員在國際社區的參與度較低，很多廠商從開源獲取很多，但貢獻卻很少，部分產品只是更換了商標及進行了中文化翻譯，用心在程式碼研發的不多，而且各自為戰。龍啟文指出，在 20 多年前，中國人就有開源的概念，大家將代碼一行一行的分享，但是後來很多公司轉向專屬軟體開發，開源的風氣反而轉弱。我們應該視微軟為第一競爭對手，如果 Linux 及開源可以做到微軟做不到的事情，自然會吸引更多人使用。」

「要推動中國的開源產業，公司間需要加強融合與合作，Linux 平台和介面需逐步統一，我們要令更多的產品走出中國，更多的風險資金進入中國市場，廣州銘源公司已經達到 ISO 9000 及 CMMI Level 3，由於發展迅速，吸引了很多風險資金的注意，不過集團現在仍未接受任何風險資金的投資，因為集團希望先打穩基礎，提升管理水平，預計到年中便是合適的時候。」

「為了支持開源，我們公司將盈利的 30% 投資在非牟利的社團項目上，包括推動中國的大學參與開源，建立 Open Lab，以及提供免費的 Linux 用戶培訓給大學生，參與的大學包括廣西大學、中山大學及一些廣東高校，有關計劃的目標是提升大學生的 Linux 技能以及競爭條件，部分優秀學員經選拔後，更可在畢業後加入廣州銘源公司工作。」

每年暑假，廣州銘源公司都會與 IBM 合作舉辦「Linux on Power」體驗營，所有營員、志願工作人員皆為在校大學生，共 60人左右，包括大專、本科、研究生，他們從來自超過 20 個省市 30 所大學的數百名報名者中筆試面視選拔而出，今年的營員專業以電腦和軟體為主，有少部分的管理、經濟、藝術等專業學生。龍啟文表示：「開源軟體最需要的是人才，中國人口眾多，擁有很多大學，每年有很多畢業生加入勞動市場，所以中國有可能成為世界最重要的開源社區之一。今天中國若不貢獻開源，將來中國軟體業便會被世界拋棄。為推動更多的企業核心應用開源架構，廣州銘源公司正在支持學生參與開源架構管理平台的開發，包括研發網路架構管理、數據庫管理及備份管理等軟體，特別是在 IBM Tivoli 上進行二次開發，目前開發的項目為 Power Data Manager。」

龍啟文總結說：「目前中國仍然沒有一個具規模的軟體鑄造場 (Forge) ，即使外國的 SourceForge 也沒有太多人參加，原因之一是語言障礙，其次是很多大學生根本不知道 SourceForge 的存在，而且很多項目已經很大，新人很難弄懂如何參與。所以在中國推動開源，較好的做法是從小開始，先由幾個人做起，然後再放到網上公開。大學環境較為開放，最適合推動開源，目前廣州銘源公司已經將幾間大學的 Open Lab 連結，各自負責開發同一系統的不同模組，達到既互相競爭，又要互相合作的效果。另外去年開始，廣州銘源公司提供十萬元，鼓勵學生利用開源創業，不同組別的學生要草擬計劃書，成功的便可獲資助。」
