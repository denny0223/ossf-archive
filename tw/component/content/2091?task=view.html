Moblin Version 2 獲得多家 Linux 散佈商支持

建立日期 2009-06-09 08:00      作者是 謝良奇/編譯
Intel 在台北舉行的 Computex 展覽上，持續大力推廣開放源碼 Moblin Version 2 行動作業系統。Canonical、Xandros、Linpus 與 Red Flag 已經宣佈，將針對 Intel Atom 為基礎的 netbook，推出以開放源碼 Moblin v2 堆疊與應用軟體框架為基礎的 Linux 散佈套件。Novell 則是在 Acer 與 MSI netbook 上，展示之前已經宣佈過的 SUSE Linux Moblin v2 版本。

日前由 Intel 支持的 Moblin.org 釋出 Moblin v2.0 的 beta 版本，在使用者介面上加入延伸升級，包括新的工具列，以及整合社群網路內容的 m_zone 首頁。率先宣佈支持的，是 Novell，該公司已經準備好 SUSE Linux Moblin v2 版本以及基於 Moblin v2 的 OpenSUSE。 Computex  展覽上，Novell 在未公開的 Acer 與 MSI netbook 上，展示了 SUSE Linux Moblin v2 版本。

Acer Aspire One netbook 目前提供 Linpus Linux Lite，至於是否持續如此，或者同時提供 Novell 與 Linpus 版本的 Moblin v2，外界仍不清楚。作為長期的 Moblin 支持者，Linpus 宣佈在 Linpus Linux Lite 上支援 Moblin v2，並且表示將提供 Linpus 與 Moblin 介面的安裝選項。Linpus Linux Lite 特別針對亞洲語言提供支援。該公司在 Computex 會場展示此套軟體。Asianux 創立公司之一的 Red Flag，也宣佈在其 Midinux 3.0 散佈套件上支援 Moblin v2。

和 Canonical 同為 Moblin 早期支持者的 Xandros，將在 Asus Eee PC 上提供 Moblin v2 的散佈套件版本。Xandros 更計畫結合 Moblin v2 核心技術與 Xandros 自有的私有技術，以提供社群網路、多媒體內容管理、個人資訊管理、Web 與其他服務。

Linux 散佈商似乎希望仰賴再次振作與鞏固的 Moblin 堆疊，重奪近幾個月來流失給微軟 Windows XP 的 Linux netbook 市佔率。當然，也包括即將推出，號稱可在 netbook 上執行的 Windows 7。同時，Moblin v2 也將為 Google Android 增加所面臨的競爭。

Moblin v2 以其快速開機技術著稱，宣稱可在短短 7 秒鐘內載入桌面。從 beta 釋出開始，Moblin 就加入了採用 Clutter 動畫框架開發的新使用者介面，該動畫框架特別針對 netbook 與 nettop 進行了最佳化。Moblin 的新上方工具列只有當游標接觸時才會顯現，進而節省桌面空間。

除了新的上方工具列，關鍵的新成員 m_zone，可作為多種活動的資訊首頁，這些活動包括同步化的日曆項目、工作、約會、最近使用過的檔案，以及來自社群網站的即時更新。

Moblin v2.0 beta 其他重點功能尚包括，修改為 Clutter shell 且以 Mozilla 為基礎的最佳化網路瀏覽器、可縮放的媒體播放程式、更新過的連線軟體程式 (ConnMan)，並透過 Gnome Mobile 技術支援多種 Linux 桌面應用軟體。

4 月初，Intel 正式將以 Linux 為基礎的 Moblin 行動裝置堆疊，移交給 Linux 基金會 (Linux Foundation) 管轄。如今，MontaVista 公佈了基於 Moblin v2 的 MontaVista Linux 6  的市場特定分布 (Market Specific Distribution，MSD)，為 Moblin 增添新的開發支援。

Intel 在 Computex 展覽中舉辦了 Moblin 經理人高峰會 (Moblin Executive Summit)，宣傳 Moblin 背後產業體系成長中的動能，並展示 Moblin v2 beta 環境的豐富性。Intel 並且公開一份支持 Moblin 的作業系統廠商名單，包括 Asianux 協會的創立者 Red Flag，Canonical、CS2C、Good OS、Linpus、Mandriva、MontaVista、Novell、Pixart、TurboLinux 與 Xandros。
