JBoss 推出新產品 鞏固中介軟體高階市場地位

建立日期 2006-03-31 08:00      最近更新在 2007-03-09 01:29 作者是 謝良奇
JBoss 日前宣佈成立兩個新的開放源碼專案，包括可做為服務導向架構基礎建設的 JBoss Messaging 與高效能 Web 伺服器 JBoss Web，將成為 JBoss 鞏固 JBoss 企業中介軟體 (JBoss Enterprise Middleware Suite, JEMS) 在企業市場地位的重要一著。

JBoss Messaging 是標準訊息平台，符合 Java Message Service 1.1 與 1.0.2b 標準，具備支援大型服務導向架構與企業服務匯流排 (enterprise service buses, ESB) 的核心訊息能力，預料將可有效地拓展 JBoss 在高階市場的能見度。 JBoss Web 伺服器針對 JSP、Java Servlet 、ASP.NET、PHP 和 CGI，為企業提供一套部署平台，並整合可處理大量資料的開放源碼相關技術，可為 Apache Tomcat 與 JBoss 應用伺服器的用戶，帶來高效能的 Web 伺服器功能。 以 Apache Tomcat 的 JSP 與 Java Servlets 容器 (container) 為建構基礎，JBoss Web 可以同時處理超過 1 萬筆以上的連線。JBoss Web 1.0 預覽版本已經推出，正式版本將可在 6 月問世。 這兩套軟體以 LGPL 授權用戶免費下載。JBoss 產品管理副總裁 Shaun Connolly 表示，收益將來自簽訂型服務，如訓練、諮詢與支援。他指出，私有軟體商的做法是把功能包裝成一個黑箱，而他們的方式則是以隨插即用的模式提供讓企業可以混搭組合的功能。 JBoss Messaging 1.0 目前已可單獨下載，預計將運用在 JBoss ESB 1.0 與 JBoss 應用伺服器 5.0 的 Java 訊息服務 (Java Message Service) 之上。 Connolly 認為 JBoss Messaging 和 JBoss Web 可說是 JBoss 技術上的另一個里程碑，針對該公司高階用戶擴展並加強 JEMS 產品線，同時也維持了 JEMS 做為服務導向架構開放源碼領導平台一貫的單純性與模組性。 近來傳出 Oracle 將收購 JBoss 的消息，JBoss 技術長 Sacha Labourey 澄清表示，JBoss 比較可能走向公開發行上市 (IPO)。Connolly 則呼籲莫輕信流言。他說，這個市場需要不同的開放源碼應用軟體廠商。
