紅帽、SOA Software 合作 JBoss 搭配 SOA 管理與安全產品

建立日期 2007-04-16 08:50      最近更新在 2007-07-03 22:57 作者是 謝良奇
SOA Software 與紅帽（Red Hat）正進行合作，讓 SOA Software 的服務導向架構（services-oriented architectures，SOA）安全、管理與控管產品結合 JBoss 的開放源碼中介軟體。

去年被紅帽收購的 JBoss，已成為紅帽部門之一，旗下的 JBoss 企業中介軟體套件（JBoss Enterprise Middleware Suite）是包含應用軟體伺服器在內的一系列產品，可供企業建構與部署應用軟體與網路服務。雙方面的這次合作，將共同行銷雙方產品，並在客戶服務上達成更為密切的合作關係。

JBoss 企業中介軟體套件可自由下載，JBoss 提供以會費訂購為主的支援服務。

此次合作，雙方將著重在 JBoss 應用伺服器內部署 SOA Software 的產品，紅帽也將與 SOA Software 一同增加對 JBoss 企業服務匯流排（JBoss Enterprise Service Bus）的支援。

SOA Software 執行副總裁 Roberto Medrano 表示，開放源碼已準備好成為企業服務導向架構的主流。SOA Software 近來面對 Vitria Technology、IBM、Sun 與 Infor 等廠商推出新的服務導向架構產品與策略，此一領域的競爭可說越演激烈。

JBoss 的服務導向架構產品線管理經理 Pierre Fricke 表示，JBoss 從 2003 年起就已開始支援服務導向架構的部署，但未曾著眼於服務導向架構環境下的控管、安全與管理，這次與 SOA Software 的合作將可補足此一缺口。

SOA Software 的安全、管理與控管工具每伺服器為 5000 美元，為了與市場上其它大廠相區隔，SOA Software 強調產品與各種應用軟體伺服器皆可相容。Medrano 說，他們可以管理所有私有與非私有平台，這一點與 IBM、BEA 或 Oracle 一向希望用戶只用他們的平台和應用軟體堆疊的做法不同。
