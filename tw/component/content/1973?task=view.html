Intel 首次釋出 Moblin 2 alpha 版本

建立日期 2009-02-05 08:00      作者是 謝良奇/編譯

在試用 Windows 7 beta 之前，先看看 Moblin 的 alpha 版本吧，這是針對 netbook 特別設計的開放源碼 Linux 作業系統。Intel 的 Moblin 2 平台首次 alpha 釋出已可公開下載，專為採用 Atom 的 netbook 設備而設計，提供快速開機與獨特的網際網路連線管理程式。

Intel 宣佈 Moblin 2 的首次 alpha 釋出。Moblin 是 Intel 為基於 Atom 的行動裝置所開發的開放源碼 Linux 平台。該平台的理念是為 netbook 用戶建構新的使用體驗，這些用戶至今購買的 netbook 多是預載微軟 Windows XP 或 Linux 作業系統。同時 Moblin 也希望協助筆記型電腦製造商，在其設備中營造更大的區隔，以維持這些廠商的價格利潤。

Intel 希望吸引到第三方開發者社群貢獻此一平台，並為其開發應用軟體。Moblin 已經由若干 Linux 散佈商所採納，例如 Linpus、GoS 與 Mandriva。這些散佈商計畫在 Moblin 核心上建構客製化衍生產品。

儘管便宜的輕量級 netbook 成為 PC 產業成長最為快速的區段，日前 HP Voodoo 事業部技術長 Rahul Sood 在其網誌上指出，netbook 目前的形式，將扼殺 PC 創新。

Netbook 原本的設計希望成為桌機與筆電的伴隨產品，但是越來越多使用者購買 netbook 作為主要的設備。Sood 認為，這意味著使用者得到的是一部完全無法提供滿意使用者體驗的精簡版筆記型電腦。

專為 netbook 處理器，例如 Intel 的 Atom，所設計的作業系統，可以協助改變此一情況，同時協助筆記型電腦與晶片製造商防止其產品相互侵蝕。Sood 表示，這有賴於你如何讓高效能機器與 netbook 區隔開來。由 Intel 贊助的 Moblin 專案正希望為 netbook 處理器專門設計一款作業系統，在這些小型裝置上提供用戶更好的體驗。

主要針對 netbook  的此一新的 alpha 釋出，將加速部份 Intel 增強技術的測試。然而，由於仍處於開發的初步階段，其使用者介面離完成仍有一大段距離。Moblin 目前採用了輕量級 Xfce 桌面環境的標準安裝，並納入由 OpenedHand 所開發的開放源碼輕量級個人資訊管理應用軟體 Pimlico。Xfce 最終將以採用 OpenedHand 的 Clutter 框架建構，更為豐富且更易於行動使用的使用者介面來取代。Intel 已於去年購併 OpenedHand。

Moblin 主要延伸自 Fedora，但包含了多項提高對行動裝置適應性的客製化功能，例如獨特的網際網路連線管理程式和更高的開機效率。Moblin 專為 Intel 為基礎的硬體，特別是採用 Atom 或 Core 2 處理器的 netbook。根據釋出摘要，該作業系統經過測試，已經可在 Acer Aspire One  與 Dell Mini 9 上運作，在 Asus Eee 901 上雖然可以開機，但並不支援 Eee 無線網路的硬體。

Moblin 2 alpha 的原始程式碼可由該專案的版本控制檔案庫下載，使用者透過 Moblin Image Creator 工具可由原始碼產生客製映像 (custom images)，此外，使用者也可下載預先建立的 ISO 映像。

Moblin 2 alpha 釋出是一個不錯的開始，但顯然仍處於開發中的狀態的 Moblin，還不是使用者願意安裝在日常使用 netbook 上的系統。其使用者體驗落後於許多基於 Ubuntu 的散佈套件，包括搭配 Dell Mini 9 的客製 Ubuntu 與像是 Easy Peasy 之類的第三方選向。當 Intel 為 Moblin 加入更有力的使用者介面後，Moblin 才會吸引來自開發者與終端使用者更多的目光。
