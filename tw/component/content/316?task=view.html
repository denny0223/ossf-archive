IE 7 beta 版釋出 追趕 Firefox 腳步

建立日期 2005-07-26 08:00      最近更新在 2007-03-09 01:30 作者是 謝良奇
微軟瀏覽器 Internet Explorer 7 的第一個 beta 版本，日前已經 釋出到開發者的手中。原先似乎並未計畫發展新版 IE 瀏覽器的微軟，或許因為外界對其安全性不佳的批評，加上近來 Mozilla 開放源碼 瀏覽器 Firefox 急速成長之故，而重新展開此瀏覽器的後續產品研發。

新版 IE 僅可於 XP SP2 上執行，預計會有 Server 2003 的版本， 但 Windows 2000 與 98SE 等平台上的用戶，則不會有新版 IE 瀏覽 器可用。微軟安全企業技術產品管理總經理 Gytis Barzdukas 表示，一連串技術上的投入已大幅更改了原有架構，對於客戶而言，換至新 作業系統是較為適當的做法。 新版 IE 對瀏覽安全上做了許多增強，其中首先是針對惡意軟體 （malware）的防治技術。IE 7 採用處理 Web 位址的單一資料處理器，微軟希望藉此減少攻擊者利用特殊 html 鏈結欺騙瀏覽器執行惡意軟 體的機會。針對跨網域腳本攻擊（cross-domain scripting），新版 IE 會為用戶執行的腳本加上來源網域名稱。 在外觀上，新版 IE 可以見到許多 Firefox 特色的影子。例如， Firefox 在位址列旁有個搜尋框，用戶可向多個知名搜尋引擎進行搜 尋，新版 IE 也有同樣的設計。新 IE 也提供了目前眾家瀏覽器皆有 的分頁瀏覽功能。此外，新 IE 也跟隨其他瀏覽器的腳步，加入了 RSS 支援，目前支援 RSS 1.0、2.0 及 Atom 0.3 格式。目前看來， 新版 IE 比較像是追趕 Firefox 之用的產品，創新面所見不多。 Mozilla 基金會認為 IE 7 的出現，將轉而成為 Firefox 在企業用 戶普及度上的助力。Mozilla 基金會社群協調人 Asa Dotzler 即表示， 他希望吸引更多無法使用 IE 7 的 Windows 2000 用戶到 Firefox 瀏覽器上。他說，他們高度期待能在企業的 Windows 2000 用戶群中， 有更好的表現。 根據 AssetMetrix 的調查，近半數採用 Windows 企業桌面仍在使用 Windows 2000，這些用戶將是 Mozilla 基金會未來的一大市場。 Dotzler 認為，微軟釋出 IE 7 將讓許多人了解到，假如他們要求更好的功能，就得花錢升級到 Windows XP。隨著 Firefox 在功能上的不斷提升，加上 IE 7 釋出帶來的效應，將可大幅增加 Firefox 瀏 覽器的市佔率。Dotzler 表示，當他們不斷為企業應用改善產品，再 加上 Windows 2000 用戶的被忽略感，將會促成顯見的骨牌效應。
