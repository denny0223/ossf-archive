網際網路收音機 Livio Radio 聆聽個人化音樂串流

建立日期 2009-04-21 08:00      作者是 謝良奇/編譯
Livio 發表了一部以 Linux 為基礎的網際網路收音機 Livio Radio。Livio Radio 可以播放播放Pandora 的客制串流，以及來自 Reciva 約 11000 個電台節目。這部收音機包含了 802.11 無線網路、有線 Ethernet 網路所用的 RJ45 port，另外還提供兩個好評 (thumbs-up) 與負評 (thumbs-down) 控制按鈕，為歌曲打評價。

由音樂基因專案 (Music Genome Project) 創造的 Pandora，是自動化的音樂推薦與網際網路廣播服務。和多數網際網路電台不同，Pandora 基於用戶喜歡的音樂與歌手，根據相似度計算，找出用戶可能會喜愛的音樂及歌手。Pandora 的技術來自音樂基因專案，從 2000 年開始，就利用音樂家與演算法的分析，將資料庫中的歌曲以 400 個音樂屬性加以索引。

根據 Pandora 表示，這些資訊會用來建立播放清單，當用戶輸入喜愛的歌曲或歌手，就能進入個人化的音樂聆聽體驗，並建立多達 100 個個人廣播電台。Livio 推出售價 150 美元的 Linux 網際網路收音機，可以播放 Pandora 的個人化串流，以及超過 11000  個其他電台，對於喜愛 Pandora 服務的人士，這部收音機相當具有吸引力。

宣稱擁有 1800 萬註冊用戶的 Pandora，除了以個人電腦使用外，Windows Mobile、iPhone、Blackberry 等各種智慧型手機上，也有相應的軟體用戶程式。不過，目前能夠播放 Pandora 的獨立設備卻是寥寥可數，同樣也是以 Linux 為基礎的定時收音機 Chumby 是其中之一，配備有 350MHz 的 ARM 處理器與 64MB 記憶體和 64MB 快閃儲存空間，以及 3.5 吋的觸控式螢幕。

號稱 Pandora 網際網路收音機的 Livio Radio，如今提供了新的選擇。看似老派桌上型收音機的 Livio Radio，配備有無線與有線網路、以 ARM 為基礎的單版電腦 (single board computer，SBC)。特別的是，Livio Radio 宣稱是第一部在前方面版與遙控器上，同時提供好評與負評按鈕，讓用戶表達對播放中音樂的意見。

Pandora 企業發展資深副總裁 Jessica Steel 表示，Pandora 的收聽者逐漸透過個人電腦以外的方式使用 Pandora，因此他們正在找尋易於使用，而且能吸引收聽者的新產品。

除了 Pandora，以網際網路廣播見長的 Reciva 也與 Livio 合作，讓 Livio Radio 得以收聽來自全球 11000  個，以 Real Audio、Windows Media 或 MP3 格式串流播放的網際網路廣播電台。

使用 Livio Radio  前，用戶得先在 Reciva 與 Pandora 網站上註冊，透過瀏覽器建立想要收聽的電台清單，這些電台會出現在 Livio Radio 的選單上，利用設備上的旋鈕和 3 行式的顯示器加以選擇。

儘管 Livio Radio 的硬體細節並未公開，過去販售的 Reciva 相容收音機都採用了由 Reciva 自行製造的兩款小型單版電腦 Barracuda 與 Stingray 其中之一。雖然 Reciva 並未揭露太多硬、軟體細節，不過根據試圖拆解與分析 Reciva 相容收音機的 Sharpfin Project 的資訊，Barracuda 採用的是 Samsung 的 S3C2410 處理器，並執行衍生自 Debian 的 Familiar Linux。
