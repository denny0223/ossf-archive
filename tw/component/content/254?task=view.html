笨牛改用自由軟體的心得分享（下）

建立日期 2006-10-15 08:00      最近更新在 2012-03-06 17:08 作者是 笨牛

[Q5]：若想更進一步試試免費的作業系統，該如何進行？

1. 其實可以做到在微軟作業系統下全面使用自由或是免費的軟體就不錯了，但是如果想要進一步使用另類免費的作業系統，就比較複雜一些。笨牛建議，先從下載及使用 Live-CD 開始，體會一下那種感覺，再評估是否要安裝。

2. Live-CD 簡單的說，就是把作業系統及常見的應用軟體放在一片光碟上，開機時只要設定光碟開機，就可以上網、打報告、製作投影片、修改照片等，常見的應用軟體全在一 片光碟，免費又合法，而且可以自行增減所需要的軟體，製作符合自己需求的 Live-CD（只是笨牛還沒有這種功力），但是存檔時要存在隨身碟，缺點是執行速度略慢一些些，因為不是安裝在硬碟，這也是優點，適用於無碟電腦教室。 據了解台灣已有多所中小學採用這種作法， 學校學一套，回家就是用那一套，父母親不用花錢買作業系統及辦公室軟體，或是 Photoshop，也不用擔心孩子用盜版軟體，因為學校有發學習光碟，使用的是功能相同的自由軟體而非商業軟體。況且這些東西是合法又免費，在用完 Live-CD 後關機，退出光碟， 完全不會損害在硬碟的資料。

3. 在此附上數個台灣常見 Linux 的網站，讀者不妨去下載 Live-CD（iso 檔，燒錄後就可以使用），各網站都有詳細的說明文件：

  1. Ubuntu
  2. Fedora
  3. Mandriva
  4. Knoppix 中文交流網
  5. 中文 OSS 桌面應用增進計劃
  6. B2D，笨牛就是用這個，迷人麥金塔的風格，這網頁有使用說明（若有看不懂的請跳過）

4. 有幾個相當不錯的教學網站，內有專為新手所寫的文章，淺顯易懂，可以降低恐懼感：

  1. 鳥哥的Linux私房菜
  2. Study Area
  3. GNU/Linux 初學之旅

5. 笨牛跟多數人一樣，喜歡自己買零件組電腦，己灌作業系統，所 以當笨牛看到這麼多免費合法、下載後燒成光碟就可以用作業系統， 忍不住試了幾種，包括 Fedora (pc and mac)、 Mandriva (pc and mac) 、Ubuntu (pc and mac)、edubuntu、 Scientific Linux、B2D、 Solaris 10、PC-BSD、Knoppix、Stanix、SimplyMepis、OpenDesktop、 台北新店大豐國小的自由軟體學習光碟 Live-CD，當然有安裝失敗的 Slackintosh (mac)、SUSE (mac) 、Gentoo 及下載後還沒試用的 FreeBSD、K12LTSP、Helix、debian。如果讀者有興趣，請到 DistroWatch看看，種類多到嚇死你。

當然笨牛這種初學者走馬看花的試用是沒資格去評論一個作業系統， 但是這就是自由軟體的好處之一，想用那套作業系統就用那套。例如 笨牛安裝 Solaris 10，抓不到網路卡，笨牛就不用這套系統，因為 笨牛不會設定網卡，二來代表這系統還不夠友善，不適合給笨牛這種 沒程度的人用。目前笨牛喜歡用 B2D，或許下個月就用 Solaris 10， 或許明年就用 FreeBSD，因為笨牛的文件都是用 OpenOffice 存成開 放格式，所以不用擔心換作業系統會有相容問題，即使哪天被迫回到 微軟的世界，一樣可以用 OpenOffice。

6. 使用跨平台軟體：幸運地，笨牛的跨平台策略見效，笨牛才可以 順利切換到 Linux（這當然是基於眾多 Linux 及跨平台軟體工作者 的成果）。在蘊釀階段，笨牛就用 OpenOffice、星際譯王、Firefox、 Gaim 即時通訊……剛好這幾個軟體都內建在是台灣幾個知名 Linux， 也就是安裝了 Linux，這幾種軟體就在裏面啦！不像微軟的作業系統， 灌完了要更新、安裝硬體驅動程式、裝防毒及防火牆、灌辦公室及影 音軟體，你需要多少軟體就要灌多少軟體（而且要花錢）。而安裝 Linux 可能只要一片光碟 (CD or DVD)，全部的東西都在裏面，這是 一開機就可以用的電腦。資訊消費新主張：請賣給我一開機就可以用 的電腦。

7. 劉兄（劉彥均，是笨牛台大農化的同學，給予笨牛很多自由軟體 方面的指導）說笨牛可以順利在短時間內全面切換到 Linux，除了笨 牛打死不退的牛脾氣外，還有兩個重要的因素：一是不玩遊戲，或是 某些只有支援微軟的特定軟體（笨牛不玩大型遊戲，只有玩玩 Linux 內附的遊戲）；二是笨牛切入的時間點很好，劉兄說以前插入或是移 除隨身碟都要打指令，如今在 Linux 眾多工作者的努力之下，安裝 及使用都弄的很簡單，進步非常快，誰又能預測 Linux 下一步又有 什麼驚人的進步！！ 笨牛找了幾個討論在 Linux 玩 game 的網頁，希望有所幫助：

  1. 有人想在 Linux上面玩魔獸世界嗎？
  2. 為什麼linux玩一些3d遊戲都很慢

還有，笨牛正在等候 wine 的成熟。wine 是一個軟體可以在 Linux 環境下跑原先只支援微軟作業系統的軟體。例如笨牛的韋氏字典光碟 不支援 Linux，所以笨牛還在等 wine 的正式版及熱心人士的教學文 件，應該快了吧！請見：一瓶花了十二年才（幾乎）釀好的酒。

8. 不能報喜不報憂：笨牛目前有一些使用上的困難，但是整體而言 沒有問題。

  1. 笨牛的印表機及掃描器都是不支援 Linux（也不支援 Mac OSX）， 所以笨牛學到一個教訓，下次要買電腦硬體時，一定要上網 查查看，確認是否支援 Linux。
  2. 對於 Linux 的操作還是不夠熟悉，但是要完成手邊的工作已 經夠用了。
  3. 對學習新的軟體需要花時間，而且多數是英文介面，例如這 些軟體（取自 debian 套件）：
    1. fastdnaml：[Biology] A tool for construction of phylogenetic trees of DNA sequences.
    2. stars：star map program that draws the night sky.
    3. grass：Geographic Resources Analysis Support System
    絕不像 OpenOffice 有那麼多人在討論如何安裝及使用，所以 一定要花時間去讀說明書。有人曾開玩笑說："free software is not free"，可以解讀成從商業軟體換到自由軟體是需要代 價的，學新的軟體也是要花時間的（還好不用花錢）。話說回來， 難道花錢買的商業軟體就不用花時間學嗎？

9. 未來計劃： 除了繼續加強自己的 Linux 功力，多試試不同的作業 系統及應用軟體，多建議家人朋友使用自由軟體外，笨牛還有幾個的 目標：

  1. 學習製作 Live-CD：這是來自大豐國小 Live-CD 的啟發，想 想~~會不會有這麼一天，到汽修科報到的新生，或是護理、土 木、心理、外文、財金等各領域學生，學校都會發給一張光碟， 內含有作業系統及所需要的應用軟體，甚至有各老師編寫的教 材及題庫，學生只要準備硬體就好（而且硬體的需求不用太高）， 剩下的都在 Live-CD，然後不定期收到系上的新軟體更新通知， 只要上網更新就好，不用再燒光碟，使用上非常有彈性。想要 只用光碟內特定專業軟體也可以（例如統計軟體），用完後存 檔，退出光碟即可，想要安裝到硬碟也不錯。

    未來，教授對研究生說，試試這片光碟，所有的東西自由免費 又合法，除了平常上網、聽音樂、即時通等常用的，你在研究 所期間需要的專業軟體也都在裏面。從簡報、文書處理、數據 作圖、數據統計與分析、畫化學式、畫實驗器材及實驗流程圖、 寫論文時的插入文獻與編排都有，覺得不夠的話可以自己加， 或是發現更好的軟體，我們可以放入下一版的 Live-CD。

    笨牛不懂技術面的問題，但是大豐國小 Live-CD 是很好的學 習範本，目前欠缺的是各領域的整合。現在網路發達，這些東 西可以透過網路來整合，節省開會時間。例如想要製作生物領 域的 Live-CD，可以上網尋找生物相關領域的大學生、研究生 及老師，然後分為兩群人， 一群人幫忙找適合的軟體及編寫 使用手冊，另一群人試用軟體進行評估，並回報使用狀況，例 如軟體的優缺點等。最後請會製作 Live-CD 的人把所有的軟 體及使用手冊放在一張光碟，無論參加的是三、五人或是三、 五百人，都可以透過網路把大家串起來，不需要每個人都是電 腦高手，各有各的貢獻。第一版出現後相信會有許多批評與建 議，此時再進行改版就好。如果有數個領域已經完成，甚至可 以把東西放在網路上，以後只要上網安裝就好。舉個例子，假 如家庭成員可能有不同領域的專業需求，讀國高中的、大學或 研究所的，或者需要股市分析的軟體，只要上網點選所需的軟 體及教材（或題庫），如國中生物、高中英文、 大學微積分， 如此一來就不需要每個領域都發行一套 Live-CD。

    笨牛的規劃是畢業後在業界工作或是從事教職，當然希望量身 訂作自己領域適用的 Live-CD，所以一方面找尋功能夠用又容 易操作的軟體及作業系統，另一方面學著製作 Live-CD（或者 不用那麼麻煩，只要 rpm 或 synaptic，滑鼠按一按就好）。

  2. 參與中文化計劃：許多好用的軟體都是英文介面，如有中文介 面有多好！國內已有軟體中文化計劃，不過以笨牛近似於零的 電腦功力及滷肉腳的英文程度，想參與中文化是不可能的任務， 但是目標總是要訂難一點，如果實力不夠無法參與時，至少笨 牛可以幫忙加油吶喊。請見自由軟體中文目錄計劃。
  3. 編寫使用心得及教學文件： 一方面當作自己的學習日記，另 一方面可以分享給家人朋友或是有需要的人，就像有這麼多熱 心人士願意在網路上分享經驗。至於教學文件，當然不是要重 新發明輪子，既然有現成的教學文件，笨牛只要把收集連結就 好。如果有自己領域常用但是網路上找不到教學文件，那再來 考慮自己寫文件。

理想說的再多再美好，還是要從本身作起，請祝福笨牛吧！

[Q6]： 既然笨牛兄只是新手，為何有膽子寫這些建議？難道不怕誤導別人？不怕被嘲笑？

1. 笨牛的臉皮比牛皮還厚，不怕別人嘲笑。上述的經驗都是笨牛跌 跌撞撞摸索出來的，換個角度來看，可以代表新手可能遇到的困難， 雖然不見得是最聰明的或最正確的，但一定是可行的。而且笨牛認為， 要讓更多人來使用自由軟體，就要先解決入門問題，讓更多人可以無 痛入門以及勇於戒毒（戒除商業軟體的毒），當然笨牛只是分享個人 戒毒的過程， 讓高手們可以知道新手可能遇到的困難，以協助新手 更快上路，也讓想要嘗試自由軟體尚未入門的人有個概念。

2. 之前與劉兄討論，劉兄非常贊成並鼓勵笨牛寫下這些心得，因為他已經忘記幾年前開始使用 Linux 的經驗了。幾次與劉兄請教一些 Linux 使用上的問題，劉兄說一句話，笨牛試了 30 分鐘才了解他在說什麼，這也是笨牛想寫這篇心得的原因之一。想想看，十年後笨牛可能從新手升級到中手，也有可能忘記那段摸索期的痛苦，所以趁現在記憶還鮮 明，趕快紀錄下來。而且笨牛很阿 Q 的認為，這篇心得若寫得不好，一定會有人跳出來批判，或看不下去自己寫一篇（這就不是拋磚引玉了嗎？），有批評才有進步。

◎ 結語

笨牛已經把自己領域所需的軟體找齊了並寫下心得，歡迎其他領域的 仁人志士分享各領域的經驗，無論是會計、語文、工程、地理、氣象、 藝術等，這樣就可以吸引更多人使用自由軟體嗎。

笨牛感謝全台灣及全世界為自由軟體奮鬥的前輩們，沒有你們的付出， 就沒有這麼多自由的軟體可以用；感謝台南縣網及 B2D 的作者們， 有了他們笨牛才有 B2D 及許多有用的資訊；感謝許多前輩在網路上 分享使用經驗；感謝劉兄的電話教學及鼓勵。也祝大家快樂的使用自 由軟體！
