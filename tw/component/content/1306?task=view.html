紅帽推出 Amazon 運算網雲的紅帽企業版 Linux

建立日期 2007-11-13 08:00      最近更新在 2007-11-26 02:28 作者是 謝良奇/編譯
紅帽（Red Hat）推出了紅帽企業 Linux 的 Amazon 彈性運算網雲版本（Red Hat Enterprise Linux on Amazon Elastic Compute Cloud），這是紅帽 Linux 自動化策略中的一步，希望給帶給用戶簡化應用軟體運作與管理的 Linux 基礎架構。在此一合作下，Amazon 彈性運算網雲（Amazon Elastic Compute Cloud，Amazon EC2）網路架構與資料中心的使用者，可以使用到紅帽企業版 Linux 的所有功能，如紅帽網路（Red Hat Network）管理服務、世界級技術支援與超過 3400 種認證應用軟體。

Amazon EC2 是一套網路服務，在網雲中提供可彈性增減的運算能力。Amazon EC2 與紅帽企業版 Linux 的結合改變了運算的成本模式，用戶只需支付實際使用到的架構軟體服務與計算能力。紅帽企業 Linux 的 Amazon 彈性運算網雲版本可以讓用戶在幾分鐘內增加或減少運算能力，免去為了偶發的需求高峰必須超量購買軟硬體資源的困擾。

紅帽企業版 Linux 整合了虛擬化技術，針對就地（on-premise）與網雲運算（cloud computing）提供無接縫的部署解決方案，方案中由紅帽網路提供跨越就地部署與網雲運算環境的共同管理與自動化工具。紅帽將針對紅帽企業 Linux 的 Amazon 彈性運算網雲版本提供技術支援與維護，成為 Amazon EC2 上第一套商業支援作業系統。

紅帽線上服務副總裁 Donald Fischer 表示，藉由與 Amazon 網路服務（Amazon Web Services）合作，他們提供用戶部署紅帽企業 Linux 平台的另種選擇，對於期盼以迅速、高成本效率部署網路規模（web-scale）服務的開發者與用戶，以及需要高規模彈性運算資源的企業來說，是相當具有吸引力的。紅帽企業版 Linux 與 Amazon EC2 服務的結合讓專業網路規模運算成為現實。

Amazon 網路服務產品管理與開發者關係副總裁 Adam Selipsky 認為，他們很樂意與紅帽合作，為 Amazon EC2 使用者提供更多選擇。此一合作可協助用戶省去部署與管理自有基礎架構時的沈重負擔。

紅帽企業 Linux 的 Amazon 彈性運算網雲版本目前僅提供非公開 beta 版，公開 beta 版本預計於 2007 第 4 季推出，基本售價為每人每月 19 美元，Amazon EC2 服務依使用者選擇小、大或超大的運算資料量，每小時為 0.21、0.53、0.94 美元，此外還包括頻寬與儲存空間費用。
