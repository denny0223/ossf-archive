Debian.Org.Tw 成員 Kanru Chen 專訪

建立日期 2005-10-10 08:00      最近更新在 2011-01-12 18:41 作者是 Jim Huang
（編按：本文作者 jserv 為自由軟體專案的開發者與維護者，亦為 DOT（Debian.Org.Tw）成員之一，希望透過一系列 DOT 人物專訪， 讓更多讀者對 Debian 台灣的社群發展有所認識，進一步肯定台灣自 由軟體社群開發者的貢獻。） jserv問（以下簡稱 "Ｑ"）：請您作自我簡介。
kanru答（以下簡稱 "Ａ"）：我是 Kanru Chen（陳侃如），通常在 網路上的暱稱是 kanru，目前是暨南大學資工系三年級學生，詳細的 資訊可參考個人網頁[1] 與 Blog[2]。

Ｑ：何時對 Debian 產生興趣，又是如何參與 Debian 的發展？
Ａ：嗯，大約是在 RedHat 8.0 或更早的國旗事件那時候（那是什麼 時候？），開始換到 Debian ，其實之前早就改用 Mandrake 加上 CLE 提供的 apt porting，優秀的套件管理系統是我想要嘗試轉換到 Debian 的主因。

Ｑ：明確來說，是高中就學的時候？那時候學校有跟 Linux 相關的社 團嗎？
Ａ：一開始的參與不能說是直接接觸到 Debian 的核心成員與活動， 主要是在 Moto 討論區問問題跟回答問題。差不多就是高三兵荒馬亂 的時候，那時候學校沒有什麼 Linux 的相關社團，不然就是我接觸 太少，不熟悉資訊社團的關係 :)

Ｑ：所以，那個時候，基本上都是從網路上擷取所需要的經驗，以及 自己的嘗試了？
Ａ：沒錯，那時候家裏剛接好 ADSL ，所以有比較好的環境可以讓我 長時間在網路上搜尋資料，Moto 上也有許多熱心的人幫忙。不過， 就像許多初學著一樣，家裏還有一大堆書，雖然不是 Debian 的書 籍，但也常常參考 :)

Ｑ：嗯，那個時候主要接觸 Debian 的哪些項目，比方說桌面系統、 server 架設，或者程式設計一類的？
Ａ：最主要是程式設計，這是我接觸 Linux 的主因，Server 架設也 有接觸，不過因為沒有可以實驗的環境，所以很多都只是把資料看過 ……桌面系統因為天天都要使用，所以常常有需要熟悉、調整，久了 經驗就多了。不過難度好像正好相反，網路上詢問 Desktop 的人最 多，Server 次之，程式設計幾乎沒看到。

Ｑ：所以就這些項目來說，你認為 Debian 與之前接觸的 Redhat 或 其他 RPM-based 的系統有什麼吸引你的因素？
Ａ：我很喜歡測試新的東西，這時後方便安裝就很重要，跟 RPM-based 的其他系統比起來，現在的 Debian 是做的比較好，要取得現在使用 的程式的 source 也很方便。

Ｑ：對 Moto 學園[3]哪些朋友較有印象，並且有互動呢？
Ａ：最早的時候就屬阿信（Asho）最活躍了，很多問題都可以得到立 即的回答，現在則是 Tetralet 很熱心的一直在回答問題。我發現我 在 Moto 上問的問題其實不多，後來就開始幫忙回答問題了，然後就 開始潛水，現在在 Moto 上認識的人不多。上大學之後，有許多的時 間可以掛在聊天室上，也才因此認識 AndrewLee、chihchun、jesse、 AceLan、jserv 等人。

Ｑ：在你接觸 Debian 的時候，台灣的 Debian 社群組織也剛起步， 你在其中扮演什麼角色呢？
Ａ：跟 Debian 比較相關的應該是套件打包為主吧，就在 IRC 上面 當測試員，有什麼新的東西就拿來玩玩，有空的話就幫忙寫 patch ，包套件（package）等等。

Ｑ：可以具體的談談參與台灣 Debian 社群的項目嗎？比方說讓你覺 得投入的動力？
Ａ：在 IRC 上，很多事情都可以參一腳。投入的動力應該就是回饋 社群吧，有些東西都是我會用到的，我想要讓 Debian 有的，既然做 了，那就分享出來給大家一起用。現在維護了幾個套件，還有 APT.DOT 服務，都是基於這個原因。APT.DOT 是類似於 mentors.debian.net 的服務，讓華文的使用者有較新的相關套件可以使用，這是在缺少官 方 developer 的情況下，可有個共同的地方可以放置自製的套件。

Ｑ：Debian 是發展自由軟體相當理想的平台，我想你應該也同意， 請談談你參與開發的自由軟體。
Ａ：第一次接觸到自由軟體的開發是幫 libchewing 寫 patch，讓 UTF8 環境下的 XCIN 也有酷音可以用，接下來又陸陸續續的參與了 許多酷音相關的計畫，像是 SCIM-chewing，OV-chewing 等等，另外 幫忙 PCMan 移植 PCManX 到 Gtk2 上。其他還陸陸續續幫一些自由 軟體專案寫了一些小小的 patch :)

Ｑ：至於網路社群以外的環境，是否有機會使用 Debian 呢？
Ａ：我在學校有幫忙管理幾台伺服器，可是這些機器都是裝 FreeBSD， 因為年代久遠，所以我也沒有轉移到 Debian 的打算，倒是有幫一些 同學、學弟裝 Debian :)

Ｑ：你現在就於讀資訊工程系，是否有機會使用 Debian 或者其他自 由軟體來處理課業需求？
Ａ：我幾乎都會以自由軟體來處理課業需求 :) 相較於其他同學，因 為我使用自由軟體，所以我可以選擇的工具多的多了，之前才幫學弟 寫了篇「一定要 MATLAB？」[4]

Ｑ：作為一個活躍的 DOT（Debian.Org.Tw）成員，你希望未來 DOT 會有什麼新的風貌？
Ａ：我希望能有更多的人參與，提出更多有趣的計畫 :) 現在 DOT 有許多熱血的計畫希望可以有更多人參與，特別是年紀較輕的新血 :) AndrewLee 有計劃在花蓮地區的國小推動 Debian 的使用，希望透過 在校園的推廣，可以吸引到更多的人參與自由軟體 :) 另外固定的聚 會，無論是網路上的，或是現實生活中的，都令人期待。

Ｑ：很高興你接受這個訪談。
Ａ：謝謝！
