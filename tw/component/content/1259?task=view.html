Ubuntu 7.10 釋出 加強硬體支援、3D 桌面效果與 Windows 相容支援

建立日期 2007-10-16 16:00      作者是 謝良奇/編譯
公認易於安裝、使用的 Linux 散佈套件 Ubuntu，新版 7.10 日前由 Ubuntu 的商業贊助商 Canonical 公佈釋出消息。鎖定一般用戶的 Ubuntu 打破了 Linux 只適合電腦技客的刻板印象。新版 Ubuntu 在桌面視覺效果、設備支援以及對 Windows 的相容性，有相當多的改進。

超人氣的 Linux 散佈套件 Ubuntu 在代號 Gutsy Gibbon 的 7.10 版中，引入了 GNOME 2.20 桌面系統，來強化系統的使用者介面。Ubuntu 與 GNOME 的團隊針對使用者介面下了許多工夫，在新的圖形化介面下，修改顯示器設定變得更為簡單，對於筆記型電腦的使用者，Gutsy Gibbon 完整支援外接 VGA，可以輕鬆支援投影機切換。對於高階使用者，甚至可以進行多重螢幕的設定。

新版的 Ubuntu 特別在硬體支援方面做了改善，包括印表機的即插即用設定以及 Broadcom Wi-Fi 網路卡的韌體自動安裝。

除了最新版 GNOME 提供了改進後的拖放支援外，Ubuntu 7.10 包含了 3D 桌面效果套件 Compiz。Ubuntu 7.10 會自動偵測硬體是否適合執行 Compiz，系統無法執行 Compiz 時，將會自動切換為一般桌面。

有趣的是，這個酷眩的桌面效果是否有助於使用者生產力，Canonical 執行長同時也是 Ubuntu 創辦人 Mark Shuttleworth 也沒有確定的答案。他說，部份使用者可能會不適應這個前衛的顯示技術。不約而同的，新一代作業系統，如 Windows Vista 與 Apple 的 OS X Leopard，也都納入了這類的 3D 特效。

多虧了開放源碼驅動程式 NTFS3G，讓 Linux 與其他 Unix 系統有了讀取與寫入 NTFS 的能力，Ubuntu 7.10 在讀取與寫入 Windows 磁碟分割區的表現上，有相當大的改進。Shuttleworth 表示，這項功能對於有 Linux、Windows 雙重開機需要的用戶，可說是一大福音。Ubuntu 使用者可以從 Windows XP 與 Vista 檔案系統中，讀取與寫入檔案。用戶甚至可以輕易在雙重開機系統中的 Ubuntu 讀取，放在 Windows 系統中的舊有專案。

Gutsy Gibbon 和前一個版本相同，都採用了 Live 光碟。Live 光碟可以讓用戶從 DVD 光碟機開機，無須動到現有系統而試用 Ubuntu。確定要進行安裝時，點選安裝選項即可進行安裝步驟。安裝過程簡易，直逼 Mac OS X。如果是選用與 Windows 系統並存的雙重開機安裝，使用者還可以讓 Ubuntu 匯入之前的設定與檔案。

許多類似 Mac OS X 的桌面功能，這次被引進 Gutsy Gibbon 中。新增的使用者切換功能，有如 OS X 無須使用者登出，即可在使用者帳號間切換。此外，Ubuntu 7.10 也有了類似 OS X Spotlight 的桌面搜尋功能，可搜尋檔案、資料夾、對話記錄與相片等。Ubuntu 7.10 的每個印表機對話視窗，都增列了虛擬的 PDF 印表機。這個 Mac OS X 使用者熟悉的印表機選項，可以讓各種應用軟體輸出 PDF 檔案。

儘管桌面系統市佔率調查公司 Net Applications 指出，Linux 桌面系統市佔率也許仍不到 1%，但包括開始供應預先安裝 Ubuntu 桌面和可攜式電腦的 Dell Computer 在內，近來有關 Linux 桌面系統的發展，顯示 Linux 桌面系統並非毫無前景。

個人電腦也不是 Linux 未來唯一的戰場。Shuttleworth 表示，除個人電腦外，各種可能使用 Linux 開發的設備，也都是 Ubuntu 的目標。他在一場會談中表示，如今許多消費電子產品，其設計基本上就是個小型電腦。他們雖然還未涉足這塊市場，但情況到了 2009 年中後，將有所轉變。屆時 Linux 將出現在消費者眼中的熱門商品內。
