Linux 上的視訊會議軟體—Skype、GnomeMeeting

建立日期 2005-02-18 08:00      最近更新在 2007-03-09 01:24 作者是 謝良奇
日前全球網路電話公司 Skype 正式推出 Skype 的 1.0 Linux 版本， 囊括更完整的功能。但如今有音訊，甚至視訊通訊需求的 Linux 用戶，除了去年開始火紅的 Skype，更多了 GnomeMeeting 可供選擇。此外，有可能移植至 Linux 的 StanaPhone 也提供了額外的選擇。 GnomeMeeting 則以視訊、音訊兼容為特色。

Skype 為全球 VoIP 的領導產品，擁有兩千三百萬註冊用戶，目前該 軟體支援 Linux、Mac OS X、Windows 與 Pocket PC 平台。1.0 版 的 Skype for Linux功能包括 Skype 對 Skype 的免費通訊、多達五人共同參與的會議通話、跨平台通訊及豐富的個人化功能，當然也有 能撥打至市內及行動電話的 SkypeOut。 Damien Sandras 於 2000 年開始發展的 GnomeMeeting，則是融合了 音訊與視訊的會議軟體。GnomeMeeting 提供了針對 Debian、Fedora Core 2、Slackware、Mandrake 和 SUSE 的預編譯版本，此外也有原始碼可供下載取得。目前處於 1.2 版階段的 GnomeMeeting 僅支援 H.323，但 GnomeMeeting 網站已表示這是最後一個僅支援 H.323 的 版本，下一版的 GnomeMeeting 將同時支援 H.323 與新的 VoIP 技術 SIP（Session Initiation Protocol）。 在硬體方面，GnomeMeeting 需要支援 Video4Linux 的攝影機，不過 視訊並非必要需求。音效卡則可使用相容於 OSS 或 ALSA 的裝置。 GnomeMeeting 提供 Configuration Druid 協助用戶設定所有元件， 在安裝上相當簡便，其中包括音訊管理、NAT 設定、視訊管理等等。 GnomeMeeting 1.2 版支援了 ENUM、ZeroConf 和 PC 對電話通訊。 ENUM 可以讓用戶將真的電話號碼連繫至 H.323 位址，ZeroConf 協 定能探測網路上的其它設備，PC 對電話通訊功能則能讓 GnomeMeeting 與一般電話建立通訊。 宣稱為第一套整合真實美國電話號碼的免費電話系統 StanaPhone，與 Skype 最大不同處在於用戶會取得真的電話號碼。StanaPhone 並 支援 SIP，理論上同樣支援 SIP 的 Gnomemeeting 應能相容於此項服務。StanaPhone 乃基於 SJLabs 的 Java 軟體，但目前僅有 Windows 與 Pocket PC 版本，未來有可能移植至 Linux、Mac OS X 與 PalmOS 平台。
