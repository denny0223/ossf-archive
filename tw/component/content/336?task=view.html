Mandriva One 2006 釋出 簡單易用為主打訴求

建立日期 2006-05-26 08:00      最近更新在 2007-03-09 01:20 作者是 謝良奇
法國 Linux 散佈套件商 Mandriva 宣佈 Mandriva One 2006 正式 iso 映像檔已經正式公開於 FTP。此版本的 live 與安裝 CD 採用的是 2.6.12 版核心，以 KDE 作為預設桌面環境，附加許多額外的應用軟體。此外，Mandriva 也公開了軟體安裝服務稱為 Mandriva Kiosk，要讓新手也能輕鬆在 Linux 上安裝開放源碼軟體。

早於三月已公開的 Mandriva One 2006，當時僅提供給 Mandriva Club 會員。如今 659 MB 的 Mandriva One 2006 Linux 散佈套件 iso 映像檔，已經可由 FTP、HTTP 映像站台公開下載。此外也提供 BitTorrent 下載管道。 Mandriva One 2006 內含括了 Firefox 1.5.0.3、Evolution 2.2.3、 GAIM 2.0 Beta 3、OpenOffice 2.0、GIMP 2.2.8 與 Mindawn 音樂播放軟體等應用程式。 Mandriva One 2006 的 Live CD 可以讓用戶在無須改動系統的情況下，先行體驗 Mandriva Linux 系統。 Mandriva Kiosk 是 Web 上的軟體安裝服務，強調只需滑鼠單點就能輕鬆存取最新版的應用軟體。Mandriva 團隊也會提供用戶建議的應用軟體，供用戶選擇之用。 軟體以套裝方式先行包裝完成，Mandriva Kiosk 的用戶不具備技術能力，就能依需求自行安裝所需環境。這種軟體套裝大幅簡化了 KDE、GNOME 或 OpenOffice.org 這類複雜系統的安裝。也因此 Mandriva Kiosk 搭配 Mandriva One 能夠滿足從 Linux 新手到專家的各種需求。
