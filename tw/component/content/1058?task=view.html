應用 Linux 與開放源碼 對環保盡一分力

建立日期 2007-06-25 07:33      最近更新在 2011-01-12 19:37 作者是 Linuxpilot
微軟希望透過 Windows Vista 推動業界發展，但是根據研究機構 Softchoice Inc. 向北美 472 家機構作出的調查，超過一半受訪者表示他們未準備好升級至 Windows Vista，同時結果顯示有 94% 的電腦未能滿足 Windows Vista Premium 的規格要求，意味機構需要升級或更換系統，相信會對環境保護產生很大影響。目前香港政府正在草擬法例，準備徵收環保稅，估計每台棄置電腦的處理成本約為港幣 60 元（約新台幣 252 元）。 當微軟的 Windows Vista 在間接鼓吹廢棄電腦的同時，香港 Linux 商會聯同多個業界組織主辦一個名為「Linux 及開放源碼是否綠色運算的正確方向」研討會，希望從積極層面鼓勵業界善用現有資源，不是一味製造更多電腦廢物，破壞環境生態，當日與會嘉賓從不同角度分享 Linux 及開放源碼軟體如何成功推動環保。

◎ 省電與環保的關係

研討會先由香港專業教育學院（屯門分校）資訊及通訊科技系講師賴國華及林振帆開始，從省電與環保的關係比較 Windows 與 Linux，相信大家都有經驗使用 Windows 開機，很多時由於安裝的應用程式太多，啟動速度變得緩慢（隨時要幾分鐘）。根據他們的研究與測試，在使用同一硬體規格的電腦（P4 2.66G 處理器、512 MB 記憶體及 20 GB 硬碟），一台新安裝的 Windows XP 需要 21 秒才能進入登入畫面，登入後再需要 20 秒才能開始工作，而 Linux 由於源碼開放，可以透過使用優化核心加快開機速度，及停止不需要的服務，結果整個開機時間只需 24 秒便完成（較 Windows 快 41%）。

由於 Linux 開機時間比 Windows 少 17 秒，可節省 0.001653KWH 電力，假設全香港有 100 萬台電腦需要每日開機一次，轉 Linux 將可節省 1653 KWH 電力，每日可節省港幣 1,488 元，一年下來便可節省港幣 54 萬 3,120 元（約新台幣 228 萬 4,055 元）。除了電力之外，由於Windows對硬體規格要求極高，如果在較低規格的系統使用 Windows，速度將遠低於 Linux，另外 Linux 的系統修補及升級檔發布速度更快，相信可提供更好的性能及安全性，最重要是 Linux 可延長電腦壽命，對環保作出貢獻。

◎ 澳門生產力中心成功個案

大會接著安排了三個用戶現身說法，其中澳門生產力暨科技轉移中心資訊科技部經理湯家旭以系統管理員身份道出轉用 Linux 的好處，湯家旭表示：「在澳門生產力中心屬下的 IT 培訓中心「數碼匯點」，我們用 Linux 作為文件伺服器、路由器/防火牆/VNP 伺服器、DNS 伺服器、DHCP 伺服器、WWW 伺服器、系統管理伺服器，以及網域控制器。在整個生產力中心，我們使用 Linux 作為電郵伺服器（同時作為過濾垃圾郵件及病毒的網關）、WWW 郵件伺服器和支持我們的核心業務系統的資料庫伺服器。」為甚麼中心會大量使用 Linux？湯家旭在會上分享背後原因：「授權費是一個因素，但它不是唯一的原因。實際上，縱使 Linux 和 Windows 的授權費相同，我也會選 Linux 而不選 Windows，因為 Linux 容易管理得多，可減輕 IT 人員的負擔。」

◎ 克服困難始能先苦後甜

Linux 這麼好，是不是人人都應該用 Linux呢？湯家旭提出其個人意見：「不是。為了得到上述好處，您的 IT 員工必須捱過入門時較艱辛的日子。通常， Linux 軟體有數以百計的配置設定和成千上萬種組合（以支援最罕見的情況） 。因此，要配置 Linux 軟體就要先花相當的時間來學習其設定。相反，要配置 Windows 的軟體通常只需要不斷點擊『確定』，而預設的配置好多時都可以工作。如果貴機構的 IT 人員較醒目而且願意學習新事物，想以更短的時間解決 IT 問題，想要達到更大的可用性，想要用最少 IT 人手達到最大的效能，機構的業務要求度身訂造的 IT 服務，Linux 將會是一個值得考慮的選擇。」

◎ Linux 系統管理較 Windows 優勝之處

1) 容易保持系統更新：在 Linux 中，每當哪個軟體包有安全性更新時，我們將收到電郵通知。相反，Windows 更新操作只為微軟的軟體服務，而第三方軟體就不行。
2) 容易升級整個系統：在 Linux 中，我們可以把整個系統升級到下一個主要版本。理論上，Windows 同樣可以這樣升級，但實際上，我曾經嘗試把一部 Windows NT 伺服器升級為 Windows 2000，升級後很多東西都不再正常工作。
3) 容易進行故障排除：在 Linux 中，如果某些東西不能運作，我們通常能在記錄 (log) 中找出問題所在，如有需要可指定 verbose 選項，因為開放源碼軟體通常會記錄很多資料。相反，Windows 的軟體一般只會告訴您一些簡短的錯誤提示，很難進行故障排除。
4) 容易自動化：在 Linux 中，我們可以使用內建的命令來編寫一個簡單的指令稿 (script)，跨越 WAN 來進行網路備份或檔案同步。我們甚至寫了一個指令稿來每日重置教室網域的系統管理員密碼。在 Windows 中，我們要辦到此事必須找尋額外的軟體或特意編寫程式。
5) 容易更新：在 Linux 中，在使用中的檔案能像普通的檔案一樣被更新。所以我們很少需要在更新後重新啟動電腦。在 Windows 中，更新瀏覽器都要重新開機！所以使用 Linux 意味著用戶可以享受更高的可用時間。

最後，Specialized Technology Resources (H.K.) Limited 資訊管理部經理 Thomas Lau 指出，轉用 Linux 對機構生產力有 25% 的提升，而智網電子商貿服務有限公司產品研發部總監馬恩明則表示使得 Linux 的開放平台，使他們成功開發了商貿易網上報關服務，未來公司將會增加更多 Linux 系統以應付業務發展需要。
