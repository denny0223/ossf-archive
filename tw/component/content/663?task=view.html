「YAPC Taipei」研討會三月登場

建立日期 2004-02-12 16:00      最近更新在 2011-01-11 20:09 作者是 elixus
繼 2003 年 11 月舉辦的「Perl入門課程」之後，台北 Perl 推廣組將於今年三月下旬，於台北開羅會議中心舉辦「YAPC::Taipei」Perl 會議。在為期兩天的議程中，將邀請國內及國際知名的 Perl 專家，就協同開發、性能調校與流程管理等軟體技術議題進行演講，同時介紹實用的 Perl 工具。 這場由台北推廣組主辦、奇科資訊、自由軟體鑄造場協辦的「YAPC::Taipei」Perl 研討會，鑑於國內 Perl 使用者日眾，決定以「Perl 與程式開發者」作為今年的主題，並且配合技術議題與開發實例討論，讓與會者對於將 Perl 當成開發工具，有豐富的資訊與進一步了解。

國外講者方面，目前已邀請到自由軟體界最受歡迎的事務追蹤系統（RT）的作者 Jesse Vincent、文件共同撰寫系統（Kwiki）作者 Brian Ingerson，以及 Mason 計畫主要開發成員之一的 Dave Rolsky。Mason 是一套應用廣泛的網站開發架構，從 2002 年起就獲得美國 Amazon 網路書店的採用。

另外，該研討會另邀請到 Perl 社群公認最為優秀的講者 － Mark-Jason Dominus 與會，暢談如何撰寫出更具效率的 Perl 程式，以及如何進行一場精采的簡報。每場國外講者的議程，均提供現場同步口譯。

台灣方面，講者包括中央研究院「自由軟體鑄造場」（OpenFoundry）技術組的黃俊穎，介紹鑄造場的實作經驗。而「自由軟體鑄造場」系統整合的主要 負責人唐宗漢（Autrijus Tang）也將公開鑄造場背後的技術架構，並說明 Perl 在這項工作中的運用。同時，最近深入研究及積極參與 Subversion 版本控制系統開發，並且用 Perl 撰寫新一代版本控制系統 SVK 的高嘉良（Chia-Liang Kao），也會針對利用 Perl 進行版本控制的應用，進行技術上的介紹。最後，該研討會還會展開「自由軟體鑄造場」的相關討論，也看看這些工具的整合能帶來什麼有趣的應用。

Perl 是由 Larry Wall 在 1987 年開發的程式語言，廣受程式開發與系統管理人員的喜愛。目前的最新穩定版本為 Perl 5.8.3、開發中的版本是 Perl 5.9.0、規劃中的版本則是 Perl 6.0。

台北 Perl 推廣組成立於 1999 年，為區域性的 Perl 使用者團體，致力於 Perl 相關的推廣活動。除每月常態性的集會之外，曾於去年 3 月承辦 P3P（Perl/Python/PHP Party）會議，也與美商歐萊禮台灣分公司與培生出版社合作，翻譯一系列的 Perl 書籍。

台北 Perl 推廣組誠摯邀請對 Perl 語言有興趣的朋友，參與此次的盛會，詳細資訊可參考 https://yapc.elixus.org/YAPC/2004。
