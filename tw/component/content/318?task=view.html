Intel 釋出支援Linux Centrino 驅動程式 試圖融入社群

建立日期 2004-06-11 08:00      最近更新在 2007-03-09 01:30 作者是 謝良奇
Linux 上終於有堪用的 Intel Centrino 驅動程式了。Intel 日前在 SourceForge 上釋出了 802.11b/g Wi-Fi 的 0.1 版 "pre-beta" Linux 驅動程式，補足了原有的 802.11b Linux 驅動程式。Intel 官方表示，最終的驅動程式程式碼暫定 2004 下半年釋出。

Intel 的 Centrino 平台佔目前輕薄型筆記型電腦市場的大宗，但該技術目前僅提供 Windows 平台的支援。對 Linux 使用者，特別是將 Linux 裝在筆記型電腦上的使用者，缺乏可用的 Centrino 驅動程式一直是個困擾。eWEEK 的 Steven J. Vaughan-Nichols 就以自己的 Centrino 使用經驗，彰顯 Linux 在硬體支援上的困境。一位擅長 Unix、Linux 技術的玩家，面對 Centrino-enabled 的筆記型電腦，仍得一番奮戰，才能驅動 Centrino Wi-Fi。即使如此，使用上還有許多問題。根據 Intel 在 SourceForge 上的公告，這個 Linux 驅動程式是在缺乏硬體文件下的社群成果。目前 Intel 802.11b Centrino 的 Linux 驅動程式 0.45 版，包含了 ad hoc 網路的基本支援。802.11a/b 則為 0.1 版。Intel 發言人 Barbara Grimes 表示，該公司尚未有針對 802.11a/b/g 驅動程式的公開開放源碼 Linux 驅動程式計畫。雖然該驅動程式會用開放源碼模式釋出，但她未說明將採用的授權模式為何。過去 Intel 一直是自行開發驅動程式。Linux 的驅動程式開發者，在沒有硬體廠商支援的情況下，往往要身兼二職，既是開發人員，又要扮演偵探的角色。Grimes 表示，Intel 正著手融入社群之中。這也許是對雙方都有利的妥協關係。Grimes 說，這對他們是或多或少有些陌生和新鮮。
