MEPIS 以 Ubuntu 取代 Debian 釋出 SimplyMEPIS 6.0 測試版

建立日期 2006-03-22 08:00      最近更新在 2007-03-09 01:19 作者是 謝良奇
Debian Linux 散佈套件商 MEPIS LLC 以 Ubuntu 取代 Debian 做為其 SimplyMEPIS Linux 散佈套件的基本核心，整合 Ubuntu Dapper套件庫的 SimplyMEPIS 6.0 測試版本也隨之釋出，成為第一套基於 Ubuntu 的 SimplyMEPIS。

MEPIS 創辦人 Warren Woodford 表示，Ubuntu 穩定的 6 個月釋出周期讓 MEPIS 得以提供用戶更可靠的釋出計畫。轉移至 Ubuntu 可提供用戶更為穩定的底層系統。他相信這次的釋出版本會證明他們有能融合 MEPIS 的使用者體驗與 Ubuntu 的優點。 Woodford 指出轉換至 Ubuntu 的過程比想像中簡單許多。他表示在發展 SimplyMEPIS 3.4.3 時，Debian Etch 的基礎套件就和 UbuntuBreezy 套件十分相似。因此 Woodford 認為整個轉換工作的難度，應該和將 Ubuntu 從 Breezy 升級到 Dapper 相去不遠。 他補充說，當然還有許多實作與風格上的差異，他們會將 Ubuntu 的差異整合到 MEPIS 之中，同時切合他們的模式與願景，此外，他們也會將成果回饋至 Ubuntu。 Woodford 認為 MEPIS 與 Ubuntu 間的互動也有助於改進 MEPIS 與Debian 間的程式碼交流。他表示 Ubuntu 與 Debian 的關係相當緊密，Ubuntu 會定期取得 Debian 尚未穩定的版本，加以改善除錯後回饋至 Debian 原始碼樹。一旦 MEPIS 與 Ubuntu 的合作有助改善 Ubuntu，最終 Debian 也會因此受惠。 Ubuntu 之父 Mark Shuttleworth 表示，與 MEPIS 的合作將有助 Ubuntu 為 KDE 用戶提供更高品質的桌面套件，他們在系統整合、桌面加強與 Linux 核心可靠度上的努力，也能因此讓更多人受惠，能與 MEPIS 這樣一個充滿活力的社群一同合作，他們也相當開心，並且同時也會尊重 Ubuntu 與 MEPIS 彼此間的獨特差異。 針對 SimplyMEPIS 6.0 測試版本，Woodford 表示有許多整合上的問題與新功能尚未一一處理，Dapper 套件庫也仍在持續改進中，先推出測試版本，是希望事先了解用戶想法。此試驗版本可由 MEPIS 的訂戶 ftp 網站下載，有任何意見可發信至 這個 E-mail 地址已經被防止灌水惡意程式保護，您需要啟用 Java Script 才能觀看 。
