縱論中外開源運動發展及其策略

建立日期 2008-04-11 08:00      最近更新在 2011-01-11 18:15 作者是 Linuxpilot
較早前在廣州召開的「開源中國，開源世界」高峰論壇圓桌會議上，中國開源軟件推進聯盟主席陸首群將中外專家的發言紀要摘錄。Linuxpilot 現轉載給各讀者參考。圓桌會議主席中國開源軟件推進聯盟主席陸首群表示：「Linux 在伺服器領域技術上已經成熟，在桌面領域已經可用。Linux 桌面尚需克服三大障礙：（1）用戶的使用習慣，（2）驅動支援，即自動識別各種硬體並順利匹配驅動程式，（3）豐富的應用軟體。中國的 Linux 系統與國際水平尚有差距，近年來這個差距正在以驚人地速度縮小。」 ◎ Linux 市場佔有率增長

「據 Saugatuck Technology 公司預測，到 2011 年底，世界上接近半數的商業系統將在桌面或伺服器領域選擇使用 Linux。今天，Linux 在移動和嵌入式領域更以驚人地速度增長。Linux 桌面系統在解決驅動支援和應用軟體方面取得了突破性地發展，像 Dell 一樣的 PC 供應商已應用戶需求開始向用戶提供 Linux 桌面系統。我們承認 Windows 是最成功的作業系統，但 Linux 現在也是主流技術平台，可以說，整個 IT 行業現在已進入了 Linux 和 Windows 兩個平台的新階段（即形成主流作業系統的雙元系統）。為了令 Linux 進一步成功，在雙元系統發展階段，Linux 要揚長避短。」國際 Linux 基金會執行主任 Jim Zemlin 說。

◎ Linux 的生態系統

「Linux 開放性的開發模式是其最大的優勢，要使中國朋友清楚地意識到，採用開源的開發模式，可使中國在 IT 領域取得更強的競爭力，其他國家已經開始行動了，中國面臨著挑戰。我認為開放源碼是中國唯一的選擇。我感謝陸主席邀我參加在中國召開的開源論壇，我也希望給他發一個邀請：如果中國開源軟件推進聯盟願與國際 Linux 基金會合作，在中國共同召開一個 Linux 開發設計技術峰會，我們將很願意把美國和歐洲的一些主要開發者請來，其中包括 Andrew Morton 和其他 Linux 主要開發者，與中國同行見面和對話。Linux 和開源在互聯網上的優勢是明顯的，Web 2.0 是在 Linux 和開源基礎上建立起來的。Linux 的今天，需要建立一個完整的生態系統。迄今為止，Linux 已形成一個數十億美元的生態系統。」JimZemlin 說。

◎ 全球開源成主流

「目前在全球開源已開始成為主流。過去我們談及的『開源發展在社區』，其實除社區作貢獻外，作為開源成功的因素還有企業。85% 的社區開發的志願者來自企業（特別是跨國企業），而不是社會上分散的志願者。」Intel 開源總監 Dirk Hohndel 說。「以 Intel OTC（開源@Intel）為例，有超過 700 名的工程師致力於開源，他們都是高級的 Linux 開發、維護人員和技術專家，他們對不少開源技術如虛擬化技術（Xen、KVM、UML）、內核性能、圖表演算法、Linux 流動裝置、圖表驅動器、固定工具箱、Java 協調，以及開源產品的功率、性能等作出無償的貢獻，他們也是 Linux 開放標準方面的主要專家，他們支持中國的開源社區和企業，為他們提供很多開源工具和開源技術，Intel 還是國際 Linux 基金會的主要
發起人之一。」

◎ 社區與企業的結合

「另外，無商業模式的社區作先導開發，與具有商業模式的企業作後續開發，兩者加起來，才能完整地為開源作貢獻。由於不同國家文化、環境、語言的差異，使開源運動的合作受到一定影響，我們要研究如何排除這些差異帶來的障礙，我們支持中國的開源社區、開源企業的成長。」Dirk Hohndel 說。

Apache 創始人 Brain Behlendorf 則表示：「Apache 開發始於 1995 年，成立後不久就成為互聯網上的主流伺服器，至今還一直保持著領先地位。我認為，利他主義與利己主義，即社區與企業合在一起，才能對開源作貢獻。開源軟件業餘開發者常常是最具創新性的，很多好的專案來源於民間。微軟其實也愛好開源軟體。開源軟體還面臨很多新任務、新挑戰，還能再次改變世界。我們絕對支持中國的開源運動。」

◎ Google 與開源

「這裏我想談一些「使用開源」的企業，像 Google，自己不出售開源軟體，而是利用開源軟體來解決問題。」 Brain Behlendorf 繼續說。「他們不把開源軟體商業化，這時他們和別人不是競爭、排斥的關係，而是增進合作的關係。如果要建立一個更好的醫療資訊系統，我建議我們可聚焦於這類用戶使用開源軟體的問題，即如何用開源技術來解決問題，然後使很多人共用經驗，促進事情更好發展，提升一個層次。」 Google 資深開源專家 Chris DiBona 則表示 ：「Google 大量使用開源軟體，作為用戶來說，開源軟體給用戶選擇權、控制權，自己可以修改、調整軟硬體的配置，這是私有商業軟體做不到的。」

◎ 中國缺乏開源文化

「中國缺乏開源文化，這是急需要解決的問題。」LUPA 主席張建華說。「開源文化和政府提倡建設和諧社會的概念是一致的，要研究其共同點，爭取政府更好的支持，促進 Linux 和開源的推廣，就要用資料來說話。昨天許洪波博士在開源峰會上講，歐盟通過開源使 56 萬人重新找到就業機會，在中國也一樣，通過開源的推廣和普及，使成千上萬的大學生走出一條創業和就業之路。」

◎ 開源軟體不是微軟軟體的克隆

現在在 Source Forge 平台上也已有中國人主持的開源開發專案了。「我們發展開源軟件不是微軟軟體的克隆或複製，而要以一種更好的、創新的方式出現，使創新的開源軟體更好。」全球最大的開源社區 Source Forge 的創始人 Larry Augustin 說。「人們選擇 Office、Linux 軟體，要更好完成工作，這是自由競爭的結果，而不是像微軟那樣壟斷的結果。在 Source Forge 的平台上，有 11 萬個開源軟體專案（100 多萬開發人員）在孕育開發之中，微軟也在 Source Forge 的平台上參與 7.7 萬個開源軟件專案的開發。在中國，我們不是簡單看現有甚麼東西，而是要創造下一代軟體，從根本上改變人們的工作方式和軟體的創新方式。」
