香港中小企業應用 Linux 成功經驗分享

建立日期 2006-11-24 08:00      最近更新在 2011-01-12 19:25 作者是 LinuxPilot
為加強中小企業對 Linux 的功能及效用有更深入的認識，香港生產 力促進局特別設立中小企應用 Linux 啟動計畫，進行 Linux 應用調 查，了解中小企業應用 Linux 的趨勢、效益、需要及困難。更優先 測試中小企業必需的基建及基礎商業軟體，為他們測試市面上各種軟 體組合的可行性，好讓中小企業於短時間能夠有更多已接受測試的軟 體包選擇。有關研討會由香港生產力促進局將主辦，主要內容包括開 放源碼軟體應用，如亞太國家如何部署、測試完成的軟體方案組合介 紹、Linux 商業大獎得獎者分享其成功經驗等等。
◎ 從海外經驗尋找借鏡

香港業界近年已開始注意到需要一個全盤的策略推動開放源碼軟體應 用，據知中國、台灣、韓國、日本、泰國、印尼、馬來西亞等已經制 定了支持開放源碼方案的採購政策，其中以中國的購買量最大，以紅 旗 Linux 為例，去年的桌面付運量達到 60 萬套，而泰國方面，泰 文 Linux TLE 得到很大的支持，政府更有意資助中小企業部署 100 萬台 Linux 個人電腦，在印尼，科研部宣布在全國部署 Linux 版的 Java Desktop System (JDS)，有關計畫名為走向開放源碼 (Indo- nesia Goes Open Source, IGOS)，第一階段會先由政府開始，再慢 慢擴展至全國。Asia Open Source Software Community 網址。

◎ Web 應用與目錄服務的整合

香港生產力促進局高級顧問吳志榮在研討會上，匯報了「中小型企 業理想的 Web 應用環境」的測試項目，包括開發工具 PHP4、Perl 及 Java，資料庫管理 MySQL、PostgreSQL 及 Berkeley DB，資源共享 Samba 及 Cups，網路及伺服器管理 Sendmail、Tomcat、Apache 及 Vsftpd，保安及授權 OpenLDAP、Clam AV 及 IP Tables，作業系統 Novell SUSE LINUX 及 Red Hat Linux，硬體方面使用 HP RX1600 伺服器及 HP XW4200 工作站。吳志榮指出很多中小企業不知如何將 多台 Linux 系統作統一管理，所以要處理多個用戶資訊及密碼組合， 產生不便，這次的測試重點是使用 OpenLDAP 目錄服務作存取控制 管理，以及集中儲存所有用戶資訊，方便用戶可以單次登入方式作 認證，結果除了 MySQL 5.0 之前版本因不能支援 Stored Proce- dures，未能用 LDAP 客戶端增加及更改 LDAP 選項外，其他所有應 用皆通過測試，十分理想，相信可對中小企業有所支持。

◎ Linux 應用於海上船舶管理

在研討會上，香港生產力促進局邀請到 Linux 商業大獎得獎者－－ 來寶船舶管理有限公司 (Fleet Management Ltd.) 的 John Barcy 分享如何利用Linux 資訊系統成功管理 145 艘船舶，來寶船舶管理 有限公司以香港為基地，其所屬船舶在全世界不同海域航行，公司需 要為他們提供支援服務，使他們可以由一個港口安全經濟地到達另一 個港口，並符合當地及國際的監管條例，公司開發了一個入口網站供職員及船主監視船舶的資訊，例如報 告、位置、調查、船員詳情及品質與安全事務，由於要求船舶與各地 辦事處保持聯繫，他們使用了特定格式的電子郵件以傳送數據，以免 除寬頻的昂貴費用，而且他們更自行開發了一個名為 PARIS 的船上 資訊系統，能夠自行將數據以XML格式傳送及與辦事處的數據同步化， 避免使用 Excel 或 Word 等笨重的數據格式，使通訊成本得以降低。

考慮到船上的環境異常惡劣，通訊成本很高，又沒有 IT 人員駐守， 系統不能不斷重新開機，所以他們以 Linux 開發了 PARIS 系統， 因為 Linux 既穩定、安全，又合乎成本效益，而且公司之前在 UNIX 上開發的 Java 軟體模組也可以再用，PARIS 系統有多個功能模組， 包括船員名單、船員工資、調查及證書、技術及位置報告、計畫維修 系統、品質與安全事務、通告及規則等，PARIS 系統作為一台迷你的 Linux 伺服器，裡面運行 Red Hat Linux、MySQL 及 Java，為船員 提供一個虛擬的 Web 系統，而且可通過衛星連線至辦事處的電腦作 雙向資料傳送。在部署時，首先在經常來港的船隻進行安裝，然後以 速遞方式送到全球不同地點，當安裝好後再輸出有時間性的數據，至 於軟體更新則以光碟形式送出。為了避免受故障影響，船上有後備的 磁碟機，另有疑難排解指引、診斷用光碟及求助桌支援。

使用 PARIS 系統後，由於船舶資訊增加，公司可以作出更好的決策， 及減輕船上工作人員的負擔，船主可以更清楚船舶的狀況及降低經營 成本，經理也可以更好地存取資料，未來的 Linux 計畫包括加入輪 船裝載計畫模組及船員訓練伺服器。來寶船舶管理有限公司的成功應 用對於航運及交通業應用 Linux 有很好的提示。
