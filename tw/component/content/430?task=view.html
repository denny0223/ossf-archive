Ubuntu Linux 獲得 IBM DB2 認證

建立日期 2005-11-11 08:00      最近更新在 2007-03-08 08:22 作者是 謝良奇
以易於安裝、使用著稱的 Ubuntu Linux 套件日前獲得 IBM 認證，Ubuntu Linux 5.04 版與其他 11 種套件同列為 IBM DB2 資料庫軟體的支援 Linux 環境。 獲得 IBM 認證的消息緊跟於 Ubuntu 10 月伺服器版本釋出後發佈，Ubuntu 更將計畫於明年 4 月釋出企業版本。Ubuntu 策略聯盟經理 Malcolm Yates 表示，該釋出將作為未來認證的基礎。

Yates 指出，鑑於 Ubuntu 專案為大型系統提供桌面用戶之易用性的目標，Ubuntu Linux 延伸至伺服器與企業版本領域僅是時間問題。

該認證是由 Ubuntu 與 IBM DB2 實驗室雙方的軟體工程師於 DB2 Linux 認證計畫（DB2 for Linux Validation program）下，進行相容性測試，為 Ubuntu Linux 取得 "Ready for IBM DB2 Software for Linux" 標章。

針對 DB2，IBM 推薦作業系統與認證支援的作業系統。該公司網站上列出了 Ubuntu Linux 在內的 12 種 Linux 支援環境，其中有 5 套推薦套件包括兩種版本的 Red Hat 企業版 Linux 以及兩種版本的 SUSE Linux 企業伺服器。

Canonical 公司贊助、Ubuntu 社群提供維護的 Ubuntu Linux，是以 Debian套件系統為基礎的非商業 Linux 套件。Ubuntu 創辦人與贊助人 Mark Shuttleworth 表示，獲得 "Being Ready for DB2 UDB" 是 Ubuntu 往前邁進的一步，他們於上個月釋出的伺服器版本，提供了主要資料庫環境的絕佳平台。

Debian 專案創辦人 Ian Murdock 表示，缺乏獨立軟體廠商的支援一向是 Debian 系列套件的一大缺點。IBM 認可以 Debian 為基礎的系統對於 Debian 專案背後的整個社群相當重要。

Murdock 也指出，假如該認證並非局限於 Ubuntu，相信對核心平台將有更大幫助。他更希望能看到軟體廠商直接對 Debian 認證，而非僅限於如 Ubuntu 這樣的延伸系統。

Ubuntu Linux 以廣為第三世界運動者與教育領域愛用而聞名。作為以桌面易用性為關注重點的 Linux 套件，Ubuntu 強調自己為人本 Linux。Ubuntu 家族中尚包括以 KDE 為基礎的 Kubuntu，以及鎖定教育領域的 Edubuntu。
