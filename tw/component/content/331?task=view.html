Linux 桌面新裝 五大桌面介面提高 Linux 親和力

建立日期 2004-03-21 08:00      最近更新在 2007-03-09 01:23 作者是 謝良奇
安全性、成本，一直是 Linux 最大的賣點和長處，伺服器成了 Linux 近來最大的舞台。然而，Linux 在介面親和性、易用性的逐步改善，一般桌面個人電腦不再是 Windows 作業系統的獨佔市場。Libra Computer Systems Ltd. 的 Libranet 2.8.1 Flagship Edition、Lindows.com Inc. 的 LindowsOS 4.5、Lycoris 的 Desktop/LX、MandrakeSoft Inc. 的 Mandrake Linux 9.2，以及 Xandros Inc. 的 Xandros Desktop OS Version 2-Deluxe Edition，這些現今的 Linux 桌面介面，在各方面已有進駐使用者桌面的實力。

可靠的安全核心，讓 Linux 在政府機構的伺服器上，逐漸佔有一席之地。過去，Linux 一直被視為只有慣於使用命令列介面的電腦高手，才會使用的系統。近來 Linux 在介面親和力的演進和改善，使 Linux 進而侵蝕 Windows 雄霸許多的桌面個人電腦版圖。許多 IT 部門因為 Linux 不易被入侵、安全弱點較少，正尋求 Linux 作為桌面投資的另種選擇。Linux 介面提供了系統管理者，更佳的桌面電腦軟體安裝控制權，避免使用者未經授權濫於安裝軟體，構成系統安全漏洞。為了保障過去在人員訓練上的投資， Xandros Linux 等介面，提供了類似 Microsoft Office 和 IBM Lotus Notes 的軟體 CrossOver Office，降低轉換至 Linux 平台的困難。Debian based 的 Libranet，是開發人員、系統管理者、和技術高手的利器。USATODAY.com 實際測試後，認為 Libranet 2.8.1 較適合 IT 部門的用戶。Libranet 2.8.1 安裝十分簡單，拜 Debian based 之賜，Libranet 的套件管理可以輕鬆地安裝、移除軟體。Lycoris Desktop/LX 和 LindowsOS 4.5 的安裝、設定簡易，可協助縮短轉移至 Linux 的時程。Lindows 有著成熟桌面的表現，類似 Windows 的介面和工具，使用者轉移平台時無須太多訓練。藉由 Click-N-Run，LindowsOS 獨特的應用軟體，取得授權的用戶可輕易安裝並更新軟體。Lycoris Desktop/LX 提供客製化的 KDE 2.X 桌面，內含 KDE 的文書處理軟體 Kword、Kspread、Kpresenter 等。Lycoris 是以 RPM 為基礎的套件，並附有圖形化的軟體安裝程式。Mandrake Linux 9.2 以 Linux 2.4.22 為基礎，有許多先進的 Linux 功能，包含 KDE 3.1.3, GNOME 2.4.0 桌面環境。針對企業用戶的 ProSuite 9.2，內含終端用戶應用軟體及服務，其中包括防毒軟體。和 Libranet 一樣，Mandrake Linux 需要較多的調整和設定，才能達到模擬 Windows 環境的目標。Xandros Desktop OS version 2 提供 CrossOver，和 Office、Lotus Notes 相似的應用軟體，簡化平台的轉換過程。USATODAY.com 測試過程中，發覺使 Xandros 與網路資源接軌，不如其他套件簡易。測試結果並非絕對，或許與使用者的實際經驗有所差異，然而 Linux 對於企業用戶，已可視為可行的桌面另種選擇。Linux 無疑地是低成本，具易用性、安全性和可靠性的一條路。
