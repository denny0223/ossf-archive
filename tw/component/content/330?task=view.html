Linux 仍為最普遍嵌入式作業系統 使用率略見下降

建立日期 2006-06-06 08:00      最近更新在 2007-03-09 01:23 作者是 謝良奇
最新的調查發現，Linux 仍是最常見的嵌入式作業系統，但是 2005 跟 2006 年的調查結果相較後顯示，嵌入式 Linux 的使用率有下跌的趨勢。然而這樣的結果與 LinuxDevices.com 不久前所做的同類型調查有相左的跡象。 CMP 針對電子商業期刊與嵌入式系統研討會 (Embedded Systems Conference) 與會者所做的調查中，Linux 仍是應用最為普遍的嵌入式作業系統。調查包括嵌入式系統設計 (Embedded Systems Design) 訂戶、嵌入式系統研討論與會者與部份 EE Times 讀者，共 1200 人。

有 16% 的受訪者表示採用非商業 Linux 或其它開放源碼作業系統，約有 12% 受訪者則使用了商業版本的 Linux 作業系統。總計使用商業與非商業 Linux 的受訪者有 28%。

調查中指出 51% 的受訪者使用的是商業作業系統，其中使用 VxWorks 的有 26%，18% 使用的是 Windows XP，18% 使用 Windows CE。

因此所有使用商業作業系統的受訪者中，Linux 以 12% 緊追在 VxWorks 之後，然而，假如將 Windows XP 與 Windows CE 歸為同一項目，微軟則是佔有率最高的商業作業系統廠商。

這份調查有趣之處在於，比較 2005 與 2006 的調查結果發現，包括商業與非商業的嵌入式 Linux 使用率有下跌趨勢。2005 年時 37% 的受訪者使用嵌入式 Linux，今年僅有 28%。2005 年表示將會使用嵌入式 Linux 的比率為 43%，今年則降為 36%。

嵌入式系統設計主編 Jim Turley 表示，Linux 現象退燒與幻想破滅是這種情況的原因。然而對照日前 LinuxDevices 所做的類似調查結果，卻有相左的現象。

LinuxDevices 針對 853 位受訪者所做的第六屆嵌入式 Linux 市場調查 (Embedded Linux Market Survey) 顯示，實際與計劃使用 Linux 的受訪者仍在持續成長。
