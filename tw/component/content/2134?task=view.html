Linux 基金會推出 Visa 白金認同信用卡

建立日期 2009-08-09 14:34      作者是 謝良奇/編譯

支援 Linux 核心發展的非營利組織 Linux 基金會 (Linux Foundation)，推出一張 Visa 白金認同卡，給希望為 Linux 作業系統盡一份心力的用戶。

認同卡早已行之有年，通常附有免稅團體如學校或慈善機構的標誌。這張已經在美國發行的 Linux 認同卡上，有 Linux 標誌兼吉祥物 Tux 企鵝的圖像，每張卡開卡後 Linux 基金會將會收到 50 美元，信用卡每筆消費的一部份也會捐給該基金會。Linux 基金會表示，該 Visa 信用卡計畫的所得，100% 將直接作為社群技術活動，以及開放源碼社群成員為推動 Linux 創新的旅費資助之用。

Linux 基金會執行董事 Jim Zemlin 表示，大家有許多方式為 Linux 做出貢獻，像是撰寫程式、行銷推廣等，現在有了一個成為社群支持者的便利方式，就是在口袋裡放入這張有企鵝圖片的信用卡。

大多數的認同卡是由美信銀行 (MBNA America) 或是明尼蘇達州的美國銀行 (US Bank of Minneapolis) 負責發行，不過，這張 Linux 信用卡卻是位於堪薩斯城由 Mariner Kemper 經營的 UMB 銀行推出。

Mariner 是 UMB 銀行 95 年歷史中第 6 位執掌該銀行的 Kemper 家族成員，因此 American Banker 在 2008 年關於該銀行的簡介中，描寫 UMB 銀行為家族經營銀行，並稱讚其服務品質。

Linux 基金會這次是與一家名為 CardPartner 的紐約公司合作，透過 UMB 銀行發行信用卡。CardPartner 去年推出超過 100 個認同卡計畫，全都是透過 UMB 銀行發行，該公司與小型慈善機構合作，例如 Gilda’s Club New York City 與導盲犬協會 (Guide Dogs for the Blind )等，推出許多認同卡計畫。

Linux 基金會的 Visa 白金卡最初將僅提供給居住在美國的用戶，該組織希望接下來數月能加以擴展。
