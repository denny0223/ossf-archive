ICOS2006會議報導：國際交流與產業經驗分享

建立日期 2006-09-08 08:00      最近更新在 2011-07-21 17:36 作者是 涂敏怡

◎ 前言

ICOS2006 開放源碼國際研討會日前於新竹國家高速網路與計算中心舉行，本屆的 ICOS 由新竹市社區大學承辦，新竹市社區大學的資訊課程以及對自由軟體的推廣不遺餘力，此次舉辦 ICOS 也吸引了不少的社大學員參與。

本次會議主題為「開放源碼：與你我對話」，以使用者、教育界和非政府組織 (NGO) 對自由軟體的應用為會議主軸，但仍不忘規劃國際與商業等議程，邀請各界開放源碼專業人士進行交流，如邀請印尼菩提心曼荼羅基金會探討自由軟體系統導入經 驗，以及 HP、NOVELL 等IT 大廠代表簡報主要自由軟體商業模式，此兩個議程亦為中研院資訊所自由軟體鑄造場 (OSSF) 共同規劃，讓使用者體驗開放源碼/自由軟體應用的不同面貌。


◎ 國際：印尼菩提心曼荼羅基金會以自由軟體協助當地教育經驗

在 本次研討會的國際議程中，印尼菩提心曼荼羅基金會創辦人釋學源暨印尼棉蘭市菩提學校主要幹部一行人應主辦單位之邀來訪，談論菩提學校與台灣新竹市三所社 區大學等單位合作經驗，以及導入自由軟體系統、佳惠菩提學校華語學子和當地社區民眾的過程，屬於台灣應用自由軟體技術進行國際公益推廣、縮減數位落差的成 功案例。

菩 提心曼荼羅基金會是印尼蘇門答臘地區的大型基金會，創辦人釋學源師父在印尼致力於創立學校興學、救災救貧、推廣環保與教育，其中位於棉 蘭市的菩提學校便是該基金會創立的典範學校，協助當地學子積極向學、脫離貧困。與台灣較為不同的是印尼的學校體制通常包含從幼稚園到高中，屬於綜合型態， 目前該校共有 1500 餘名學生。

為了營造有助學習的環境，菩提學校近年開始引進個人電腦設備與Internet。2004 年底南亞海嘯重創印尼，該基金會便長期投入災區救援與重建，並與台灣梵音公益協會、新竹市社區大學和國家高速網路與計算中心合作「亞齊社區數位學習中心」 計畫，讓當地受災學生與民眾能透過網路免費學習，並結合社區進行經濟重建。

釋學源師父表示，他個人本來反對網路科技，但自從三年前至新加坡參觀網路化小學後留下深刻印象，並思考如果從小學就開始讓孩童接觸網路，將來這群孩童會變成如何的問題，於是不再堅持排斥網路、同意讓菩提學校透過電腦和網路進行教學。

菩提學校行政執行長沈茂林表示，學校目前已將初中和高中的物理與化學課以電腦融入教學，學校老師亦以分期付款的方式購買筆記型電腦，為了統 籌學校電腦與網路設備，學校成立了專門的 IT 部門管理，並在 2004、2005 年分別和新竹市社大交流，由新竹市社大至該校舉辦夏令營教導當地學童學習電腦，國家高速網路與計算中心的蕭志榥組長亦於今年六月間至該校推廣 Linux 技術。

菩提學校 IT 經理李垂泰表示，該校現有桌上型電腦與 thin client 共 51 台，其中 38 台給學生使用，其中包含寫作業、上網、找資料等。另有安裝 Linux server， 為 Ubuntu 6.06 版，用途是收藏夏令營的成果檔案。未來希望將 Linux server 作為討論區與教學平台系統、網路公共論壇與 mailing list 等用途。針對學校目前應用IT 於教學的挑戰，李垂泰認為每個部門有不同的應用軟體，因此將來想參考類似 moodle 的軟體創造一個教學討論平台系統，將學校各部門使用的不同軟體能整合在一起，避免資料無法交換或是無法協調的問題。至於學生出缺席的問題也可以開發相關軟 體作為因應。

國家高速網路與計算中心副研究員、計算軟體組組長蕭志榥表示，由於他同時在社大教授 Linux 課程，與社大有淵源，故於今年二月至六月間分別至亞齊與棉蘭菩提學校推廣自由軟體理念，並以國網中心開發的 DRBL 跟 Clonezilla，以兩個工作天的時間將亞齊社區數位學習中心的 33 台電腦系統安裝完成。但蕭志榥不諱言，因印尼的正版軟體與網路連線的價格均非常高昂，若以不到 2M 的 1/3 頻寬上網每個月就需要 700 多元美金，而一套商業作業系統加辦公室套裝軟體更相當於一般人八個月的薪水，故該中心後續將使用商業軟體或自由軟體亦尚待規劃，但就印尼國內情況而論，採 用自由軟體應該是較可能且自然的理由。

◎ 產業-商業：商業模式的重組與創新

自 由軟體除了在非營利組織領域有極佳的應用外，在凡事講求經濟與效益的商業世界亦同樣受到重視，並獲得多家電腦大廠的背書支持。但是，面對 客戶各式各樣的需求，以及百家爭鳴的多款自由軟體，廠商若想要獲利，就不能再謹守單一的獲利模式，如何搭配不同的自由軟體解決方案進行應用創新進而獲利， 將是意圖以自由軟體做為獲利模式的廠商未來所需面對的課題，在 ICOS 的商業議程中便以此主題進行討論。

產業- 商業議程主持人為翔威國際的劉龍龍總經理，開場白時便明確指出開放源碼是從使用者需求為出發的運動，期許在場聽眾能夠從使用者的角 度開始參與開放源碼發展，如果認同自由軟體的話亦應給開發者一些鼓勵。參與座談的廠商則包括支持開放源碼的 HP 和 Novell代表，以及活躍國內自由軟體社群的 STEPS 智新資通執行長陳貴成(kc)。

HP 和 Novell 代表在簡報時均不約而同以自家產品為例，說明以Linux 為主的商業解決方案的發展潛力，以及自家公司如何支持社群的開發。自由軟體以社群作為開發與支援模式，能創造出具有創新意涵的自由軟體，使用者也因此有更 多樣彈性化的選擇，並能自由選擇從使用者轉為開發者角色，讓自由軟體更符合自己的需求，著名 IT公司如 Google 便是採用社群模式開發的例子之一。HP 惠普科技技術顧問沈仲杰表示，從市場的自由軟體採用趨勢來看，對Linux作業系統的採用已成為自由軟體採用的主要選擇，中介軟體則在早期採用者的階段， 至於應用軟體的採用對客戶而言尚屬於尖端技術 (cuttingedge) 階段，離普及化還有一段距離，市場將來應會逐漸成熟。

相較於兩位大廠代表，STEPS 智新資通執行長陳貴成則以新創中小企業的角度出發來看自由軟體商業創新議題。他表示國內的以自由軟體做為 解決方案的中小企業多以承接專案做為獲利模式，多專注於自己公司強項的領域承接專案，以滿足客戶需求建立口碑，若能透過自由軟體的重組進行應用創新，進而 創造出滿足客戶需求的解決方案對此領域的中小企業而言將是未來的機會，例如將Linux放入USB中做成可攜式裝置、進一步做Terminal Services， Blog與Wiki的解決方案（如 Wikia）等，並期許自己的新創公司將來能朝著這個方向找出明確的商業模式。

最後，針對民眾對於自由軟體商業模式的提問，沈仲杰認為國人尚未建立起針對使用服務付費的觀念，但自由軟體商業模式未來將以服務為主要導 向，因此培養國內使用者以服務做為獲利模式的概念將是當務之急，如此才能讓開發者專心致力於軟體研發，「要讓有 idea 的人活得好好的，讓他們把心力放在創造力上，而不是花在陪笑臉上」沈仲杰說道。

◎ 結語

透 過印尼菩提學校與新竹市社區大學、國網中心交流的成功案例，以及商業大廠代表與在場聽眾的腦力激盪，證明自由軟體不但可以促進國際間的交 流對話，透過台灣的技術優勢，成功協助偏遠地區縮減數位落差，也讓與會者了解身為消費者可以不只是使用特定的商業電腦系統，在自由軟體的世界中更有成千上 萬種的自由軟體解決方案可供選擇，落實了本次大會意圖讓使用者與開放源碼/自由軟體對話的宗旨。相信對參加的使用者來說，不論是國際或是產業的經驗分享， 都是一次很好的開放源碼/自由軟體經驗學習。
