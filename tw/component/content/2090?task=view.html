Linux netbook 大熱門 Moblin 2.0、Android 產品蓄勢待發

建立日期 2009-06-04 08:00      作者是 謝良奇/編譯
本次台北舉行的 Computex 展覽中，從相關消息的數量可以看出，Linux netbook 熱門的程度。綜觀而言，Linux netbook 今年趨勢包括 Moblin 2.0、Android 即將加入 netbook Linux 競爭，私有媒體播放軟體進駐 Linux 桌面等。

首先是 Moblin 2.0，這個出自 Intel 如今納於 Linux 基金會 (Linux Foundation) 旗下的開放源碼專案，在 netbook Linux 作業系統競賽中，看似拔得了頭籌。Ubuntu Netbook Remix 背後的 Canonical，加上 Linpus、Novell、Red Flag、Xandros 等廠商，皆已宣佈將為自家的散佈套件，推出支援 Intel Atom 架構的 Moblin 版本。

在這些散佈套件之中，儘管 Ubuntu 最受矚目，Novell 卻在與 OEM 廠商獲得實際合作上，佔據領先地位。大張旗鼓投入 Moblin 的白牌個人電腦商 Acer，看來將與 Novell 進行合作。MSI 投入 SUSE Moblin 的懷抱， 而 Asus 的 Moblin 合作夥伴則是 Xandros。預料第 4 季，Moblin netbook 就會在市場上銷售。

另外一方面，儘管 Google 仍舊不鬆口 Android Linux 是否進駐桌面，相關 netbook 卻已經悄悄起跑。Acer 已經計畫在第 3 季推出採用 Android 的 Aspire One。在此值得注意的是，可能感受到來自微軟的壓力，Asus 在 Android netbook 上表現退縮的態度。

在 Computex 展覽上，Qualcomm  展示一款採用該公司 ARM Snapdragon 晶片的 Asus Eee PC，並執行 Google 的 Android Linux。一款運用 ARM 的 Asus 新 netbook，加上熱門當道的 Android，價格足以力拼 Intel 的 Moblin 2.0，正當一切聽來十分美好時，Asus 董事長 Jonney Shih 卻在隔天為了這款 Android Eee PC 的出現，表示道歉。

Shih 表示，該公司已經決定不展示這個產品，雖然有些人已經在 Qualcomm 的攤位看過這些裝置，但事實上，該公司至今的決定是，不展出此一裝置。他願為此致歉。然而，這款展出裝置並非不應該公開亮相的實驗性專案，而是可立即製造出貨的產品，導致外界猜想，Shih 致歉的理由，是 Asus 不願意激怒微軟。

在 netbook 市場，ARM 處理器的腳步似乎一直落後於 Intel Atom。然而，Asus、Compal、Foxconn、HTC (High Tech Computer)、Inventec、Toshiba 與 Wistron 在這次展覽中，都展出了採用 Snapdragon 的裝置。外界認為 ARM/Snapdragon 主要是在等待 Android。一旦 Google Android 就緒，netbook 市場恐怕將到處充斥運用 Android 的 netbook。

此外，缺乏 Linux 桌面私有媒體播放軟體，這個一直困擾許多用戶的問題，即將隨著 RealNetworks 認真看待 Linux 桌面市場，而迎刃而解。長久以來銷售私有與提供 Linux 開放源碼媒體播放軟體 Helix 的 RealNetworks，日前宣佈一連串的合作案。

RealNetworks 的新私有媒體播放軟體 RealPlayer for Mobile Devices，即將隨附於 Canonical、Xandros、Phoenix Technologies 與 DeviceVM 的 Linux 散佈套件中。Canonical 是 Ubuntu 母公司，Xandros 則有同名的散佈套件，以及新的隨開即用 (instant-on) Linux 系統 Presto。Phoenix 和 DeviceVM 分別是嵌入式隨開即用 Linux 散佈套件 HyperSpace 與Splashtop 背後的廠商。

RealPlayer for Mobile 支援 ARM 與 Intel Atom 的 netbook。隨著 SplashTop 已經預先安裝於 Lenovo、Asus、HP 以及 LG 的新筆電與 netbook 上，各地使用者很快將接觸到這套新播放軟體。此一 RealPlayer 版本將可播放所有開放媒體格式與 Windows Media 和 Apple QuickTime 等常見的私有格式。然而，Adobe Flash 並不在支援之列。

在此一個人電腦市場低價區段，外界仍持續看好 Linux netbook。Windows 7 SE (Starter Edition) 這套實際上的 Windows netbook 版本，儘管移除了只能運行 3 個應用軟體的限制，恐怕還是難以披敵。畢竟，Win 7 SE 缺少 Aero Glass、遠端桌面、網域名稱(network domain) 或 AD (Active Directory) 支援，也存在無法播放 DVD 與執行 XP 模式等限制。反觀 netbook Linux 在價格上應該低於 Windows 7 SE，更可以在虛擬機器上執行 XP，Linux 實在有足夠理由拿下此一桌面市場。
