自由軟體授權模式眾多 OSSF自由軟體授權指引提供建議

建立日期 2007-01-29 22:45      最近更新在 2011-01-06 22:03 作者是 webmaster

自由軟體授權模式的選取，攸關軟體開發專案的命運。然而，目前的授權模式眾多，光是自由軟體組織OSI（Open Source Initiative）認可的授權方式，就有54種之多。對此，目前並無相關網站提供相關的協助。為協助專案開發者選擇授權方式，自由軟體鑄造場日前推出法律授權指引（License Wizard），讓駐站的專案開發者在幾分鐘內，就可以從自由軟體社群最常用的十數種授權條款中，取得適合專案性質的授權模式。


 

雖然國外已經有SourceForge、 Tigris等具有相當規模的網站，提供自由軟體專案開發者，將 其專案進駐網站，與其它社群開發者進行交流，進行協同開發。不過，對於複雜的授權問題，尚未有網站提供專案開發者相關的建議與協助。自由軟體鑄造場為協助 專案開發者解決此一問題，日前將License Wizard上線，在現有的專案管理網站中，屬於創舉。未來自由軟體鑄造場，也將持續針對專案開發者的需求，持續強化授權方面的服務。有意試用者可連上網址使用。

自由軟體鑄造場（OSSF is Seeking Software Freedom, OSSF）計畫主持人中研院資訊所研究員何建明表示，OSSF為經濟部工業局所補助的計畫，主要目地在協助國內社群進行自由軟體的專案開發。除了 License Wizard以外，此一網站免費提供網頁空間以及事務追蹤（Issue Tracking）、通信論壇、共同筆記、版本控制、檔案下載等服務等專案開發者所需的功能。

何建明表示，「自由軟體授權指引」整合各授權條款內容，轉換成淺顯易懂的問答題，透過互動對話的方式，軟體工作者只需幾個基本的步驟，就找到適合自身開發的程式授權需求，可以大幅降低過往，自由軟體工作者必須閱讀每一授權條款內容的時間成本與心力。

OSSF 計畫協同主持人中研院資訊所副研究員莊庭瑞指出，自由軟體鑄造場提供自由軟體授權指引的構想，主要是因為智慧財產權法令限制越來越多，授權方式的 選擇，攸關專案開發未來的各種應用，或者是商業授權模式，甚至關係專案開發的成敗，往往選擇不同的授權模式，專案開發就會衍生不同的樣貌。但絕大多數軟體 程式設計師因不熟稔法律條文，面臨尋找授權條款不易問題。因此OSSF在計畫一開始即成立法律政策組，針對授權問題進行規劃。

莊庭瑞說：「『法律授權指引』看似簡單，但其實每一種自由軟體的授權條文，都必須花時間研究整理，必須先進行作很多準備累積成果，才能完成這個簡單易於使用的介面。」他表示，先前OpenFoundry針對相關授權問題，完成「自由軟體授權常見問答（FAQ）」，以及「常用OSI approved自由軟體授權條款比較」。授權指引這項服務，就是根據以往累積的自由軟體授權模式的研究成果，再結合技術團隊共同開發出來的。

值得注意的是，除了「自由軟體授權指引」外，OSSF 也將授權服務延伸到自由軟體以外的授權指引服務。針對圖文授權部份，包含網站、音樂、影片、攝影、文 學、教材等的創作，在OSSF網站上可以找到由Creative Commons製作的Commons License Wizard，讓非軟體設計者也可找到適用的CC授權條款(Creative Commons Public License)。

此一公共授權條款，以條列式的簡易條件，透過各種排列，組合成6種不用的授權條款。創作者經由Commons License Wizard，可簡單地挑選出最適的授權條款，將自身作品釋出給大眾使用，同時也保障自己權益。

同時也是tw-iCommons國際合作計畫主持人的莊庭瑞指出，CC的授權方式，對創作者而言，有利於作品的流通及創作理念的推廣。對一般大眾而言，可以清楚了瞭懂授權條件，「因為怕有侵權疑慮，不敢使用志在流通的作品」這類「因噎廢食」的困境，也便得以避免。

莊庭瑞強調，選擇使用CC 授權條款並不意味著放棄著作權，而是讓著作權人藉由控制著作的使用方式，確保大眾對於著作的接近與使用的機會與權利，使著作能保 持一定的公共性。相較於傳統的「All Rights Reserved」的宣告，Commons License Wizard具彈性的設計，可以協助創作者挑選出適合的CC授權方式，節省成本，並有助於作品流通與理念散布。

OSSF 在法律授權方面的努力也受到聯合國組織UNDP（United Nation Development Programme）下的亞太資訊發展計畫（Asia-Pacific Development Information Programme, 簡稱APDIP）的注意。

何建明表示，UNDP 已經委託OSSF撰寫授權入門手冊（License Primer）。先前也曾受越南官方邀請，到河內與當地軟體開發人員進行交流。他也提醒，此一授權指引僅針對自由軟體工作者所設定之條件，提供一般性之建 議，並不負任何法律責任。若需專業協助及有特殊需求，仍須求助專業的法律諮詢。

對 於此項創新的線上法律服務，法學專家中央大學產業經濟研究所副教授劉靜怡表示，「自由軟體授權指引」能讓自由軟體程式開發者以簡易的方式選擇出符合個案 需求的授權條款，這個授權指引的功能，也展現出與傳統法律服務相當不同的呈現方式。為了要能在數個問答中瞭解程式開發者的授權需求，並且從中找到符合需求 的授權條款，研究人員必須通盤瞭解各種授權條款的特性，歸納出具有重要法律效果的差異點，才能完成這樣的設計。

活躍於台灣自由軟體社群的開發者唐宗漢表示，很高興看到自由軟體鑄造場能針對社群朋友的需求推出這一套線上指引服務，的確可以省去社群開發者蒐集、了解符合自己需求的自由軟體授權條款的時間。

何建明指出，OpenFoundry.org自由軟體鑄造場專案開發平台系統的 建置，提供國內自由軟體社群一個永續經營的網路服務系統。除了提供「自由軟體授權指引」、Commons License Wizard外，OpenFoundry.org亦提供事務追蹤（Issue Tracking）、通信論壇、共同筆記、版本控制、檔案下載等服務，並規劃建置嵌入式系統特殊鑄造場、撰寫國內自由軟體社群調查報告、人才資料庫等，另 舉辦計畫說明會推廣工作坊與成果發表會，與提供FAQ、電子報等加值服務。

使用者若有其他系統上的問題或建議，可寫信至 這個 E-mail 地址已經被防止灌水惡意程式保護，您需要啟用 Java Script 才能觀看 或 OSSF專案，與自由軟體鑄造場計畫聯絡。

關於自由軟體鑄造場計畫

本專案為累積國內自由軟體/開放源碼經驗與知識之先導計畫，目標為建立開發軟體專案之網站，以作為發展自由軟體的基礎平台，並舉辦工作坊與研討會，整理專案網站使用手冊、翻譯名詞對照與字彙表、國內自由軟體人才與社群活動資訊、常用授權條款比較與常見問答小手冊等文件。
