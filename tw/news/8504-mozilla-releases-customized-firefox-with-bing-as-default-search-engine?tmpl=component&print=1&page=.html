Mozilla 釋出以 Bing 為預設搜尋引擎的 Firefox 版本

建立日期 2011-11-07 10:16      最近更新在 2011-11-18 09:47 作者是 謝良奇／編譯
Mozilla 日前釋出 Firefox 瀏覽器的客製化版本 Firefox with Bing，該版本使用 Microsoft 的 Bing 作為預設搜尋引擎。此外，AwesomeBar 也將 Bing.com 設定為使用者首頁。除了到 https://www.firefoxwithbing.com/ 下載，其他版本的 Firefox 用戶也能透過安裝 Bing Search for Firefox 這個附加元件，達到同樣效果。

Mozilla 在一份簡短聲明中表示，採用 Bing 的 Firefox 瀏覽器版本，擴展了與 Microsoft 在去年簽訂的合作關係。事實上，從 2010 年 10 月 Microsoft 與 Mozilla 合作協議宣布後，Mozilla 已經將 Bing 設為搜尋引擎選項之一，允許用戶從預設的 Google 切換至 Bing。在此之前，用戶僅能透過手動方式，將 Bing 加入選單中。

 

▲ Firefox with Bing 網頁

Microsoft 則樂見此次 Firefox 的更新。該公司表示 Firefox 用戶中的 Bing 愛好者如今可以依照自己喜歡的方式使用 Firefox with Bing 瀏覽網路，無須透過額外步驟以客製對 Bing 的設定。

在 2010 年與 Microsoft 的合作中，Mozilla 與該公司簽署了財務協議，雙方將共享由 Firefox 發送至 Bing 的流量帶來的營收。該協議的細節至今並未公開，Mozilla 宣布推出此一客製化瀏覽器時，也未提及財務部份。

多年來 Mozilla 將用戶導向 Google 利潤豐厚的搜尋與廣告系統，藉此從 Google 獲得其大部分的營收。然而，近來 Mozilla 逐漸多元化其業務，慢慢減少對 Google 的依賴。

在兩家公司於 2004 年達成的協議下，來自 Google 的獲利幾乎占了 Mozilla 所有的營收。但是 Google 與 Mozilla 的合約，即將於今年到期。到目前為止，外界似乎並未獲得有關該合約是否續約的消息。

無疑地，Mozilla 可以透過推廣 Bing，分散對 Google 的依賴，然而用戶對於預設 Microsoft 為搜尋引擎的反應為何，仍有待觀察。

從 Mozilla 與 Microsoft、Google 的合作關係中，外界不禁要問，Mozilla 是否應該找出更好的營收模式，不論是 Google、Microsoft 或其他公司，過分依賴單一獲利來源，長期而言恐將發生問題。

Mozilla 總監與 Firefox 桌面瀏覽器市場經理 Asa Dotzler 表示，Mozilla 也許有機會以新的方式，協助推廣新興的搜尋組織或功能。

2010 年宣布與 Microsoft 的合作時，Mozilla 曾表示，他們認為 Bing 提供了對用戶有價值的使用者體驗，鑑於該搜尋引擎的普及率在去年顯著提升，他們將 Bing 納為英語用戶的一般搜尋選項。

Mozilla 同時也宣布了近 20 種 Firefox 的客製化版本，將透過其合作夥伴如 Twitter、Yahoo!、United Internet 散布全球。
