MozTW 軟體自由日《Code Rush》紀錄片放映會

建立日期 2012-09-11 12:06      最近更新在 2012-10-04 16:04 作者是 林旅強

自由軟體愛好者前進電影院！MozTW 將在 9/15 國際軟體自由日 (Software Freedom Day) 前進電影院，播映講述 Mozilla 創立的紀錄片《Code Rush》。

Code Rush（源碼奔馳，暫譯）紀錄片由美國公共電視所製作，David Winton 導演於 2000 年間拍攝。當時矽谷 Netscape 公司面臨微軟 IE6 的強烈競爭，為度過危機而採取激烈手段「Mozilla 」專案，將公司的最高機密 — Netscape 瀏覽器組成程式碼完全公開，開放予以大眾自由使用，成為開放原始碼運動的濫觴。當時工程師們為趕上預定的釋出時程，日以繼夜的犧牲日常生活與家庭，完成 Mozilla 軟體，並歷經十年演進，最終成為現今的 Firefox（火狐）瀏覽器，帶來網路世界大革命。

本片為 David Winton 導演於 1998 年至 2000 年間拍攝的紀錄片，紀錄了矽谷的 Netscape 工程師們將瀏覽器原始碼釋出，成為 Mozilla 專案的經過；同時 Netscape 也正經歷被 AOL 併購的過程。本片描繪了 Netscape 的工程師們，犧牲日常生活與家庭，努力防止他們的公司倒閉的經歷。

2002 年釋出的 Mozilla 1.0 歷經十年演進，最終成為現今的 Firefox 瀏覽器，引領網路發展、帶來世界變革。明年是 Mozilla 基金會成立十週年及 Mozilla 計畫 15 週年。在這個新世代行動瀏覽器大戰的起點，就讓我們一起回顧 Mozilla 的起源，迎接未來的挑戰。

Firefox 作為為全世界數億人上網的門戶，由 Mozilla 全球社群數十萬志工共同開發，提供超過 86 種語言版本。Code Rush 一片記載 Mozilla 專案的誕生經過，本事件同時也是「開放原始碼」一詞的起源，及網際網路發展的重要契機。

歡迎對自由軟體、開放原始碼感興趣，熱愛網際網路及 Firefox 瀏覽器的你，千萬不可錯過！現在就立即報名吧！
