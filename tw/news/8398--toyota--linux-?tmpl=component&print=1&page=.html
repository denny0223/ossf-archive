汽車製造商 Toyota 加入 Linux 基金會

建立日期 2011-07-21 00:00      作者是 謝良奇／編譯

Toyota 成為最新一間加入 Linux 基金會 (Linux Foundation) 的公司，希望強化該公司在車內資訊與娛樂服務上的創新。Linux 基金會是以促進開放源碼作業系統 Linux 成長為目標的非營利組織。


Toyota 表示該公司加入 Linux 基金會為金卡會員 (Gold member)，將最大化在 Linux 上的投資，同時促進汽車產業生態的開放創新。身為金卡會員，Toyota 是 Linux 基金會少數並非技術供應商的主要會員之一。該基金會其他大型團體會員包括 Google、BT、Fujitsu、Hitachi、HP、Oracle、Intel、IBM、Nokia。

對於試圖尋求將智慧型手機、MP3 播放器、GPS 導航等資訊娛樂系統，與車內控制系統加以整合的汽車製造商來說，Linux 的重要性越趨顯著。Toyota 電子技術發展專案總經理 Kenichi Murata 指出，Linux 給予該公司演進其車載資訊娛樂 (in-vehicle-infotainment，IVI) 與通訊系統，以滿足其客戶期待，所需要的彈性和技術成熟度。

過去許多汽車製造商使用由德國汽車公司 BMW、Opel、Volkswagen 與其他組織於 1993 年開發的 OSEK OS 作為標準，以整合電子設備。然而該產業領導廠商如 BMW、General Motors、Peugeot Citroen 逐漸開始嘗試運用 Linux 來整合資訊娛樂系統。如今 Toyota 正式加入 Linux 基金會的決定，更強化了該開放源碼作業系統對於汽車廠商逐漸成長的重要性。

Murata 表示 Linux 基金會提供給該公司一個中立論壇，在該論壇中 Toyota 能夠與全球技術領導廠商，在促進上述演進的開放創新上進行合作。

Linux 基金會執行董事 Jim Zemlin 表示，Toyota 的領導地位與展現的創新力，將為 Linux 的發展帶來重要的貢獻。該公司對 Linux 的投資證明了該作業系統已經普及至隨處可見，並且具備足以支援最新市場需求的能力。
