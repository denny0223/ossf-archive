Linux 音樂服務不缺席 Spotify 釋出預覽版本軟體

建立日期 2010-07-15 00:00      作者是 謝良奇/編譯

音樂串流服務 Spotify 日前為桌面 Linux 釋出旗下點對點 (peer-to-peer，P2P) 音樂應用軟體的預覽版本，提供用戶下載 Debian Squeeze/Ubuntu 10.04 上的軟體套件。這套軟體有著 Windows 和 Mac OS X 平台 Spotify 版本的大多數功能，不過目前僅提供給 Spotify Premium 的付費會員使用。

位於盧森堡的 Spotify，是歐洲的音樂服務提供者，除了 Windows 和 Mac OS X 桌面之外，Spotify 也在 Android、Symbian、iPhone行動裝置上提供付費服務。

 

Spotify 允許用戶透過 P2P 技術分享音樂，並提供音樂搜尋、客製化播放列表、音樂熱門榜、歌曲購買、網路廣播，其他還包括音樂人資料、相關音樂人推薦、Facebook 整合等功能。Spotify 另一項有如音樂人廣播的音樂資料庫功能，會搜尋網路電台，並播放特定樂團或音樂人的連續音樂串流。

該服務之前公佈了新的價格方案，日前則為 Linux 桌面使用者提供了用戶端軟體的 beta 版本。根據 Spotify 全球社群經理 Andres Sehr 的部落格文章，該公司的開發者發展 Linux 版本的用戶端，部分原因是為了自己。Sher 表示，他們有許多開發者在使用 Linux，他們顯然希望在寫程式時，能夠一面聽音樂。從他們所收到的用戶回應，這些開發者似乎並不孤單。因此他們非常高興能為 Linux 推出 Spotify 的預覽版本。

Sehr 指出，Linux 預覽版本提供的功能，和 Windows、Mac 版本大致相同。雖然 Sehr 承諾將盡可能確保 Linux 版本追上 Windows、Mac 版本，不過 Linux 版本目前還落後其他兩個版本一步。由於 Linux 平台上本地端音樂解碼的問題，目前的 Spotify Linux 版本無法播放本地端音樂檔案。

此外，因為該公司尚未找出處理廣告的可靠方式，Spotify 的 Linux 版本目前僅提供給 Spotify Premium 的訂戶使用，這項服務每個月要價 10 英鎊，約 15 美元。Spotify 並未提到這些問題何時，或者在最終釋出時是否會獲得解決，而且也沒有提出最終釋出的時間表。

Spotify 目前並未在美國推出服務，不過該公司表示正與唱片業者以及當地的出版權協會交涉授權協議。Spotify 現在僅於芬蘭、法國、挪威、荷蘭、西班牙、瑞典、英國各國提供服務。

儘管 Spotify 的 Linux 在功能上有著若干不足，Linux 桌面的支持者在各部落格、線上論壇上，倒是給了 Spotify 許多充滿讚美與興奮的回應。鑑於這樣的結果，不禁令人猜想，是不是會有越來越多開發者慫恿他們的公司，為他們的軟體釋出 Linux 版本。
