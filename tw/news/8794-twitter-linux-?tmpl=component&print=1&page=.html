Twitter 將加入 Linux 基金會

建立日期 2012-09-06 10:18      最近更新在 2012-09-06 10:24 作者是 黃湘婷／編譯

Twitter 於日前宣布將加入 Linux 基金會，以每年 15 萬美元的會費成為銀等級的會員。Linux 基金會為一致力於促進 Linux 發展的非營利組織，成員還包括了 Intel, Red Hat, Oracle 以及 6 月初才升級為白金等級會員的 Samsung 等等，眾多對於 Linux 開發貢獻良多的知名公司。

Twitter 的開放原始碼經理 Chris Aniszczyk 表示，Twitter 即將加入 Linux 基金會，除了支持 Linux 外，也將促進 Linux 在這塊領域未來的發展，此外他也認為 Twitter 加入基金會後，亦可透過內部的整合，在 Linux 快速成長的同時，令 Twitter 也能夠持續的進步。

對於 Twitter 的加入，Linux 基金會的執行董事 Jim Zemlin 表示，這正顯示了有越來越多大型網路公司是以 Linux 系統為基礎運行，並正努力建立自己的基礎設備能夠完全為開放原始碼軟體。

另外，Google 也是大型網路公司加入 Linux 基金會的例子。同時還有一個對於網路使用者有重大影響力的社群網站：Facebook，雖然 Facebook 尚未加入該基金會，但 Linux 基金會發言人表示，他們非常樂意與 Facebook 創辦人 Mark Zuckerberg 合作，也持續看好該公司在網路世界無可限量的潛力。
