Microsoft 與緯創簽署專利協議 範圍涵蓋 Android 與 Chrome

建立日期 2011-07-14 00:00      最近更新在 2011-07-26 11:50 作者是 謝良奇／編譯

Microsoft 宣佈與台灣設備製造商緯創 (Wistron) 簽署專利授權協議，針對該製造商採用 Google Android 作業系統所製造的設備，此一協議給予廣泛的專利授權 (broad coverage)，意指一旦 Microsoft 針對 Android 向 Google 提出專利侵害訴訟，緯創將可免於牽涉在內。這是兩週以來 Microsoft 所簽署的第四個類似協議，顯示 Microsoft 在 Android 的專利授權接連取勝，仍在持續當中。

從這項專利協議可看到 Microsoft 近日來和設備製造商 Velocity Micro、Itronix、Onkyo 交涉的同一模式。然而，此一最新的消息卻出現一項新的轉折，與緯創簽訂的協議範圍更為擴大，不僅僅涉及之前的 Android，更涵蓋了 Google 的 Chrome OS 作業系統。作為 Acer 前合約製造單位的緯創，正是 Google 新 Chromebook 電腦製造商的供應商。


Microsoft 組織副總裁 Horacio Gutierrez 在聲明中指出，他們樂見緯創能利用他們全產業的授權計畫，這項計畫的成立為的是幫助公司解決 Android 的智財權議題。

Microsoft 在新聞稿中表示，與緯創的協議為緯創的平板電腦、手機、電子書閱讀器和其他執行 Android 或 Chrome 的消費者設備，提供了 Microsoft 專利組合下的廣泛專利授權。和其他涉及 Android 有關的專利授權協議一樣，這次協議中的財務條款與牽涉的特定專利均未公開。Microsoft 僅提到在此一協議下該公司將收取權利金。

Microsoft 於 1 年多前和該公司長久以來的手機合作夥伴 HTC，簽署了第一份的 Android 專利協議。微軟並於今年 3 月控告 Barnes & Noble，指稱該公司以 Android 為基礎的 Nook 電子書閱讀器侵害其專利。

今年 5 月，Citi 分析師 Walter Pritchard 在一份報告中表示，HTC 每售出 1 支 Android 裝置，Microsoft 便可收取 5 美元。Pritchard 估計 Microsoft 向其他 Android 裝置製造商收取的費用，約為 7.50 到 12.50 美元之間。對於該產業而言，這筆權利金構成不小的難題。某分析師估計，Microsoft 從 HTC 的 Android 裝置上所賺到的專利權利金，甚至多於該公司自己的 Windows Phone。

除了 HTC 之外，Microsoft 從 Android 裝置製造商收取權利金的動作，至今仍局限於利基廠商。不過該公司正逐漸向更大型的製造商採取行動，有報告指出 Microsoft 正向 Samsung 要求收取每支 Android 裝置 15 美元的權利金。

Microsoft 在智財權領域上一直採取積極的態度，從 2003 年 12 月以來已經累積 700 個專利授權協議。與緯創的協議則是 Microsoft 轉進涉及 Google Chrome 潛在訴訟授權協議的第一個已知案例。之前 Google 才在其實驗性的 CR-48 筆記型電腦上，讓該公司的雲端作業系統首次登場。第一款以 Chrome 為基礎的消費者筆電 Chromebooks，也剛剛於 6 月時由 Samsung 和 Acer 推出。
