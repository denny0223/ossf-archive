Valve 證實將在 Ubuntu 上推出 Steam 客戶端及遊戲「惡靈勢力 2」

建立日期 2012-07-23 16:42      最近更新在 2012-08-14 13:51 作者是 蕭惟杰 (RJ)／編譯

以「戰慄時空」(Half life) 系列、「絕對武力」(Counter-Strike) 系列、「傳送門」(Portal)、Steam 發布系統及 Source 引擎等產品廣為人知的美國知名遊戲開發公司 Valve Software，在官方部落格公開宣佈，將會在 Ubuntu 上發佈他們的 Steam 客戶端及頂級遊戲「惡靈勢力 2」。雖然這已經不算什新聞了，先前已有諸多消息指出 Valve 正試圖開發能在 Linux 上執行的遊戲，但是官方正式發佈，除了證實這些消息不再只是空穴來風之外，也讓許多 Linux 使用者感到興奮，這預期將會提升了其他作業系統使用者轉換到 Linux 的吸引力，讓 Linux 不再只是以穩定性著稱的 Server 端作業系統，同時也能洗刷「Linux 上沒辦法玩遊戲」的渾名（儘管事實並非如此）。


功能齊全的 Steam 客戶端

Valve 在他們的官方部落格分享他們的計劃：

「Steam 客戶端計劃的最終目標就是 能在 Ubuntu 12.04 上執行全功能的 Steam 客戶端。 今年我們已經取得了不錯的進展，現在 Ubuntu 上面的 Steam 客戶端已經能夠執行所有主要的功能。」

可是上面還沒有遊戲？

「…我們還將惡靈勢力 2 移植到 Ubuntu。除了作為在 Steam 客戶端中與遊戲相關的功能的測試之外，同時也作為在 Ubuntu 上面的玩遊戲的測試。」

該公司表示他們在 Linux 上面努力的3個重點為：

  • 將 Steam 客戶端的所有功能移植到Linux上面。
  • 調整「惡靈勢力 2」使其在 OpenGL 上能有高幀率。
  • 將更多 Valve 的招牌遊戲帶到 Linux 上。

時程尚未確定

Valve 目前還未公佈相關的時程，不過他們表示將會在不久的將來進行內部測試，以調教遊戲的功能及機制。而 Valve 未來也將會把他們的計劃擴展到其他的 Linux 發行版上面。


不會是唯一一家

Valve 並不是第 1 家在 Ubuntu 上面看到價值的遊戲大廠。早在年初，美商藝電 (EA) 就將他們目前上線中的兩款招牌遊戲上架到 Ubuntu 的軟體中心。
