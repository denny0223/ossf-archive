國中小教師組 S4A 社群將 Arduino 導入教材

建立日期 2012-03-01 10:13      最近更新在 2012-03-27 09:59 作者是 李婉婷

前言

還記得花博夢想館的「花械花開 (Florabot)」嗎？隨著人的接近，電子花朵會立刻感應到人的動態，接著產生光影形體的變化，科技與藝術結合的互動設計著實讓人驚豔！而這個頗受各界好評的互動裝置，就是用 Arduino 做出來的。隨著開源軟體逐漸被大家所熟知，其概念也從軟體延伸到硬體，由於踏入開源硬體 Arduino 比開源軟體所需的門檻低，近年來也吹起一陣 Arduino 熱潮。而現在這股熱潮也吹往國中小學老師的方向，一群熱愛自由軟體的國中小老師籌組 S4A 社群，互相分享玩互動式機器人的經驗，研究如何將這有趣的互動裝置當作教材，藉此提高學生的學習興趣。
 

 

Arduino 簡介

我們生活當中存在著許多互動裝置，當經過暗巷時會有照明燈亮起，還有大家熟知的遊戲機 - Wii ，更是家喻戶曉的互動裝置。而 Arduino 是一個開放源碼硬體，是一張微控制電路板，在這張板子上可以加裝馬達、LED 燈、開關，以及各種 sensor（感應器），像是光源感應器、溫度感應器、聲音感應器...等。也因為可以加上諸多種類的感應器，如此便可以利用機器人做出不同的應用（例如：運用開放源碼軟硬體打造自動划拳手套）。

由於 Arduino 進入門檻低，只要懂簡單的機械原理，加上創意，就可以製作獨一無二的機器人，即便沒有電子、電機相關科系的背景，任何人都可以很容易學會如何玩 Arduino，S4A 社群國中小的老師們也因此群聚起來，分享 Arduino 可能的應用教材、討論如何帶著學生們從互動機器人中學習新知識，並期望這樣活潑的教學方式，可以給學生多一些啟發。

 

S4A 社群

S4A (Scratch for Arduino) 社群是由一群熱愛自由軟體的國中小老師所組成，由於國中小電腦課程都只有教軟體，老師們也希望運用更多元的教材來吸引學生的注意力，增添教學樂趣。有鑑於此，佛光大學許惠美老師便號召其他有在玩互動裝置的老師們一同研究教案，2011 年 11 月、S4A 社群因而誕生，社群創立之初便獲得熱烈響應，老師們紛紛主動加入，大家的共同目標是希望透過彼此分享機器人教材的運用，一同研究未來將硬體機器人導入教材的具體作法，藉由社群討論的力量讓教學更上層樓。

其實用來組裝機器人的硬體電路板有非常多種，像是 Arduino、WeDo、Kinect 或 Picoboard...等，但若是要成為教材，其前提是價格必須要一般家庭可負擔得起，也因此、老師們十分期待運用 Arduino 這個開源的低價面板來達到節省教材成本的效果。由於 Arduino 是個開源硬體，電路路設計圖或韌體都可以在網路上下載，不論在價格、易用性都非常適宜作為教學教材。此外 Arduino 開源、開放的特性使得網路上有豐碩的資源得以參考，開發軟體用的 IDE 可免費下載，Arduino 的電路設計圖也可以在網路上找得到。看中 Arduino 開源、便宜、網路教學資源豐碩等諸多特性，S4A 便將希望投注在此，S4A 名稱中的「A」即是代表社群主要討論著重在以 Arduino 這種開源電路板為主。

 

未來如何運用 Arduino 教學

蟠桃國小的范運平老師在製作機器人實作領域中可以說是十分有研究，他多次帶領學生參加 WRO 國際奧林匹克機器人大賽，並且是全國賽前三名的常勝軍，戰績輝煌。在 S4A 的討論聚會中，范運平老師分享了許多他曾實踐過的 Arduino 發想應用。像是有一次，學校舉辦票選模範生的全校性活動，范運平老師就利用硬體 Arduino 搭配圖形化程式設計軟體 Scratch 建立一個投票工具，透過 Arduino 及其他配件輸入訊息，Scratch 程式接收訊息後，會將原本的訊息轉換成動畫的方式呈現在螢幕上。因此小朋友只要調動接連著 Arduino 面板的橫桿，就可以看著螢幕選擇想投的候選人，接著再按下同樣接連著 Arduino 面板的按鈕，於是完成投票的動作。

除此之外，范運平老師也將 Arduino 運用在體育競賽上面，他曾經試過在籃球比賽中用 Arduino 來計分，運用球進網時觸動的微動開關為原理來計算進幾球，並搭配投影和音效來增加趣味性。范運平老師表示，他自己時常會去思考這些互動裝置可以運用在生活中的哪些地方。未來他也想嘗試利用 Arduino 連接光感應器計算做伏地挺身的次數，起身時身體擋住光線、下壓時照明充足即算一次。范運平老師分享許多創意點子，讓社群成員眼睛為之一亮，原來 Arduino 也可以這樣玩。

▲圖1： 蟠桃國小的范運平老師

雖然是研究機器人，但 S4A 成員不限於教電腦課、生活科技課的老師，有許多教不同領域的老師也加入其中，跟著大家一起研究運用機器人作為教材的可行方法。文德國小英文老師陳英志老師表示，每個資訊科技產品都是工具，只要你善用它，它便可以拿來套用在不同課程中。相信在 S4A 社群的努力之下，往後將會有更豐富有趣的課程等著學生們一起從玩樂中學習。
