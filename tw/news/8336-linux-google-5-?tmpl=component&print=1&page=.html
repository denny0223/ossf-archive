Linux 專利訴訟認定侵權 Google 遭罰 5 百萬美元

建立日期 2011-04-26 00:00      最近更新在 2011-05-09 16:55 作者是 謝良奇／編譯
Google 日前在美國德州東區聯邦地方法院所舉行的專利訴訟中，被陪審團裁決侵權，必須支付 5 百萬美元損害賠償，給位於德州東區的 Bedrock Computer Technologies。這項裁決對科技產業造成的影響廣泛，因為訴訟的焦點是 Bedrock 手上一項與 Linux 核心有關的專利，一旦該專利與侵權判決成立，Bedrock 將可向使用 Linux 或其衍生作業系統的所有公司，收取授權費用與權利金。 

Bedrock 於 2009 年 6 月提起的訴訟，針對的是美國 5,893,120 號專利，其涵蓋了在資訊儲存與擷取上使用帶有外部鏈結的雜湊技術與即時移除過時資料的方法與設備。Bedrock 宣稱這項技術被運用在 Linux 當中，並據此控告 Google、Yahoo、MySpace、PayPal、Amazon、Match.com、AOL、CME Group，以及位於德州東區的兩家小公司 Softlayer 和 Citiware。專利分析師 Florian Mueller 表示，最後兩家公司之所以列入 Bedrock 的控告清單，很顯然是為了確保案件能夠在德州東區法院進行審理。而位於德州東區的聯邦地方法院，所做的宣判向來對專利持有者有利而出名。 

在訴訟中，Bedrock 指稱 2.4.22 和之後版本，或 2.6.25 和之後版本的 Linux 核心，包含了侵害其專利的程式碼。隨著案件的進行，若干同時也是紅帽 (Red Hat) 用戶的被告，要求紅帽出手干預。該公司於是在 2009 年 12 月提起另一場訴訟，隨後直接試圖讓該專利無效。儘管該公司的直接干預遭到阻擋，但提起的訴訟仍在進行中。 

這次的裁決將對自 Linux 衍生出來的 Android 造成重要影響。Android 行動作業系統在手機的佔有率節節上升，早已是一連串引人注目訴訟案所鎖定的對象，其中包括 Oracle 就其 Java 虛擬機器技術所提起的訴訟。影響所及，Google 或許被迫將修改 Android 的核心，以確保該作業系統無須支付授權費用或共同侵權損害賠償。 

Mueller 指出，對於其他 41 件進行中的 Android 相關專利侵權案，此次裁決結果並不是個好預兆，如果 Google 沒辦法成功對抗來自德州一家小公司的專利，面對 Oracle 手上 7 項虛擬機器專利又該如何？開放源碼專家 Steven Vaughan-Nichols 則有不同意見，他認為這個專利裁決出自有偏好專利流氓傾向的地方法院，是個糟糕的專利決策。 

Bedrock 對 Google 和其他公司的訴訟到此還未告一段落，產業觀察者預料 Google 不但會進行上訴，也會試圖讓 Bedrock 的專利失效。Bedrock 則已經向法院申請針對 Google 的禁制令，但法院尚未對此做出裁決。被告之一的 Match.com，已於日前和 Bedrock 達成和解，和解內容則未公佈。 

儘管存在像開放創新網路 (Open Invention Network) 這樣的組織，專利授權仍然是開放源碼產業體系的風險來源之一。由 IBM、NEC、Novell、Phillips、Red Hat、Sony 共同成立於 2005 年的開放創新網路，是為了捍衛 Linux 產業體系的專利授權團體，其工作為取得專利並以免收權利金的方式，授權給開放源碼社群使用。 

多年來開放創新網路持續成長，在 2011 年第一季該團體就增加了超過 70 個授權者，包括 HP、Facebook、Juniper Networks、OpenStack 等。該團體的授權者同意不使用其 Linux 相關專利對付其他人，並且因此可免費使用該團體獲得的專利集合。儘管如此，開放創新網路仍無法完全消除專利風險。Oracle 與 Google 之間的訴訟即為一例。 

Oracle 和 Google 都是開放創新網路的成員，然而兩家公司彼此卻陷入法律角力戰中。在簽署開放創新網路的免權利金授權協議時，成員同意不運用其專利權，對付所謂的 Linux 系統。然而這一點並不妨礙 Oracle 針對在 Android 中運用 Java 來控告 Google。 

開放創新網路執行長 Keith Bergelt 指出，Oracle 的訴訟超出了開放創新網路授權之下的義務範圍。早在 Oracle 提起訴訟前，Muller 等人已經批評過開放創新網路，表示該組織無法確實提供所預期的保護。Muller 指出沒有證據顯示該組織曾經幫助過任何受專利訴訟威脅的公司。 

開放創新網路不僅以授權保護 Linux 為目標，該團體也擁有用來找出現有技術 (prior art) 的若干計畫，以限制針對 Linux 而來的專利訴訟的影響。該團體於 2008 年推出了 Linux Defenders 計畫，其中包含用來協助 Peer to Patent 的專利審閱計畫，以幫助美國專利商標局 (Patent and Trademark Office) 找出現有技術來拒絕專利申請。
