﻿英國開放大學提供 Linux 認證

建立日期 2011-03-31 00:00      最近更新在 2011-04-12 14:32 作者是 謝良奇／編譯
假如你想要取得基本的 Linux 專業資格，參加一個為期 10 週的線上課程，或許是好的開始。因為全球最知名的 Linux 認證機構 Linux Professional Institute (LPI) 和英國開放大學 (Open University UK) 宣佈，參加開放大學 Linux 入門課程的學生，將有資格拿到由 LPI 所主持提供的 CompTIA Linux+ 認證的促銷券。取得 CompTIA Linux+ 認證的學生及 IT 專業人士更自動獲得資格，得以進行 LPIC-1 (Linux Professional Institute certification Level 1) 認證。

早在此一新計畫前，英國開放大學曾於 2010 年 5 月成功針對成年大學生，推出線上 Linux 遠距課程。去年有超過 1000 名學生結束這一項學術課程，取得學位的所需學分。2011 年 5 月的課程預計將有 500 名學生註冊。

這項針對初學者的課程，將帶領學生介紹開放源碼的歷史與價值，學習作業系統，也學習網頁伺服器的安裝、管理和基本功能。

LPI 總裁兼任執行長 Jim Lacey 表示，該項計畫顯示了 LPI 長期以來與產學界推展 Linux 專業化與開放源碼軟體的歷史與意願。英國開放大學對專業認證價值的認可，以及讓其學生獲得 Linux 教育和職業機會所做的努力，都讓他們感到讚賞。

英國開放大學 Linux 課程主席 Andrew Smith 指出，他們發現其 Linux 入門課程很受成人學習者的歡迎，因為結束此項線上學習單元者，同時也經過認證、取得產業公認的操作 Linux 能力，這為課程帶來了重要的附加價值。學生取得 LPI 所主持提供的 CompTIA Linux+ 認證後，可進一步取得 LPIC-1 以及接下來的 Novell Certified Linux Administrator 認證，大大提高了此一課程的優勢。作為一名希望學生獲得最佳工作機會的教育者，他不禁要稱讚 LPI 推動 Linux 認證產業合作的領導地位。

去年，這個針對 Linux 專業人士的 LPIC-1 認證計畫，已於去年成為英國國家認可的中學學生 IT 認證與職業學校學生 IT 認證。提供上述認證的 LPI 是一非營利組織，主要任務是針對 Linux 系統管理者與開發者進行專業認證，並提供廠商獨立測驗。該機構由 Linux 社群於 1999 年成立，主要財務支援來自 IBM、Linux Journal、Linux Magazine、Novell、SGI、HP、IDG 等。LPI 的認證計畫由跨越 5 大洲的會員網絡所支持，散佈於全球、使用各種不同語言的測驗中心超過 7000 家。
