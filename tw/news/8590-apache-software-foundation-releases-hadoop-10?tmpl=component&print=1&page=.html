Apache 軟體基金會釋出 Hadoop 1.0

建立日期 2012-01-16 15:16      最近更新在 2012-01-16 17:33 作者是 謝良奇／編譯

Apache 軟體基金會 (Apache Software Foundation) 釋出了分散式運算軟體框架 Hadoop 的第一個生產版本 Hadoop 1.0。

開放源碼軟體專案 Hadoop 在歷時 6 年的開發後，終於達到了第一個完整釋出的里程碑。Hadoop 是提供可靠、可延展分散式運算的軟體框架，可連接數以千計的伺服器，為雲端運算和其他應用軟體處理大規模資料。

Hadoop 是 Doug Cutting 在 Yahoo 時為了支援 Nutch 搜尋引擎所開發，該框架的誕生受到 Google MapReduce 與 Google File System 論文的啟發。2006 年該框架納入 Apache 軟體基金會底下，成為該基金會的主要專案之一，並且培育出許多子專案以解決分散式運算各個層面的不同挑戰。

1.0 釋出版本號的重要性，在於它意味著 Apache 宣告 Hadoop 已經為企業應用做好準備。Apache 稱 Hadoop 為雲端運算的基礎。Hadoop 1.0 是以 Hadoop 0.20 的安全分支 Hadoop 0.20.205.0 作為根基。

仿照 Google Bigtable 的 Hadoop 大規模數據資料庫 Hbase 也被整合到 1.0 版本中，該支援也包含了針對該資料庫本地檔案的效能增強存取 (performance enhanced access)。運算節點上的安全性則依賴以 Kerberos 為基礎的身分認證。

Apache Hadoop 副總裁 Arun Murthy 在聲明中表示，此次釋出，是許多艱苦工作以及來自 Apache 社群團隊協同合作的成果。這個包含了專責軟體開發者與提交者的活躍社群將 Hadoop 專案的穩定性與專業技術提升到新的水準。

Murthy 表示，Hadoop 逐漸成為實際上的資料平台，讓組織得以儲存、處理、查詢龐大的資料，而新版本的釋出代表了效能、穩定性、安全性的重要進展。

Hadoop 框架已經受到許多大型組織的採用，包括 Amazon、Ebay、IBM、Apple、Facebook、Yahoo。這些公司也協助測試與開發該軟體，使其足以穩定至 1.0 版。在這些公司中，Yahoo 對該專案有著顯著的貢獻，該公司建構出超過 4 萬 2000 個節點的最大 Hadoop 生產環境。

儘管釋出 Hadoop 最終生產版本主要是為了宣告該框架的正式推出，1.0 版與之前的版本相較，還包含許多臭蟲修正、效能及安全性的加強，尤其是允許透過 HTTP 對 Hadoop 分散式檔案系統 (Hadoop Distributed File System，HDFS) 進行寫入。該 webhdfs 檔案系統的舊版本僅支援讀取。

Yahoo 雲端平台集團的資深副總裁 Jay Rossiter 表示，1.0 版的釋出是 Apache Hadoop 社群的重要成就，也是多年來辛苦開發工作與共同學習的成果。Apache Hadoop 將持續屬於 Yahoo 重要的投資領域。今日 Yahoo 的每一個點擊背後，都仰賴 Hadoop 協助將個人化內容與體驗遞送到全球超過 7 億用戶的手中。
