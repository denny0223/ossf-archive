35 美元的超迷你電腦 Raspberry Pi

建立日期 2012-03-12 12:58      最近更新在 2012-04-23 11:26 作者是 黃湘婷／編譯

科技世界的產品不斷推陳出新，目前最受矚目的不外乎消費者預覽版的 Windows 8 以及最新的 iPad3 傳聞，但與 2 月 29 日英國 Raspberry Pi 基金會最新發表的電腦 Raspberry Pi 相比，這款信用卡尺寸的超迷你電腦價格親民多了，僅要價 25 或 35 美元，就能把他帶回家。

▲ 圖片來源：Singularity Hub

這台超小型電腦不僅沒有鍵盤、滑鼠或螢幕，甚至也沒有機殼，需要再外接才能運作，Raspberry Pi 希望這款低價電腦能普及給兒童使用，透過這樣的裝置改變學校電腦課程的上課方式，提供孩子們撰寫基本程式的環境，培養他們的興趣。

雖然外觀超級迷你，但 35 美元的 Model B Raspberry Pi 包含 700MHz ARM11 處理器，256 MB 的記憶體，HDMI、RCA video、3.5mm 耳機（喇叭）的輸出孔，一個乙太網路連接埠，兩個 USB 2.0 連接埠，以及 SD 記憶卡擴充槽。未來將推出較便宜的 Model A 版本，售價 25 美元，規格減少一個 USB 連接埠與乙太網路連接埠，並對記憶體做相對的調整。

由於只需 35 美元，這款迷你電腦首批製造的1萬部開放預購不到一天便銷售一空，Raspberry Pi 的銷售商網站 element14 與 RS Components 甚至因為瞬間湧入太多的預購者，造成網站癱瘓。
