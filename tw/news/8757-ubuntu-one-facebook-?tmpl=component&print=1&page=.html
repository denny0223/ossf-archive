Ubuntu One 不再支援 Facebook 的同步

建立日期 2012-07-23 16:10      最近更新在 2012-07-24 15:35 作者是 黃湘婷／編譯

Ubuntu One 是由 Canonical 公司經營的雲端服務，提供檔案儲存與同步，擁有 5GB 的免費空間。除了 Ubuntu 外，也支援 Windows 的版本，以及行動裝置 Android 與 iOS 上的使用。Ubuntu One 時常更新內部的功能，並持續創新、改善缺點，於日前宣布了一個重要的決定：不再支援與 Facebook 的同步。

這個重大的決定並不是 Ubuntu One 今年唯一的改變，早在年初 Ubuntu One 也取消了對於 Online Notes 的服務。

Ubuntu One 團隊的開發者表示，為了繼續改善服務品質與更新，並推出新功能，才會做出如此決定。據了解，目前部分的同步服務都依賴舊版本數據同步為基礎，而未來幾個月他們將會以 U1DB 為基礎建立新的架構，使更多的功能具備發展性與未來性。

至於原先使用 Ubuntu One 中 Facebook 同步功能的使用者也不用擔心，Ubuntu One 不會停止對原先使用此功能用戶的服務，使用者未來還是可以正常使用，而在這項改變之後才使用 Ubuntu One 的用戶則沒有 Facebook 同步功能的選項可以選取，換句話說對於新加入 Ubuntu One 的使用者其實沒有太大的影響。

目前這項策略對 Ubuntu One 後續的發展與影響還不明顯，但對於接下來將推出的新改版與功能是值得期待的。
