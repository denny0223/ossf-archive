Asus 平板裝置 Eee Pad 將採用 Android 而非 Windows

建立日期 2010-08-19 00:00      最近更新在 2010-08-30 20:38 作者是 謝良奇/編譯
Asus 的平板電腦 Eee Pad 將捨棄 Windows，改採 Google 的開放源碼作業系統 Android。這一款搭載 10 吋觸控式螢幕的平板電腦 Eee Pad EP101TC，是在今年的 Computex 大會上首次公開亮相，當時運行的是 Windows 7 的嵌入式版本。如今看來 Asus 似乎不願意自家平板裝置上，安裝微軟過於龐大的作業系統，轉而選擇了基於 Linux 的 Android 系統。

據報導，Asus 已經決定等到 Google 釋出代號薑餅 （Gingerbread） 的 Android 3.0 後再推出Eee Pad，目前預料 Android 3.0 會在今年底釋出。因此 Eee Pad 可望在該作業系統釋出後不久推出，很可能選擇在國際消費電子展（CES）上公開問世。

微軟於今年 Computex 大會上賣力拉攏眾多廠商，在其平板裝置上運行精簡過的 Windows Embedded Compact 7。這套作業系統針對的是功能受限的裝置，例如無鍵盤的平板電腦。不過現在看來 Asus 似乎在安全性與可用性方面有更多需求。

Google 的 Android 作業系統被認為是 Apple Iphone OS 在智慧型手機的主要競爭對手。不過 Android 尚未能在平板裝置領域造成有如在智慧型手機市場的轟動。但是情況似乎將有所改變，因為 Lenovo 也和 Asus 一樣，宣佈將在該公司的平板電腦上運行 Android，而非微軟的作業系統。

儘管微軟的 Windows 在小筆電市場有凌駕 Linux 的趨勢，如今在平板電腦上看來兩者的角色卻互相對調了。目前尚未得知 Asus 另一款 12 吋平板裝置 Eee Pad EP121，是否同樣會揚棄微軟的作業系統，早先於 Computex 大會上，該款裝置運行的是 Windows 7 的全功能版本，而非 EP101TC 使用的修改版本。一旦 Asus 決定全面捨棄 Windows，對於日前在全球合作夥伴大會（Worldwide Partner Conference）上，宣佈 Asus 將是 Windows 7 平板裝置合作夥伴之一的微軟而言，將會是一大打擊。
