美國白宮於 GitHub 建立永續專案以 Open Source 工具推動 Open Data！

建立日期 2013-05-10 15:31      作者是 黃郁文／編譯

對美國政府而言，開放政府資料 (Open Government Data) 是國家在當代可資運用的重要資源與策略，因為將政府資料集以開放的態度進行妥善的管理與釋出，不但能夠強化民主政體發展的穩固性，也可以增進政府施政的效能；在這些面向之外，如果運轉得宜，更可以創造新的經濟發展契機，並提升公民社會裡每一個參與者的生活品質。而為了讓開放政府資料的運動，不會侷限於政府機關自我的格局，而要能夠吸引民眾廣泛且積極的參與，美國白宮 (The White House) 近期創舉式的在知名的自由開源軟體 (Free and Open Source Software, FOSS) 開發與託管網站 GitHub 上，建置了名為 Project Open Data 的 Open Data 專案！

4 月 20 日是 2013 年的硬體自由日

建立日期 2013-04-02 00:00      最近更新在 2013-04-15 10:46 作者是 謝良奇／編譯

硬體自由日 (Hardware Freedom Day) 的主辦單位已敲定 4 月 20 日為該活動第二屆的日期。由非營利組織數位自由基金會 (Digital Freedom Foundation) 於 2012 年所發起的硬體自由日，是為了建立軟體自由日 (Software Freedom Day) 的硬體版本。軟體自由日這個自由軟體的慶祝活動，同樣是由數位自由基金會主辦。該組織已經在世界各地舉辦過許多活動，讓人們了解並參與那些在他們周圍發展的自由軟體、開放源碼與社群。硬體自由日是為了促進開放硬體 (Open Hardware) 的理念，鼓勵人們不加限制地分享設計，並基於這些設計創造實際硬體。
