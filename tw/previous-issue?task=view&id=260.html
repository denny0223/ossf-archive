雜談網頁親和力（四）自然語言的親和力原則
Jedi 2007/05/16
「自然語言」這個說法，是相對於「人造語言」（例如電腦程式語言）的稱呼，簡單來說也就是一般人日常溝通所會使用的語言。人造語言通常有著（由少數人）自創的詞彙，以及嚴格、絲毫不通融的文法，再加上受限的表意範圍，所以對普羅大眾來說，往往會是不容易上手、但也不難完全掌握的語言類別。

香港培正教育中心在學界播放 Linux 種子
Linuxpilot 2007/05/16
香港培正教育中心成立於 2000 年，是一所非牟利教育機構，中心成立目的旨在推廣資訊科技教育及資優教育，其中「I.T.種子」計畫更被本地學界所稱譽。「I.T.種子」計畫獲勞工處資助，提供多個見習培訓計畫，學歷在大學學位以下的 15 至 24 歲青少年均可報讀，而參與學校須提供在職培訓職位，並給予學員每星期一天在職進修例假參加「I.T.教學行政支援」證書課程，以加強其專業技能，為學校推動資訊科技教育及學校行政提供支援服務。而「數碼領袖」培訓計畫，則為完成「I.T.教學行政支援」證書課程的青少年，提供一個持續進修階梯，以加強學員的專業和學術技能，為日後升學或成為資訊科技從業員作好準備。

Sun 釋出 Java SE 平台原始碼
謝良奇 2007/05/16

Sun Microsystems 宣佈將 Java 平台標準版 (Java Platform Standard Edition，Java SE) 的 Java 開發套件 (JDK)，以 GNU 通用公共授權 (General Public License) 第 2 版釋出給 OpenJDK 社群，成為自由軟體。基於 OpenJDK 的實作將可藉由 Java SE 6 技術相容套件 (Java SE 6 Technical Compatibility Kit, JCK)，達成與 Java SE 6 相容的目標。此外 Sun 並且公佈了 OpenJDK 社群的臨時管理委員會。
