Topic Information 議程簡介

第一部份：Ubuntu Studio 自 8.04LTS 開始，已經是一個成熟度相當高的數位音樂製作平台，與商業音樂製作軟體相比作品品質豪不遜色，而且先天整合度更高，本課程淺介如何在 Ubuntu Studio 下進行音樂創作；第二部份：Wubi 是一個基於Ubuntu所開發的安裝工具。藉由簡易的方式，使 Windows 的使用者可以不必切割硬碟空間、調整 BIOS、光碟開機，即可享用 Linux 作業系統。並且擁有與一般 Windows 系統下的程式相同的新增移除功能，想玩 Linux 就是那麼的簡單。

詳細的內容時間描述與報名資訊如下：

  • 對象：對自由軟體音樂創作有興趣的學生、音樂老師、音樂相關科系學生。
  • 人數：約 40 人。
  • 費用：全程免費。
  • 平台：請自行攜帶筆記型電腦。
