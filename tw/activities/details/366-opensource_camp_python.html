議程簡介

Python 是一個泛用型 (general-purpose) 的動態語言 (dynamic language)。從嵌入式系統、系統工具、科學計算、測試腳本、桌面應用、網站應用等都可以看到 Python 的身影。
相對於 Perl，Python 的語言哲學強調一件事最好只有一種解決方法，使程式設計師可以專注在解決問題上，因為工具集相對較小，學習起來也相對快速、也因其語法的優良設計，使得寫起來更加快速。
這兩天的學習營將會完整地介紹 Python 的特性，以及相關的應用與函數庫，讓學員在上完課之後，都能將 Python 融於自己的日常工作中。
  • 對象：大專院校資訊相關系所的學生，或是對自由軟體有興趣者。
  • 人數：約 50 人。
  • 費用：免費。
  • 平台：Fedora 14。
