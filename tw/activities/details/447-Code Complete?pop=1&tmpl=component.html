活動內容：

本次講演主要目的為學生與初學者分享，開發高品質軟體應具備的概念，以如何寫「好的」程式為引導，並以 "Code Complete" 一書為本出發，提供簡短的閱讀指南與案例心得分享。

  1. What is High-Quality Code?
  2. How to create High-Quality Code?
  3. Variables (Good and Bad example)
  4. Statements (Good and Bad example)
  5. Code Improvements
  6. Software Craftsmanship

Code Complete 一書是目前被公認為給年輕的軟體工程師[1]，最需要且重要的必讀之作，若想提升軟體開發的能力，這本書千萬不要錯過。

[1] Stackoverflow -- What is the single most influential book every programmer should read?

講者介紹：

蘇展(Johnson J. Su)，自由軟體鑄造場臨時水電工。

對軟體開發流程有濃厚的興趣，特別重視良好的的開發方法與適合的工具，如古人云：工欲善其事，必先利其器，期許自己可以減少不必要的時間浪費軟體開發上。
