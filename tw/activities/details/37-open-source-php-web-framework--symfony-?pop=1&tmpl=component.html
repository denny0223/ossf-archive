課程簡介

symfony 是一個基於 php5 的 Web Framework。 根據網站編寫經驗，symfony 能夠提高網站應用程式建立及維護的效率，且以有趣的控制項來取代重複性程式撰寫工作。

本課程共三小時，詳細的議程內容描述與報名資訊如下所示：

 

  • 對象：php 程式設計已有經驗，並對 php 的 MVC Framework 有興趣並想了解的資訊相關人員。
  • 人數：每班約 25 人 ( 406 教室-實機操作 )。
  • 平台：Windows。
  • 費用：全程免費。
