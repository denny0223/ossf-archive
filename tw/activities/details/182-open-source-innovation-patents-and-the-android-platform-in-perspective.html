議程簡介

去年 3 月蘋果電腦控告宏達電生產的 Android 手機侵害其專利權，開啟了 Android 手機相關技術的專利訴訟戰爭，同時也使自由開源軟體與軟體專利的議題白熱化，我國身為行動裝置與資訊硬體生產大國，更是深受影響。有鑑於此，自由軟體鑄造場籌辦本次的法律工作坊，邀請國內外的專家從 Android 平台的訴訟案件為引，為與會者解析自由開源軟體與軟體專利相關的議題。

議程將由政治大學法律系李治安助理教授開場，以中文介紹軟體專利的概念，接著由國際知名的專利組織 Open Invention Network (OIN) 執行長 Keith Bergelt 進行主場演講。Bergelt 先生將會以開源創新、專利與 Android 平台為演講的構成要素，來深入解析軟體專利在自由軟體領域中，為大家所關心的各項議題。OIN 為一個致力於建立 Linux Kernel 專利安全防衛領域的組織，透過收購、接受捐贈與簽訂交互授權協議等方式，逐年建立起了一個 Linux Kernel 相關的專利池 (Patent Pool)，以保障 Linux Kernel 與相關自由開源軟體產業的持續發展。而執行長 Bergelt 先生本身投入智財產業數十年，熟稔專利與相關的產業動態，對於自由開源軟體的專利議題具有獨到的見解。

在演講之後，並將安排討論座談的時間，讓所有與會者與講者可以進行意見交流與討論。會後並將提供與會者於中研院福華哲思軒，自費參加講者餐敘的機會，以深入討論進階議題並增進彼此的認識。[餐敘報名頁面]

講者簡介

Keith Bergelt

  • Open Invention Network 執行長
  • Keith Bergelt 目前為 Open Invention Network 的執行長，他同時也是一位深具經驗的開放源碼經理人，專長於組織規劃、政府政策、商業發展、授權策略、智慧資產管理、智慧財產財務與外交策略。在加入 OIN 之前，Bergelt 先生曾擔任兩間避險基金－Paradox Capital 與 IPI－的總裁與執行長。更早之前，他在 Motorola 公司內部成立智慧資產策略管理部門，同時擔任這個部門的總經理以及 Motorola 的技術策略主任。而在開始進入私人公司服務之前，Bergelt 先生從事了12年的外交工作，曾擔任派駐在聯合國紐約總部以及美國在日本東京大使館的外交官。
  • Keith Bergelt 詳盡的背景資訊： https://www.openinventionnetwork.com/about.php
  • 網路履歷： https://www.linkedin.com/pub/keith-bergelt/5/b6b/856

李治安

  • 李治安是政治大學法律科際整合研究所暨智慧財產研究所合聘助理教授、經濟部智慧財產局著作權法修法諮詢委員，美國史丹福大學法學博士、哈佛大學法學碩士。李治安於臺灣大學法律學系法學組畢業後，即通過律師高考、司法官特考暨公務員高考，並獲得同校商學研究所碩士學位，曾任臺北地方法院公證人、協合國際法律事務所律師及美國史丹福大學 John M.
