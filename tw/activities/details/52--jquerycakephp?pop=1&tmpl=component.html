課程簡介

CakePHP 簡化了 PHP 應用的開發流程，只是將焦點轉移到使用者操作介面時，會發現許多需求並不是 PHP 所能夠直接滿足的，這時候會希望透過額外的工具輔助， javascript 是許多人的首選。而 javascript 的程式設計一樣會面臨開發速度的問題，所以選擇 jQuery 作為 javascript framework ，除了 jQuery 本身簡化了許多應用的開發流程，由社群朋友所分享的外掛非常多，幾乎可以滿足大部分的需求。在這個課程中，焦點會放在如何在 CakePHP 應用開發中結合 jQuery 進行各種功能的延伸。

課程的進行以實作為主，每次講解完一個段落就會讓參與者進行同樣操作，藉此希望能夠有比較實際的體驗。
