議程簡介

將詳細介紹 Python 這套程式語言，從最基礎的流程控制到高階函式應用及物件導向，以期能從容易撰寫的 Python 來讓學員迅速寫出一些對自己有幫助的小工具，並以 Python 做為學習其他相對較複雜程式語言的基石。課程內容會視實際情況彈性調整，每堂課都會有實作的部份。
