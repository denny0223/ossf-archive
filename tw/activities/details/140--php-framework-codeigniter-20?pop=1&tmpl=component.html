議程簡介

CodeIgniter 是一個 PHP5 的 Web Framework 。 對於還沒有使用 Web Framework 開發網站的使用者，這會是一個非常好入門的 Framework 。它擁有清楚完整的繁體中文說明文件，提供簡易的介面和邏輯結構來使用豐富的函式庫，其目的是讓使用者可以加快開發速度。

詳細的內容時間描述與報名資訊如下：

  • 對象：初學 PHP 程式設計，以及對於 MVC Framework 初學都可。
    （建議對於 PHP 有一定的基本瞭解）
  • 人數：約 28 人。
  • 費用：全程免費。
  • 平台：提供 28 台電腦實機操作，使用 Windows 系統。
