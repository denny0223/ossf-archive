Topic Information 議程簡介

本議程前半段以介紹軟體逆向工程為主，並透過實例了解軟體逆向工程的運用。後半段將會利用淺顯易懂的教學，讓一般使用者與網站開發者了解 Web Security 的重要。透過對攻擊手法的了解以及對網站的實際模擬測試，學習修補其中的弱點，達到網站安全的基本需求。

詳細的內容時間描述與報名資訊如下：

  • 對象：資訊相關系所的學生，或是對資訊安全、自由軟體有興趣者。
    （本課程要求具備基本程式設計能力，例如 PHP / MySQL、JSP、ASP 等動態網頁開發經驗、或熟 x86 組合語言者。）
  • 人數：約 20 人。
  • 費用：全程免費。
  • 平台：現場電源有限，如需要請自備充好電的筆記型電腦。
