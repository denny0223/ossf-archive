議程簡介

Amazon Web Service（AWS）是目前雲端解決方案之一，而且是少數三種雲端架構服務（IaaS/PaaS/SaaS）都擁有的雲端大廠，本課程會由 AWS 中的 Elastic Compute Cloud（EC2），帶領每一個人進入雲端，簡單建立 Apache、Mysql 架設一個 Open Source Blog（WordPress）。
請各位特別注意，因為 AWS 有提供免費方案（AWS Free Usage Tier），過程中按照課程的步驟，基本上不會花費到任何的費用，但需要利用信用卡認證帳號。

詳細的內容時間描述與報名資訊如下：

  • 對象：對於雲端運算有興趣者即可。
  • 人數：約 28 人。
  • 費用：全程免費。
  • 平台：座位電腦為 Windows 系統，自行攜帶筆電者則不限制。
