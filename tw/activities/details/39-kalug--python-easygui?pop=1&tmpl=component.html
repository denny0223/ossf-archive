課程簡介

Python EasyGUI - EasyGUI 是個 python 模組，它非常簡單，且容易使用。它提供個一個跨平台且簡單撰寫的方式，方便你利用它開發一系列容易使用的圖形操作設定工具。

本課程共三小時，詳細的內容時間描述與報名資訊如下：

  • 對象：有 python 基礎。
  • 人數：每班約 25 人 ( 實機操作 )。
  • 平台：Linux or Windows XP。
  • 費用：全程免費。
時間 內容
09:20-10:30 EasyGUI 簡介，buttonboxes , choicebox，Inputbox 的使用。
10:30-10:50 休息
10:50-12:00 文字顯示, 檔案與目錄的使用。程式撰寫實作。/td>
