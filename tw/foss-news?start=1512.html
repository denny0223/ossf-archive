YAPC::Taipei 2005 報名即將截止

建立日期 2005-03-09 16:00      最近更新在 2007-02-03 00:27 作者是 webmaster
台灣Perl使用者的年度盛會 ─ YAPC::Taipei 2005，將於3月26、27日於台灣大學資訊系館舉行，目前正進行熱烈報名，由於僅開放65位報名名額，歡迎有興趣了解Perl最新應用的同好盡速報名。

今年邀請的國外講者，除了著名的追蹤管理系統Request Tracker(RT)作者Jesse Vincent與Kwiki作者Brian Ingerson(Ingy)二度來台參與會議外，還另外邀請了來自英國YAPC::Eu的發起人Leon Brocard、紐西蘭的Tangram(七巧板)模組的maintainer Sam Valain、以及six apart(知名產品有部落格網站程式Movable Type)日本分公司的宮川達彥(Tatsuhiko Miyagawa)這幾位在國際Perl社群非常知名的技客(geek)

此次YAPC::Taipei的主要內容，包括Brian Ingerson將帶來Kwiki的最新介紹，以及Jesse Vincent描繪RT的未來發展，此外宮川達彥將介紹如何利用一些部落格(Blog) APIs，如AtomAPI與XML-RPC等各樣有趣的工具與方法，將一些網站如TypePad、Flickr、Last.fm、del.icio.us、Bloglines以及Amazon網站上的資料，整合到使用者的部落格上。Leon Brocard亦將介紹Perl如何協助像是Google、Amazon、eBay等以網頁服務(Web Services)為主的企業解決互通性(interoperability)與延展性(scalability)的問題。Sam Valain則將帶領大家探討Perl的物件永續性(Object Persistence)，以及其他與功能與Tangram類似的模組介紹，如：Pixie, Class::DBI, Alzabo等。

推廣自由軟體不遺餘力的自由軟體鑄造場，除了身為這次的協辦單位外，也規劃在今年的研討會中，跟與會者分享RT的廣泛技術運用及如何使用自由軟體做程式開發等經驗交流；在國內講者的部份，也推出多位深具實力的Perl開發者，向與會者介紹Perl上最新與最受歡迎的程式或模組，如唐宗漢(Autrijus)介紹Perl6計畫最新發展，高嘉良(clkao)介紹RunApp與Profiler、張琮翔(vgod)探討著名的垃圾信殺手SpamAssassin，以及劉康民(gugod) 和宮川達彥介紹Perl應用程式的最新發展。

自去年YAPC::Taipei 2004順利結束後，台北Perl推廣組開始舉辦每個月的固定聚會，大多以技術討論為主。另舉辦多場關於Perl Debugger、Bricolage出版系統、Kwiki、Template::Toolkit、極道程式設計法(Extreme Programming)等多場主題討論。繼 YAPC::Taipei 2004 受到廣泛好評之後，台北 Perl 推廣組今年與中央研究院資訊科學研究所的自由軟體鑄造場計畫，以及著名的PTT BBS合作舉辦 YAPC::Taipei 2005。
本場研討會為期兩天，演講主要以中、英文為主，依據講者的主要使用語言決定，但是英文演講將會有現場翻譯。有興趣的民眾可至 https://register.elixus.org/yapc2005/
進行線上報名。

嵌入式系統技術工作坊

建立日期 2005-02-23 16:00      最近更新在 2011-01-06 23:37 作者是 webmaster

Linux 在嵌入式系統領域的發展前景極受曙目，但進入門檻又常讓許多人望而卻步。中央研究院資訊科學研究所 自由軟體鑄造場計畫特別邀請光寶科技公司 新技術暨產品研發中心 曾瑞源資深經理來分享他個人在這領域的專業經驗。曾瑞源資深經理早年以曾桑為名，引介 Linux 學習及應用的入門知識，如今，曾經理把累積多年的 Embedded Linux 開發及教學經驗，整理成一套 Minidoochun (迷你豆漿) 教學環境，內含最佳化的 Red Hat 9.0 Linux 系統，使用者可在Windows 及 VMware 作業環境進行練習，不會影響到既有PC內的資料。

Minidoochun 是一套嵌入式系統 rootfs 的模具組。利用這樣的模具，可以快速製作定型化的 rootfs，而且修改是非常容易的，只需在預定的rootfs 放置所需的應用程式以及函式庫，十分鐘後就可以製作出全新的 rootfs。以 Minidoochun 做為 Embedded Linux 教學環境，搭配 VMware 這套硬體平台模擬軟體，使用者可以獨立在 Windows XP 作業環境上運行及測試，快速練習各式Embedded Linux 的設定與應用程式開發；甚至搭配 skyeye 軟體可模擬多款 ARM CPU環境，練習開發 ARM 環境上的驅動程式與應用程式。
