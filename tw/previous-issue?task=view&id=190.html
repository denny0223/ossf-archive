社群計畫搶鮮報
涂敏怡 2004/12/28

經過一天半的緊湊行程，主辦單位安排了 Lightning Talk，介紹國 內社群中最新的開放源碼專案現況。一開始由來自義大利的 Debian 資深軟體開發者 Enrico Zini 介紹 "DebTag"，這是一套在 Debian 裡為 package 進行分類的管理程式。

目前在台灣工作的 Arne Goetje 則介紹他目前投入的中文 Unicode 繁簡字體擴充計畫。接著由在國內推廣台語文白話字（POJ）的陳柏 中（pektiong）介紹台語文網誌與POJ的顯示與輸入問題。Kaffe.org 的官方開發者jserv在現場介紹了 FreeJava 計畫，與他最近積極投 入的新酷音輸入法計畫。

鑄造場經驗交流：本地化、輸入法與國際合作
涂敏怡 2004/12/28

除了關切政府、商業跟學術界如何促進帶動國內的自由軟體使用風潮外，另外一個為國內的自由軟體社群持續關心的自由軟體議題，就是自由軟體的本地化與中文輸入的問題。

此次會議邀請國內主要從事自由軟體中文化以及輸入法的開發者一起聊聊開發甘苦，包括中文編碼、中文字型、在社群中受到歡迎的新酷音輸入法、與使用於MacOSX的開放香草 (OpenVanilla) 輸入法等等。

開放源碼歐洲與荷蘭發展現況
涂敏怡 2004/12/28

Souabi 在 ICOS 開幕演說「歐洲公部門的開放標準與開放源碼軟體」 中提到，荷蘭政府自 2002 年開始支持開放標準與開放源碼的推廣， 預計至 2006 年政府將全面支援開放標準，2007 年將有 65% 的政府 服務可以從網路取得。

Brahaim Souabi 是荷蘭國家資訊傳播科技組織 ICTU 所屬開放標準 與開放源碼 OSOSS 計畫的主持人，負責推廣促成荷蘭政府與公部門 使用開放源碼。

開放源碼：社群、產業與政府綜合座談
涂敏怡 2004/12/28

大會第一天下午有「開放源碼：社群、產業與政府」的綜合座談，邀 請各界代表一起探討產、官、學、社群如何就自由/開放源碼軟體進 行合作。

關於政府近年來推行自由軟體的政策規劃，行政院科技顧問組副執行 秘書蔡志宏表示主要方向有三，一是讓政府採購成為自由軟體主要力 量之一；二是自由軟體的支援不限 Linux、也可延伸到商業軟體上； 三是產業發展跟使用者權益的平衡。蔡志宏並表示，行政院國家資訊 通訊發展推動小組（NICI）在落實自由軟體相關政策上有三大方向： 一是促成自由軟體產業可以與其他產品結合，例如網路設備、智慧型 手機等；二是教育推廣自由軟體以縮減數位落差；三是與資訊安全的 結合。
